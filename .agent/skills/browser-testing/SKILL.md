## Browser Testing URL

When testing in the browser use this base URL: `http://jmeryar-notebooklm.test/jmeryar/1`

## Common Routes

- **Dashboard:** `/jmeryar/1`
- **Accounting Resources:** `/jmeryar/1/accounting/*`
- **Sales Resources:** `/jmeryar/1/sales/*`  
- **Purchase Resources:** `/jmeryar/1/purchase/*`
- **Inventory Resources:** `/jmeryar/1/inventory/*`
- **HR Resources:** `/jmeryar/1/hr/*`

## Authentication

The application uses Filament for authentication. Developer logins are available via `filament-developer-logins` package in development.
