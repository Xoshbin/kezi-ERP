# پرۆسێسکردنی مووچە: بەڕێوەبردنی پێدراوەکانی کارموند

ئەم ڕێبەرە ڕوونکردنەوە دەدات لەسەر چۆنییەتی بەڕێوەبردن و پرۆسێسکردنی مووچە لە ناو مۆدیوڵی سەرچاوە مرۆییەکان. لە ئەژمارکردنی مووچەوە تا تۆمارکردنی تۆمارە ڕۆژانەکان و ئەنجامدانی پارەدان، هەموو هەنگاوەکان ڕوون دەکەینەوە.

---

## سووڕی مووچە (Payroll Run) چییە؟

**سووڕی مووچە** گوزارشت دەکات لە ئەژمارکردنی مووچە بۆ کارمەندێکی دیاریکراو لە ماوەیەکی دیاریکراودا (زۆربەی کات مانگێک). ئەمە کۆکەرەوەی مەرجەکانی گرێبەستی کارمەند، زانیارییەکانی ئامادەبوون، و هەر دەستکارییەکی تایبەتە (وەک پاداشت یان بڕدراوەکان) بۆ دیاریکردنی مووچەی پاک (Net Pay).

**بۆچی ئەمە گرنگە؟**
1. **دروستی (Accuracy)**: دڵنیابوونەوە لەوەی کارمەندان بە دروستی مووچە وەردەگرن بەپێی گرێبەستەکانیان.
2. **پەیوەستبوون بە ژمێریاری**: بە شێوەیەکی ئۆتۆماتیکی تۆمارە ڕۆژانەکان دروست دەکات بۆ تۆمارکردنی خەرجییەکانی مووچە و شایستە داراییەکان.
3. **شوێنپێی وردبینی**: پارێزگاری دەکات لە تۆمارێکی ڕوون کە کێ مووچەی وەرگرتووە، چەند بووە، و کێ پەسەندی کردووە.

---

## لە کوێ دەیدۆزیتەوە

بڕۆ بۆ: **سەرچاوە مرۆییەکان (HR) ← مووچەکان (Payrolls)**

هەروەها دەتوانیت تۆمارەکانی مووچە لە ناو **پرۆفایلی کارمەند** لە بەشی تۆمارە پەیوەندیدارەکان ببینیت.

---

## هەنگاوەکانی کار (Workflow)

پرۆسێسکردنی مووچە لە جمیار چوار قۆناغی سەرەکی دەبڕێت:

┌───────────┐      ┌──────────────┐      ┌───────────┐      ┌───────────┐
│ ڕەشنووس  │ ──▶ │ پرۆسێسکراوە  │ ──▶ │ پارەدراوە │ ──▶ │ داخراوە   │
└───────────┘      └──────────────┘      └───────────┘      └───────────┘
      📝                 ✅                  💰                 🔒

### ١. دروستکردنی مووچەیەکی نوێ (ڕەشنووس)

کاتێک تۆمارێکی نوێی مووچە دروست دەکەیت، دەبێت **کارمەند**، **ماوەی دارایی** (ڕێکەوتی دەستپێک/کۆتایی)، و **ڕێکەوتی پارەدان** دیاری بکەیت.

| کێڵگە | وەسف |
|-------|-------------|
| **کارمەند** | ئەو ئەندامەی ستاف کە پارەکەی بۆ خەرج دەکرێت. |
| **ڕێکەوتەکانی ماوە** | زۆربەی کات لە ١ تا کۆتایی مانگ. |
| **مووچەی بنەڕەتی** | بە ئۆتۆماتیکی لە گرێبەستی کارەکەیەوە وەردەگیرێت. |
| **پێدراوەکان (Allowances)** | پێشوەختە لە گرێبەستەکەوە پڕکراوەتەوە (خانووبەرە، گواستنەوە، و هتد). |

### ٢. پێداچوونەوە و ئەژمارکردن

کاتێک زانیارییە بنەڕەتییەکان داخڵ دەکرێن، سیستەمەکە کۆی گشتییەکان ئەژمار دەکات.

> [!NOTE]
> لە ئێستادا، سیستەمەکە ڕێگە دەدات بە دەستکاری کردنی بڕەکانی کاری زیادە، پاداشت، و بڕدراوە تایبەتەکان لە قۆناغی ڕەشنووسدا.

### ٣. پەسەندکردن (ناردن)

دوای پێداچوونەوە بە ژمارەکاندا، بەڕێوەبەرێکی ڕێپێدراو دەبێت کرتە لەسەر **پەسەندکردن (Approve)** بکات.

**چی ڕوودەدات لە پشت پەردەوە:**
- بارودۆخ دەگۆڕێت بۆ **پرۆسێسکراوە (Processed)**.
- **تۆمارێکی ڕۆژانە (Journal Entry)** بە ئۆتۆماتیکی لە مۆدیوڵی ژمێریاری دروست دەبێت.
- سیستەمەکە تۆمارێکی **قەرزدار (Debit)** بۆ خەرجی مووچە و **قەرزدەر (Credit)** بۆ مووچەی شایستە (Salary Payable) ئەنجام دەدات.

┌─────────────────────────────────────────────────────────────┐
│  Dr. خەرجی مووچە (٦٠١٠٠١)           $١,٠٠٠.٠٠               │
│      Cr. مووچە شایستەکان (٢٠١٠٠١)             $١,٠٠٠.٠٠      │
└─────────────────────────────────────────────────────────────┘

### ٤. ئەنجامدانی پارەدان

کاتێک پرۆسێسکرا، کرداری **پارەدانی کارمەند (Pay Employee)** بەردەست دەبێت. کرتەکردن لەمە:
1. پەنجەرەی پارەدان دەکاتەوە.
2. تۆمارێک لە مۆدیوڵی **پارەدان (Payment)** دروست دەکات بۆ بڕی مووچەی پاک (Net Salary).
3. پارەدانەکە بە تۆماری مووچەکەوە دەبەستێتەوە بۆ گەڕانەوەی ئاسان.
4. بارودۆخی مووچەکە وەک **پارەدراوە (Paid)** دیاری دەکات.

---

## نموونەیەک: کاریگەری مۆڵەتی بێ مووچە

با سەیری چۆنییەتی کاریگەری ئامادەنەبوون بکەین لەسەر پارەدانی کۆتایی.

**بارودۆخەکە**:
کارمەندێک بە ناوی ئازاد، مووچەی بنەڕەتی مانگانەی **$١,٠٠٠**ـە. لە مانگی یەکدا، **٢ ڕۆژ مۆڵەتی بێ مووچەی** وەرگرتووە.

**چۆنییەتی ئەژمارکردن**:
- **کۆی ڕۆژەکانی کار لە مانگدا**: ٢٢ ڕۆژ (بۆ نموونە).
- **نرخی ڕۆژانە**: $١,٠٠٠ / ٢٢ = $٤٥.٤٥
- **بڕینی مۆڵەتی بێ مووچە**: ٢ ڕۆژ × $٤٥.٤٥ = **$٩٠.٩٠**

**لە تۆماری مووچەکەدا**:
1. ئازاد و ماوەی مانگی یەک هەڵبژێرە.
2. مووچەی بنەڕەتی $١,٠٠٠ نیشان دەدات.
3. لە ژێر **بڕدراوەکانی دیکە (Other Deductions)**، بڕی $٩٠.٩٠ بنووسە لەگەڵ تێبینی "مۆڵەتی بێ مووچە - ٢ ڕۆژ".
4. **مووچەی پاک (Net Salary)** بە ئۆتۆماتیکی دەبێتە **$٩٠٩.١٠**.

---

## پرسیارە دووبارەبووەکان

**پ: ئایا دەتوانم دەستکاری مووچە بکەم دوای پەسەندکردنی؟**
نەخێر. کاتێک پەسەندکرا (Processed)، قفڵ دەکرێت بۆ پاراستنی دروستی ژمێریاری. ئەگەر هەڵەیەک هەبوو، دەبێت مامەڵەکە هەڵوەشێنیتەوە یان پێچەوانەی بکەیتەوە.

**پ: لە کوێ تۆمارە ڕۆژانەکە (Journal Entry) ببینم؟**
تۆماری مووچەکە بکەرەوە؛ تۆمارە ڕۆژانەکە لە بەشی **زانیارییەکانی سیستەم** یان **Workflow** بەستراوەتەوە. کرتەی لێ بکە بۆ بینینی وردەکارییەکان.

**پ: مووچە بە فرە-پارە (Multi-currency) چۆن کار دەکات؟**
سیستەمەکە ئەو دراوە بەکاردەهێنێت کە لە گرێبەستی کارمەندەکەدا دیاریکراوە. ئەگەر جیاواز بوو لە دراوی بنەڕەتی کۆمپانیا، سیستەمەکە نرخی ئاڵوگۆڕی ڕۆژی **پارەدان** بەکاردەهێنێت.

---

## باشترین ڕێکارەکان

- **سەرەتا گرێبەستەکان بپشکنە**: دڵنیابەوە لەوەی کارمەند گرێبەستێکی چالاک و پەسەندکراوی هەیە پێش دەستپێکردنی مووچە.
- **پرۆسێسکردنی بەکۆمەڵ**: فلتەرەکان بەکاربهێنە بۆ گرووپکردنی کارمەندان بەپێی بەشەکان بۆ پێداچوونەوەی ئاسانتر.
- **پشکنینی دووبارە بۆ بڕدراوەکان**: هەمیشە پێداچوونەوە بە کێڵگەی "بڕدراوەکانی دیکە" بکە بۆ هەر دەستکارییەکی دەستی کە بۆ ئەو ماوەیە پێویستە.
