# هاوتاکردنی بانکی: دڵنیابوون لە دروستی پارە و کۆنترۆڵی دارایی

ئەم ڕێبەرییە گشتگیرە ڕونیدەکاتەوە جۆن هاوتاکردنی بانکی کار دەکات لە سیستمی ژمێریاریدا، چۆن هاوتاکردن بڕی پارەکان کار دەکات، و چۆن بڕە ناهاوتاەکان هاوتا دەکەیتەوە. وە چۆن چارەسەری جیاوازی بڕی پارەکان دەکەیت لە نێوان پارە جیاوازەکاندا. ئەم ڕێبەرییە نووسراوە بۆ هەموو بەکارهێنەرێک — ژمێریار بێت یان بەکارهێنەری ئاسایی — لەم ڕێبەرییەدا هەوڵدراوە بە نمونەی کردەیی هەنگاوەکان باس بکرێت کە پەیڕەوی لە باشترین شێوازەکانی ژمێریاری دوو-تۆماری دەکات.

---

## هاوتاکردنی بانکی چییە؟

هاوتاکردنی بانکی پرۆسەی بەراوردکردنی تۆمارەکانی پارەی کۆمپانیاکەتە لەگەڵ ڕاپۆرتی هەژماری بانک بۆ دیاریکردن و چارەسەرکردنی جیاوازیەکان لە نێوان تۆمارەکانی پارەدانی ناوخۆیی و تۆمارەکانی مامەڵەکانی بانک.

-   **_ تۆمارەکانی پارەی کۆمپانیا (تۆماری پارەدانەکان) _**: مەبەستمان ئەو تۆمارانەیە کە لەسیستمی ژمێریارییەکەتدا تۆمارکراوە.
-   **_ ڕاپۆرتی هەژماری بانک (تۆمارە دەرەکییەکان) _**: مەبەستمان لە ڕاپۆرتی بانکی یان لیستی تۆمارەکانی (بانک یان بانکەکانە) کە بەکاریان دێنیت بۆ مامەڵەی درایی. ئەم تۆمارە بانک دەیبەخشێت و لیستی مامەڵە داراییەکانت نیشان دەدات و پێویستە لە ناو تۆمارەکانی سیستمی ژمێریارییەکەتدا تۆماریان بکەیت بۆ هاوتاکردن یان هاوتاکردنیان لەگەل تۆمارەکانی ناو سیستمی ژمێریارییەکەت.

-   **دڵنیابوونەوەی دروستی**: دڵنیابوون لەوەی تۆماری پارەدانەکان لەگەڵ تۆمارەکانی بانک هاوتان.
-   **دیاریکردنی جیاوازییەکان**: دۆزینەوەی هەڵەکان یان جیاوازی لە تۆمارەکا ناوخۆیی و دەرەکییەکاندا.
-   **نوێکردنەوەی تۆمارەکان**: نیشانکردنی پارەدانەکان وەک هاوتاکراو دوای هاوتاکردنەوەیان.
-   **چارەسەرکردنی بڕگە جیاوازەکان یان تۆمارنەکراوەکان**: چارەسەکردنی جیاوازییەکان یان لێکۆڵینەوەی تۆمارە نادیارەکان.

**بنەمای ژمێریاری**: تۆمارەکانی پارەدان دەبێت لەگەڵ مامەڵەکانی ڕاپۆرتی بانک بگونجێت دوای ژماردن و دیاریکردنی جیاوازی لە کات، پێویستە لە کۆتاییدا ڕاپۆرتی هەژماری کاش دروست بێت هەڵەی تێدا نەبێت.

## پێداویستیەکانی سیستەم

### ڕێکخستنەکانی کۆمپانیا

-   **چالاککردنی ڕێکخستن**: دەبێت لە ڕێکخستنەکانی کۆمپانیادا چالاک کرابێت.
-   **هەژمارە بنەڕەتیەکان**: هەژمارەکانی بانک و هەژمارەکانی ڕێکخستن دەبێت ڕێکخراو بن

### پێشمەرجەکان

1. **ڕاپۆرتی هەژماری بانک دروستکراو**: [ڕاپۆرتی هەژماری بانک](bank-statements.ckb.md) دەبێت دروستکرابێت لەگەڵ تۆمارەکانی بانک.
2. **تۆمارەکانی پارەدان**: تۆماری پارەدانەکان دەبێت پشتڕاست کرابێتەوە و ئامادە بێت بۆ هاوتاکردن
3. **هاوڕێیی پارە**: هەردوو ڕاپۆرتەکانی هەژماری بانک و پارەدانەکان دەبێت پارەی گونجاو بەکاربهێنن

---

## چۆن لە ڕووکاری بەکارهێنەردا بیدۆزینەوە

بڕۆ بۆ **ژمێریاری → بانک و پارەی نەقد → ڕاپۆرتەکانی هەژماری بانک**

هاوتای بانکی لەم بەشانەشدا دەردەکەون:

-   **لیستی ڕاپۆرتەکانی هەژماری بانک**: دوگمەی کردار "هاوتای" (ئایکۆنی هاوتاکردن سەوز)
-   **تۆمارەکانی پارەدان**: نیشانکەرەکانی دۆخی هاوتاکردن.

**تێبینی**: کرداری هاوتای تەنها کاتێک دەردەکەوێت کە هاوتای لە ڕێکخستنەکانی کۆمپانیادا چالاککرابێت.

## دەستپێکردنی هاوتای بانکی

بڕۆ بۆ **ژمێریاری → بانک و پارەی نەقد → ڕاپۆرتەکانی هەژماری بانک**

### هەنگاوی ۱: دەستگەیشتن بە ڕووکاری هاوتای

**ڕاپۆرتی هەژماری بانک بدۆزەرەوە**: ئەو ڕاپۆرتە بدۆزەرەوە کە دەتەوێت هاوتای بکەیت لە لەگەڵ لیستەکە
**کلیک لە کردەوەی هاوتای بکە**: دوگمەی "هاوتای" بەکاربهێنە (ئایکۆنی هاوتاکردن سەوز)

![bank-statements.jpeg](/images/docs/bank-reconcilation/bank-statements.ckb.png)

### هەنگاوی ۲: تێگەیشتن لە ڕووکاری بەکارهێنەر

پەنجەرەی هاوتای دیزاینێکی کارلێکی سێ پانێڵی بەکاردەهێنێت:

**پانێڵی ڕاست - مامەڵەکانی بانک**: ڕیزەکانی ڕاپۆرتی هەژماری بانکی هاوتا نەکراو پیشاندەدات

-   بینینی هەموو مامەڵەکانی بانکی بەردەست
-   دەستنیشانکردنی مامەڵەکان بە بەکارهێنانی چوارچێوەکانی هەڵبژاردن
-   بەشی کورتەی هاوتاکردن، خۆکارانە دەیژمێرێت و هاوسەنگی نیشاندەدات

**پانێڵی چەپ - تۆماری پارەدانەکان**: پارەدانە پشتڕاستکراوەکان بەبێ هاوتای پیشاندەدات

-   بینینی پارەدانە پشتڕاستکراو و هاوتا نەکراوەکان
-   پشتگیری پارە جیاوازەکان کە خۆکارانە جیاوازی ئاڵوگۆڕی پارە دەژمێرێت
-   دەستنیشانکردنی پارەدانەکان بە بەکارهێنانی چوارچێوەکانی هەڵبژاردن

**پانێڵی خوارەوە - کورتەی هاوتاکردن**: نیشاندانی هاوسەنگی ڕاستەوخۆ

-   کۆی بانک (کۆی بڕگە دەستنیشانکراوەکانی بانک)
-   کۆی تۆماری پارەدانەکان (کۆی پارەدانە هەڵبژێردراوەکان)
-   ژماردنی جیاوازی
-   نیشاندانی دۆخی هاوسەنگی

### هەنگاوی ۳: هاوتاکردنی مامەڵەکان

**هەڵبژاردنی مامەڵەکانی بانک**: کلیک لە چوارچێوەکان بکە لە پانێڵی چەپدا بۆ هەڵبژاردنی ڕیزەکانی ڕاپۆرتی هەژماری بانک
**هەڵبژاردنی تۆماری پارەدانەکان**: کلیک لە چوارچێوەکان بکە لە پانێڵی ڕاستدا بۆ هەڵبژاردنی پارەدانە پەیوەندیدارەکان
**چاودێری یەکە**: چاودێری پانێڵی کورتەی هاوتاکردن بکە بۆ دڵنیابوون لە هاوتای لە بڕی پارەکاندا.
**جێبەجێکردنی هاوتاکردن**: کلیک لە "هاوتاکردنی هەڵبژێردراوەکان" بکە کاتێک کۆی ژمارەکان یەکدەگرنەوە

## هاوتاکردنی پارە جیاوازەکان

### ئاڵوگۆڕی پارە خۆکارانە

سیستەم خۆکارانە سیناریۆکانی ئاڵوگۆڕی پارە بەڕێوەدەبات:

**تۆماری پارەدانەکان کە بە پارەی جیاواز تۆمارکراون**: پارەدانەکان کە بە پارەی جیاواز تۆمارکراون لە ڕاپۆرتی داراییدا دەگۆڕدرێن بۆ پارەی بنچینەیی کۆمپانیا بە نرخی ڕۆژی تۆمارکراو
**ڕێژەی کاتی ڕاستەقینە**: نرخی رۆژی پارە بەکاردەهێنرێت بۆ ژماردن لە ڕاپۆرتەکاندا
**پیشاندانی زانیاری**: هەردوو بڕی ڕەسەن و گۆڕدراو پیشاندەدات
**ژماردنی هاوتای**: هەموو کۆکان بە پارەی بنچینەیی کۆمپانیا دەژمێردرێن

### هێماکانی پارە

-   **هەمان پارە**: هێمای سەوز بۆ پارە هاوتاکان
-   **پارەی جیاواز**: هێمانی زەرد پیشاندانی ئاڵوگۆڕی جێبەجێکراو
-   **وردەکاریەکانی گۆڕین**: سەرنج بدە یان کلیک بکە بۆ زانیاری یان جیاوازی ڕێژە

**نموونە**: هاوتای پارەدانێکی دۆلاری ئەمریکی لە دژی ڕاپۆرتی هەژماری بانکی دیناری عێڕاقی:

-   پارەدانی سیستەم: $500 دۆلاری ئەمریکی
-   نرخی ئاڵوگۆڕ: 1 دۆلاری ئەمریکی = 1,310 دیناری عێڕاقی
-   بڕی ئاڵوگۆڕکراو: 655,000 دیناری عێڕاقی
-   ڕیزی ڕاپۆرتی هەژماری بانک: 655,000 دیناری عێڕاقی (یەکدەگرێتەوە دوای ئاڵوگۆڕ)

## پاککردنەوە (پاککردنەوە، مانای سڕینەوە یان ڕەشکردنەوە نادات)

### چارەسەرکردنی بڕگە ناهاوتاکان

بۆ جیاوازییە بچووکەکان یان کرێی بانکەکان (کرێی بانکی مامەڵە داراییەکان) کە پارەدانی پەیوەندیداری ناوخۆییان نییە:

### هەنگاوی ۱: دیاریکردنی مامەڵە ناهاوتاکان

**پێداچوونەوەی ڕیزەکانی ڕاپۆرتی هەژماری بانک**: مامەڵەکان بدۆزەرەوە کە پارەدانی پەیوەندیداریان نییە
**نموونە باوەکان**: کرێی بانک، کرێی سوود، جیاوازییە بچووکەکان

### هەنگاوی ۲: جێبەجێکردنی پاککردنەوە

**کلیک لە دوگمەی پاککردنەو**: ئایکۆنی "X" لە خشتەی مامەڵەکانی بانکدا بەکاربهێنە
![write-off.jpeg](/images/docs/bank-reconcilation/write-off.ckb.png)

**هەژماری خەرجی هەڵبژێرە**: هەژماری گونجاو هەڵبژێرە (نموونە "کرێی بانک")
![write-off-modal.jpeg](/images/docs/bank-reconcilation/write-off-modal.ckb.png)

**هۆکار تۆماربکە**: ڕوونکردنەوەی پێویست بۆ تۆمارە نوێکە دیاریبکە
**تۆماری نوێ دروست بکە**: سیستەم خۆکارانە تۆماری ڕۆژانە دروستدەکات و ڕیزەکە وەک هاوتاکراو نیشاندەدات.

### نموونەکانی پاککردنەوە

-   **کرێی بانک**: کرێی چاودێری مانگانە ($15-25)
-   **کرێی سوود**: کرێی زیادەپارە، سوودی قەرز
-   **جیاوازییە بچووکەکان**: جیاوازی خولاندنەوە کەمتر لە $5

## کاریگەری لەسەر تۆماری ڕۆژانە

### هاوتای سەرکەوتوو

```
قەرزدار (مدين) هەژماری خەرجی بانک    $25.00
    قەرزدەر (دائن) هەژماری بانک              $25.00
```

دوای هاوتاکردنی پارەدانەکان بە سەرکەوتویی، سیستەم تۆمارێکی ڕۆژانە دروستدەکات کە تۆماری پارەدانەکان دەگوازێتەوە بۆ هەژماری بانک. کە ئەمەش هاوتای (تسویە) دروستدەکات لە نێوانی بانک و پارەدانەکاندا.

### مامەڵەی پاککردنەوە

```
قەرزدار (مدين). خەرجی کرێی بانک    $25.00
    قەرزدەر (دائن). هەژماری بانک              $25.00
```

پاککردنەوە تۆماری ڕۆژانەی نوێ دروستدەکات بۆ تۆمارکردنی کرێی بانک یان جیاوازی لە کاتدا لە هەمان کاتدا هێڵەکانی ڕاپۆرتی هەژماری بانک وەک هاوتاکراو نیشاندەدات.

## سیناریۆ باوەکان

### سیناریۆی ۱: پرۆسەی هاوتای ستاندارد

هاوتاکردنی تەواوی تۆمارەکانی بانک و تۆماری پارەدانەکان:

**هەنگاوەکان**:

1. دەستگەیشتن بە تۆماری مامەڵە داراییەکانی بانک
2. هەڵبژاردنی مامەڵەی بانکی (گواستنەوەی بانکی $1,500)
3. هەڵبژاردنی تۆماری ناوخۆیی پارەدانی پەیوەندیدار (پارەدانی فرۆشیار $1,500)
4. دڵنیابوونەوەی لە هاوتایی "هاوتای" پیشاندەدات لەگەڵ جیاوازی سفر
5. کلیک لە "هاوتاکردنی هەڵبژێردراو" بکە بۆ تەواوکردنی هاوتاکردنیان

**ئەنجام**:

-   دۆخی پارەدان نوێدەکرێتەوە بۆ "هاوتاکراو"
-   ڕیزی ڕاپۆرتی هەژماری بانک بە تۆماری پارەدان دەبەسترێتەوە
-   ڕووکار هەڵبژاردنەکان پاکدەکاتەوە بۆ کۆمەڵەی داهاتوو نیشانیان ناداتەوە

### سیناریۆی ۲: پاککردنەوەی کرێی بانک بڕی بچووک

چاودێریکردن و هاوتاکردنی کرێی بانک بەبێ تۆماری ناوخۆیی پارەدانی پەیوەندیدار:
بە گشتی لە دۆخی ئاساییدا کرێی مامەڵە داراییەکان لە ڕێی بانکەوە کە (فی) پێدەوترێت، لای ژمێریار ڕۆژانە تۆمار ناکرێت لە تۆماری پارەدانەکاندا بەهۆی کەمی بڕی پارەکانیان وە لە هەمان کاتدا زۆری ژمارەی مامەڵەکانی ڕۆژانە. سەرەڕای ئەوەش کرێی مامەڵەی بانکی ڕاستەوخۆ ڕەنگناداتەوە لە مامەڵە داراییەکاندا بەڵکو دەبێت ژمێریار تا کۆتایی مانگ چاوەڕێ بکات تا تۆماری مامەڵە بانکییەکانی مانگی ڕابردووی بەردەستکەوێت تا تۆماریان بکات. لەم کاتە کە مامەڵە داراییەکان لە (ڕاپۆرتەکانی بانک) تۆماردەکات و هاوتای دروستدەکات لە نێوان تۆماری پارەدانەکان و تۆمارە بانکییەکان هەندێک بڕ یان تۆمار لە تۆماری بانکیدا بە ناهاوتای دەمێنێتەوە کە زۆربەی کات کرێ بانکییەکانن و پێویستە بە شێوازی تۆماری نوێ هاوتا بکرێن.

**هەنگاوەکان**:

1. دیاریکردنی هێڵی ڕاپۆرتی هەژماری بانکی نایەکگرتوو: -$25.00 "کرێی خزمەتگوزاری مانگانە"
2. کلیک بکە لە کردەوەی پاککردنەوە (ئایکۆنی X) لە مامەڵەی بانکدا
3. هەڵبژاردنی هەژماری خەرجی "کرێی خزمەتگوزاریەکانی بانک"
4. هۆکار تۆماربکە: "کرێی چاودێری هەژماری مانگانە"
5. سیستەم تۆماری خەرج دروستدەکات و ڕیزەکە وەک هاوتاکراو نیشاندەدات

**ئەنجام**:

-   تۆماری ڕۆژانە: قەرزدار (مدين) کرێی خزمەتگوزاریەکانی بانک $25، قەرزدەر (دائن) هەژماری بانک $25
-   ڕیزی ڕاپۆرتی هەژماری بانک وەک هاوتاکراو هێماکراوە
-   پێویستی بە کردەوەی زیاتر نییە

### سیناریۆی ۳: هاوتاکردنی پارەدانەکان کە بە پارەی جیاواز تۆمارکراون

هاوتای پارەدانێکی دۆلاری ئەمریکی لە دژی ڕاپۆرتی هەژماری بانکی دیناری عێڕاقی:

**نموونە**:

-   ڕاپۆرتی هەژماری بانک: 655,000 دیناری عێڕاقی گواستنەوەی دەرچوو
-   پارەدانی سیستەم: $500 دۆلاری ئەمریکی پارەدانی فرۆشیار
-   نرخی ئاڵوگۆڕ: 1 دۆلاری ئەمریکی = 1,310 دیناری عێڕاقی (تۆمارکراو لە کاتی پارەداندا)
-   بڕی ئاڵوگۆڕکراو: 655,000 دیناری عێڕاقی

**هەنگاوەکان**:

1. هەڵبژاردنی ڕیزی ڕاپۆرتی هەژماری بانکی دیناری عێڕاقی (655,000 دیناری عێڕاقی)
2. هەڵبژاردنی پارەدانی سیستەمی دۆلاری ئەمریکی ($500 دۆلاری ئەمریکی)
3. سیستەم ئاڵوگۆڕ پیشاندەدات: $500 × 1,310 = 655,000 دیناری عێڕاقی
4. دڵنیابوونەوەی کۆکان تەواو هاوتان
5. تەواوکردنی هاوتای بە شێوەی ئاسایی

**ئەنجام**:

-   هاوتای پارە جیاوەزەکان سەرکەوتوو بوو
-   نرخی رۆژی ئاڵوگۆڕی پارە تۆمارکراوە و پارێزراوە
-   هەردوو تۆمار وەک هاوتاکراو نیشانکراون

## باشترین شێوازەکان مامەڵەکردن لەگەڵ هاوتاکردن یان هاوتاکردن

### ئامادەکاری

-   **چالاککردنی هاوتاکردن**: هاوتاکردن چالاک بکە لە بەشی ڕێکخستنی کۆمپانیا پێش دەستپێکردن
-   **ڕاپۆرتەکانی تەواو**: دڵنیابە هەموو مامەڵەکانی بانک لە [ڕاپۆرتەکانی بانک](bank-statements.ckb.md)دا تۆمارکراون
-   **دڵنیابوونەوەی پارەدان**: دڵنیابەرەوە کە تۆماری پارەدانەکان لە دۆخی "پشتڕاستکراو"دان
-   **دروستی پارە**: پێداچونەوە بکە بە تۆمارەکاندا کە بە پارەی جیاواز مامەڵەیان پێوەکراوە

### پرۆسەی هاوتاکردن

-   **ڕێگەی سیستەماتیک**: بە شێوەی سیستماتیک بە مامەڵەکاندا بچۆرەوە بەپێی بەروار یان بڕ
-   **دڵنیابوونەوەی کاتی ڕاستەقینە**: بە ووردی چاودێری هاسەنگی بڕی پارەکان بکە کاتێک هێڵەکانی هەردوو پانێڵی ڕاست و چەپ دەستنیشان دەکەیت
-   **پاککردنەوەی دەستبەجێ**: جیاوازییە بچووکەکان بە خێرایی بە بەکارهێنانی پاککردنەوی بڕە ناهاوتاکان چارەسەربکە (دوای دڵنیابوونەوە لە تۆمارەکە)
-   **بەڵگەکردنی ڕوون**: هۆکاری ورد بۆ هەموو تۆمارەکانی پاککردنەوە تۆماربکە

### کۆنترۆڵی کوالیتی

-   **هاوتاکردنی بەردەام**: دوای بەدەستگەیشتنی تۆماری مامەڵە بانکییەکان لەلایەن بانکەوە لە ماوەی چەند ڕۆژێکی زۆر کەمدا هاوتایان بکە
-   **پێداچوونەوەی بڕگە ناهاوتاکان**: جیاوازییەکان بپشکنە کە لە سنووری پێشبینیکراو زیاترن
-   **بەدواداچوون و چاودێریکردن**: بڕگە ماوەکان (نەناسراوەکان) لە چەندین کاتی جیاوازدا چاودێری بکە

### پشکنین و پابەندبوون

-   **ڕێڕەوی پشکنین**: سیستەم مێژووی تەواوی هاوتاکردن و هاسەنگی دەپارێزێت
-   **بەڵگە پشتگیریکەرەکان**: ڕاپۆرتەکانی بانک و پشتڕاستکردنەوەکانی پارەدان ببەستەرەوە بەیەکەوە
-   **پرۆسەی پێداچوونەوە**: پێویستە سەرپەرشتیار پێداچوونەوە بۆ پاککردنەوە گەورەکان بکات
-   **ڕێکارەکانی کۆتایی مانگ**: هەموو هاوتاکردنەکان پێش داخستنی ماوە تەواو بکە

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

**پرسیار: کردەوەی هاوتای لە ڕاپۆرتی هەژماری بانکدا دیار نییە**
وەڵام: ڕێکخستنەکان و مۆڵەتەکان بپشکنە:

-   دڵنیابەرەوە کە "چالاککردنی هاوتای" لە ڕێکخستنەکانی کۆمپانیادا چالاککراوە
-   دڵنیابەرەوە کە بەکارهێنەر مۆڵەتی گونجاوی هەیە بۆ کردارەکانی هاوتای
-   دڵنیابەرەوە کە ڕاپۆرتی هەژماری بانک بەردەستەو تۆمارکراوە

**پرسیار: کۆکان هاوتا نابن لە مامەڵەی پارە جیاوازەکانداد**
وەڵام: پێداچوونەوە بکە لە بەشی بەڕێوەبردنی ئاڵوگۆڕی پارە:

-   نرخەکانی ئاڵوگۆڕی پارە بپشکنە کە بۆ پارەدان و هاوتای بەکارهاتوون
-   دڵنیابەرەوە بڕی پارەکان بە دروستی لە بەڵگەنامەکاندا تۆمارکراون
-   پێداچونەوە بکە بە جیاوازی بچووکی خولاندنەوە (بە شێوەی ئاسایی بڕی پارەکانیان کەمترە لە $1)
-   پاککردنەوە بەکاربێنە بۆ جیاوازی یان ناهاوتاییە بچوکەکان

**پرسیار: هیچ پارەدانێک لە پانێڵی تۆماری پارەدانەکاندا دەرناکەوێت**
وەڵام: دڵنیابەرەوە لە دۆخی پارەدان و فلتەرەکان:

-   دڵنیابەرەوە پارەدانەکان لە دۆخی "پشتڕاستکراوە"دان (نەک ڕەشنووس)
-   دڵنیابەرەوە کە پێشتر پارەدانەکان هاوتانەکراون
-   دڵنیابەرەوە کە پارەدانەکان سەر بە کۆمپانیا و هەژماری بانکی دروستن
-   وێبگەڕ نوێبکەرەوە ئەگەر ڕووکار کۆن دیاربێت

**پرسیار: ناتوانرێت پرۆسەی هاوتای تەواو بکرێت**
وەڵام: دڵنیابەرەوە لە دەستنیشانکراوەکان و هاوتاییان:

-   دڵنیابەرەوە کە کۆکان هاوتان (جیاوازی = 0)
-   دڵنیابەرەوە کە لانیکەم یەک بڕگە لە هەر پانێڵێکدا هەڵبژێردراوە
-   چاودێری هەڵەکانی پشتڕاستکردنەوە یان پەیامەکانی سیستەم بکە
-   هەڵبژاردنەکان پاکبکەرەوە و سەرلەنوێ دەست پێبکەرەوە ئەگەر پێویست بوو

### هەڵە پەیامەکان

**"هاوتای بۆ ئەم کۆمپانیایە چالاک نەکراوە"**

-   پەیوەندی بە بەڕێوەبەرەوە بکە بۆ چالاککردنی هاوتای لە ڕێکخستنەکانی کۆمپانیادا
-   دڵنیابەرە کە لە ڕێکخستنەکانی کۆمپانیادا هاوتای یان هاوتاکردن چالاککراوە

**"کۆی پارەدانە هەڵبژێردراوەکان لەگەڵ کۆی بانک ناگونجێت"**

-   بە وردی پێداچوونەوە بکە بە بڕە هەڵبژێردراوەکان
-   کێشەکانی ئاڵوگۆڕی پارە بپشکنە
-   بیر لە بەکارهێنانی پاککردنەوە بکەرەوە بۆ جیاوازییە بچووکەکان

## پرسیارە دووبارەبووەکان

**پرسیار: چی بەسەر پارەدانەکان دێت دوای ئەوەی هاوتا دەکرێن؟**
وەڵام: پارەدانە هاوتاکراوەکان بە دۆخی "هاوتاکراو" نیشان دەکرێن و بە ڕیزەکانی تایبەتی ڕاپۆرتی هەژماری بانک دەبەسترێنەوە. لە ڕووکارەکانی هاوتای داهاتوودا دەرناکەون بەڵام لە تۆمارەکانی پارەداندا دیارن لەگەڵ دۆخی هاوتای.

**پرسیار: دەتوانم هاوتاکردنێک بگەڕێنمەوە ئەگەر هەڵەم کرد؟**
وەڵام: سیستەم ڕێڕەوی پشکنین دەپارێزێت بەڵام هاوتایەکان دەبێت وەک هەمیشەیی بیری لێبکرێنەوە. پەیوەندی بە بەڕێوەبەری سیستەمەوە بکە ئەگەر پێویستت بە گەڕاندنەوەی تۆمارەکانی هاوتای هەیە بۆ ڕاستکردنەوە.

**پرسیار: چۆن مامەڵەکانی بانک بەڕێوەببەم کە دەبنە هۆی سپام لە نێوان ماوەکاندا؟**
وەڵام: مامەڵەکانی بانک دەبێت لە ماوەیەکدا هاوتا بکرێن کە لە ڕاپۆرتی هەژماری بانکدا دەردەکەون. ئەگەر تۆماری پارەدانەکان لە ماوەیەکی جیاوازدا تۆمارکراون، هێشتا دەتوانرێت لە کاتی هاوتایدا یەکبخرێنەوە.

**پرسیار: چی لەگەڵ ئەو بڕگانە بکەم کە ناهاوتان وە بڕی پارەکانیان زۆرە؟**
وەڵام: بە وردی لێکۆڵینەوە بکە پێش پاککردنەوە. بگەڕێ بۆ تۆماری پارەدانەکان کە ونبوون، بڕە هەڵەکان بپشکنە، یان جیاوازی کات. جیاوازییە گەورەکان لەوانەیە نیشانە بن بۆ هەڵە کە پێویستیان بە ڕاستکردنەوەیە نەک پاککردنەوە.

## فەرهەنگ

### زاراوە ژمێریارییەکان

-   **هاوتاکردنی بانکی** (Bank Reconciliation) - پرۆسەی هاوتاکردنی تۆمارەکانی پارەی کۆمپانیا لەگەڵ ڕاپۆرتی هەژماری بانک
-   **ڕاپۆرتی هەژماری بانک** (Bank Statement) - بەڵگەنامەی فەرمی بانک کە مامەڵەکانی هەژمار پیشاندەدات
-   **پارەدان** (Payment) - گواستنەوەی پارە لە کۆمپانیا بۆ لایەنێکی دیکە
-   **پاککردنەوە** (Write-off) - تۆمارکردنی بڕێک وەک لەدەستچوو یان نەگەڕاوە
-   **هاوتاکردن** (Matching) - هاوتاکردنی مامەڵەیەکی بانک لەگەڵ پارەدانێکی ناوخۆیی
-   **یەکەکردن** (Balancing) - دڵنیابوون لەوەی کۆکان یەکدەگرنەوە
-   **ژمێریاری دوو-تۆمار** (Double-Entry Accounting) - سیستەمێکی ژمێریاری کە هەر مامەڵەیەک لە لانیکەم دوو هەژماردا تۆماردەکات
-   **قەرزدار (مدين)** (Debit/Dr.) - لایەنی چەپی تۆماری ژمێریاری
-   **قەرزدەر (دائن)** (Credit/Cr.) - لایەنی ڕاستی تۆماری ژمێریاری
-   **تۆماری ڕۆژانە** (Journal Entry) - تۆمارکردنی مامەڵەیەک لە کتێبە ژمێریاریەکاندا
-   **هەژماری خەرج** (Expense Account) - هەژمارێک کە خەرجییەکانی کۆمپانیا تۆماردەکات

### زاراوە تەکنیکیەکانی سیستەم

-   **ڕووکاری کارلێک** (Interactive Interface) - ڕووکارێک کە ڕەوتی دووڕەفەی زانیاری ڕێگەپێدەدات
-   **پشتڕاستکردنەوە** (Validation) - پشکنینی دروستی داتا
-   **ئاڵوگۆڕی پارە** (Currency Conversion) - ئاڵوگۆڕی بڕێکی دیاریکراو لە پارەیەکەوە بۆ پارەیەکی دیکە
-   **نرخی ئاڵوگۆڕ** (Exchange Rate) - نرخی ئاڵوگۆڕی پارە
-   **ڕاستەوخۆ** (Real-time) - زانیاری کە دەستبەجێ نوێدەکرێتەوە

### دۆخەکانی سیستەم

-   **پشتڕاستکراوە** (Posted) - دۆخی پارەدانێک کە پشتڕاستکراوە بۆ تۆماری ڕۆژانە
-   **هاوتاکراو** (Reconciled) - دۆخی پارەدانێک یان مامەڵەیەک کە هاوتاکراوە
-   **ڕەشنووس** (Draft) - دۆخی سەرەتایی بەڵگەنامەیەک پێش ناردن بۆ تۆماری ڕۆژانە
-   **ماوە** (Outstanding) - مامەڵەیەک کە هێشتا هاوتا نەکراوە

---

هاوتاکردنی بانکی دروست، دڵنیات دەکاتەوە لە دروستی تۆمارەکانی کاش و متمانە دەبەخشێت بە ڕاپۆرتە داراییەکانت. ڕووکاری کارلێک پرۆسەی هاوتاکردن کاراتر دەکات لە هەمان کاتدا ڕێگای تەواوی پشکنین و کۆنترۆڵە دروستەکانی ژمێریاری پارێزراو دەکات.
