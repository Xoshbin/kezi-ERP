# ڕاپۆرتی پەرتووکی گشتی: مێژووی تەواوەتی

ئەم ڕێبەرە ڕوونی دەکاتەوە کە چۆن ڕاپۆرتی پەرتووکی گشتی (General Ledger) بەکاربهێنیت بۆ بینینی مێژووی وردی هەژمارەکانت. لە کاتێکدا تەرازووی پێداچوونەوە (Trial Balance) پوختەی هەژمارەکانت پیشان دەدات، پەرتووکی گشتی هەموو ئەو مامەڵانە پیشان دەدات کە بوونەتە هۆی دروستبوونی ئەو باڵانسە.

---

## پەرتووکی گشتی چییە؟

**پەرتووکی گشتی** بریتییە لە "پەرتووکی تۆماری کۆتایی". تۆماری وردی هەموو ئەو مامەڵانەی تێدایە کە بۆ هەر هەژمارێک لە سیستەمەکەدا نێردراون (Posted).

**بۆچی ئەمە گرنگە؟**
1.  **وردەکاری**: بە تەواوی دەریدەخات *بۆچی* هەژمارێک خاوەنی باڵانسێکی دیاریکراوە.
2.  **لێکۆڵینەوە**: ئامرازی سەرەکییە بۆ دۆزینەوەی هەڵەکان (بۆ نموونە، "بۆچی خەرجی ئۆفیسەکەم ئەوەندە زۆرە؟ ئا، دەبینم پێنج مامەڵەی کڕینی کەلوپەلی نوێی تێدایە").
3.  **شوێنپێهەڵگرتنی وردبینی**: تۆمارێکی هەمیشەیی بۆ هەموو جووڵە داراییەکان دابین دەکات.

---

## لەکوێ دەیدۆزیتەوە؟

بڕۆ بۆ: **ژمێریاری ← ڕاپۆرتەکان ← پەرتووکی گشتی**

---

## چۆنێتی بەکارهێنانی ڕاپۆرتەکە

### 1. دیاریکردنی فلتەرەکانت
ڕاپۆرتەکە بەهێزە چونکە دەتوانیت فلتەری بکەیت بۆ ئەوەی بە تەواوی ئەوەی دەتەوێت بیبینیت.

| فلتەر | وەسف |
|-------|------|
| **ڕێکەوتی دەستپێک** | سەرەتای ئەو ماوەیەی کە دەتەوێت پێداچوونەوەی بۆ بکەیت (بە شێوەیەکی بنەڕەتی سەرەتای مانگی ئێستایە). |
| **ڕێکەوتی کۆتایی** | کۆتایی ماوەکە (بە شێوەیەکی بنەڕەتی کۆتایی مانگی ئێستایە). |
| **هەژمارەکان** | بە بەتاڵی جێی بهێڵە بۆ بینینی هەموو هەژمارەکان، یان گەڕان بکە و دانەی تایبەت هەڵبژێرە (بۆ نموونە، "کاش"، "فرۆشتن"). |

### 2. دروستکردنی ڕاپۆرتەکە
کلیك لەسەر دوگمەی **دروستکردنی ڕاپۆرت** بکە (بەدوای ئایکۆنی <heroicon-o-play class="w-4 h-4 inline"/> بگەڕێ).

---

## تێگەیشتن لە ستوونەکان

ڕاپۆرتەکە مامەڵەکان بەپێی **هەژمار** کۆدەکاتەوە. بۆ هەر هەژمارێک، سەردێڕێک دەبینیت کە ئەمانەی تێدایە:
-   **کۆد و ناوی هەژمار**
-   **کردنەوەی باڵانس**: بەهای هەژمارەکە *پێش* ڕێکەوتی دەستپێکی دیاریکراو.
-   **داخستنی باڵانس**: بەهای هەژمارەکە *دوای* هەموو مامەڵە لیستکراوەکان.

لە ناو هەر هەژمارێکدا، ئەم دێڕانەی مامەڵە دەبینیت:

| ستوون | وەسف |
|-------|------|
| **ڕێکەوت** | کەی مامەڵەکە ڕوویداوە. |
| **ژمارەی بەڵگە** | ژمارەی بەڵگەنامەکە (بۆ نموونە، `INV/2024/0001`، `BILL/2024/005`). |
| **وەسف** | تێبینییەکی کورت لەسەر ئەوەی مامەڵەکە چی بووە. |
| **هەژماری بەرامبەر** | ئەو *هەژمارەی تر* کە کاریگەری ئەم مامەڵەیەی لەسەرە (یارمەتیت دەدات لە دوو-تۆمار تێبگەیت). |
| **قەرزدار (Debit)** | بڕی زیادکراو بۆ سامان/خەرجییەکان یان کەمکراوە لە قەرز/داهاتەکان. |
| **قەرزدەر (Credit)** | بڕی زیادکراو بۆ قەرز/داهاتەکان یان کەمکراوە لە سامان/خەرجییەکان. |
| **باڵانس** | باڵانسی بەردەوامی هەژمارەکە دوای ئەم دێڕە دیاریکراوە. |

---

## سیناریۆ باوەکان

### سیناریۆی 1: لێکۆڵینەوە لە باڵانسێکی گوماناوی
**دۆخ**: هەژماری "خەرجی گەشت" باڵانسی 5,000 دۆلار نیشان دەدات، کە زۆر دێتە بەرچاو بۆ ئەم مانگە.
**کردار**:
1.  ڕاپۆرتی پەرتووکی گشتی بۆ هەژماری "خەرجی گەشت" دەربخە.
2.  ستوونەکانی **وەسف** و **بڕ** بپشکنە.
3.  **ئەنجام**: مامەڵەیەکی هەڵە بۆ 2,000 دۆلار دەدۆزیتەوە کە دەبوایە "بەشداریکردنی سۆفتوێر" بوایە. ئێستا دەتوانیت ڕاستی بکەیتەوە.

### سیناریۆی 2: هاوتاکردنی هەژمارێکی بانکی دیاریکراو
**دۆخ**: دەتەوێت مامەڵەکانی "بانکی USD" لە سیستەمەکەتدا لەگەڵ ڕاپۆرتی ڕاستەقینەی بانکەکەت هاوتا بکەیت.
**کردار**:
1.  فلتەر بکە بەپێی هەژماری "بانکی USD".
2.  ڕێکەوتەکان دابنێ بۆ ئەوەی لەگەڵ ماوەی ڕاپۆرتی بانکەکەت بگونجێت.
3.  **ئەنجام**: بەراوردی **باڵانسی بەردەوام** بە شێوەی دێڕ بە دێڕ لەگەڵ ڕاپۆرتی بانکەکەت دەکەیت بۆ دۆزینەوەی هەر تۆمارێکی ونبوو یان زیادە.

### سیناریۆی 3: ئامادەکاری بۆ پێداچوونەوەی وردبین
**دۆخ**: وردبینێک داوای بینینی هەموو مامەڵەکانی پەیوەست بە "سەرمایەی پشکدار" دەکات بۆ ساڵەکە.
**کردار**:
1.  فلتەر بکە بەپێی "سەرمایەی پشکدار".
2.  مەودای ڕێکەوتەکە بۆ تەواوی ساڵی دارایی دابنێ.
3.  **ئەنجام**: لیستێکی پاک لە هەموو تێخستن یان کێشانەوەیەکی سەرمایە، ئامادەیە بۆ پێداچوونەوە.

## باشترین پراکتیزەکان

### 🔍 ووردبوونەوە (Drill Down)
ژمارەی بەڵگەکان بەکاربهێنە! ئەگەر مامەڵەی `INV/2024/001` دەبینیت کە سەیر دیارە، دەتوانیت بۆ ئەو ژمارەی پسووڵەیە لە سیستەمەکەدا بگەڕێیت بۆ بینینی بەڵگەنامە ڕەسەنەکە.

### 📅 چاودێری کردنەوەی باڵانس بکە
ئەگەر ڕاپۆرتەکە بۆ *ئازار (مانگی 3)* دەردەکەیت، **کردنەوەی باڵانس** نوێنەرایەتی هەموو ئەو شتانە دەکات کە *تا 28ی شوبات* ڕوویان داوە. ئەگەر کردنەوەی باڵانس هەڵە دیار بێت، هەڵەکە لە ماوەیەکی پێشوودا ڕوویداوە.

### 🧹 بە خاوێنی ڕایبگرە
پێداچوونەوەی بەردەوامی هەژمارە سەرەکییەکان (پشکنینی کاش، قەرزەکان لای خەڵک، قەرزەکان لەسەر ئێمە) لە پەرتووکی گشتیدا ڕێگری دەکات لە سەرئێشەی کۆتایی ساڵ.
