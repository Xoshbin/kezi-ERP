ئەم ڕێبەرە گشتگیرە ڕوونی دەکاتەوە چۆن گواستنەوەی نێوان کۆگاکان بەڕێوەببەیت بە بەکارهێنانی ڕێڕەوی دوو-هەنگاوی ناردن و وەرگرتن (ship and receive)، کە باس لە دروستکردنی گواستنەوە، چاودێریکردنی کاڵا لە ڕێگەدا، و بینینی تەواوی جوڵەی کۆگا لە نێوان کۆگاکان دەکات. نووسراوە بۆ هەموو بەکارهێنەران — بەڕێوەبەرانی کۆگا، ڕێکخەرانی لۆجیستیک، و ژمێریاران — ڕێنمایی پراکتیکی دابین دەکات بۆ بەڕێوەبردنی ئۆپەراسیۆنی ئاڵۆزی فرە-کۆگا.

---

## گواستنەوەی نێوان کۆگاکان چییە؟

گواستنەوەی نێوان کۆگاکان بریتین لە جوڵەی تایبەتی کۆگا کە شوێنپێی کاڵاکان هەڵدەگرن لە نێوان شوێنە جیاوازەکانی کۆگا بە بەکارهێنانی ڕێڕەوی دوو-هەنگاوی (ناردن → وەرگرتن) لەگەڵ بینینی کاڵا لە ڕێگەدا (in-transit)، کە دڵنیایی دەدات لە کۆنترۆڵی تەواو و لێپرسینەوە لە کاتی پرۆسەی گواستنەوەکەدا.

- **ڕێڕەوی دوو-هەنگاوی**: جیاکردنەوەی ناردن و وەرگرتن بۆ کۆنترۆڵی باشتر
- **چاودێریکردنی ناو-ڕێگە (In-Transit)**: چاودێریکردنی کاڵا کاتێک لە نێوان شوێنەکاندایە
- **هەڵگرتنی کاڵا (Reservation)**: هەڵگرتنی کاڵا لە سەرچاوە کاتێک گواستنەوە پشتڕاست دەکرێتەوە
- **تۆماری وردبینی تەواو**: شوێنپێی کێ ناردوویەتی، کەی، و کێ وەریگرتووە
- **پرۆسەکردنی نەرم**: پشتگیری ناردن و وەرگرتنی بەشەکی (partial)

**ئامانجی ژمێریاری**: گواستنەوەی نێوان کۆگاکان تۆماری وردی کۆگا دەپارێزێت لە چەندین شوێندا لە هەمان کاتدا تۆماری وردبینی دابین دەکات کە پێویستە بۆ کۆنترۆڵی ناوخۆیی و مطابقەکردنی نێوان کۆگاکان.

---

## پێداویستییەکانی سیستەم

### ڕێکخستنی شوێن
- **شوێنی سەرچاوە**: کۆگای ناوخۆیی کە کاڵای تێدایە
- **شوێنی مەبەست**: کۆگای ناوخۆیی بۆ وەرگرتنی کاڵا
- **شوێنی گواستنەوە (Transit)**: شوێنی خەیاڵی بۆ چاودێریکردنی کاڵا لە ڕێگەدا (ئەگەر پێویست بێت خۆکار دروست دەکرێت)

### مەرجە پێشوەختەکان
1. **چەندین کۆگا**: لانی کەم دوو شوێنی کۆگای ناوخۆیی ڕێکخرابن
2. **بەرهەمەکان**: بەرهەمی کۆگایی (Storable) کە چاودێریکردنی کۆگایان چالاک بێت
3. **دەسەڵاتی بەکارهێنەر**: دەسەڵاتی دروستکردن و پرۆسەکردنی هەڵگرتنی کۆگا (stock pickings)
4. **بەردەستبوونی کاڵا**: کاڵای پێویست لە شوێنی سەرچاوە بەردەست بێت

---

## لە کوێ دەیدۆزیتەوە لە ڕووکارەکەدا

بڕۆ بۆ **کۆگا (Inventory) ← هەڵگرتنی کۆگا (Stock Pickings)**

فیلتەر بە **جۆر: ناوخۆیی (Internal)** بکە بۆ بینینی فەرمانەکانی گواستنەوە

گواستنەوەی نێوان کۆگاکان هەروەها دەردەکەون لە:
- **داشبۆرد**: ویجێتی گواستنەوەکان لە ڕێگەدا
- **بڕەکانی کۆگا**: بڕە هەڵگیراوەکان لە سەرچاوە
- **جوڵەکانی کۆگا**: مێژووی وردی جوڵەکان
- **ڕاپۆرتەکان**: مێژووی گواستنەوە و ڕاپۆرتی ناو-ڕێگە

**تێبینی**: دوگمەی Help/Docs لە سەرەوە ئەم ڕێبەرە دەکاتەوە.

---

## تێڕوانینی گشتی بۆ ڕێڕەوی دوو-هەنگاوی

### بۆچی دوو هەنگاو؟

ڕێڕەوی دوو-هەنگاوی ئەمانە دابین دەکات:
- **لێپرسینەوە**: بەرپرسیارێتی جیاواز بۆ ناردن و وەرگرتن
- **بینین**: چاودێریکردنی کاڵا کاتێک لە ڕێگایە
- **کۆنترۆڵ**: پشتڕاستکردنەوەی بڕەکان لە هەردوو لا
- **نەرمی**: مامەڵەکردن لەگەڵ دواکەوتن لە نێوان ناردن و وەرگرتن
- **تۆماری وردبینی**: تۆماری تەواوی کێ چی کردووە و کەی

### دۆخەکانی ڕێڕەوی کار

```
ڕەشنووس (Draft) → پشتڕاستکراو (Confirmed) → نێردراو (Shipped) → تەواو (Done)
```

**ڕەشنووس**: فەرمانی گواستنەوە دروستکراوە، دەستکاری دەکرێت
**پشتڕاستکراو**: کاڵا لە سەرچاوە هەڵگیراوە (reserved)، ئامادەیە بۆ ناردن
**نێردراو**: کاڵا لە ڕێگایە، چاوەڕێی وەرگرتنە
**تەواو**: گواستنەوە تەواو بووە، کاڵا گەیشتۆتە مەبەست

---

## دروستکردنی فەرمانی گواستنەوە

بڕۆ بۆ **کۆگا ← هەڵگرتنی کۆگا ← دروستکردنی هەڵگرتنی کۆگا**

### هەنگاوی ١: زانیاری بنەڕەتی

**جۆری گواستنەوە**:
- **جۆر**: "ناوخۆیی" (Internal) هەڵبژێرە بۆ گواستنەوەی نێوان کۆگاکان
- **سەرچاوە**: ژمارەی گواستنەوە کە خۆکار دروست دەبێت
- **بەرواری دیاریکراو**: کەی گواستنەوەکە دەبێت ئەنجام بدرێت

**ڕێکخستنی شوێن**:
- **شوێنی سەرچاوە**: ئەو کۆگایەی کاڵاکە دەنێرێت
- **شوێنی گواستنەوە (Transit)**: بە بەتاڵی جێی بهێڵە (خۆکار دروست دەبێت) یان شوێنی بوونی هەڵبژێرە
- **شوێنی مەبەست**: ئەو کۆگایەی کاڵاکە وەردەگرێت

### هەنگاوی ٢: دێڕەکانی بەرهەم

**زیادکردنی بەرهەمەکان**:
1. کلیک لەسەر **Add Product Line** بکە
2. **بەرهەم**: بەرهەمە کۆگاییەکە هەڵبژێرە
3. **بڕ**: بڕی گواستنەوە
4. **لە شوێنی**: دیفاڵت بۆ شوێنی سەرچاوە
5. **بۆ شوێنی**: دیفاڵت بۆ شوێنی مەبەست

### هەنگاوی ٣: زانیاری زیادە

**زانیاری ئارەزوومەندانە**:
- **تێبینییەکان**: تێبینی ناوخۆیی دەربارەی گواستنەوەکە
- **سەرچاوە (Origin)**: ئاماژە بۆ بەڵگەنامەی بنچینەیی
- **پێشینە (Priority)**: ئاستی گرنگی گواستنەوەکە

**هەڵبژاردنەکانی پاشەکەوتکردن**:
- **پاشەکەوت وەک ڕەشنووس**: دواتر دەستکاری بکە
- **پاشەکەوت و پشتڕاستکردنەوە**: دەستبەجێ گواستنەوەکە پشتڕاست بکەرەوە

---

## هەنگاوی ١: پشتڕاستکردنەوە و ناردن

### پشتڕاستکردنەوەی گواستنەوە

بڕۆ بۆ فەرمانی گواستنەوە ← کلیک لەسەر **Confirm** بکە

**چی ڕوودەدات**:
- **هەڵگرتنی کاڵا**: کاڵا لە شوێنی سەرچاوە هەڵدەگیرێت (reserved)
- **گۆڕانی دۆخ**: گواستنەوە دەچێتە دۆخی "Confirmed"
- **ڕاستاندن**: سیستەم پشکنینی بەردەستبوونی کاڵا دەکات
- **داخستن**: وردەکاری گواستنەوەکە دادەخرێت (دەستکاری ناکرێت)

### ناردنی گواستنەوە

بڕۆ بۆ گواستنەوەی پشتڕاستکراو ← کلیک لەسەر **Ship** بکە

**کرداری ناردن**:
- **جوڵەی کۆگا**: جوڵەیەک دروست دەکات لە سەرچاوەوە بۆ گواستنەوە (transit)
- **گۆڕانی دۆخ**: گواستنەوە دەچێتە دۆخی "Shipped"
- **کات**: بەروار و کاتی ناردن تۆمار دەکات
- **تۆماری بەکارهێنەر**: تۆماری دەکات کێ ناردنەکەی ئەنجام داوە

**چی ڕوودەدات**:
1. **جوڵەی کۆگا دروست دەبێت**: لە سەرچاوەوە بۆ شوێنی transit
2. **کۆگای فیزیایی نوێ دەکرێتەوە**: لە کۆگای سەرچاوە کەمدەبێتەوە
3. **کۆگای Transit دروست دەبێت**: زیاد دەبێت بۆ شوێنی ناو-ڕێگە
4. **هەڵگرتن ئازاد دەکرێت**: کاڵا چیتر لە سەرچاوە هەڵگیراو نییە

---

## هەنگاوی ٢: وەرگرتنی گواستنەوە

### کرداری وەرگرتن

بڕۆ بۆ گواستنەوەی نێردراو ← کلیک لەسەر **Receive** بکە

**پرۆسەی وەرگرتن**:
- **جوڵەی کۆگا**: جوڵەیەک دروست دەکات لە transitـەوە بۆ مەبەست
- **گۆڕانی دۆخ**: گواستنەوە دەچێتە دۆخی "Done"
- **تەواوکردن**: گواستنەوەکە وەک تەواو دیاری دەکات

**چی ڕوودەدات**:
1. **جوڵەی کۆگا دروست دەبێت**: لە transitـەوە بۆ شوێنی مەبەست
2. **کۆگای فیزیایی نوێ دەکرێتەوە**: لە transit لادەبرێت، بۆ مەبەست زیاد دەکرێت
3. **گواستنەوە تەواو دەبێت**: دۆخ دەبێتە "Done"

---

## چاودێریکردنی کاڵا لە ڕێگەدا (In-Transit)

### بینینی گواستنەوەکان لە ڕێگەدا

بڕۆ بۆ **کۆگا ← هەڵگرتنی کۆگا**

**هەڵبژاردنەکانی فیلتەر**:
- **جۆر**: Internal
- **دۆخ (State)**: Shipped

### ویجێتی داشبۆرد

**ویجێتی گواستنەوەکان لە ڕێگەدا**:
- **ژمارە**: ژمارەی گواستنەوەکان کە ئێستا لە ڕێگەن
- **کۆی بەها**: بەهای ئەو کاڵایانەی لە ڕێگەن
- **تەمەن (Aging)**: چەند کاتە گواستنەوەکان لە ڕێگەن
- **ئاگادارییەکان**: گواستنەوە دواکەوتووەکان

---

## کەی گواستنەوەی سادە یان دوو-هەنگاوی بەکاربهێنیت

### کەی گواستنەوەی سادە بەکاربهێنیت (Standard Transfer)
ئەمانە بەکاربهێنە کاتێک:
- **هەمان شوێن**: جوڵە لە ناو هەمان کۆگای فیزیایی
- **دەستبەجێ**: گواستنەوە دەستبەجێ ڕوودەدات
- **هیچ کاتێکی گواستنەوە نییە**: هیچ کاتێک نییە لە نێوان ناردن و وەرگرتن
- **یەک کەس**: هەمان کەس هەردوو لا بەڕێوەدەبات

**نموونە**: گواستنەوەی کاڵا لە "کۆگای A - ناوچەی ١" بۆ "کۆگای A - ناوچەی ٢"

### کەی گواستنەوەی دوو-هەنگاوی بەکاربهێنیت
ئەمانە بەکاربهێنە کاتێک:
- **شوێنی جیاواز**: گواستنەوە لە نێوان کۆگای جیاواز
- **کاتی گواستنەوە**: کاڵا کاتی دەوێت لە ڕێگەدا
- **کەسی جیاواز**: ناردن و وەرگرتن لەلایەن کارمەندی جیاوازەوە دەکرێت
- **لێپرسینەوە**: پێویستت بە پشتڕاستکردنەوەی جیاوازە بۆ ناردن و وەرگرتن

**نموونە**: گواستنەوەی کاڵا لە "کۆگای A - سەرەکی" بۆ "کۆگای B - سەنتەری دابەشکردن"

---

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

**پرسیار: بۆچی دوگمەی Ship نابینم؟**
وەڵام: دوگمەی Ship تەنها بۆ گواستنەوەی جۆری "Internal" دەردەکەوێت لە دۆخی "Confirmed". دڵنیابەرەوە جۆرەکە "Internal"ـە و دۆخەکە "Confirmed"ـە.

**پرسیار: بۆچی دوگمەی Receive نابینم؟**
وەڵام: دوگمەی Receive تەنها بۆ گواستنەوەکان دەردەکەوێت لە دۆخی "Shipped". دڵنیابەرەوە سەرەتا گواستنەوەکە نێردراوە.

**پرسیار: کاڵاکانی ناو-ڕێگەم بۆ کوێ ڕۆیشتن؟**
وەڵام: سەیری شوێنی Transit بکە لە "بڕەکانی کۆگا". کاڵا لە ڕێگەدا لە شوێنی transit دەردەکەوێت تا وەردەگیرێت.

**پرسیار: ئایا دەتوانم گواستنەوەیەکی نێردراو ڕەت بکەمەوە (Cancel)؟**
وەڵام: نەخێر، گواستنەوە نێردراوەکان ڕەت ناکرێنەوە. دەبێت وەریبگریت، پاشان گواستنەوەیەکی پێچەوانە (reverse) دروست بکەیت ئەگەر پێویست بوو.

---

گواستنەوەی نێوان کۆگاکان کۆنترۆڵ، بینین، و لێپرسینەوە دابین دەکات بۆ بەڕێوەبردنی کۆگا لە نێوان چەندین شوێن، دڵنیایی دەدات لە تۆماری وردی کۆگا و تۆماری وردبینی تەواو بۆ هەموو جوڵەکانی نێوان شوێنەکان.
