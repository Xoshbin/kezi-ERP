# ڕێکخستنی بانک: دڵنیابوون لە دروستی پارە و کۆنترۆڵی دارایی

ئەم ڕێنماییە ڕوونکردنەوەی چۆنێتی ئەنجامدانی ڕێکخستنی بانک لە سیستەمی ژمێریاریتدا دەکات. ڕێکخستنی بانک پرۆسەی یەکخستنی تۆمارەکانی پارەی ناوخۆیی لەگەڵ تۆمارەکانی بانکە بۆ دڵنیابوون لە دروستی و دۆزینەوەی جیاوازیەکان. ئەم تایبەتمەندییە پێویستی بە [ڕاپۆرتەکانی بانک](bank-statements.ckb.md) هەیە کە سەرەتا دروست بکرێن.

**گرنگ**: ڕێکخستنی بانک تەنها کاتێک بەردەستە کە ڕێکخستنی "چالاککردنی ڕێکخستن" لە ڕێکخستنەکانی کۆمپانیاکەتدا چالاک بکرێت.

---

## ڕێکخستنی بانک چییە؟

ڕێکخستنی بانک پرۆسەی بەراوردکردنی تۆمارەکانی پارەی کۆمپانیاکەت لەگەڵ [ڕاپۆرتی بانک](bank-statements.ckb.md)ە بۆ:
- **دڵنیابوون لە دروستی**: دڵنیابوون لەوەی تۆمارە ناوخۆییەکان لەگەڵ تۆمارەکانی بانک دادەگرن
- **دۆزینەوەی جیاوازی**: دۆزینەوەی هەڵەکان یان جیاوازی کاتی
- **نوێکردنەوەی تۆمارەکان**: نیشانکردنی پارەدانەکان وەک ڕێکخراو
- **بەڕێوەبردنی بڕگە نایەکگرێتەوەکان**: جێهێشتنی جیاوازی بچووکەکان یان لێکۆڵینەوە لە گەورەترەکان

**بنەمای ژمێریاری**: تۆمارەکانی پارەدانت دەبێت لەگەڵ مامەڵەکانی ڕاپۆرتی بانک بگونجێت دوای بیرکردنەوە لە جیاوازیەکانی کاتی.

---

## پێداویستیەکانی سیستەم

### ڕێکخستنەکانی کۆمپانیا
- **چالاککردنی ڕێکخستن**: دەبێت لە ڕێکخستنەکانی کۆمپانیادا چالاک بکرێت
- **هەژمارە بنەڕەتیەکان**: هەژمارەکانی بانک و هەژمارەکانی ڕێکخستن دەبێت ڕێکخراو بن
- **مۆڵەتەکانی بەکارهێنەر**: بەکارهێنەران پێویستیان بە دەستپێگەیشتنی گونجاو هەیە بۆ تایبەتمەندیەکانی ڕێکخستن

### پێشمەرجەکان
1. **ڕاپۆرتی بانک دروستکراو**: [ڕاپۆرتی بانک](bank-statements.ckb.md) دەبێت هەبێت لەگەڵ ڕیزەکانی مامەڵە
2. **تۆمارەکانی پارەدان**: پارەدانەکانی سیستەم دەبێت پشتڕاست بکرێت و ئامادە بێت بۆ یەکخستن
3. **ڕێکخستنی دراو**: هەردوو ڕاپۆرتی بانک و پارەدانەکان دەبێت دراوی گونجاو بەکاربهێنن

---

## تێڕوانینی گشتی پرۆسەی ڕێکخستن

### ۱. کۆنترۆڵی دەستپێگەیشتن
- بپشکنە ڕێکخستن بۆ کۆمپانیاکەت چالاککراوە
- دڵنیابە مۆڵەتت هەیە بۆ ئەنجامدانی ڕێکخستن
- دڵنیابە ڕاپۆرتی بانک بەردەستە بۆ پرۆسەکردن

### ۲. ڕێکخستنی ڕووکار
- ڕاپۆرتی بانکی بۆ ڕێکخستن هەڵبژێرە
- سیستەم ڕووکاری کارلێکی ڕێکخستن بارددەکات
- بینینی سێ پانێڵ مامەڵەکانی بانک، پارەدانەکانی سیستەم و کورتە پیشاندەدات

### ۳. قۆناغی یەکخستن
- ڕیزەکانی ڕاپۆرتی بانک لە پانێڵی چەپەوە هەڵبژێرە
- پارەدانە پەیوەندیدارەکانی سیستەم لە پانێڵی ڕاستەوە هەڵبژێرە
- حیسابکردنی بڕی کاتی ڕاستەقینە دۆخی یەکخستن پیشاندەدات

### ۴. قۆناغی چارەسەرکردن
- بڕگە تەواو یەکگرتووەکان ڕێکبخە
- بڕە بچووکە نایەکگرێتەوەکان جێبهێڵە
- جیاوازی گرنگەکان لێکۆڵینەوە بکە

---

## دەستپێکردنی ڕێکخستنی بانک

بڕۆ بۆ **ژمێریاری → بانک و پارەی نەقد → ڕاپۆرتەکانی بانک**

### دەستپێگەیشتن بە ڕێکخستن
1. **ڕاپۆرتی بانک بدۆزەرەوە**: ئەو ڕاپۆرتە بدۆزەرەوە کە دەتەوێت ڕێکی بخەیت
2. **کلیک لە کردەوەی ڕێکخستن بکە**: دوگمەی "ڕێکخستن" بەکاربهێنە (ئایکۆنی قەپانی سەوز)
3. **کۆنترۆڵی دەستپێگەیشتن**: کردەوەکە تەنها کاتێک دەردەکەوێت کە ڕێکخستن چالاک بێت

**تێبینی**: کردەوەی ڕێکخستن تەنها کاتێک دیارە کە کۆمپانیاکەت ڕێکخستنی لە ڕێکخستنەکاندا چالاککردووە.

### ڕووکاری ڕووکاری ڕێکخستن
شاشەی ڕێکخستن ڕووکارێکی کارلێکی Livewire بەکاردەهێنێت لەگەڵ سێ بەشی سەرەکی:

**پانێڵی چەپ - مامەڵەکانی بانک**: ڕیزەکانی ڕاپۆرتی بانکی ڕێکنەخراو پیشاندەدات
**پانێڵی ڕاست - پارەدانەکانی سیستەم**: پارەدانە پشتڕاستکراوەکان بە بێ ڕێکخستن پیشاندەدات
**پانێڵی خوارەوە - کورتەی ڕێکخستن**: کۆی کاتی ڕاستەقینە و دڵنیابوونەوەی بڕ

---

## یەکخستنی مامەڵەکان

### پرۆسەی هەڵبژاردنی کارلێک
سیستەم هەڵبژاردن و حیسابکردنی کاتی ڕاستەقینە دابیندەکات:

#### هەڵبژاردنی مامەڵەی بانک
1. **بینینی ڕیزە بەردەستەکان**: پانێڵی چەپ ڕیزەکانی ڕاپۆرتی بانکی ڕێکنەخراو پیشاندەدات
2. **هەڵبژاردنی بڕگەکان**: کلیک لە چوارچێوەکان بکە بۆ هەڵبژاردنی مامەڵەکانی بانک
3. **کۆی کاتی ڕاستەقینە**: سیستەم کۆی بڕگە هەڵبژێردراوەکانی بانک حیسابدەکات
4. **پیشاندانی دراو**: هەموو بڕەکان بە دراوی ڕاپۆرت پیشاندەدرێن

#### هەڵبژاردنی پارەدانی سیستەم  
1. **بینینی پارەدانە بەردەستەکان**: پانێڵی ڕاست پارەدانە پشتڕاست و ڕێکنەخراوەکان پیشاندەدات
2. **هەڵبژاردنی بڕگەکان**: کلیک لە چوارچێوەکان بکە بۆ هەڵبژاردنی پارەدانەکانی سیستەم
3. **پشتگیری زۆر دراوی**: پارەدانەکان لە دراوی جیاوازدا بە شێوەی خۆکار دەگۆڕدرێن
4. **کۆی کاتی ڕاستەقینە**: سیستەم کۆی پارەدانە هەڵبژێردراوەکان حیسابدەکات (لەگەڵ گۆڕینی دراو)

### دڵنیابوونەوەی بڕ
کورتەی ڕێکخستن فیدباکی دەستبەجێ دابیندەکات:
- **کۆی بانک**: کۆی ڕیزەکانی ڕاپۆرتی بانکی هەڵبژێردراو
- **کۆی سیستەم**: کۆی پارەدانەکانی سیستەمی هەڵبژێردراو (لەگەڵ گۆڕینی دراو)
- **جیاوازی**: جیاوازی حیسابکراو لە نێوان کۆکان
- **دۆخی بڕ**: سەوزی "بڕ" کاتێک جیاوازی سفرە

### جێبەجێکردنی ڕێکخستن
کاتێک کۆکان یەکدەگرنەوە:
1. **کلیک لە "ڕێکخستنی هەڵبژێردراو" بکە**: یەکخستنەکە پرۆسە بکە
2. **نوێکردنەوەی سیستەم**: دۆخی پارەدان دەگۆڕێت بۆ "ڕێکخراو"  
3. **دروستکردنی بەستنەوە**: ڕیزەکانی ڕاپۆرتی بانک بە تۆمارەکانی پارەدان دەبەسترێنەوە
4. **پاککردنەوەی خۆکار**: هەڵبژاردنەکان پاکدەکرێنەوە بۆ کۆمەڵەی داهاتووی مامەڵەکان

---

## بەڕێوەبردنی بڕگە نایەکگرێتەوەکان

### تایبەتمەندی جێهێشتن
بۆ جیاوازی بچووکەکان یان کرێی بانکەکان کە پارەدانی پەیوەندیداری سیستەمیان نییە:

#### جێهێشتنی ڕیزی ڕاپۆرتی بانک
1. **نایەکگرێتەوە دیاری بکە**: مامەڵەی بانک بدۆزەرەوە کە پارەدانی پەیوەندیداری نییە
2. **کلیک لە کردەوەی جێهێشتن بکە**: ئایکۆنی "X" لە خشتەی مامەڵەکانی بانکدا بەکاربهێنە
3. **هەژماری خەرجی هەڵبژێرە**: هەژماری خەرجی گونجاو هەڵبژێرە (نموونە "کرێی بانک")
4. **هۆکار تێبخە**: ڕوونکردنەوە بۆ جێهێشتنەکە بدە
5. **تۆماری دروست بکە**: سیستەم تۆماری ژۆرنالی دروستدەکات و ڕیزەکە وەک ڕێکخراو نیشاندەدات

#### نموونەکانی جێهێشتن
- **کرێی بانک**: کرێی چاودێری مانگانە، کرێی مامەڵەکان
- **کرێی سوود**: کرێی زیادەپارە، سوودی قەرز
- **جیاوازی بچووک**: جیاوازی خولاندنەوە، جیاوازی حیسابکردنی بچووک

### پێویستی بە لێکۆڵینەوە
بۆ جیاوازی گەورەتر:
1. **وردەکارییەکانی مامەڵە پێداچوونەوە بکە**: بەروار، بڕ و پێناسەکان بپشکنە
2. **پارەدانەکانی سیستەم دڵنیابکەرەوە**: دڵنیابە هەموو پارەدانەکان بە دروستی تۆمارکراون
3. **کات بپشکنە**: بگەڕێ بۆ پارەدانەکان کە لەوانەیە لە ماوەی داهاتوودا بپەڕنەوە
4. **تۆمارە ونبووەکان دروست بکە**: مامەڵە پشتگوێخراوەکان تۆمار بکە ئەگەر پێویست بێت

---

## ڕێکخستنی زۆر دراوی

### گۆڕینی خۆکاری دراو
سیستەم سیناریۆکانی زۆر دراوی بە شێوەی خۆکار بەڕێوەدەبات:

#### پارەدانەکانی دراوی جیاواز
- **گۆڕینی خۆکار**: پارەدانەکان لە دراوی جیاوازدا بۆ دراوی ڕاپۆرت دەگۆڕدرێن
- **ڕێژەی کاتی ڕاستەقینە**: ڕێژەی گۆڕانی ئێستا بۆ حیسابکردنەکان بەکاردەهێنێت
- **زانیاری پیشاندان**: هەردوو بڕی ڕەسەن و گۆڕدراو پیشاندەدات
- **حیسابکردنی بڕ**: هەموو کۆکان بە دراوی ڕاپۆرت حیسابدەکرێن

#### نیشانکەرەکانی دراو
- **هەمان دراو**: نیشانکەری سەوز بۆ دراوە یەکگرتووەکان
- **دراوی جیاواز**: نیشانکەری زەرد پیشاندانی گۆڕینی جێبەجێکراو
- **وردەکاریەکانی گۆڕین**: سەرنج بدە یان کلیک بکە بۆ زانیاری ڕێژە

### بیرکردنەوەکانی ڕێکخستن
- **گۆڕانی ڕێژە**: جیاوازی بچووک لەوانەیە ڕووبدات بەهۆی گۆڕانی ڕێژەی گۆڕانەوە
- **ڕێکخستنەکانی تۆڵەرانس**: سیستەم لەوانەیە جیاوازی بچووک ڕێگەپێبدات بۆ گۆڕینەکانی دراو
- **بەڵگەکردن**: وردەکاریەکانی گۆڕین بۆ مەبەستەکانی پشکنین تۆمارکراون

---

## تەواوکردنی ڕێکخستن

### کارگێڕی ڕێکخستن
ڕووکاری کارلێک فیدباکی بەردەوام دابیندەکات:

#### چاودێری دۆخی کاتی ڕاستەقینە
- **نیشانکەری بڕ**: "یەکگرتووە" یان "یەکنەگرتووە" پیشاندەدات
- **بەدواداچوونی بڕ**: کۆی بانک، کۆی سیستەم و جیاوازی پیشاندەدات
- **ژمارەی هەڵبژاردن**: ژمارەی بڕگە هەڵبژێردراوەکان لە هەر پانێڵێکدا پیشاندەدات
- **زانیاری دراو**: هەر گۆڕینێکی دراو پیشاندەدات

#### ڕێکخستنی سەرکەوتوو
کاتێک مامەڵەکان بە سەرکەوتوویی ڕێکدەخرێن:
1. **نوێکردنەوەی دۆخ**: تۆمارەکانی پارەدان دەگۆڕێت بۆ دۆخی "ڕێکخراو"
2. **دروستکردنی بەستنەوە**: ڕیزەکانی ڕاپۆرتی بانک بە تۆمارەکانی پارەدان دەبەسترێنەوە
3. **ڕیسێتکردنی هەڵبژاردن**: ڕووکار هەڵبژاردنەکان پاکدەکاتەوە بۆ کۆمەڵەی ڕێکخستنی داهاتوو
4. **ئاگاداری**: پەیامی سەرکەوتن تەواوبوونی ڕێکخستن پشتڕاستدەکاتەوە

### کۆتاییهێنان بە ڕاپۆرت
دوای ڕێکخستنی هەموو مامەڵە گونجاوەکان:
1. **پێداچوونەوەی بڕگە ماوەکان**: مامەڵە نایەکگرتووەکان بپشکنە
2. **بەڕێوەبردنی جێهێشتنەکان**: جیاوازی بچووکەکان لە ڕێگەی تایبەتمەندی جێهێشتنەوە پرۆسە بکە
3. **لێکۆڵینەوە لە بڕگە گەورەکان**: بڕە نایەکگرتووە گرنگەکان پێداچوونەوە بکە
4. **بەڵگەکردنی بڕیارەکان**: تۆماری بڕیارەکانی ڕێکخستن پارێزراو بکە

### ئامادەکردنی ڕاپۆرتی داهاتوو
- **بڕگە ماوەکان**: مامەڵە نایەکگرتووەکان لە ڕێکخستنەکانی داهاتوودا دەردەکەون
- **جیاوازیەکانی کاتی**: بڕگەکان لەوانەیە چارەسەر بن کاتێک [ڕاپۆرتەکانی بانکی](bank-statements.ckb.md) دواتر پرۆسەدەکرێن
- **پێویستی بە دواپێچوون**: بڕگەکان بەدوادا بخە کە پێویستیان بە لێکۆڵینەوە یان ڕاستکردنەوەیە

---

## باشترین ڕێگاکان

### ئامادەکردن
- **چالاککردنی ڕێکخستن**: دڵنیابە ڕێکخستنی کۆمپانیا چالاککراوە پێش دەستپێکردن
- **ڕاپۆرتەکانی تەواو**: دڵنیابەرەوە [ڕاپۆرتەکانی بانک](bank-statements.ckb.md) بە تەواوی تێخراون لەگەڵ هەموو مامەڵەکان
- **دڵنیابوونەوەی پارەدان**: دڵنیابە هەموو پارەدانەکانی سیستەم لە دۆخی "پشتڕاستکراو"دان
- **پشکنینی دراو**: مامەڵەکانی زۆر دراوی بۆ دروستی پێداچوونەوە بکە

### پرۆسەی ڕێکخستن
- **ڕێگەی سیستەماتیک**: بە ڕێکوپێکی لە مامەڵەکاندا کاربکە
- **دڵنیابوونەوەی کاتی ڕاستەقینە**: نیشانکەری بڕ بەکاربهێنە بۆ دڵنیابوونەوە لە یەکخستن
- **جێهێشتنی دەستبەجێ**: جیاوازی بچووکەکان بە خێرایی بە بەکارهێنانی تایبەتمەندی جێهێشتن بەڕێوەببە
- **بەڵگەکردن**: هۆکاری ڕوون بۆ هەموو تۆمارەکانی جێهێشتن پارێزراو بکە

### کۆنترۆڵی کوالیتی
- **ڕێکخستنی ئاسایی**: ڕێکخستن بە خێرایی دوای دروستکردنی ڕاپۆرت ئەنجام بدە
- **پێداچوونەوەی بڕگە نایەکگرتووەکان**: جیاوازی گرنگەکان بە وردی لێکۆڵینەوە بکە
- **بەدواداچوونی دواپێچوون**: بڕگە ماوەکان لە چەندین ماوەدا چاودێری بکە
- **ڕاهێنانی بەکارهێنەر**: دڵنیابە تیمەکە ڕووکاری کارلێک تێدەگات

---

## سیناریۆ باوەکان

### سیناریۆی ۱: پرۆسەی ڕێکخستنی ستاندارد
ڕێکخستنی ڕاپۆرتێکی مانگانە لەگەڵ یەکگرتنەوەی تەواو:

1. دەستپێگەیشتن بە ڕێکخستن لە لیستی ڕاپۆرتی بانکەوە
2. مامەڵەی بانکی یەکگرتوو و پارەدانی سیستەم هەڵبژێرە
3. دڵنیابەرەوە بڕەکە "یەکگرتووە" پیشاندەدات 
4. کلیک لە "ڕێکخستنی هەڵبژێردراو" بکە
5. سیستەم دۆخی پارەدان نوێدەکاتەوە و هەڵبژاردنەکان پاکدەکاتەوە
6. بۆ مامەڵەکانی ماوە دووبارە بکەرەوە

### سیناریۆی ۲: جێهێشتنی کرێی بچووکی بانک
بەڕێوەبردنی کرێیەکی چاودێری بانک:

1. ڕیزی ڕاپۆرتی بانکی نایەکگرتوو بۆ کرێی ١٥ دینار دیاری بکە
2. کلیک لە کردەوەی جێهێشتن (ئایکۆنی X) لە مامەڵەی بانکدا بکە
3. هەژماری خەرجی "کرێی بانک" هەڵبژێرە
4. هۆکار تێبخە: "کرێی چاودێری مانگانە"
5. سیستەم تۆماری ژۆرنالی دروستدەکات و ڕیزەکە وەک ڕێکخراو نیشاندەدات

### سیناریۆی ۳: پارەدانی زۆر دراوی
ڕێکخستنی پارەدانێکی USD لە دژی ڕاپۆرتی بانکی IQD:

1. پارەدانی USD لە پانێڵی پارەدانەکانی سیستەمەوە هەڵبژێرە
2. سیستەم بە شێوەی خۆکار بۆ IQD لە ڕێژەی ئێستادا دەگۆڕێت
3. ڕیزی ڕاپۆرتی بانکی IQDی پەیوەندیدار هەڵبژێرە
4. دڵنیابەرەوە کۆکان یەکدەگرنەوە (ڕێگەپێدان بە جیاوازی بچووکی گۆڕین)
5. ڕێکخستن بە شێوەی ئاسایی تەواو بکە

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

**کێشە**: کردەوەی ڕێکخستن دیار نییە
**چارەسەر**:
- بپشکنە "چالاککردنی ڕێکخستن" لە ڕێکخستنەکانی کۆمپانیادا چالاککراوە
- مۆڵەتەکانی گونجاوی بەکارهێنەر دڵنیابکەرەوە
- دڵنیابە ڕاپۆرتی بانک هەیە و دەستپێگەیشتنی هەیە

**کێشە**: کۆکان بە وردی یەکناگرنەوە
**چارەسەر**:
- جیاوازیەکانی گۆڕینی دراو بپشکنە
- بڕەکانی مامەڵە بە دروستی تێخراون دڵنیابکەرەوە
- بگەڕێ بۆ پارەدانی بەشی یان کرێکان
- جێهێشتن بۆ جیاوازی ناگرنگ بەکاربهێنە

**کێشە**: هیچ پارەدانێک لە پانێڵی پارەدانەکانی سیستەمدا دەرناکەوێت
**چارەسەر**:
- دڵنیابەرەوە پارەدانەکان لە دۆخی "پشتڕاستکراو"دان
- بپشکنە پارەدانەکان پێشتر ڕێکنەخراون
- دڵنیابە پارەدانەکان سەر بە کۆمپانیا و ژۆرنالی دروستن

**کێشە**: ناتوانرێت ڕێکخستن تەواو بکرێت
**چارەسەر**:
- دڵنیابە کۆکان یەکدەگرنەوە (جیاوازی = ٠)
- دڵنیابەرەوە لانیکەم یەک بڕگە لە هەر پانێڵێکدا هەڵبژێردراوە
- بگەڕێ بۆ هەڵەکانی سیستەم یان کێشەکانی دڵنیابوونەوە
- تاقیکردنەوەی ڕووکار ئەگەر پێویست بێت

## یەکخستن لەگەڵ ڕاپۆرتەکانی بانک

ڕێکخستنی بانک بە توندی لەگەڵ سیستەمی [بەڕێوەبردنی ڕاپۆرتی بانک](bank-statements.ckb.md) یەکخراوە:

### گەرانی دراوە
1. **دروستکردنی ڕاپۆرت**: [ڕاپۆرتەکانی بانک](bank-statements.ckb.md) دراوەی مامەڵەی دەرەکی دابیندەکەن
2. **پرۆسەی ڕێکخستن**: ئەم ڕووکارە دراوەی دەرەکی لەگەڵ پارەدانەکانی ناوخۆیی یەکدەخات  
3. **نوێکردنەوەی دۆخ**: بڕگە ڕێکخراوەکان نیشانکراو و بەستراوەتەوە لە هەردوو سیستەمدا
4. **ڕاپۆرتکردن**: دراوەی یەکخراو پشتگیری ڕاپۆرتکردنی گشتگیری بەڕێوەبردنی پارە دەکات

### زنجیرەی کارگێڕی
1. **دروستکردنی ڕاپۆرتی بانک**: لەگەڵ [دروستکردنی ڕاپۆرتی بانک](bank-statements.ckb.md) دەست پێبکە
2. **تێخستنی مامەڵەکان**: هەموو مامەڵەکانی بانک بۆ ڕاپۆرتەکە زیاد بکە
3. **دڵنیابوونەوەی ڕاپۆرت**: دڵنیابە بڕی دەستپێک/کۆتایی دروستن
4. **دەستپێکردنی ڕێکخستن**: ئەم پرۆسەی ڕێکخستنە بەکاربهێنە بۆ یەکخستنی مامەڵەکان
5. **تەواوکردنی سوور**: ئامادەبوون بۆ ڕاپۆرتی بانک و ڕێکخستنی ماوەی داهاتوو

---

ڕێکخستنی بانک دروستی تۆمارەکانی پارەت دڵنیادەکاتەوە و متمانە بە ڕاپۆرتکردنی داراییت دەبەخشێت. ڕووکاری کارلێک پرۆسەی یەکخستن کاراتر دەکات لە هەمان کاتدا ڕێگای تەواوی پشکنین و کۆنترۆڵە دروستەکانی ژمێریاری پارێزراو دەکات.
