# ڕێکخستنی بانک: دڵنیابوون لە دروستی پارە و کۆنترۆڵی دارایی

ئەم ڕێنماییە گشتگیرە ڕوونکردنەوەی چۆنێتی کارکردنی ڕێکخستنی بانک لە سیستەمی ژمێریاریتدا دەکات، بەربەستکردنی پرۆسەی یەکخستن، جێهێشتن، و بەڕێوەبردنی زۆر دراو. نووسراوە بۆ هەموو بەکارهێنەران — ژمێریار و نا-ژمێریارەکان — ڕێنماییەکی کردەیی دابیندەکات کە پایڕەوی لە باشترین شێوازەکانی ژمێریاری دووقات ئێنتری دەکات.

---

## ڕێکخستنی بانک چییە؟

ڕێکخستنی بانک پرۆسەی بەراوردکردنی تۆمارەکانی پارەی کۆمپانیاکەت لەگەڵ ڕاپۆرتی بانکە بۆ دیاریکردن و چارەسەرکردنی جیاوازیەکان لە نێوان تۆمارەکانی پارەدانی ناوخۆیی و تۆمارەکانی مامەڵەکانی بانک.

- **دڵنیابوونەوەی دروستی**: دڵنیابوون لەوەی تۆمارە ناوخۆییەکان لەگەڵ تۆمارەکانی بانک دەگونجێن
- **دیاریکردنی ناکۆکیەکان**: دۆزینەوەی هەڵەکان یان جیاوازی کاتی
- **نوێکردنەوەی تۆمارەکان**: نیشانکردنی پارەدانەکان وەک ڕێکخراو کاتێک یەکدەگرنەوە
- **بەڕێوەبردنی بڕگە نایەکگرێتەوەکان**: جێهێشتنی ناکۆکی بچووکەکان یان لێکۆڵینەوە لە گەورەترەکان

**بنەمای ژمێریاری**: تۆمارەکانی پارەدانت دەبێت لەگەڵ مامەڵەکانی ڕاپۆرتی بانک بگونجێت دوای بیرکردنەوە لە جیاوازیەکانی کاتی، دڵنیابوون لە ڕاپۆرتکردنی دروستی پارە.

---

## پێداویستیەکانی سیستەم

### ڕێکخستنەکانی کۆمپانیا
- **چالاککردنی ڕێکخستن**: دەبێت لە ڕێکخستنەکانی کۆمپانیادا چالاک بکرێت
- **هەژمارە بنەڕەتیەکان**: هەژمارەکانی بانک و هەژمارەکانی ڕێکخستن دەبێت ڕێکخراو بن
- **مۆڵەتەکانی بەکارهێنەر**: بەکارهێنەران پێویستیان بە دەستپێگەیشتنی گونجاو هەیە بۆ تایبەتمەندیەکانی ڕێکخستن

### پێشمەرجەکان
1. **ڕاپۆرتی بانک دروستکراو**: [ڕاپۆرتی بانک](bank-statements.ckb.md) دەبێت هەبێت لەگەڵ ڕیزەکانی مامەڵە
2. **تۆمارەکانی پارەدان**: پارەدانەکانی سیستەم دەبێت پشتڕاست بکرێت و ئامادە بێت بۆ یەکخستن
3. **هاوڕێیی دراو**: هەردوو ڕاپۆرتەکانی بانک و پارەدانەکان دەبێت دراوی گونجاو بەکاربهێنن

---

## لە کوێ دەیدۆزیتەوە لە UI دا

بڕۆ بۆ **ژمێریاری → بانک و پارەی نەقد → ڕاپۆرتەکانی بانک**

ڕێکخستنی بانک هەروەها دەردەکەوێت لە:
- **لیستی ڕاپۆرتەکانی بانک**: دوگمەی کردار "ڕێکخستن" (ئایکۆنی قەپانی سەوز)
- **تۆمارەکانی پارەدان**: نیشانکەرەکانی دۆخی ڕێکخستن

**تێبینی**: کردەوەی ڕێکخستن تەنها کاتێک دیارە کە ڕێکخستن لە ڕێکخستنەکانی کۆمپانیادا چالاک بێت.

---

## دەستپێکردنی ڕێکخستنی بانک

بڕۆ بۆ **ژمێریاری → بانک و پارەی نەقد → ڕاپۆرتەکانی بانک**

### هەنگاوی ۱: دەستپێگەیشتن بە ڕووکاری ڕێکخستن

**ڕاپۆرتی بانک بدۆزەرەوە**: ئەو ڕاپۆرتە بدۆزەرەوە کە دەتەوێت ڕێکی بخەیت لە لیستەکە
**کلیک لە کردەوەی ڕێکخستن بکە**: دوگمەی "ڕێکخستن" بەکاربهێنە (ئایکۆنی قەپانی سەوز)
**کۆنترۆڵی دەستپێگەیشتن**: کردەوەکە تەنها کاتێک دەردەکەوێت کە ڕێکخستن چالاک بێت

### هەنگاوی ۲: تێگەیشتن لە دیزاینی ڕووکار

شاشەی ڕێکخستن دیزاینێکی کارلێکی سێ پانێڵی بەکاردەهێنێت:

**پانێڵی چەپ - مامەڵەکانی بانک**: ڕیزەکانی ڕاپۆرتی بانکی ڕێکنەخراو پیشاندەدات
- بینینی هەموو مامەڵەکانی بانکی بەردەست
- هەڵبژاردنی مامەڵەکان بە بەکارهێنانی چوارچێوەکانی هەڵبژاردن
- حیسابکردنی کۆی کاتی ڕاستەقینە

**پانێڵی ڕاست - پارەدانەکانی سیستەم**: پارەدانە پشتڕاستکراوەکان بەبێ ڕێکخستن پیشاندەدات
- بینینی پارەدانە پشتڕاست و ڕێکنەخراوەکان
- پشتگیری زۆر دراوی لەگەڵ گۆڕینی خۆکار
- هەڵبژاردنی پارەدانەکان بە بەکارهێنانی چوارچێوەکانی هەڵبژاردن

**پانێڵی خوارەوە - کورتەی ڕێکخستن**: دڵنیابوونەوەی تەرازووی کاتی ڕاستەقینە
- کۆی بانک (کۆی بڕگە هەڵبژێردراوەکانی بانک)
- کۆی سیستەم (کۆی پارەدانە هەڵبژێردراوەکان)
- حیسابکردنی جیاوازی
- نیشانکەری دۆخی تەرازوو

### هەنگاوی ۳: یەکخستنی مامەڵەکان

**هەڵبژاردنی مامەڵەکانی بانک**: کلیک لە چوارچێوەکان بکە لە پانێڵی چەپدا بۆ هەڵبژاردنی ڕیزەکانی ڕاپۆرتی بانک
**هەڵبژاردنی پارەدانەکانی سیستەم**: کلیک لە چوارچێوەکان بکە لە پانێڵی ڕاستدا بۆ هەڵبژاردنی پارەدانە پەیوەندیدارەکان
**چاودێری تەرازوو**: پانێڵی کورتە بچاودێرە بۆ دۆخی "تەرازوو"
**جێبەجێکردنی ڕێکخستن**: کلیک لە "ڕێکخستنی هەڵبژێردراو" بکە کاتێک کۆکان یەکدەگرنەوە

## ڕێکخستنی زۆر دراوی

### گۆڕینی خۆکاری دراو
سیستەم سیناریۆکانی زۆر دراوی بە شێوەی خۆکار بەڕێوەدەبات:

**پارەدانەکانی دراوی جیاواز**: پارەدانەکان لە دراوی جیاوازدا بۆ دراوی ڕاپۆرت دەگۆڕدرێن
**ڕێژەی کاتی ڕاستەقینە**: ڕێژەی گۆڕانی ئێستا بۆ حیسابکردنەکان بەکاردەهێنێت
**زانیاری پیشاندان**: هەردوو بڕی ڕەسەن و گۆڕدراو پیشاندەدات
**حیسابکردنی تەرازوو**: هەموو کۆکان بە دراوی ڕاپۆرت حیسابدەکرێن

### نیشانکەرەکانی دراو
- **هەمان دراو**: نیشانکەری سەوز بۆ دراوە یەکگرتووەکان
- **دراوی جیاواز**: نیشانکەری زەرد پیشاندانی گۆڕینی جێبەجێکراو
- **وردەکاریەکانی گۆڕین**: سەرنج بدە یان کلیک بکە بۆ زانیاری ڕێژە

**نموونە**: ڕێکخستنی پارەدانێکی USD لە دژی ڕاپۆرتی بانکی IQD:
- پارەدانی سیستەم: $500 USD
- ڕێژەی گۆڕان: 1 USD = 1,310 IQD
- بڕی گۆڕدراو: 655,000 IQD
- ڕیزی ڕاپۆرتی بانک: 655,000 IQD (یەکدەگرێتەوە دوای گۆڕین)

## تایبەتمەندی جێهێشتن

### بەڕێوەبردنی بڕگە نایەکگرێتەوەکان

بۆ ناکۆکی بچووکەکان یان کرێی بانکەکان کە پارەدانی پەیوەندیداری سیستەمیان نییە:

### هەنگاوی ۱: دیاریکردنی مامەڵەی نایەکگرێتەوە

**پێداچوونەوەی ڕیزەکانی ڕاپۆرتی بانک**: مامەڵەکان بدۆزەرەوە کە پارەدانی پەیوەندیداریان نییە
**نموونە باوەکان**: کرێی بانک، کرێی سوود، ناکۆکی بچووکەکان

### هەنگاوی ۲: جێبەجێکردنی جێهێشتن

**کلیک لە کردەوەی جێهێشتن بکە**: ئایکۆنی "X" لە خشتەی مامەڵەکانی بانکدا بەکاربهێنە
**هەژماری خەرجی هەڵبژێرە**: هەژماری گونجاو هەڵبژێرە (نموونە "کرێی بانک")
**هۆکار تێبخە**: ڕوونکردنەوەی ڕوون بۆ جێهێشتنەکە دابینبکە
**تۆماری دروست بکە**: سیستەم تۆماری ژۆرنالی دروستدەکات و ڕیزەکە وەک ڕێکخراو نیشاندەدات

### نموونەکانی جێهێشتن
- **کرێی بانک**: کرێی چاودێری مانگانە ($15-25)
- **کرێی سوود**: کرێی زیادەپارە، سوودی قەرز
- **ناکۆکی بچووکەکان**: جیاوازی خولاندنەوە لە ژێر $5

## کاریگەری تۆماری ژۆرنالی

### ڕێکخستنی سەرکەوتوو
```
تۆماری ژۆرنالی دروست نابێت - نیشانکردنی پارەدانە هەبووەکان وەک ڕێکخراو
```

### مامەڵەی جێهێشتن
```
بەدەین. خەرجی کرێی بانک    $25.00
    بەخێر. هەژماری بانک              $25.00
```

جێهێشتنەکە تۆماری ژۆرنالی دروستدەکات بۆ تۆمارکردنی کرێی بانک یان ناکۆکیەکە لە هەمان کاتدا ڕیزی ڕاپۆرتی بانک وەک ڕێکخراو نیشاندەدات.

## سیناریۆ باوەکان

### سیناریۆی ۱: پرۆسەی ڕێکخستنی ستاندارد
یەکخستنی تەواوی تۆمارەکانی بانک و سیستەم:

**هەنگاوەکان**:
1. دەستپێگەیشتن بە ڕێکخستن لە لیستی ڕاپۆرتی بانکەوە
2. هەڵبژاردنی مامەڵەی بانکی یەکگرتوو (گواستنەوەی تەل $1,500)
3. هەڵبژاردنی پارەدانی سیستەمی پەیوەندیدار (پارەدانی فرۆشیار $1,500)
4. دڵنیابوونەوەی تەرازوو "تەرازوو" پیشاندەدات لەگەڵ جیاوازی سفر
5. کلیک لە "ڕێکخستنی هەڵبژێردراو" بکە بۆ تەواوکردنی یەکخستن

**ئەنجام**:
- دۆخی پارەدان نوێدەکرێتەوە بۆ "ڕێکخراو"
- ڕیزی ڕاپۆرتی بانک بە تۆماری پارەدان دەبەسترێتەوە
- ڕووکار هەڵبژاردنەکان پاکدەکاتەوە بۆ کۆمەڵەی داهاتوو

### سیناریۆی ۲: جێهێشتنی کرێی بچووکی بانک
بەڕێوەبردنی کرێی چاودێری بانک بەبێ پارەدانی پەیوەندیدار:

**هەنگاوەکان**:
1. دیاریکردنی ڕیزی ڕاپۆرتی بانکی نایەکگرتوو: -$25.00 "کرێی خزمەتگوزاری مانگانە"
2. کلیک لە کردەوەی جێهێشتن (ئایکۆنی X) لە مامەڵەی بانکدا
3. هەڵبژاردنی هەژماری خەرجی "کرێی خزمەتگوزاریەکانی بانک"
4. هۆکار تێبخە: "کرێی چاودێری هەژماری مانگانە"
5. سیستەم تۆماری خەرج دروستدەکات و ڕیزەکە وەک ڕێکخراو نیشاندەدات

**ئەنجام**:
- تۆماری ژۆرنالی: بەدەین کرێی خزمەتگوزاریەکانی بانک $25، بەخێر هەژماری بانک $25
- ڕیزی ڕاپۆرتی بانک وەک ڕێکخراو نیشانکراو
- پێویستی بە کردەوەی زیاتر نییە

### سیناریۆی ۳: ڕێکخستنی پارەدانی زۆر دراوی
ڕێکخستنی پارەدانێکی USD لە دژی ڕاپۆرتی بانکی IQD:

**نموونە**:
- ڕاپۆرتی بانک: 655,000 IQD گواستنەوەی دەرچوو
- پارەدانی سیستەم: $500 USD پارەدانی فرۆشیار
- ڕێژەی گۆڕان: 1 USD = 1,310 IQD (گیراو لە کاتی پارەداندا)
- بڕی گۆڕدراو: 655,000 IQD

**هەنگاوەکان**:
1. هەڵبژاردنی ڕیزی ڕاپۆرتی بانکی IQD (655,000 IQD)
2. هەڵبژاردنی پارەدانی سیستەمی USD ($500 USD)
3. سیستەم گۆڕین پیشاندەدات: $500 × 1,310 = 655,000 IQD
4. دڵنیابوونەوەی کۆکان تەواو تەرازوون
5. تەواوکردنی ڕێکخستن بە شێوەی ئاسایی

**ئەنجام**:
- ڕێکخستنی زۆر دراوی تەواوکراو
- بەڵگەکردنی ڕێژەی گۆڕان پارێزراو
- هەردوو تۆمار وەک ڕێکخراو نیشانکراون

## باشترین شێوازەکان

### ئامادەکردن
- **چالاککردنی ڕێکخستن**: ڕێکخستنی کۆمپانیا چالاکبکە پێش دەستپێکردن
- **ڕاپۆرتەکانی تەواو**: دڵنیابە هەموو مامەڵەکانی بانک لە [ڕاپۆرتەکانی بانک](bank-statements.ckb.md)دا تێخراون
- **دڵنیابوونەوەی پارەدان**: پشتڕاستبکە پارەدانەکانی سیستەم لە دۆخی "پشتڕاستکراو"دان
- **دروستی دراو**: مامەڵەکانی زۆر دراوی بۆ ڕێژە دروستەکان پێداچوونەوە بکە

### پرۆسەی ڕێکخستن
- **ڕێگەی سیستەماتیک**: بە شێوەی ڕێکوپێک لە مامەڵەکاندا کاربکە بە بەروار یان بڕ
- **دڵنیابوونەوەی کاتی ڕاستەقینە**: نیشانکەری تەرازوو بە بەردەوامی لە کاتی هەڵبژاردندا چاودێری بکە
- **جێهێشتنی دەستبەجێ**: ناکۆکی بچووکەکان بە خێرایی بە بەکارهێنانی تایبەتمەندی جێهێشتن بەڕێوەببە
- **بەڵگەکردنی ڕوون**: هۆکاری ورد بۆ هەموو تۆمارەکانی جێهێشتن دابینبکە

### کۆنترۆڵی کوالیتی
- **ڕێکخستنی ئاسایی**: لە ماوەی چەند ڕۆژدا دوای وەرگرتنی ڕاپۆرتەکانی بانک ئەنجام بدە
- **پێداچوونەوەی بڕگە نایەکگرێتەوەکان**: ناکۆکیەکان بپشکنە کە لە سنووری پێشدیاریکراو زیاترن
- **بەدواداچوونی فۆلۆئاپ**: بڕگە هەڵواسراوەکان لە چەندین ماوەدا چاودێری بکە
- **ڕاهێنانی بەکارهێنەر**: دڵنیابە تیمەکە تایبەتمەندیەکانی ڕووکاری کارلێک تێدەگات

### پشکنین و پابەندبوون
- **ڕێڕەوی پشکنین**: سیستەم مێژووی تەواوی ڕێکخستن دەپارێزێت
- **بەڵگە پشتگیریکەرەکان**: ڕاپۆرتەکانی بانک و پشتڕاستکردنەوەکانی پارەدان بەستبدەرەوە
- **پرۆسەی پێداچوونەوە**: پێداچوونەوەی سەرپەرشتیار بۆ جێهێشتنە گەورەکان جێبەجێ بکە
- **ڕێکارەکانی کۆتایی مانگ**: هەموو ڕێکخستنەکان پێش داخستنی ماوەکان تەواو بکە

- "چالاککردنی ڕێکخستن" لە ڕێکخستنەکانی کۆمپانیادا چالاککراوە دڵنیابکەرەوە
- بەکارهێنەر مۆڵەتی گونجاوی هەیە بۆ ڕێکخستن پشتڕاستبکە
- ڕاپۆرتی بانک هەیە و دەستگەیشتنی بۆ هەیە دڵنیابکەرەوە

**پرسیار: کۆکان لە ڕێکخستنی زۆر دراویدا تەواو تەرازوو نین**
وەڵام: بەڕێوەبردنی گۆڕینی دراو پێداچوونەوە بکە:
- ڕێژەکانی گۆڕان بپشکنە کە بۆ پارەدان و ڕێکخستن بەکارهاتوون
- بڕەکان بە دروستی لە بەڵگە سەرچاوەکاندا تێخراون دڵنیابکەرەوە
- جیاوازی بچووکی خولاندنەوە بیرلێبکەرەوە (بە شێوەی ئاسایی لە ژێر $1)
- جێهێشتن بۆ جیاوازی ناگرنگی گۆڕین بەکاربهێنە

**پرسیار: هیچ پارەدانێک لە پانێڵی پارەدانەکانی سیستەمدا دەرناکەوێت**
وەڵام: دۆخی پارەدان و فلتەرەکان دڵنیابکەرەوە:
- پارەدانەکان لە دۆخی "پشتڕاستکراو"دان دڵنیابکەرەوە (نەک ڕەشنووس)
- پارەدانەکان پێشتر ڕێکنەخراون بپشکنە
- پارەدانەکان سەر بە کۆمپانیا و هەژماری بانکی دروست دڵنیابکەرەوە
- وێبگەڕ نوێبکەرەوە ئەگەر ڕووکار کۆن دیاربێت

**پرسیار: ناتوانرێت پرۆسەی ڕێکخستن تەواو بکرێت**
وەڵام: هەڵبژاردن و تەرازووکردن دڵنیابکەرەوە:
- کۆکان تەرازووکراون دڵنیابکەرەوە (جیاوازی = 0)
- لانیکەم یەک بڕگە لە هەر پانێڵێکدا هەڵبژێردراوە دڵنیابکەرەوە
- هەڵەکانی پشتڕاستکردنەوە یان پەیامەکانی سیستەم بپشکنە
- هەڵبژاردنەکان پاکبکەرەوە و لە نوێوە دەست پێبکەرەوە ئەگەر پێویست بوو

### پەیامەکانی هەڵە

**"ڕێکخستن بۆ ئەم کۆمپانیایە چالاک نەکراوە"**
- پەیوەندی بە بەڕێوەبەرەوە بکە بۆ چالاککردنی ڕێکخستن لە ڕێکخستنەکانی کۆمپانیادا
- ڕێکخستنی کۆمپانیا مۆڵەتەکانی ڕێکخستن لەخۆدەگرێت دڵنیابکەرەوە

**"کۆی پارەدانە هەڵبژێردراوەکان لەگەڵ کۆی بانک ناگونجێت"**
- بڕە هەڵبژێردراوەکان بە وردی پێداچوونەوە بکە
- کێشەکانی گۆڕینی دراو بپشکنە
- بیرلێکردنەوەی بەکارهێنانی جێهێشتن بۆ جیاوازی بچووکەکان

---

## پرسیارە دووبارەبووەکان

**پرسیار: چی لە پارەدانەکان دەبێت دوای ئەوەی ڕێکدەخرێن؟**
وەڵام: پارەدانە ڕێکخراوەکان بە دۆخی "ڕێکخراو" نیشان دەکرێن و بە ڕیزەکانی تایبەتی ڕاپۆرتی بانک دەبەسترێنەوە. لە ڕووکارەکانی ڕێکخستنی داهاتوودا دەرناکەون بەڵام لە تۆمارەکانی پارەداندا دیارن لەگەڵ نیشانکەرەکانی ڕێکخستن.

**پرسیار: دەتوانم ڕێکخستنێک بگەڕێنمەوە ئەگەر هەڵەم کردووە؟**
وەڵام: سیستەم ڕێڕەوی پشکنین دەپارێزێت بەڵام ڕێکخستنەکان دەبێت وەک هەمیشەیی بیربکرێنەوە. پەیوەندی بە بەڕێوەبەری سیستەمەوە بکە ئەگەر پێویستت بە گەڕاندنەوەی تۆمارەکانی ڕێکخستن هەیە بۆ ڕاستکردنەوە.

**پرسیار: چۆن مامەڵەکانی بانک بەڕێوەبەم کە چەندین ماوە دەپشکنن؟**
وەڵام: مامەڵەکانی بانک دەبێت لە ماوەیەکدا ڕێکبخرێن کە لە ڕاپۆرتی بانکدا دەردەکەون. ئەگەر پارەدانەکانی سیستەم لە ماوەیەکی جیاوازدا تۆمارکراون، هێشتا دەتوانرێت لە کاتی ڕێکخستندا یەکبخرێنەوە.

**پرسیار: چی لەگەڵ بڕگە گەورە نایەکگرێتەوەکان بکەم؟**
وەڵام: بە وردی لێکۆڵینەوە بکە پێش جێهێشتن. بگەڕێ بۆ پارەدانەکانی سیستەمی ونبوو، بڕە هەڵەکان، یان جیاوازی کاتی. ناکۆکی گەورەکان لەوانەیە نیشانی هەڵە بن کە پێویستیان بە ڕاستکردنەوەیە نەک جێهێشتن.

## فەرهەنگ - Kurdish Technical Terms

### زاراوە تەکنیکیەکان
- **ڕێکخستنی بانک** (Bank Reconciliation) - پرۆسەی یەکخستنی تۆمارەکانی پارەی کۆمپانیا لەگەڵ ڕاپۆرتی بانک
- **ڕاپۆرتی بانک** (Bank Statement) - بەڵگەی فەرمی بانک کە مامەڵەکانی هەژمار پیشاندەدات
- **پارەدان** (Payment) - گواستنەوەی پارە لە کۆمپانیا بۆ لایەنێکی دیکە
- **جێهێشتن** (Write-off) - تۆمارکردنی بڕێک وەک لەدەستچوو یان نەگەڕاوە
- **یەکخستن** (Matching) - یەکخستنی مامەڵەیەکی بانک لەگەڵ پارەدانێکی سیستەم
- **تەرازووکردن** (Balancing) - دڵنیابوون لەوەی کۆکان یەکدەگرنەوە

### زاراوە ژمێریاریەکان
- **ژمێریاری دووقات ئێنتری** (Double-Entry Accounting) - سیستەمێکی ژمێریاری کە هەر مامەڵەیەک لە لانیکەم دوو هەژماردا تۆماردەکات
- **بەدەین** (Debit/Dr.) - لایەنی چەپی تۆماری ژمێریاری
- **بەخێر** (Credit/Cr.) - لایەنی ڕاستی تۆماری ژمێریاری
- **تۆماری ژۆرنالی** (Journal Entry) - تۆمارکردنی مامەڵەیەک لە کتێبە ژمێریاریەکاندا
- **هەژماری خەرج** (Expense Account) - هەژمارێک کە خەرجەکانی کۆمپانیا تۆماردەکات

### زاراوە تەکنیکیەکانی سیستەم
- **ڕووکاری کارلێک** (Interactive Interface) - ڕووکارێک کە ڕەوتی دووڕەفەی زانیاری ڕێگەپێدەدات
- **پشتڕاستکردنەوە** (Validation) - پشکنینی دروستی داتا
- **گۆڕینی دراو** (Currency Conversion) - گۆڕینی بڕێک لە دراوێکەوە بۆ دراوێکی دیکە
- **ڕێژەی گۆڕان** (Exchange Rate) - ڕێژەی گۆڕینی دراو
- **کاتی ڕاستەقینە** (Real-time) - زانیاری کە دەستبەجێ نوێدەکرێتەوە

### دۆخەکانی سیستەم
- **پشتڕاستکراو** (Confirmed) - دۆخی پارەدانێک کە پشتڕاستکراوەتەوە
- **ڕێکخراو** (Reconciled) - دۆخی پارەدانێک یان مامەڵەیەک کە ڕێکخراوە
- **ڕەشنووس** (Draft) - دۆخی سەرەتایی بەڵگەیەک پێش پشتڕاستکردنەوە
- **هەڵواسراو** (Outstanding) - مامەڵەیەک کە هێشتا ڕێکنەخراوە

---

ڕێکخستنی بانک دروستی تۆمارەکانی پارەت دڵنیادەکاتەوە و متمانە بە ڕاپۆرتکردنی داراییت دەبەخشێت. ڕووکاری کارلێک پرۆسەی یەکخستن ڕەوانتر دەکات لە هەمان کاتدا ڕێڕەوی تەواوی پشکنین و کۆنترۆڵە دروستەکانی ژمێریاری پارێزراو دەکات.

## باشترین شێوازەکان

### ئامادەکردن
- **چالاککردنی ڕێکخستن**: ڕێکخستنی کۆمپانیا چالاکبکە پێش دەستپێکردن
- **ڕاپۆرتەکانی تەواو**: دڵنیابە هەموو مامەڵەکانی بانک لە [ڕاپۆرتەکانی بانک](bank-statements.ckb.md)دا تێخراون
- **دڵنیابوونەوەی پارەدان**: پشتڕاستبکە پارەدانەکانی سیستەم لە دۆخی "پشتڕاستکراو"دان
- **دروستی دراو**: مامەڵەکانی زۆر دراوی بۆ ڕێژە دروستەکان پێداچوونەوە بکە

### پرۆسەی ڕێکخستن
- **ڕێگەی سیستەماتیک**: بە شێوەی ڕێکوپێک لە مامەڵەکاندا کاربکە بە بەروار یان بڕ
- **دڵنیابوونەوەی کاتی ڕاستەقینە**: نیشانکەری تەرازوو بە بەردەوامی لە کاتی هەڵبژاردندا چاودێری بکە
- **جێهێشتنی دەستبەجێ**: ناکۆکی بچووکەکان بە خێرایی بە بەکارهێنانی تایبەتمەندی جێهێشتن بەڕێوەببە
- **بەڵگەکردنی ڕوون**: هۆکاری ورد بۆ هەموو تۆمارەکانی جێهێشتن دابینبکە

### کۆنترۆڵی کوالیتی
- **ڕێکخستنی ئاسایی**: لە ماوەی چەند ڕۆژدا دوای وەرگرتنی ڕاپۆرتەکانی بانک ئەنجام بدە
- **پێداچوونەوەی بڕگە نایەکگرێتەوەکان**: ناکۆکیەکان بپشکنە کە لە سنووری پێشدیاریکراو زیاترن
- **بەدواداچوونی فۆلۆئاپ**: بڕگە هەڵواسراوەکان لە چەندین ماوەدا چاودێری بکە
- **ڕاهێنانی بەکارهێنەر**: دڵنیابە تیمەکە تایبەتمەندیەکانی ڕووکاری کارلێک تێدەگات

### پشکنین و پابەندبوون
- **ڕێڕەوی پشکنین**: سیستەم مێژووی تەواوی ڕێکخستن دەپارێزێت
- **بەڵگە پشتگیریکەرەکان**: ڕاپۆرتەکانی بانک و پشتڕاستکردنەوەکانی پارەدان بەستبدەرەوە
- **پرۆسەی پێداچوونەوە**: پێداچوونەوەی سەرپەرشتیار بۆ جێهێشتنە گەورەکان جێبەجێ بکە
- **ڕێکارەکانی کۆتایی مانگ**: هەموو ڕێکخستنەکان پێش داخستنی ماوەکان تەواو بکە

---

## سیناریۆ باوەکان

### سیناریۆی ۱: پرۆسەی ڕێکخستنی ستاندارد
ڕێکخستنی ڕاپۆرتێکی مانگانە لەگەڵ یەکگرتنەوەی تەواو:

1. دەستپێگەیشتن بە ڕێکخستن لە لیستی ڕاپۆرتی بانکەوە
2. مامەڵەی بانکی یەکگرتوو و پارەدانی سیستەم هەڵبژێرە
3. دڵنیابەرەوە بڕەکە "یەکگرتووە" پیشاندەدات 
4. کلیک لە "ڕێکخستنی هەڵبژێردراو" بکە
5. سیستەم دۆخی پارەدان نوێدەکاتەوە و هەڵبژاردنەکان پاکدەکاتەوە
6. بۆ مامەڵەکانی ماوە دووبارە بکەرەوە

### سیناریۆی ۲: جێهێشتنی کرێی بچووکی بانک
بەڕێوەبردنی کرێیەکی چاودێری بانک:

1. ڕیزی ڕاپۆرتی بانکی نایەکگرتوو بۆ کرێی ١٥ دینار دیاری بکە
2. کلیک لە کردەوەی جێهێشتن (ئایکۆنی X) لە مامەڵەی بانکدا بکە
3. هەژماری خەرجی "کرێی بانک" هەڵبژێرە
4. هۆکار تێبخە: "کرێی چاودێری مانگانە"
5. سیستەم تۆماری ژۆرنالی دروستدەکات و ڕیزەکە وەک ڕێکخراو نیشاندەدات

### سیناریۆی ۳: پارەدانی زۆر دراوی
ڕێکخستنی پارەدانێکی USD لە دژی ڕاپۆرتی بانکی IQD:

1. پارەدانی USD لە پانێڵی پارەدانەکانی سیستەمەوە هەڵبژێرە
2. سیستەم بە شێوەی خۆکار بۆ IQD لە ڕێژەی ئێستادا دەگۆڕێت
3. ڕیزی ڕاپۆرتی بانکی IQDی پەیوەندیدار هەڵبژێرە
4. دڵنیابەرەوە کۆکان یەکدەگرنەوە (ڕێگەپێدان بە جیاوازی بچووکی گۆڕین)
5. ڕێکخستن بە شێوەی ئاسایی تەواو بکە

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

**کێشە**: کردەوەی ڕێکخستن دیار نییە
**چارەسەر**:
- بپشکنە "چالاککردنی ڕێکخستن" لە ڕێکخستنەکانی کۆمپانیادا چالاککراوە
- مۆڵەتەکانی گونجاوی بەکارهێنەر دڵنیابکەرەوە
- دڵنیابە ڕاپۆرتی بانک هەیە و دەستپێگەیشتنی هەیە

**کێشە**: کۆکان بە وردی یەکناگرنەوە
**چارەسەر**:
- جیاوازیەکانی گۆڕینی دراو بپشکنە
- بڕەکانی مامەڵە بە دروستی تێخراون دڵنیابکەرەوە
- بگەڕێ بۆ پارەدانی بەشی یان کرێکان
- جێهێشتن بۆ جیاوازی ناگرنگ بەکاربهێنە

**کێشە**: هیچ پارەدانێک لە پانێڵی پارەدانەکانی سیستەمدا دەرناکەوێت
**چارەسەر**:
- دڵنیابەرەوە پارەدانەکان لە دۆخی "پشتڕاستکراو"دان
- بپشکنە پارەدانەکان پێشتر ڕێکنەخراون
- دڵنیابە پارەدانەکان سەر بە کۆمپانیا و ژۆرنالی دروستن

**کێشە**: ناتوانرێت ڕێکخستن تەواو بکرێت
**چارەسەر**:
- دڵنیابە کۆکان یەکدەگرنەوە (جیاوازی = ٠)
- دڵنیابەرەوە لانیکەم یەک بڕگە لە هەر پانێڵێکدا هەڵبژێردراوە
- بگەڕێ بۆ هەڵەکانی سیستەم یان کێشەکانی دڵنیابوونەوە
- تاقیکردنەوەی ڕووکار ئەگەر پێویست بێت

## یەکخستن لەگەڵ ڕاپۆرتەکانی بانک

ڕێکخستنی بانک بە توندی لەگەڵ سیستەمی [بەڕێوەبردنی ڕاپۆرتی بانک](bank-statements.ckb.md) یەکخراوە:

### گەرانی دراوە
1. **دروستکردنی ڕاپۆرت**: [ڕاپۆرتەکانی بانک](bank-statements.ckb.md) دراوەی مامەڵەی دەرەکی دابیندەکەن
2. **پرۆسەی ڕێکخستن**: ئەم ڕووکارە دراوەی دەرەکی لەگەڵ پارەدانەکانی ناوخۆیی یەکدەخات  
3. **نوێکردنەوەی دۆخ**: بڕگە ڕێکخراوەکان نیشانکراو و بەستراوەتەوە لە هەردوو سیستەمدا
4. **ڕاپۆرتکردن**: دراوەی یەکخراو پشتگیری ڕاپۆرتکردنی گشتگیری بەڕێوەبردنی پارە دەکات

### زنجیرەی کارگێڕی
1. **دروستکردنی ڕاپۆرتی بانک**: لەگەڵ [دروستکردنی ڕاپۆرتی بانک](bank-statements.ckb.md) دەست پێبکە
2. **تێخستنی مامەڵەکان**: هەموو مامەڵەکانی بانک بۆ ڕاپۆرتەکە زیاد بکە
3. **دڵنیابوونەوەی ڕاپۆرت**: دڵنیابە بڕی دەستپێک/کۆتایی دروستن
4. **دەستپێکردنی ڕێکخستن**: ئەم پرۆسەی ڕێکخستنە بەکاربهێنە بۆ یەکخستنی مامەڵەکان
5. **تەواوکردنی سوور**: ئامادەبوون بۆ ڕاپۆرتی بانک و ڕێکخستنی ماوەی داهاتوو

---

ڕێکخستنی بانک دروستی تۆمارەکانی پارەت دڵنیادەکاتەوە و متمانە بە ڕاپۆرتکردنی داراییت دەبەخشێت. ڕووکاری کارلێک پرۆسەی یەکخستن کاراتر دەکات لە هەمان کاتدا ڕێگای تەواوی پشکنین و کۆنترۆڵە دروستەکانی ژمێریاری پارێزراو دەکات.
