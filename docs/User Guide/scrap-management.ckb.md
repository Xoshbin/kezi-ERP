# بەڕێوەبردنی کاڵای تێکچوو و فڕێدان

ئەم ڕێبەرە ڕوونی دەکاتەوە چۆن کاڵای تێکچوو، خراپبوو، و فڕێدراو بەڕێوەببەیت لە JMeryar ERP. بەڕێوەبردنی دروستی کاڵای تێکچوو (Scrap) دڵنیایی دەدات لە وردبینی بڕی کۆگا و ڕاپۆرتکردنی دروستی زیانە داراییەکان.

## تێڕوانینی گشتی

**بەڕێوەبردنی سکراپ** بریتییە لە پرۆسەی لابردنی کاڵاکان لە کۆگای چالاک کاتێک چیتر شیاوی فرۆشتن یان بەکارهێنان نین بەهۆی:
- **زیانلێکەوتن** (شکاوی، کەموکورتی)
- **بەسەرچوون** (خۆراک و دەرمانی بەسەرچوو)
- **بەسەرچوونی مۆدێل** (تەکنەلۆژیا یان مۆدێلی کۆن)
- **دزی/ونبوون** (کەمبوونی کۆگا)

لە JMeryar ERP، سکراپکردن لە ڕێگەی **جوڵەکانی کۆگا** یان **ڕاستکردنەوەی کۆگا** ئەنجام دەدرێت، بە گواستنەوەی کاڵا لە شوێنێکی فیزیاییەوە بۆ شوێنێکی مەجازی "زیان".

## چەمکە سەرەکییەکان

### شوێنی مەجازی سکراپ/ڕاستکردنەوە
بە پێچەوانەی کۆگای فیزیایی، شوێنی سکراپ **شوێنێکی مەجازییە**. کاتێک کاڵا دەگوازیتەوە ئێرە:
1. لە کۆگای بەردەستت لادەبرێت.
2. سیستەمەکە دەزانێت کە لە بەهای سەرمایەی کۆمپانیا دەرچووە.
3. قەیدێکی ژمێریاری دروست دەکرێت بۆ لابردنی تێچووی سەرمایەکە.

### کاریگەری ژمێریاری
کاتێک کردارێکی سکراپ پشتڕاست دەکەیتەوە، سیستەمەکە بە ئۆتۆماتیکی:
- **Credit** دەدات بە هەژماری سەرمایەی کۆگا (کەمکردنەوەی بەهای سەرمایە).
- **Debit** دەدات بە هەژماری ڕاستکردنەوەی کۆگا/زیان (تۆمارکردنی خەرجی).

---

## چۆن کاڵا سکراپ دەکەیت

باشترین ڕێگا بۆ سکراپکردنی کاڵا بریتییە لە دروستکردنی **جوڵەی کۆگا** بە جۆری **Adjustment**.

### هەنگاوی ١: دروستکردنی جوڵەی کۆگا
1. بڕۆ بۆ **کۆگا ← کردارەکان ← جوڵەکانی کۆگا (Stock Moves)**.
2. کلیک لەسەر **Create** بکە.

### هەنگاوی ٢: ڕێکخستنی جوڵەکە
زانیارییەکان پڕ بکەرەوە:

- **Reference**: ناوی سەرچاوە بنووسە (نموونە: `SCRAP/2024/001`).
- **Move Type**: جۆری **Adjustment** هەڵبژێرە.
- **Description**: هۆکاری سکراپکردنەکە بنووسە (نموونە: "پالێتی شکاو"، "شیری بەسەرچوو").

### هەنگاوی ٣: زیادکردنی بەرهەمەکان
لە بەشی **Product Lines**:
1. **Product**: ئەو بەرهەمە هەڵبژێرە کە سکراپ دەکرێت.
2. **Quantity**: ئەو بڕە بنووسە کە کەم دەکرێتەوە.
3. **From Location**: شوێنی فیزیایی هەڵبژێرە (نموونە: `WH/Stock`).
4. **To Location**: شوێنی مەجازی **Inventory Adjustment** (یان Scrap) هەڵبژێرە.
   > **تێبینی**: ئەمە دڵنیایی دەدات کە کاڵاکە بە دروستی لە بەها دەخرێت.

### هەنگاوی ٤: پشتڕاستکردنەوە و بڕیاردان
1. کلیک لەسەر **Save** بکە بۆ دروستکردنی ڕەشنووس.
2. وردەکارییەکان ببینەوە.
3. کلیک لەسەر **Confirm Movement** بکە.

دوای پشتڕاستکردنەوە:
- بڕی کۆگا دەستبەجێ کەم دەکرێتەوە.
- قەیدی دارایی تۆمار دەکرێت.

---

## سکراپکردن بەرامبەر گەڕانەوە (Returns)
- **سکراپ بەکاربهێنە**: کاتێک کاڵاکە هیچ بەهایەکی نەماوە و تێکدەشکێنرێت یان فڕێ دەدرێت.
- **گەڕانەوەی دابینکەر بەکاربهێنە**: کاتێک کاڵاکە زیانی بەرکەوتووە بەڵام دەتوانرێت بگەڕێنرێتەوە بۆ دابینکەر بۆ قەرەبوو (دەفتەری قەرز - Debit Note بەکاربهێنە).

## ڕاپۆرت و شیکاری
بۆ بینینی مێژووی سکراپ:
1. بڕۆ بۆ **کۆگا ← کردارەکان ← جوڵەکانی کۆگا**.
2. فیلتەر بکە بەپێی **Move Type: Adjustment**.
3. هەروەها دەتوانیت فیلتەر بکەیت بەپێی "To Location" = Adjustment/Scrap بۆ بینینی هەموو فڕێدانەکان.

### شیکاری تێچوو
تێچووی کاڵای سکراپکراو هەژمار دەکرێت بەپێی ڕێگای بەهاکدانانی بەرهەمەکە (FIFO/Average Cost) لە کاتی جوڵەکەدا. ئەم تێچووە دەچێتە ناو ڕاپۆرتی **قازانج و زیان (P&L)** لەژێر **خەرجییەکانی زیانی کۆگا**.
