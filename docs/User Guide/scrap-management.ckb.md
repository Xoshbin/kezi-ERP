# بەڕێوەبردنی فڕێدان و کەرەستە تێکچووەکان (Scrap Management)

ئەم ڕێبەرە ڕوونی دەکاتەوە کە چۆن بەڕێوەبردنی کەرەستە فڕێدراو، تێکچوو، یان بەسەرچووەکان لە سیستەمی Jmeryar ERP ئەنجام دەدرێت. بەڕێوەبردنی دروستی ئەم پرۆسەیە دڵنیایی دەدات لە ڕاستی ژمارەی کۆگا و ڕاپۆرتی دارایی ورد.

## تێڕوانینی گشتی

**بەڕێوەبردنی فڕێدان (Scrap)** بریتییە لە پرۆسەی دەرکردنی کاڵا لە کۆگای چالاک کاتێک چیتر شایەنی فرۆشتن یان بەکارهێنان نین بەهۆی:
- **زیان (Damage)**: شکاو یان تێکچوو.
- **بەسەرچوون (Expiry)**: کاڵایەک کە بەروارەکەی بەسەرچووە.
- **کۆن بوون (Obsolescence)**: تەکنەلۆژیا یان مۆدێلی کۆن.
- **ونبوون/دزی**: کەمبوونی کۆگا.

لە Jmeryar ERP، ئەم پرۆسەیە لە ڕێگەی **جوڵەی کۆگا (Stock Moves)** یان **ڕاستکردنەوەی کۆگا** ئەنجام دەدرێت، بە گواستنەوەی کاڵاکان لە شوێنی فیزیایی بۆ شوێنێکی گریمانەیی "زیان".

## چەمکە سەرەکییەکان

### شوێنی گریمانەیی بۆ فڕێدان
بە پێچەوانەی کۆگای ئاسایی، شوێنی فڕێدان **شوێنێکی گریمانەیی (Virtual Location)**. کاتێک کاڵا دەگوێزیتەوە بۆ ئێرە:
1. لە کۆگای بەردەستت کەمدەبێتەوە.
2. سیستەم دەزانێت کە ئەمە چیتر بەشێک نییە لە سەرمایەی کۆمپانیا.
3. تۆمارێکی ژمێریاری دروست دەکرێت بۆ سڕینەوەی تێچووی کاڵاکە.

### کاریگەری ژمێریاری
کاتێک پرۆسەکە پشتڕاست دەکەیتەوە، سیستەمەکە بە شێوەیەکی ئۆتۆماتیکی:
- **قەرزدەر (Credit)** دەکاتە هەژماری سامانی کۆگا (Inventory Asset) - کەمکردنەوەی بەها.
- **قەرزدار (Debit)** دەکاتە هەژماری خەرجی ڕاستکردنەوە/زیان (Loss Account) - تۆمارکردنی زیانەکە.

---

## چۆنیەتی فڕێدانی کاڵا

باشترین ڕێگە بۆ فڕێدانی کاڵا دروستکردنی **جوڵەی کۆگا (Stock Move)**ـیە بە جۆری **ڕاستکردنەوە (Adjustment)**.

### هەنگاوی ١: دروستکردنی جوڵەی کۆگا
1. بچۆ بۆ **Inventory > Operations > Stock Moves**.
2. کلیک لە **Create** بکە.

### هەنگاوی ٢: دیاریکردنی زانیارییەکان
زانیارییەکان پڕ بکەرەوە:

- **Reference**: ژمارەیەک بنووسە (نموونە: `SCRAP/2024/001`).
- **Move Type**: هەڵبژێرە **Adjustment**.
- **Description**: هۆکاری فڕێدانەکە بنووسە (نموونە: "شکاوە"، "بەسەرچووە").

### هەنگاوی ٣: زیادکردنی بەرهەمەکان
لە بەشی **Product Lines**:
1. **Product**: ئەو بەرهەمە دیاری بکە کە دەتەوێت فڕێی بدەیت.
2. **Quantity**: بڕەکەی دیاری بکە.
3. **From Location**: شوێنی ئێستای کۆگا دیاری بکە (نموونە: `WH/Stock`).
4. **To Location**: شوێنی **Inventory Adjustment** (یان Scrap) دیاری بکە.
   > **تێبینی**: دڵنیابە کە شوێنی "ڕاستکردنەوە" دیاری دەکەیت بۆ ئەوەی حسابی زیان بکرێت.

### هەنگاوی ٤: پشتڕاستکردنەوە
1. کلیک لە **Save** بکە بۆ پاشەکەوتکردن.
2. وردەکارییەکان بپشکنە.
3. کلیک لە **Confirm Movement** بکە.

دوای پشتڕاستکردنەوە:
- بڕی کۆگا کەمدەبێتەوە.
- تۆماری ژمێریاری دروست دەکرێت.

---

## ڕاپۆرت و شیکاری
بۆ بینینی مێژووی فڕێدانەکان:
1. بچۆ بۆ **Inventory > Operations > Stock Moves**.
2. فیلتەر بکە بەپێی **Move Type: Adjustment**.
3. دەتوانیت فیلتەر بکەیت بەپێی "To Location" بۆ بینینی هەموو فڕێدانەکان.

### شیکاری تێچوو
تێچووی کاڵا فڕێدراوەکان بەپێی شێوازی هەڵسەنگاندنی بەرهەمەکە (FIFO/Average) ئەژمار دەکرێت و دەچێتە ناو ڕاپۆرتی **قازانج و زیان (P&L)** لەژێر بەشی **Inventory Loss/Adjustment Expenses**.
