# پارەدانەکان: جووڵەی پارە و بەڕێوەبردنی جووڵەی پارە

ئەم ڕێنمایی تەواوە ڕوونکردنەوەی چۆنیەتی کارکردنی پارەدانەکان لە سیستەمی ژمێریاری دەکات، دروستکردن، بەڕێوەبردن، پشتگیری فرە دراو، و هاوسەنگی بانکی دەگرێتەوە. بۆ هەموو بەکارهێنەران نووسراوە — ژمێریار و غەیری ژمێریار — ڕێنمایی کردەیی دابین دەکات بە پەیڕەوکردنی باشترین پراکتیسەکانی ژمێریاری دووقات.

---

## پارەدان چییە؟

پارەدان نوێنەرایەتی جووڵەی پارە لە نێوان کۆمپانیاکەت و هاوبەشێکی بازرگانی (کڕیار یان فرۆشیار) دەکات. پارەدانەکان بنەڕەتین بۆ بەڕێوەبردنی جووڵەی پارە و تەسویەی هەژمارەکانی وەرگرتنی/پارەدانی.

### جۆرەکانی پارەدان
- **پارەدانی ناوەوە** — پارەی وەرگیراو لە کڕیارەکان (کەمکردنەوەی هەژمارەکانی وەرگرتنی)
- **پارەدانی دەرەوە** — پارەی نێردراو بۆ فرۆشیارەکان (کەمکردنەوەی هەژمارەکانی پارەدانی)

### پۆلەکانی پارەدان
- **پارەدانی تەسویە** — بەستراوە بە پسووڵەکانی فرۆشتن یان پسووڵەکانی کڕینی تایبەت بۆ تەسویەی بڕە ماوەکان
- **پێش پارەدان/قەرزی هاوبەش** — پارەدانی سەربەخۆ بۆ پێش پارەدانی کڕیار یان پێش پارەدانی فرۆشیار

هەر پارەدانێک ئەمانە لەخۆدەگرێت:
- پەرتووکی ڕۆژانە (هەژماری بانک یان کاش کە پارەکە جووڵاوە)
- بەرواری پارەدان و دراو
- بڕ لە هەردوو دراوی پارەدان و دراوی بنەڕەتی کۆمپانیا
- ڕێگای پارەدان و سەرچاوەی ئیختیاری
- زانیاریەکانی هاوبەش و شوێنکەوتنی دۆخ

---

## پێداویستیەکانی سیستەم

### هاوسەنگی پەرتووکی ڕۆژانە
- **پەرتووکەکانی ڕۆژانەی بانک**: لانیکەم یەک پەرتووکی ڕۆژانەی بانک ڕێکخراو بۆ پرۆسەی پارەدان
- **ڕێگاکانی پارەدان**: ڕێگاکانی پارەدان دەبێت ڕێک بخرێن بۆ بژاردەکانی پرۆسە
- **هاوسەنگی دراو**: دراوەکان دەبێت چالاک بن و نرخەکانی ئاڵوگۆڕ ڕێکبخرێن
- **تۆمارەکانی هاوبەش**: کڕیار و فرۆشیارەکان دەبێت هەبن بۆ بەستەری پارەدان

### پێشمەرجەکان
1. **هاوسەنگی هەژماری بانک**: پەرتووکەکانی ڕۆژانەی بانک بە شێوەیەکی گونجاو ڕێکخراو لەگەڵ یەکگرتنی ژمێریاری
2. **زانیاریەکانی هاوبەش**: تۆمارەکانی کڕیار یان فرۆشیار دروستکراو لەگەڵ وردەکاریەکانی پارەدان
3. **تۆمارەکانی پسووڵەکانی فرۆشتن/پسووڵەکانی کڕین**: بۆ پارەدانە بەستەرکراوەکان، پسووڵەکانی فرۆشتن یان پسووڵەکانی کڕین دەبێت نێردراو بن
4. **ڕێگەپێدانەکانی بەکارهێنەر**: دەسەڵاتی گونجاو بۆ تایبەتمەندیەکانی بانک و پارەدان

---

## لە کوێ لە ڕووکاری بەکارهێنەردا بیدۆزیتەوە

ڕێگەی گەیشتن: **ژمێریاری → بانک و کاش → پارەدانەکان**

پارەدانەکان هەروەها دەردەکەون لە:
- **پسووڵەکانی فرۆشتنی کڕیار**: کردەی "تۆمارکردنی پارەدان" بۆ پسووڵەکانی فرۆشتن نێردراوە نەپارەدراوەکان
- **پسووڵەکانی کڕینی فرۆشیار**: کردەی "تۆمارکردنی پارەدان" بۆ پسووڵەکانی کڕین نێردراوە نەپارەدراوەکانی فرۆشیار  
- **هاوسەنگی بانک**: پارەدانە پشتڕاستکراوەکان بەردەستن بۆ هاوتاکردن لەگەڵ ڕاپۆرتەکانی بانک
- **ڕۆژمێر**: چالاکی پارەدانی نوێ و کورتەی جووڵەی پارە

**ئامۆژگاری**: دوگمەی یارمەتی/بەڵگەنامەکان لە سەرپەڕە ئەم ڕێنماییە دەکاتەوە.

---

## ڕێڕەوی دۆخی پارەدان

پارەدانەکان ڕێڕەوێکی کۆنترۆڵکراو پەیڕەو دەکەن بۆ دڵنیابوون لە یەکگرتووی داتا:

### ڕەشنووس
- پارەدانی تازە دروستکراو، بە تەواوی دەستکاریکراو
- دەتوانرێت گۆڕانکاری بکرێت یان بسڕدرێتەوە
- هێشتا کاریگەری ژمێریاری نییە
- هیچ تۆماری ڕۆژنامەیەک دروست نەکراوە

### پشتڕاستکراوەتەوە
- پارەدان کۆتایی هاتووە و بۆ ژمێریاری پۆست کراوە
- تۆمارەکانی ڕۆژنامە بە خۆکاری دروست دەبن
- نەگۆڕ بۆ پاراستنی شوێنی پشکنین
- ناتوانرێت دەستکاری بکرێت یان بسڕدرێتەوە
- ڕێژەکانی ئاڵوگۆڕ لە پشتڕاستکردنەوەدا قوڵف دەبن

### ڕێکخراو
- پارەدان لەگەڵ هێڵی ڕاپۆرتی بانک هاوتا کراوە
- پشتڕاستی دەکاتەوە کە مامەڵەی ڕاستەقینەی بانک ڕوویداوە
- تۆماری ڕۆژنامەی زیادەی ڕێکخستن دروست دەبێت
- دۆخی کۆتایی لە ڕێڕەوی ئاسایی

### هەڵوەشێنراوەتەوە
- پارەدانی پشتڕاستکراوەتەوە کە پێچەوانە کراوەتەوە
- تۆماری ڕۆژنامەی پێچەوانەکردنەوە دروست دەبێت
- شوێنی تەواوی پشکنین پارێزراوە
- ناتوانرێت پاشگەز بکرێتەوە کاتێک هەڵوەشێنراوەتەوە

**گرنگ**: تەنها پارەدانەکانی ڕەشنووس دەتوانرێن دەستکاری بکرێن یان بسڕدرێنەوە. پارەدانە پشتڕاستکراوەکان پێویستیان بە ڕێکارەکانی گونجاوی پێچەوانەکردنەوەیە بۆ پاراستنی یەکگرتووی ژمێریاری.

## ڕێگاکانی پارەدان

سیستەمەکە پشتگیری چەندین ڕێگای پارەدان دەکات:

- **دەستی** — تۆمارکردنی گشتی دەستی
- **چێک** — پارەدانەکانی چێکی کاغەزی
- **گواستنەوەی بانکی** — گواستنەوەی ئەلیکترۆنی بانکی
- **کارتی کریدیت** — مامەڵەکانی کارتی کریدیت
- **کارتی دەبیت** — مامەڵەکانی کارتی دەبیت
- **کاش** — پارەدانەکانی کاشی فیزیکی
- **گواستنەوەی سیمی** — گواستنەوەی سیمی نێودەوڵەتی
- **ACH** — گواستنەوەکانی خۆکارانەی پاککردنەوەی ماڵ
- **SEPA** — گواستنەوەکانی ناوچەی یەک یۆرۆیی
- **پارەدانی ئۆنلاین** — پلاتفۆرمەکانی پارەدانی دیجیتاڵ

## پارەدانەکان لە کوێ لە سیستەمدا دەردەکەون؟

ڕێگەی گەیشتن بە پارەدانەکان لە ڕێگەی: **ژمێریاری → بانک و کاش → پارەدانەکان**

پارەدانەکان هەروەها دەردەکەون لە:
- **پسووڵەکانی فرۆشتن**: کردەی تۆمارکردنی پارەدان بۆ پسووڵەکانی فرۆشتن نێردراوەکان (نەپارەدراوەکان)
- **پسووڵەکانی کڕین**: کردەی تۆمارکردنی پارەدان بۆ پسووڵەکانی کڕین نێردراوەکانی فرۆشیار (نەپارەدراوەکان)
- **هاوسەنگی بانک**: پارەدانە پشتڕاستکراوەکان لەگەڵ هێڵەکانی ڕاپۆرتی بانک هاوتا دەکرێن
- **تۆمارەکانی پەرتووکی ڕۆژانە**: بینینی کاریگەری ژمێریاری پارەدانە پشتڕاستکراوەکان

---

## دروستکردنی پارەدانەکان

### ١) پارەدانی سەربەخۆ لە مێنووی پارەدانەکان

ئەمە بەکاربهێنە بۆ جوڵەی پارە کە ڕاستەوخۆ بەستراو نییە بە پسوڵە/پسوڵەی تایبەت (پێش پارەدانی کڕیار، پێش پارەدانی فرۆشیار، یان پارەدانی ڕاستەوخۆ کە دواتر بەستەر دەکرێت).

**هەنگاوەکان:**
1. بڕۆ بۆ **ژمێریاری → بانک و کاش → پارەدانەکان**
2. کلیک لە **دروستکردن**
3. فۆرمی پارەدان پڕبکەرەوە:
   - **جۆری پارەدان**: وەرگرتن (ناوەوە) یان ناردن (دەرەوە) هەڵبژێرە
   - **هاوبەش**: کڕیار یان فرۆشیار هەڵبژێرە
   - **پەرتووکی ڕۆژانە**: هەژماری بانک یان کاش هەڵبژێرە
   - **بەرواری پارەدان**: کاتێک پارەکە جووڵاوە
   - **دراو و بڕ**: دراوی پارەدان و بڕ
   - **ڕێگای پارەدان**: لە ڕێگا بەردەستەکان هەڵبژێرە
   - **سەرچاوە**: تێبینی ناوخۆیی یان سەرچاوەی بانکی ئیختیاری
4. **پاشەکەوت** بکە بۆ دروستکردن وەک ڕەشنووس
5. **پشتڕاستکردنەوە** کاتێک ئامادەیت بۆ پۆستکردن بۆ ژمێریاری

**ئەنجام:**
- پارەدانی ڕەشنووس دروست دەبێت و دەستکاریکراوە
- لە پشتڕاستکردنەوەدا، تۆمارەکانی پەرتووکی ڕۆژانە بە خۆکاری نێردراو دەکرێن
- پارەدان بەردەست دەبێت بۆ تەسویە یان هاوسەنگی

### ٢) تۆمارکردنی پارەدان لەسەر پسووڵەکانی فرۆشتن (پارەدانی کڕیار)

ئەمە بەکاربهێنە بۆ کۆکردنەوەی پارە بۆ پسووڵەکانی فرۆشتنی نێردراوی کڕیار.

**پێشمەرجەکان:**
- پسووڵەکانی فرۆشتن دەبێت نێردراو بێت (نەک ڕەشنووس)
- بڕی ماوەی شایستە دەبێت هەبێت

**هەنگاوەکان:**
1. پسووڵەکانی فرۆشتنی نێردراو بکەرەوە
2. کلیک لە **تۆمارکردنی پارەدان** (ئایکۆنی پارەکان)
3. دیالۆگی پارەدان پڕبکەرەوە:
   - **پەرتووکی ڕۆژانە**: بەزۆری هەژماری بانکی سەرەکیت
   - **بەرواری پارەدان**: کاتێک پارەکە وەرگیراوە
   - **بڕ**: بنەڕەت بۆ بڕی ماوە؛ ڕێکبخە بۆ پارەدانی بەشی
   - **سەرچاوە**: سەرچاوەی بانکی یان تێبینی ئیختیاری
4. **پشتڕاستکردنەوە** بکە بۆ دروستکردن و نێردانی پارەدان

**چی ڕوودەدات:**
- پارەدانی ناوەوە دروست دەبێت و ڕاستەوخۆ پشتڕاست دەکرێتەوە
- پارەدان بە پسووڵەکانی فرۆشتن بەستەر دەکرێت، بڕی شایستە کەم دەکاتەوە
- دۆخی پسووڵەکانی فرۆشتن نوێ دەکرێتەوە (بەشی پارەدراو یان پارەدراو)
- تۆمارەکانی پەرتووکی ڕۆژانە بە خۆکاری نێردراو دەکرێن
- پارەدان بەردەست دەبێت بۆ هاوسەنگی بانک

### ٣) تۆمارکردنی پارەدان لەسەر پسووڵەکانی کڕینی فرۆشیار (پارەدانی فرۆشیار)

ئەمە بەکاربهێنە بۆ پارەدانی پسووڵەکانی کڕینی نێردراوی فرۆشیار.

**پێشمەرجەکان:**
- پسووڵەکانی کڕین دەبێت نێردراو بێت (نەک ڕەشنووس)
- بڕی ماوەی شایستە دەبێت هەبێت

**هەنگاوەکان:**
1. پسووڵەکانی کڕینی نێردراوی فرۆشیار بکەرەوە
2. کلیک لە **تۆمارکردنی پارەدان** (ئایکۆنی پارەکان)
3. دیالۆگی پارەدان پڕبکەرەوە:
   - **پەرتووکی ڕۆژانە**: بەزۆری هەژماری بانکی سەرەکیت
   - **بەرواری پارەدان**: کاتێک پارەکە نێردراوە
   - **بڕ**: بنەڕەت بۆ بڕی ماوە؛ ڕێکبخە بۆ پارەدانی بەشی
   - **سەرچاوە**: سەرچاوە یان تێبینی ئیختیاری
4. **پشتڕاستکردنەوە** بکە بۆ دروستکردن و نێردانی پارەدان

**چی ڕوودەدات:**
- پارەدانی دەرەوە دروست دەبێت و ڕاستەوخۆ پشتڕاست دەکرێتەوە
- پارەدان بە پسووڵەکانی کڕین بەستەر دەکرێت، بڕی شایستە کەم دەکاتەوە
- دۆخی پسووڵەکانی کڕین نوێ دەکرێتەوە (بەشی پارەدراو یان پارەدراو)
- تۆمارەکانی پەرتووکی ڕۆژانە بە خۆکاری نێردراو دەکرێن
- پارەدان بەردەست دەبێت بۆ هاوسەنگی بانک

## پشتڕاستکردنەوەی پارەدان و بەڕێوەبردن

### پشتڕاستکردنەوەی پارەدانەکان

**پارەدانەکانی ڕەشنووس** دەتوانرێن پشتڕاست بکرێنەوە بۆ پۆستکردنیان بۆ ژمێریاری:

1. پارەدانی ڕەشنووس بکەرەوە
2. کلیک لە **پشتڕاستکردنەوەی پارەدان**
3. کردەوەکە پشتڕاست بکەرەوە کاتێک داوا دەکرێت

**لە پشتڕاستکردنەوەدا:**
- تۆمارەکانی پەرتووکی ڕۆژانە بە خۆکاری دروست دەبن
- نرخەکانی ئاڵوگۆڕ قوڵف دەبن (بۆ فرە دراو)
- پارەدان نەگۆڕ دەبێت
- دۆخ دەگۆڕێت بۆ پشتڕاستکراوەتەوە

### سنووردارکردنەکانی دەستکاری

- **پارەدانەکانی ڕەشنووس**: بە تەواوی دەستکاریکراو و سڕینەوە
- **پارەدانە پشتڕاستکراوەکان**: نەگۆڕ، ناتوانرێت دەستکاری بکرێت یان بسڕدرێتەوە
- **پارەدانە ڕێکخراوەکان**: دۆخی کۆتایی، هیچ گۆڕانکارییەک ڕێگەپێدراو نییە

### یاساکانی سڕینەوە

- **پارەدانەکانی ڕەشنووس**: بە سەلامەتی دەتوانرێن بسڕدرێنەوە
- **پارەدانە پشتڕاستکراوە/ڕێکخراوەکان**: ناتوانرێن بسڕدرێنەوە
- هەڵوەشاندنەوە بەکاربهێنە بۆ پارەدانە پشتڕاستکراوەکان کە پێویستیان بە پێچەوانەکردنەوەیە

---

## پارەدانەکانی فرە دراو

سیستەمەکە پارەدانەکان لە هەر دراوێکی ڕێکخراو بە گۆڕینی خۆکار بەڕێوە دەبات:

### بەڕێوەبردنی دراو
- **دراوی پارەدان**: ئەو دراوەی کە پارەدانەکە تێدا کراوە
- **دراوی کۆمپانیا**: دراوی بنەڕەتی کۆمپانیاکەت بۆ ڕاپۆرتکردن
- **ڕێژەی ئاڵوگۆڕ**: بە خۆکاری لە بەرواری پارەداندا گیراوە
- **گۆڕین**: گۆڕینی خۆکار بۆ دراوی کۆمپانیا

### بەڕێوەبردنی نرخی ئاڵوگۆڕ
- نرخەکان لە بەرواری پارەداندا گیراون
- لە پشتڕاستکردنەوەدا قوڵف دەبن بۆ ڕێگریکردن لە دەستکاری
- بەکاردەهێنرێن بۆ حیسابکردنی قازانج/زیانی ڕاستەقینە
- لە وردەکاریەکانی پارەداندا پیشان دەدرێن بۆ شوێنی پشکنین

### قازانج/زیانی ڕاستەقینەی ئاڵوگۆڕ
کاتێک پسووڵەکانی فرۆشتن/پسووڵەکانی کڕینی دراوی بیانی تەسویە دەکرێن:
- سیستەمەکە جیاوازی لە نێوان نرخەکانی پسووڵەکان و پارەدان حیساب دەکات
- تۆمارەکانی پەرتووکی ڕۆژانەی خۆکار بۆ قازانج/زیانی ڕاستەقینە
- نێردراو دەکرێن بۆ هەژمارەکانی ڕێکخراوی قازانج/زیانی ئاڵوگۆڕ
- ڕاپۆرتی داراییی ورد پارێزراوە

**نموونە**: 
- پسووڵەکانی فرۆشتن: ١٠٠٠ دۆلاری ئەمریکی لە نرخی ١٣٠٠ دینار/دۆلار = ١٣٠٠٠٠٠ دینار
- پارەدان: ١٠٠٠ دۆلاری ئەمریکی لە نرخی ١٣١٠ دینار/دۆلار = ١٣١٠٠٠٠ دینار
- زیانی ڕاستەقینە: ١٠٠٠٠ دینار نێردراو دەکرێت بۆ هەژماری زیانی ئاڵوگۆڕ

---

## پەیوەندیەکانی پارەدان و بەستەرەکانی بەڵگەنامە

### بەڵگەنامە بەستەرکراوەکان
پارەدانەکان دەتوانرێن بە چەندین بەڵگەنامە بەستەر بکرێن:
- **پسووڵەکانی فرۆشتن**: تەسویەکانی پارەدانی کڕیار
- **پسووڵەکانی کڕین**: تەسویەکانی پارەدانی فرۆشیار
- **بڕی جێبەجێکراو**: شوێنکەوتنی بڕە تایبەتەکان کە بۆ هەر بەڵگەنامەیەک تەرخان کراون

### بینینی پەیوەندیەکان
هەر پارەدانێک زانیاریە پەیوەندیدارەکان پیشان دەدات:
- **تابی پسووڵەکانی فرۆشتن**: هەموو پسووڵەکانی فرۆشتنی بەستەرکراوی کڕیار
- **تابی پسووڵەکانی کڕین**: هەموو پسووڵەکانی کڕینی بەستەرکراوی فرۆشیار
- **تابی تۆمارەکانی پەرتووکی ڕۆژانە**: هەموو تۆمارەکانی ژمێریاری
- **تابی هێڵەکانی ڕاپۆرتی بانک**: مێژووی هاوسەنگی

### تەسویەی بەڵگەنامە
- پارەدانەکان دەتوانن چەندین بەڵگەنامە تەسویە بکەن
- تەسویەی بەشی پشتگیری دەکرێت
- بالانسە ماوەکان بە خۆکاری شوێنکەوتوون
- مێژووی تەسویە بۆ پشکنین پارێزراوە

---

## هاوسەنگی بانک

### پڕۆسەی هاوسەنگی
1. ڕاپۆرتەکانی بانک هاوردە بکە یان دروست بکە
2. بڕۆ بۆ **ژمێریاری → بانک و کاش → ڕاپۆرتەکانی بانک**
3. ڕاپۆرت بکەرەوە و کلیک لە **هاوسەنگی**
4. پارەدانە پشتڕاستکراوەکان لەگەڵ هێڵەکانی ڕاپۆرت هاوتا بکە
5. سیستەمەکە دۆخی پارەدان نوێ دەکاتەوە بۆ هاوسەنگکراو

### کاریگەرییەکانی هاوسەنگی
- دۆخی پارەدان دەگۆڕێت بۆ هاوسەنگکراو
- هێڵی ڕاپۆرتی بانک وەک هاوسەنگکراو نیشان دەکرێت
- تۆماری پەرتووکی ڕۆژانەی زیادەی هاوسەنگی دروست دەبێت
- پشتڕاستی دەکاتەوە کە مامەڵەی ڕاستەقینەی بانک ڕوویداوە

### پێداویستیەکانی هاوسەنگی
- پارەدانەکان دەبێت پێش هاوسەنگی پشتڕاست بکرێنەوە
- بڕەکان دەبێت لە نێوان پارەدان و هێڵی ڕاپۆرتدا هاوتا بن
- ئاڵوگۆڕی دراو بە خۆکاری بەڕێوە دەبرێت
- چەندین پارەدان دەتوانن بۆ یەک هێڵی ڕاپۆرت هاوسەنگ بکرێن

---

## باشترین پراکتیسەکان و ئامۆژگارییەکان

### بەڕێوەبردنی پارەدان
- **بە وریاییەوە پشتڕاست بکەرەوە**: پارەدانە پشتڕاستکراوەکان ناتوانرێن دەستکاری بکرێن
- **سەرچاوەکان بەکاربهێنە**: سەرچاوەی بانک زیاد بکە بۆ ڕێکخستنی ئاسانتر
- **ڕێکخستنی بەردەوام**: کتێبەکان لەگەڵ ڕاپۆرتەکانی بانک هاوڕێ بگرە
- **پارەدانی بەشی**: بۆ کۆکردنەوە/پارەدانی قیستی بەکاربهێنە

### باشترین پراکتیسەکانی فرە دراو
- **چاودێری ڕێژەکانی ئاڵوگۆڕ**: گۆڕانکاری گرنگی ڕێژە کاریگەری لەسەر جۆڕی پارە دەکات
- **مەترسی دراو بپارێزە**: کۆنتراتەکانی پێشەوە بۆ بڕە گەورەکان لەبەرچاو بگرە
- **قازانج/زیان پێداچوونەوە بکە**: کاریگەری قازانج/زیانی ئاڵوگۆڕ لەسەر قازانج چاودێری بکە

### پشکنین و گونجان
- **بەڵگەنامە بپارێزە**: بەڵگەنامەی پشتگیری بۆ هەموو پارەدانەکان بهێڵەرەوە
- **تۆمارەکانی ڕۆژنامە پێداچوونەوە بکە**: کاریگەری ژمێریاری پارەدانەکان بسەلمێنە
- **گونجانی بەرواری قوڵف**: ڕێز بۆ بەروارەکانی قوڵفی خولی ژمێریاری بگرە
- **جیاکردنەوەی ئەرکەکان**: دروستکردن و پشتڕاستکردنەوەی پارەدان جیا بکەرەوە

---

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

**پ: ناتوانم پارەدانی پشتڕاستکراوە دەستکاری بکەم**
وەڵام: پارەدانە پشتڕاستکراوەکان نەگۆڕن. تۆماری پێچەوانەکردنەوە دروست بکە ئەگەر گۆڕانکاری پێویست بێت.

**پ: پارەدان لە هاوسەنگیدا دەرناکەوێت**
وەڵام: دڵنیابە کە پارەدان پشتڕاستکراوەتەوە و لەگەڵ دراو/بڕی ڕاپۆرت دەگونجێت.

**پ: نرخی ئاڵوگۆڕ هەڵە دیارە**
وەڵام: نرخەکانی ئاڵوگۆڕ لە پشتڕاستکردنەوەدا قوڵف دەبن. سەرچاوەی نرخ و بەروار بپشکنەرەوە.

**پ: ناتوانم پارەدان بسڕمەوە**
وەڵام: تەنها پارەدانەکانی ڕەشنووس دەتوانرێن بسڕدرێنەوە. هەڵوەشاندنەوە بۆ پارەدانە پشتڕاستکراوەکان بەکاربهێنە.

### پەیامەکانی هەڵە

**"تەنها پارەدانەکانی ڕەشنووس دەتوانرێن پشتڕاست بکرێنەوە"**
- پارەدان پێشتر پشتڕاستکراوەتەوە یان لە دۆخی هەڵەدایە
- پەڕە نوێ بکەرەوە و دۆخی ئێستا بپشکنەرەوە

**"بەرواری پارەدان لە خولی قوڵفکراودایە"**
- بەرواری پارەدان دەکەوێتە ناو خولی قوڵفکراوی ژمێریاری
- بەروار بگۆڕە یان داوای کردنەوەی خول لە بەڕێوەبەر بکە

**"بڕی ماوە بەس نییە"**
- هەوڵدەدەیت زیاتر لە بالانسی ماوەی بەڵگەنامە جێبەجێ بکەیت
- دۆخی بەڵگەنامە و پارەدانە پێشووەکان بپشکنەرەوە

---

## پرسیارە دووپاتکراوەکان

**پ: دەتوانم پارەدانی پشتڕاستکراوە بسڕمەوە؟**
وەڵام: نەخێر. بەڵگەنامە دارایییەکان کاتێک پشتڕاست دەکرێنەوە نەگۆڕ دەبن بۆ پاراستنی شوێنی پشکنین. هەڵوەشاندنەوە بۆ پێچەوانەکردنەوە بەکاربهێنە.

**پ: دەتوانم پارەدانی بەشی بکەم؟**
وەڵام: بەڵێ. بڕێکی بچووکتر داخڵ بکە کاتێک پارەدان تۆمار دەکەیت. بەڵگەنامەکە بەشی پارەدراو دەمێنێتەوە لەگەڵ بالانسی نوێکراوەوە.

**پ: دراوی بیانی چۆن کار دەکات؟**
وەڵام: ئێمە بڕەکان لە هەردوو دراوی پارەدان و کۆمپانیادا هەڵدەگرین. قازانج/زیانی ئاڵوگۆڕ بە خۆکاری حیساب دەکرێت و نێردراو دەکرێت کاتێک بەڵگەنامەکان تەسویە دەکرێن.

**پ: جیاوازی لە نێوان تەسویە و پێش پارەدانەکاندا چییە؟**
وەڵام: پارەدانەکانی تەسویە بە پسووڵەکانی فرۆشتن/پسووڵەکانی کڕینی تایبەت بەستەر دەکرێن. پێش پارەدانەکان سەربەخۆن و دەتوانرێن بۆ بەڵگەنامەکانی داهاتوو جێبەجێ بکرێن.

**پ: دەتوانێت یەک پارەدان چەندین پسووڵەکانی فرۆشتن تەسویە بکات؟**
وەڵام: بەڵێ. بەستەرەکانی بەڵگەنامە بەکاربهێنە بۆ دابەشکردنی بڕەکانی پارەدان بەسەر چەندین پسووڵەکانی فرۆشتن یان پسووڵەکانی کڕیندا.

**پ: چۆن کۆمسیۆنەکانی پارەدان بەڕێوە دەبەم؟**
وەڵام: کۆمسیۆنەکان بە جیا لە ڕێگەی ڕاپۆرتەکانی بانک یان تۆمارەکانی پەرتووکی ڕۆژانەی جۆراوجۆر تۆمار بکە. بڕەکانی پارەدان دەبێت بڕە بەدەستهاتوو/نێردراوەکانی پاک ڕەنگبدەنەوە.

## فەرهەنگ و دەستەواژە تایبەتەکان

### زانیاریەکانی بنەڕەتی
- **هاوبەش (Partner)**: کڕیار یان فرۆشیار لە سیستەمدا
- **پەرتووکی ڕۆژانە (Journal)**: هەژماری بانک یان کاش کە مامەڵەکان تێدا تۆمار دەکرێن
- **تەسویە (Settlement)**: جێبەجێکردنی پارەدان بۆ پسووڵەکانی فرۆشتن یان پسووڵەکانی کڕینی تایبەت
- **هاوسەنگی (Reconciliation)**: هاوتاکردنی پارەدانەکانی سیستەم لەگەڵ هێڵەکانی ڕاپۆرتی بانک
- **بەستەری بەڵگەنامە (Document Link)**: پەیوەندی لە نێوان پارەدان و پسووڵەکانی فرۆشتن/پسووڵەکانی کڕین لەگەڵ بڕی جێبەجێکراو

### دراو و ئاڵوگۆڕ
- **نرخی ئاڵوگۆڕ (Exchange Rate)**: نرخی ئاڵوگۆڕی دراو کە لە پشتڕاستکردنەوەی پارەداندا قوڵف دەبێت
- **قازانج/زیانی ڕاستەقینە (Realized Gain/Loss)**: جیاوازی ئاڵوگۆڕ کاتێک بەڵگەنامەکانی دراوی بیانی تەسویە دەکرێن
- **دراوی پارەدان (Payment Currency)**: ئەو دراوەی کە پارەدانەکە تێدا کراوە
- **دراوی کۆمپانیا (Company Currency)**: دراوی بنەڕەتی کۆمپانیاکەت بۆ ڕاپۆرتکردن

### دۆخەکانی پارەدان
- **ڕەشنووس (Draft)**: پارەدانی تازە دروستکراو، بە تەواوی دەستکاریکراو
- **نێردراوە (Posted)**: پارەدان کۆتایی هاتووە و بۆ ژمێریاری نێردراوە
- **هاوسەنگکراو (Reconciled)**: پارەدان لەگەڵ هێڵی ڕاپۆرتی بانک هاوتا کراوە
- **هەڵوەشێنراوەتەوە (Canceled)**: پارەدانی پشتڕاستکراوەتەوە کە پێچەوانە کراوەتەوە

### جۆرەکانی پارەدان
- **پارەدانی ناوەوە (Inbound Payment)**: پارەی وەرگیراو لە کڕیارەکان
- **پارەدانی دەرەوە (Outbound Payment)**: پارەی نێردراو بۆ فرۆشیارەکان
- **پارەدانی تەسویە (Settlement Payment)**: بەستراوە بە پسوڵە یان پسوڵەی فرۆشیاری تایبەت
- **پێش پارەدان/قەرزی هاوبەش (Partner Advance/Credit)**: پارەدانی سەربەخۆ

### ڕێگاکانی پارەدان
- **دەستی (Manual)**: تۆمارکردنی گشتی دەستی
- **چێک (Check)**: پارەدانەکانی چێکی کاغەزی
- **گواستنەوەی بانکی (Bank Transfer)**: گواستنەوەی ئەلیکترۆنی بانکی
- **کارتی کریدیت (Credit Card)**: مامەڵەکانی کارتی کریدیت
- **کارتی دەبیت (Debit Card)**: مامەڵەکانی کارتی دەبیت
- **کاش (Cash)**: پارەدانەکانی کاشی فیزیکی
- **گواستنەوەی سیمی (Wire Transfer)**: گواستنەوەی سیمی نێودەوڵەتی

### تەکنیکی و ژمێریاری
- **تۆماری پەرتووکی ڕۆژانە (Journal Entry)**: تۆماری ژمێریاری کە لە پشتڕاستکردنەوەی پارەداندا دروست دەبێت
- **هەژمارەکانی وەرگرتنی (Accounts Receivable)**: هەژمارەکان کە کڕیارەکان قەرزارن
- **هەژمارەکانی پارەدانی (Accounts Payable)**: هەژمارەکان کە کۆمپانیا قەرزاری فرۆشیارەکانە
- **بڕی ماوە (Remaining Amount)**: بڕی نەپارەدراوی پسووڵەکانی فرۆشتن یان پسووڵەکانی کڕین
- **بڕی جێبەجێکراو (Amount Applied)**: بڕێک کە بۆ بەڵگەنامەیەکی تایبەت تەرخان کراوە

### پرۆسەکان
- **پشتڕاستکردنەوە (Confirmation)**: پرۆسەی کۆتایی هێنانی پارەدان و نێردانی بۆ ژمێریاری
- **هەڵوەشاندنەوە (Cancellation)**: پێچەوانەکردنەوەی پارەدانێکی پشتڕاستکراوەتەوە
- **هاوسەنگی بانک (Bank Reconciliation)**: پرۆسەی هاوتاکردنی پارەدانەکان لەگەڵ ڕاپۆرتەکانی بانک
- **تەسویەی بەڵگەنامە (Document Settlement)**: جێبەجێکردنی پارەدان بۆ پسووڵەکانی فرۆشتن یان پسووڵەکانی کڕین

### ڕاپۆرت و شوێنکەوتن
- **شوێنی پشکنین (Audit Trail)**: ڕیکۆردی تەواوی گۆڕانکارییەکان و کردەوەکان
- **ڕاپۆرتی بانک (Bank Statement)**: بەڵگەنامەی بانک کە مامەڵەکان پیشان دەدات
- **هێڵی ڕاپۆرت (Statement Line)**: یەک مامەڵە لە ڕاپۆرتی بانکدا
- **جووڵەی پارە (Cash Flow)**: جووڵەی پارە لە ناو کۆمپانیادا
