# بەڕێوەبردنی کۆگا: بەدواداچوون و بەها دانانی کۆگا

ئەم ڕێبەرییە گشتگیرە ڕوونیدەکاتەوە چۆن بەڕێوەبردنی کۆگا کار دەکات لە سیستمی ژمێریاریدا، کە خۆی دەبینێتەوە لە ڕێکخستنی کۆگا، شێوازەکانی بەها دانان، شوێنەکانی کۆگا، و بەدواداچوونی جووڵە. ئەم ڕێبەرییە نووسراوە بۆ هەموو بەکارهێنەرێک — ژمێریار بێت یان بەکارهێنەری ئاسایی — لەم ڕێبەرییەدا هەوڵدراوە بە نمونەی کردەیی هەنگاوەکان باس بکرێت کە پەیڕەوی لە باشترین شێوازەکانی ژمێریاری دوو-تۆماری دەکات.

---

## بەڕێوەبردنی کۆگا چییە؟

بەڕێوەبردنی کۆگا بریتییە لە بەدواداچوونی سیستماتیک و بەها دانانی کۆگا لە سەرانسەری کارە بازرگانییەکانتدا، کە زانیاری تێچوونی دروست بۆ ڕاپۆرتی دارایی و بڕیاردانی کارگێڕی دابین دەکات.

-   **بەدواداچوونی کۆگا**: چاودێریکردنی بڕ و شوێنەکانی کۆگا لە کاتی ڕاستەقینەدا
-   **بەها دانانی تێچوون**: ژماردنی تێچووەکانی کۆگا بە بەکارهێنانی شێوازەکانی FIFO, LIFO, AVCO, یان نرخی ستاندارد
-   **تۆمارکردنی جووڵە**: بەڵگەکردنی هەموو جووڵەکانی کۆگا بە ڕێڕەوی پشکنینی تەواو
-   **یەکخستنی ژمێریاری**: تۆماری ڕۆژانەی خۆکارانە بۆ مامەڵەکانی کۆگا

**ئامانجی ژمێریاری**: بەڕێوەبردنی کۆگا دڵنیایی دەدات لە بەها دانانی دروستی کۆگا بۆ ڕاپۆرتی پەڕەی باڵانس و ژماردنی دروستی تێچوونی کاڵای فرۆشراو بۆ دروستی لیستی داهات.

---

## پێداویستیەکانی سیستەم

### ڕێکخستنی بەرهەم
-   **بەرهەمە کۆگاکراوەکان**: بەرهەمەکان دەبێت وەک جۆری "بەرهەمی کۆگاکراو" ڕێکخرابن بۆ بەدواداچوونی کۆگا
-   **شێوازەکانی بەها دانان**: شێوازەکانی FIFO, LIFO, AVCO (تێچووی تێکڕا)، یان نرخی ستاندارد دەبێت هەڵبژێردرێن
-   **دانانی هەژمار**: هەژمارەکانی کۆگا، تێچوونی کاڵای فرۆشراو، و هاتووی کۆگا دەبێت ڕێکخرابن
-   **پێکهاتەی شوێن**: شوێنەکانی کۆگا دەبێت دروستکرابن و ڕێکخرابن

### پێشمەرجەکان
1.  **نەخشەی هەژمارەکان**: هەژماری سامانی کۆگا، خەرجی تێچوونی کاڵای فرۆشراو، و هەژمارەکانی قەرزی هاتووی کۆگا ڕێکخرابن
2.  **کەتەلۆگی بەرهەم**: بەرهەمەکان بە ڕێکخستنی کۆگای گونجاوەوە دانرابن
3.  **پەیژەی شوێن**: پێکهاتەی کۆگا و شوێن دامەزرابێت
4.  **مۆڵەتەکانی بەکارهێنەر**: دەستگەیشتن بە تایبەتمەندییەکانی بەڕێوەبردنی کۆگا و ژمێریاری

---

## چۆن لە ڕووکاری بەکارهێنەردا بیدۆزینەوە

بڕۆ بۆ **کۆگا → بەرهەمەکان** یان **کۆگا → شوێنەکانی کۆگا**

بەڕێوەبردنی کۆگا لەم بەشانەشدا دەردەکەون:
-   **جووڵەکانی کۆگا**: بینین و دروستکردنی جووڵەی کۆگا
-   **پسووڵەکانی فرۆشیار**: جووڵەی کۆگای خۆکارانە کاتێک پسووڵە بۆ بەرهەمە کۆگاکراوەکان دەنێردرێت
-   **پسووڵەکانی کڕیار**: جووڵەی کۆگای خۆکارانە کاتێک پسووڵە بۆ بەرهەمە کۆگاکراوەکان دەنێردرێت
-   **ڕاپۆرتەکان**: ڕاپۆرتەکانی بەها دانانی کۆگا و جووڵە

**تێبینی**: دوگمەی یارمەتی/بەڵگەنامە لە سەرەوە ئەم ڕێبەرییە دەکاتەوە.

---

هەروەها دەتوانیت جووڵەی دەستی دروست بکەیت:
1. بڕۆ بۆ **جووڵەکانی کۆگا** لە بەشی **کۆگا**
2. کلیک لە **دروستکردنی جووڵەی کۆگا** بکە
3. وردەکارییەکانی جووڵەکە پڕ بکەرەوە:
    - بەرهەمەکە هەڵبژێرە
    - شوێنەکانی لە/بۆ هەڵبژێرە
    - بڕ و جۆری جووڵە بنووسە
    - بەرواری جووڵە دابنێ

## شێوازەکانی بەها دانان ڕوونکراوەتەوە

### FIFO (یەکەم هاتوو، یەکەم دەرچوو)
- کۆنترین کۆگا یەکەم جار بەکاردەهێنرێت
- چینەکانی تێچوون بەدواداچوون بۆ هەر کڕینێک بە جیا دەکەن
- باشە بۆ کاڵا خراپبووەکان یان کاتێک تێچووەکان بەرز دەبنەوە

### LIFO (کۆتا هاتوو، یەکەم دەرچوو)
- نوێترین کۆگا یەکەم جار بەکاردەهێنرێت
- چینەکانی تێچوون بەدواداچوون بۆ هەر کڕینێک بە جیا دەکەن
- بەسوودە کاتێک تێچووەکان دادەبەزن

### AVCO (تێچووی تێکڕا)
- ژماردنی تێچووی تێکڕای کێشدار
- یەک تێچووی تێکڕا بۆ هەر بەرهەمێک دەپارێزرێت
- سادەترین شێوازە بۆ زۆربەی بازرگانییەکان

### نرخی ستاندارد
- تێچووی جێگیر بۆ هەر یەکەیەک
- جیاوازییەکان لە هەژماری جیاوازی نرخدا بەدواداچوونیان بۆ دەکرێت
- باشە بۆ بەرهەمهێنان بە تێچووی ستاندارد

## باشترین شێوازەکان

### ۱. ڕێکخستنی هەژمار
- هەژماری کۆگای تایبەت بۆ هەر پۆلێنێکی بەرهەم دابنێ
- هەژماری تێچوونی کاڵای فرۆشراوی جیاواز بۆ هێڵە جیاوازەکانی بەرهەم بەکاربهێنە

### ۲. پێکهاتەی شوێن
- پەیژەی کۆگای لۆژیکی دروست بکە
- ناوی ڕوونکەرەوە بەکاربهێنە بۆ ناسینەوەی ئاسان
- شوێنەکانی فرۆشیار و کڕیار جیا بکەرەوە

### ۳. بەڕێوەبردنی جووڵە
- ژمارەی بەڵگە بەکاربهێنە بۆ بەدواداچوونی باشتر
- جووڵەکان بە خێرایی پشتڕاست بکەرەوە بۆ پاراستنی دروستی
- بە شێوەی بەردەوام پێداچوونەوە بە چینەکانی تێچوون بکە بۆ بەرهەمەکانی FIFO/LIFO

### ٤. ڕاپۆرتکردن و شیکردنەوە
- چاودێری تێچووە تێکڕاکان بکە بۆ بڕیارەکانی نرخ دانان
- پێداچوونەوە بە مێژووی جووڵەی کۆگا بکە بۆ مەبەستی پشکنین
- زانیاری چینەکانی تێچوون بەکاربهێنە بۆ شیکردنەوەی وردی تێچوون

## یەکخستن لەگەڵ مۆدیۆلەکانی تر

سیستەمی بەڕێوەبردنی کۆگا بە شێوەیەکی یەکپارچە لەگەڵ بەشەکانی تری بەرنامەکەدا یەکدەگرێتەوە:

-   **[پسووڵەکانی کڕیار](customer-invoices.ckb.md)**: خۆکارانە جووڵەی کۆگای دەرچوو دروستدەکات کاتێک پسووڵەکان دەنێردرێن
-   **[پارەدانەکان](payments.ckb.md)**: پرۆسەی پارەدان کاریگەری لەسەر لایەنە داراییەکانی مامەڵەکانی کۆگا هەیە
-   **[هاوتای بانکی](bank-reconciliation.ckb.md)**: هاوتاکردنی پارەدانەکانی پەیوەندیدار بە کڕین و فرۆشتنی کۆگا

## تایبەتمەندییە پێشکەوتووەکان

### تۆماری ڕۆژانەی خۆکارانە
هەموو جووڵەکانی کۆگا تۆماری ڕۆژانەی هاوتا دروستدەکەن:
- کۆگای هاتوو: قەرزدار(مدين) کۆگا، قەرزدەر هاتووی کۆگا
- کۆگای دەرچوو: قەرزدار(مدين) تێچوونی کاڵای فرۆشراو، قەرزدەر کۆگا
- ڕێکخستنەکانی کۆگا: قەرزدار(مدين)/قەرزدەر کۆگا و هەژمارەکانی ڕێکخستنی هاوتا

### پاراستنی بەرواری قوفڵکراو
سیستەم بەروارەکانی قوفڵکراوی ژمێریاری جێبەجێ دەکات بۆ ڕێگریکردن لە گۆڕانکاری نایاسایی لە داتای کۆگای مێژووییدا.

### بەڕێوەبردنی چینی تێچوون
بۆ بەرهەمەکانی FIFO و LIFO، سیستەم چینەکانی تێچوونی ورد دەپارێزێت کە نیشانی دەدەن:
- بەروار و بڕی کڕین
- تێچووی یەکە و بەهای گشتی
- بڕ و بەهای ماوە
- ژمارەی بەڵگەی سەرچاوە

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

**پرسیار: بۆچی ناتوانم خانەکانی کۆگا لەسەر بەرهەمەکەم ببینم؟**
وەڵام: دڵنیابەرەوە جۆری بەرهەم وەک "بەرهەمی کۆگاکراو" دانراوە. خانەکانی کۆگا تەنها بۆ بەرهەمە کۆگاکراوەکان دیارن.

**پرسیار: بۆچی چینەکانی تێچوونم بەتاڵن؟**
وەڵام: چینەکانی تێچوون تەنها بۆ شێوازەکانی بەها دانانی FIFO و LIFO دروستدەکرێن. بەرهەمەکانی AVCO یەک تێچووی تێکڕا بەکاردەهێنن.

**پرسیار: دەتوانم جووڵەیەکی کۆگای تەواوکراو دەستکاری بکەم؟**
وەڵام: نەخێر، تەنها جووڵەکانی ڕەشنووس دەتوانرێت دەستکاری بکرێن. جووڵەکانی کۆگا کە بارودۆخی "تەواوکراو"یان هەیە ناتوانرێت دەستکاری بکرێن بۆ پاراستنی یەکپارچەیی داتا و ڕێڕەوی پشکنین.

**پرسیار: جیاوازی نێوان بارودۆخەکانی جووڵە چییە؟**
وەڵام: ڕەشنووس (دەستکاریدەکرێت) → پشتڕاستکراو (قوفڵکراو) → تەواوکراو (تەواوکراوە بە تۆماری ڕۆژانەوە)

**پرسیار: چۆن کۆگا بۆ کاڵا زیانلێکەوتووەکان ڕێکبخەم؟**
وەڵام: جووڵەیەکی کۆگا بە جۆری "ڕێکخستن" دروست بکە لە کۆگاکەتەوە بۆ شوێنێکی ڕێکخستن.

## پشتگیری فرە-زمان

سیستەمی بەڕێوەبردنی کۆگا پشتگیری زمانەکانی ئینگلیزی، کوردی (سۆرانی)، و عەرەبی دەکات. هەموو توخمەکانی ڕووکار، ناونیشانەکان، و پەیامەکان بە تەواوی وەرگێڕدراون. بەکارهێنەران دەتوانن لە نێوان زمانەکاندا بگۆڕن بە بەکارهێنانی هەڵبژاردنی زمان لە سەرەوەی بەرنامەکەدا.

## یەکخستن لەگەڵ ژمێریاری

جووڵەکانی کۆگا خۆکارانە تۆماری ڕۆژانە دروستدەکەن کە:
- هەژمارەکانی سامانی کۆگا نوێ دەکەنەوە
- تێچوونی کاڵای فرۆشراو تۆمار دەکەن
- بەدواداچوون بۆ قەرزەکانی هاتووی کۆگا دەکەن
- مامەڵە لەگەڵ جیاوازی نرخ و ڕێکخستنەکان دەکەن

ئەمە دڵنیایی دەدات لەوەی بەهای کۆگاکانت هەمیشە لەگەڵ پەرتووکی گشتییەکەتدا هاوکاتن.
