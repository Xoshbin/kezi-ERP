# ڕێنمایی بەڕێوەبردنی کۆگا

## تێگەیشتنی گشتی

سیستەمی بەڕێوەبردنی کۆگا توانای تەواوی بێشوێنکردنی کۆگا و پێوانی نرخەکانی دەستەبەر دەکات. پشتگیری دەکات لە چەندین شێوازی پێوان، پێکهاتەی پلەبەندی ئەمبارەکان، و ڕێگەی تەواوی ئاگاداری بۆ هەموو جووڵەکانی کۆگا.

## تایبەتمەندییە سەرەکییەکان

### 📦 **ڕێکخستنی کۆگای بەرهەم**
- **شێوازەکانی پێوان**: یەکەم هات-یەکەم چوو (FIFO)، دواین هات-یەکەم چوو (LIFO)، تێچووی ناوەند (AVCO)، و نرخی ستاندارد
- **هەژمارەکانی کۆگا**: ڕێکخستنی خۆکار بۆ هەژمارەی سامانی کۆگا، تێچووی کاڵای فرۆشراو، هەژمارەی هاتنەژوورەوەی کۆگا
- **بێشوێنکردنی تێچووی ناوەند**: ژمارەکردن و پیشاندانی ڕاستەوخۆ بۆ بەرهەمە AVCO کان
- **مێژووی جووڵەی کۆگا**: ڕێگەی تەواوی ئاگاداری بۆ هەموو جووڵەکانی بەرهەم
- **بینینی چینەکانی تێچوو**: بینینی چینەکانی تێچووی FIFO/LIFO بۆ بێشوێنکردنی ورد
- **یەکگرتنی تۆمارکردنی ژورنال**: تۆمارکردنی خۆکاری حیسابداری بۆ هەموو مامەڵەکانی کۆگا

### 🏢 **بەڕێوەبردنی شوێنەکانی کۆگا**
- **پێکهاتەی پلەبەندی**: دروستکردنی پەیوەندی باوک-منداڵ لە نێوان شوێنەکاندا
- **جۆرەکانی شوێن**: ئەمبارە ناوخۆییەکان، شوێنەکانی کڕیار، شوێنەکانی فرۆشیار، و شوێنەکانی ڕاستکردنەوەی کۆگا
- **پشتگیری فرە-کۆمپانیا**: بەڕێوەبردنی شوێن تایبەت بە کۆمپانیا
- **دۆخی چالاک/ناچالاک**: کۆنترۆڵکردنی ئەوەی کام شوێنەکان بۆ بەکارهێنان بەردەستن

### 📋 **بێشوێنکردنی جووڵەی کۆگا**
- **جۆرەکانی جووڵە**: هاتنەژوورەوە، چوونەدەرەوە، گواستنەوەی ناوخۆیی، و ڕاستکردنەوە
- **کارپێڕاوی دۆخ**: ڕەشنووس ← پشتڕاستکراوە ← تەواوبووە (لەگەڵ پشتگیری هەڵوەشاندنەوە)
- **یەکگرتنی بەڵگەی سەرچاوە**: دروستکردنی خۆکار لە پاڵ ئینڤۆیسەکانی فرۆشیار و ئینڤۆیسەکانی کڕیار
- **ژمارەکانی ئاماژە**: بێشوێنکردنی ئاماژەی ئیختیاری بۆ باشترکردنی ڕێکخستن
- **پاڵاوتنی بەپێی بەروار**: دۆزینەوەی جووڵەکان بەپێی مەودای بەروار
- **جێبەجێکردنی بەروواری داخستن**: ڕێگری لە گۆڕانکاری لە ماوەکانی داخراو

## دەستپێکردن

### 1. ڕێکخستنی بەرهەمەکان بۆ کۆگا

1. بڕۆ بۆ **بەرهەمەکان** لە بەشی **کۆگا**
2. بەرهەمێک دروست بکە یان دەستکاری بکە
3. **جۆر**ەکە لەسەر "بەرهەمی کۆگاکراو" دابنێ
4. لە بەشی **بەڕێوەبردنی کۆگا**دا:
   - **شێوازی پێوان**ێک هەڵبژێرە (FIFO، LIFO، AVCO، یان نرخی ستاندارد)
   - **هەژمارە پێویستەکان** ڕێکبخە:
     - هەژمارەی کۆگا (هەژمارەی سامان بۆ نرخی کۆگا)
     - هەژمارەی تێچووی کاڵای فرۆشراو (هەژمارەی خەرجی بۆ کاڵا فرۆشراوەکان)
     - هەژمارەی هاتنەژوورەوەی کۆگا (هەژمارەی قەرز بۆ کۆگای وەرگیراو)

### 2. ڕێکخستنی شوێنەکانی کۆگا

1. بڕۆ بۆ **شوێنەکانی کۆگا** لە بەشی **کۆگا**
2. پێکهاتەی ئەمبارەکانت دروست بکە:
   - **ئەمباری سەرەکی** (جۆری ناوخۆیی)
   - **ژێرشوێنەکان** لە ناو ئەمبارەکاندا (پەیوەندی باوک دابنێ)
   - **شوێنەکانی فرۆشیار** (جۆری فرۆشیار)
   - **شوێنەکانی کڕیار** (جۆری کڕیار)
   - **شوێنەکانی ڕاستکردنەوە** (جۆری ڕاستکردنەوەی کۆگا) بۆ چاکسازییەکان

### 3. بێشوێنکردنی جووڵەی کۆگا

جووڵەی کۆگا خۆکار دروست دەبن کاتێک:
- پاڵ ئینڤۆیسەکانی فرۆشیار تۆمار دەکرێن (جووڵەی هاتنەژوورەوە دروست دەکات)
- ئینڤۆیسەکانی کڕیار تۆمار دەکرێن (جووڵەی چوونەدەرەوە دروست دەکات)

هەروەها دەتوانی جووڵەی دەستی دروست بکەیت:
1. بڕۆ بۆ **جووڵەی کۆگا** لە بەشی **کۆگا**
2. کلیک لە **دروستکردنی جووڵەی کۆگا** بکە
3. وردەکارییەکانی جووڵە پڕ بکەوە:
   - بەرهەمەکە هەڵبژێرە
   - شوێنەکانی لە/بۆ هەڵبژێرە
   - بڕ و جۆری جووڵە بنووسە
   - بەرواری جووڵە دابنێ

## شێوازەکانی پێوان ڕوونکراوەتەوە

### FIFO (یەکەم هات، یەکەم چوو)
- کۆنترین کۆگا سەرەتا بەکاردێت
- چینەکانی تێچوو هەر کڕینێک بە جیا بێشوێن دەکەن
- باشە بۆ کاڵا خراپبووەکان یان کاتێک تێچووەکان بەرز دەبنەوە

### LIFO (دواین هات، یەکەم چوو)
- نوێترین کۆگا سەرەتا بەکاردێت
- چینەکانی تێچوو هەر کڕینێک بە جیا بێشوێن دەکەن
- سوودبەخشە کاتێک تێچووەکان نزم دەبنەوە

### AVCO (تێچووی ناوەند)
- ژمارەکردنی تێچووی ناوەندی کێشراو
- تێچووی ناوەندی یەک بۆ هەر بەرهەمێک پارێزراوە
- سادەترین شێواز بۆ زۆربەی بازرگانییەکان

### نرخی ستاندارد
- تێچووی جێگیر بۆ هەر یەکەیەک
- جیاوازییەکان لە هەژمارەی جیاوازی نرخدا تۆمار دەکرێن
- باشە بۆ بەرهەمهێنان لەگەڵ تێچووە ستاندارەکان

## باشترین پراکتیسەکان

### 1. ڕێکخستنی هەژمار
- هەژمارە تایبەتەکانی کۆگا بۆ هەر پۆلێکی بەرهەم دابنێ
- هەژمارە جیاوازەکانی تێچووی کاڵای فرۆشراو بۆ هێڵە جیاوازەکانی بەرهەم بەکاربهێنە
- هەژمارەکانی جیاوازی نرخ بۆ بێشوێنکردنی جیاوازی ڕێکبخە

### 2. پێکهاتەی شوێن
- پلەبەندی لۆژیکی ئەمبار دروست بکە
- ناوی وەسفکار بۆ ناسینەوەی ئاسان بەکاربهێنە
- شوێنەکانی فرۆشیار و کڕیار بە جیا بهێڵەوە

### 3. بەڕێوەبردنی جووڵە
- ژمارەکانی ئاماژە بۆ بێشوێنکردنی باشتر بەکاربهێنە
- جووڵەکان بە خێرایی پشتڕاست بکە بۆ پاراستنی وردی
- چینەکانی تێچوو بە بەردەوامی بۆ بەرهەمەکانی FIFO/LIFO پێداچوونەوە بکە

### 4. گوزارشت و شیکاری
- تێچووە ناوەندەکان بۆ بڕیارەکانی نرخدانان چاودێری بکە
- مێژووی جووڵەی کۆگا بۆ مەبەستەکانی ئاگاداری پێداچوونەوە بکە
- زانیاری چینەکانی تێچوو بۆ شیکاری ورد بەکاربهێنە

## یەکگرتن لەگەڵ مۆدیولەکانی تر

سیستەمی بەڕێوەبردنی کۆگا بە نەرمی لەگەڵ بەشەکانی ترەوەی ئەپڵیکەیشنەکە یەکدەگرێتەوە:

- **[ئینڤۆیسەکانی کڕیار](customer-invoices.ckb.md)**: خۆکار جووڵەی چوونەدەرەوەی کۆگا دروست دەکات کاتێک ئینڤۆیسەکان تۆمار دەکرێن
- **پاڵ ئینڤۆیسەکانی فرۆشیار**: خۆکار جووڵەی هاتنەژوورەوەی کۆگا دروست دەکات کاتێک پاڵەکان پشتڕاست دەکرێن
- **[پارەدانەکان](payments.ckb.md)**: پڕۆسەی پارەدان کاریگەری لەسەر لایەنە دارایی ەکانی مامەڵەکانی کۆگا دەبێت
- **[یەکگرتنەوەی بانک](bank-reconciliation.ckb.md)**: یەکگرتنەوەی پارەدانەکان کە پەیوەستن بە کڕین و فرۆشتنی کۆگا

## تایبەتمەندی پێشکەوتووەکان

### تۆمارکردنی خۆکاری ژورنال
هەموو جووڵەکانی کۆگا تۆمارکردنی ژورنالی هاوتای دروست دەکەن:
- کۆگای هاتنەژوورەوە: قەرز کۆگا، کڕەدیت هاتنەژوورەوەی کۆگا
- کۆگای چوونەدەرەوە: قەرز تێچووی کاڵای فرۆشراو، کڕەدیت کۆگا
- ڕاستکردنەوەی کۆگا: قەرز/کڕەدیت کۆگا و هەژمارە هاوتاکانی ڕاستکردنەوە

### پاراستنی بەرواری داخستن
سیستەمەکە بەرواری داخستنی حیسابداری جێبەجێ دەکات بۆ ڕێگری لە گۆڕانکاری نەمۆکی زانیاری کۆگای مێژوویی.

### بەڕێوەبردنی چینەکانی تێچوو
بۆ بەرهەمەکانی FIFO و LIFO، سیستەمەکە چینەکانی ورد تێچوو پارێز دەکات کە پیشان دەدەن:
- بەرواری کڕین و بڕ
- تێچووی یەکە و نرخی گشتی
- بڕ و نرخی ماوە
- ئاماژەی بەڵگەی سەرچاوە

## چارەسەرکردنی کێشە

### کێشە باوەکان

**پ: بۆچی ناتوانم خانەکانی کۆگا لەسەر بەرهەمەکەم ببینم؟**
وەڵام: دڵنیابە کە جۆری بەرهەمەکە لەسەر "بەرهەمی کۆگاکراو" دانراوە. خانەکانی کۆگا تەنها بۆ بەرهەمە کۆگاکراوەکان دیار دەبن.

**پ: بۆچی چینەکانی تێچووم بەتاڵن؟**
وەڵام: چینەکانی تێچوو تەنها بۆ شێوازەکانی پێوانی FIFO و LIFO دروست دەکرێن. بەرهەمەکانی AVCO تێچووی ناوەندی یەک بەکاردەهێنن.

**پ: دەتوانم جووڵەی کۆگای تەواوبووم دەستکاری بکەم؟**
وەڵام: نەخێر، تەنها جووڵەی دۆخی "ڕەشنووس" دەتوانرێت دەستکاری بکرێت. ئەمە یەکڕیزی زانیاری و ڕێگەی ئاگاداری پارێز دەکات.

**پ: چی جیاوازییە لە نێوان دۆخەکانی جووڵەدا؟**
وەڵام: ڕەشنووس (دەستکاریکراو) ← پشتڕاستکراوە (داخراو) ← تەواوبووە (تەواو لەگەڵ تۆمارکردنی ژورنال)

**پ: چۆن کۆگا بۆ کاڵا زیانلێکەوتووەکان ڕاست بکەمەوە؟**
وەڵام: جووڵەی کۆگا لەگەڵ جۆری "ڕاستکردنەوە" لە ئەمبارەکەتەوە بۆ شوێنی ڕاستکردنەوە دروست بکە.

## پشتگیری فرە-زمان

سیستەمی بەڕێوەبردنی کۆگا پشتگیری لە زمانەکانی ئینگلیزی و کوردی (سۆرانی) و عەرەبی دەکات. هەموو توخمەکانی ڕووکار، نیشانەکان، و پەیامەکان بە تەواوی وەرگێڕدراون. بەکارهێنەران دەتوانن لە نێوان زمانەکاندا بگۆڕن بە بەکارهێنانی هەڵبژێری زمان لە سەری ئەپڵیکەیشنەکەدا.

## یەکگرتن لەگەڵ حیسابداری

جووڵەی کۆگا خۆکار تۆمارکردنی ژورنال دروست دەکەن کە:
- هەژمارەکانی سامانی کۆگا نوێ دەکەنەوە
- تێچووی کاڵای فرۆشراو تۆمار دەکەن
- قەرزەکانی هاتنەژوورەوەی کۆگا بێشوێن دەکەن
- جیاوازی و ڕاستکردنەوە نرخەکان مامەڵە دەکەن

ئەمە دڵنیایی دەدات کە نرخەکانی کۆگاکەت هەمیشە لەگەڵ کتێبی گشتی سەرەکیت یەکگیراون.
