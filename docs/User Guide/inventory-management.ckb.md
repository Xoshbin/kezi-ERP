# بەڕێوەبردنی کۆگا: تێڕوانینی گشتی سیتەمی تەواو

ئەم ڕێبەرە گشتگیرە ڕوونی دەکاتەوە چۆن سیستەمی تەواوی بەڕێوەبردنی کۆگا کاردەکات لە سیستەمە ژمێریارییەکەتدا، کە هەموو لایەنەکان دەگرێتەوە لە چاودێریکردنی بنەڕەتی کۆگاوە تا بەڕێوەبردنی پێشکەوتووی ژمارەی وەجبە (Lot) و ڕاپۆرتەکان. نووسراوە بۆ هەموو بەکارهێنەران — ژمێریاران و ئەوانەشی ژمێریار نین — ڕێنمایی پراکتیکی دابین دەکات بە پەیڕەوکردنی باشترین پراکتیزەکانی ژمێریاری دوو-لایەنە.

---

## بەڕێوەبردنی کۆگا چییە؟

بەڕێوەبردنی کۆگا سیستەمێکی گشتگیرە کە چاودێری، بەها، و بەڕێوەبردنی هەموو لایەنەکانی کۆگاکەت دەکات بە درێژایی سوڕی ژیانی، لە کڕینەوە تا فرۆشتن، کە بینین و کۆنترۆڵی تەواو دابین دەکات بەسەر کارەکانی کۆگاکەتدا.

- **چاودێریکردنی ڕاستەوخۆ**: چاودێریکردنی بڕەکان، شوێنەکان، و جوڵەکان بەشێوەی ڕاستەوخۆ
- **بەهاکدانانی فرە-رێگە**: پشتگیری بۆ ڕێگاکانی FIFO, LIFO, AVCO, و Standard Price
- **چاودێریکردنی وەجبە (Lot Tracking)**: شوێنپێهەڵگرتنی تەواو بە کۆدی وەجبە و بەرواری بەسەرچوون
- **چاودێریکردنی ژمارەی زنجیرەیی**: شوێنپێهەڵگرتنی یەکە تاکەکان بە مێژوو و شوێنی تایبەت
- **گواستنەوەی نێوان کۆگاکان**: ڕێڕەوی دوو-هەنگاوی ناردن/وەرگرتن لەگەڵ چاودێریکردنی ناو-ڕێگە
- **سیستەمی هەڵگرتن (Reservation)**: هەڵگرتنی کاڵا بۆ داواکاری دیاریکراو و ڕێگریکردن لە فرۆشتنی زیاد لە پێویست
- **یاساکانی داواکردنەوە (Reordering Rules)**: پێشنیاری ئۆتۆماتیکی بۆ داواکردنەوە بەپێی ئاستەکانی کەمترین/زۆرترین

**ئامانجی ژمێریاری**: سیستەمی بەڕێوەبردنی کۆگا دڵنیایی دەدات لە بەهاکدانانی وردی کۆگا بۆ ڕاپۆرتی تەرازووی گشتی (balance sheet)، هەژمارکردنی دروستی تێچووی کاڵای فرۆشراو (COGS)، و تۆماری وردبینی تەواو بۆ هەموو مامەڵەکانی کۆگا.

---

## پێداویستییەکانی سیستەم

### ڕێکخستنی بەرهەم
- **بەرهەمە کۆگاییەکان**: بەرهەمەکان دەبێت وەک جۆری "Storable Product" ڕێکبخرێن بۆ چاودێریکردنی کۆگا
- **ڕێگاکانی بەهاکدانان**: هەڵبژاردن لە نێوان FIFO, LIFO, AVCO (تێچووی ناوەند)، یان Standard Price
- **ڕێکخستنی هەژمار**: هەژماری سەرمایەی کۆگا، خەرجی COGS، و ئیلتزامی هاتنی کۆگا (stock input) دەبێت ڕێکبخرێن
- **پەیکەری شوێن**: شوێنەکانی کۆگا و شوێنەکانی کڕیار/دابینکەر دەبێت دابمەزرێن

---

## لە کوێ دەیدۆزیتەوە لە ڕووکارەکەدا

بڕۆ بۆ کۆمەڵەی **کۆگا (Inventory)** لە ڕێڕەوی سەرەکی

سیستەمی بەڕێوەبردنی کۆگا ئەم بەشە سەرەکییانە لەخۆدەگرێت:
- **بەرهەمەکان**: بەڕێوەبردنی کەتەلۆگی بەرهەم و ڕێکخستنەکانی کۆگا
- **بڕەکانی کۆگا (Stock Quantities)**: بینینی ئاستی ئێستای کۆگا بەپێی بەرهەم و شوێن
- **جوڵەکانی کۆگا (Stock Movements)**: چاودێریکردنی هەموو جوڵەکان و دروستکردنی دەستکاری دەستی
- **شوێنەکانی کۆگا**: بەڕێوەبردنی شوێنەکانی کۆگا و هەرەمی شوێن
- **وەجبەکان (Lots)**: چاودێریکردنی کۆدی وەجبە، بەرواری بەسەرچوون، و جوڵەکانی وەجبە
- **یاساکانی داواکردنەوە**: دانیانی پێشنیاری ئۆتۆماتیکی بۆ داواکردنەوە
- **ڕاپۆرتەکان**: دەستگەیشتن بە ڕاپۆرتەکانی بەها، تەمەن، سوڕانەوە، و شوێنپێهەڵگرتن

**تێبینی**: دوگمەی Help/Docs لە سەرەوە ڕێبەرە پەیوەندیدارەکان دەکاتەوە بۆ هەر بەشێک.

---

## کارە سەرەکییەکانی کۆگا

### بەڕێوەبردنی بڕەکانی کۆگا

بڕۆ بۆ **کۆگا ← بڕەکانی کۆگا**

**بینینی کۆگای ئێستا**:
- **بەرهەم**: ناوی بەرهەم و SKU
- **شوێن**: کۆگا/شوێن
- **بڕ**: بڕی بەردەست لە ئێستادا
- **هەڵگیراو (Reserved)**: بڕی هەڵگیراو بۆ داواکارییەکان
- **بەردەست (Available)**: بڕی بەردەست بۆ داواکاری نوێ (بڕ - هەڵگیراو)
- **وەجبە (Lot)**: کۆدی وەجبەی پەیوەندیدار (ئەگەر کارا بێت)

### دروستکردنی جوڵەی کۆگا

بڕۆ بۆ **کۆگا ← جوڵەکانی کۆگا**

**هەنگاوی ١: دروستکردنی جوڵەی نوێ**
کلیک لەسەر **Create Stock Movement** بکە

**هەنگاوی ٢: زانیاری بنەڕەتی**
- **بەرهەم**: بەرهەمەکە هەڵبژێرە
- **لە شوێنی**: شوێنی سەرچاوە
- **بۆ شوێنی**: شوێنی مەبەست
- **بڕ**: بڕی گواستنەوە (ژمارەی موجەب)
- **جۆری جوڵە**: وەرگرتن، ناردن، گواستنەوەی ناوخۆیی، یان دەستکاری (Adjustment)

**هەنگاوی ٣: پشتڕاستکردنەوە**
- **Draft**: پاشەکەوت وەک ڕەشنووس
- **Confirm**: داخستنی وردەکارییەکان
- **Process**: تەواوکردنی جوڵەکە و نوێکردنەوەی ئاستەکانی کۆگا

### ڕێکخستنی کۆگای بەرهەم

بڕۆ بۆ **کۆگا ← بەرهەمەکان**

**ڕێکخستنەکانی کۆگا**:
- **ڕێگای بەهاکدانان**: هەڵبژاردنی FIFO, LIFO, AVCO, یان Standard Price
- **چاودێریکردنی وەجبە**: کاراکردنی چاودێری وەجبە بۆ شوێنپێهەڵگرتن
- **هەژماری دیفاڵتی کۆگا**: هەژماری سەرمایە بۆ بەهای کۆگا
- **هەژماری داهات**: هەژماری داهات بۆ فرۆشتن
- **هەژماری خەرجی**: هەژماری COGS بۆ دانپێدانانی تێچوو

---

## ڕوونکردنەوەی ڕێگاکانی بەهاکدانان

### FIFO (یەکەم-هاتوو، یەکەم-ڕۆیشتوو)
- **چەمک**: کۆنترین کۆگا یەکەم جار بەکاردێت (دەفرۆشرێت)
- **کەڵکی هەیە بۆ**: کاڵای تێکچوو، ژینگەی بەرزبوونەوەی تێچوو
- **ژمێریاری**: چینەکانی تێچووی ورد دروست دەکات بە بەرواری کڕین و تێچووەکان

### LIFO (دوا-هاتوو، یەکەم-ڕۆیشتوو)
- **چەمک**: نوێترین کۆگا یەکەم جار بەکاردێت
- **کەڵکی هەیە بۆ**: کاڵای تێکنەچوو، ژینگەی دابەزینی تێچوو
- **ژمێریاری**: نوێترین تێچووەکان بەکاردەهێنێت بۆ هەژمارکردنی COGS

### AVCO (تێچووی ناوەند)
- **چەمک**: هەژمارکردنی تێچووی ناوەندی کێشکراو
- **کەڵکی هەیە بۆ**: کاڵا گشتییەکان، ژینگەی تێچووی جێگیر
- **ژمێریاری**: تێچووی ناوەند دووبارە هەژمار دەکاتەوە لەگەڵ هەر وەرگرتنێک

---

## ڕاپۆرتکردنی کۆگا

### ڕاپۆرتەکانی بەها

بڕۆ بۆ **کۆگا ← ڕاپۆرتەکان ← ڕاپۆرتی بەها**

**ناوەڕۆکی ڕاپۆرت**:
- **وردەکاری بەرهەم**: ناو، SKU، ڕێگای بەهاکدانان
- **بڕەکان**: لە-دەست، هەڵگیراو، بەردەست
- **تێچووی یەکە**: تێچووی ئێستا بۆ هەر یەکەیەک
- **کۆی بەهاکان**: بەهای درێژکراوەی کۆگا
- **مطابقەی GL**: بەراوردکاری لەگەڵ تۆمارە ژمێریارییەکان

### ڕاپۆرتەکانی تەمەن (Aging)

بڕۆ بۆ **کۆگا ← ڕاپۆرتەکان ← ڕاپۆرتی تەمەن**

**تایبەتمەندییەکانی شیکاری**:
- **بەها بەپێی تەمەن**: بەهای کۆگا لە هەر ماوەیەکدا (٠-٣٠، ٣١-٦٠، هتد)
- **پێوەرەکانی سوڕانەوە**: فریکوێنسی جوڵە
- **ئاگاداری بەسەرچوون**: ئەو وەجبانەی نزیکن لە بەسەرچوون

- **[پسوڵەکانی کڕیار](customer-invoices.md)**: دەرچوونی ئۆتۆماتیکی کۆگا کاتێک پسوڵەکان پەسەند دەکرێن
- **[پارەدانەکان](payments.md)**: لایەنە داراییەکانی مامەڵەکانی کۆگا
