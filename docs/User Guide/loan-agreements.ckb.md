
# ڕێنمایی تەواو بۆ ڕێکەوتنی قەرز

ئەم ڕێنماییە ڕوون دەکاتەوە چۆن قەرز بەڕێوە ببرێت لە سیستەمەکەدا لە سەرەتای دامەزراندنەوە تا کارکردنی ڕۆژانە، لەوانەش خشتەکانی قسەط، کۆکردنەوەی سۆد، پارەدانەکان و دووبارە هاوپۆلکردنی بەشی کاتی. بۆ ژمێرەوان و نەژمێرەوان هەردووکیان نووسراوە و بە بنەمایی یاساکانی ژمێرەوانی دوو-تەر کار دەکات.

---

## ڕێکەوتنی قەرز چییە؟
ڕێکەوتنی قەرز نیشاندەری ئەوەیە کە یان کۆمپانیاکەت قەرز وەرگرتووە (دەربەدە/Payable) یان تۆ پارە بە هاوبەشێک داوە (وەرگر/Receivable). هەر قەرزێک خۆی خۆی دراو/دراوی خۆی هەیە، بڕی سەرەکی (Principal)، مەرجی سۆد و خشتەی قسەطی خۆی.

خاسیەتە سەرەکییەکان:
- جۆری قەرز: وەرگر (تۆ قەرز دەدەیت) یان دەربەدە (تۆ قەرز وەردەگری)
- هاوبەش: دژی کارەکەت/کڕیار/فرۆشیار یان هاوبەشێکی تر
- دراو: دراوی بەڵگەنامە بۆ بڕی سەرەکی و هەموو بڕە خشتەکان
- بەروارەکان:
  - بەرواری قەرز: کاتێک قەرزەکە دروست دەبێت یان ناسراو دەبێت
  - دەستپێکی قسەط: کاتێک خشتەی قسەط دەست دەکات
  - بەرواری تەواوبوون: هەڵبژاردەیەتی
- بڕی سەرەکی: بڕی سەرەتایی قەرز
- ماوە (مانگ): ژمارەی قسەطەکان لە خشتەکەدا
- شێوازی خشتەکردن:
  - ئەنویتی (Annuity): قسەطی گشتی یەکسان؛ بەشی سۆد کەم دەبێت و بەشی سەرەکی زیاتر دەبێت بە کاتدا
  - سەرەکی هاوسەنگ (Straight Line Principal): هەر مانگ یەکسانی سەرەکی؛ قسەطی گشتی بەھاوەخت کەم دەبێت
- ڕێژەی سۆدی ساڵانەی ناوی (nominal): بۆ ژمێری خشتەکە بەکار دەچێت (گەڕانەوەی EIR جیاوازە)
- پشتگیری EIR (ڕێژەی سۆدی کاریگەر): هەڵبژاردەیەتی بۆ دووبارە-ژمێرین بە بنەمای EIR

دۆخ:
- ڕەشنووس/درافت: دەگۆڕدرێت
- مُرحەڵ/کاریگەر (وەک جۆرێکی جێبەجێکردن): لە هەنگاوەکاندا بەکاردێت؛ کارەکان قیدە ژورنالی ڕەشنووس دەکەن

---

## لە کەی ئەوە لە UI ــدا بدۆزیتەوە
- ژمێرەوانی → ڕێکەوتنی قەرز: دروستکردن، پیشاندانی وردەکاری و بەڕێوەبردن
- لە لاپەڕەی بینینی قەرز، دوگمەکانی کرداری ئەمانەن: خشتە-ژمێرە، دووبارە-ژمێرە EIR، کۆکردنەوەی سۆد، پۆستی پارەدان، و دووبارە هاوپۆلکردنی بەشی کاتی
- خشتەی پەیوەندیدار لە بینینەکەدا:
  - تۆمارەکانی خشتە: پلانێکی قسەط بە مانگان (بڕی پارەدان، بەشی سەرەکی، بەشی سۆد، ماوە)
  - هێڵەکانی خەرجی/فی: خەرجیە یەکجارەکان یان مانگانە کە دەکرێت رەسمەلە بکرێن
  - گۆڕانکارییەکانی ڕێژە: گۆڕانی بەروارداری ڕێژەی ساڵانە بۆ خشتەکە

ئاماژە: دوگمەی یارمەتی/بەڵگەنامەکانی سەرپەڕە هەمان ئەم ڕێنماییەی دەکاتەوە.

---

## دروستکردنی ڕێکەوتنی قەرز
لە Create بەکاربەرەوە بۆ دیاریکردنی بنچینەکان:
1) هاوبەش و دراو: هاوبەشت هەڵبژێرە و دراوی قەرزەکە
2) بەروار و ماوە: بەرواری قەرز، دەستپێکی قسەط و ژمارەی مانگەکان دابنێ؛ تەواوبوون هەڵبژاردەیەتییە
3) بڕەکان: بڕی سەرەکی بنووسە (ماوەی سەرەکی بەشەوە خۆکارانە چاودێری دەکرێت)
4) مەرج: شێوازی خشتە هەڵبژێرە و ڕێژەی سۆدی ساڵانە؛ EIR چالاک بکە ئەگەر بەکاری دەهێنیت
5) پاشەکەوت

ئەنجام: قەرزێک لە دۆخی ڕەشنووس دروست دەبێت. ئێستا دەتوانیت خشتەکە ژمێرە و بە کۆکردنەوەی سۆد و پارەدان بەردەوام بیت.

---

## ژمێرەکردنی خشتەی پارەدان
ئامانج: دروستکردنی خشتەی ئەمۆرتیزەکردن کە بۆ کۆکردنەوەی سۆد و پارەدانەکان بەکاردێت.

هەنگاوەکان:
1) قەرزەکە بکەرەوە → لاپەڕەی بینین
2) کرتە لە Compute Schedule بکە
3) سیستەمەکە هەر تۆماری پێشووی خشتە دەسڕێتەوە و خشتەی تەواوەتی دووبارە دەژمێرێت بە بنەمای:
   - بڕی سەرەکی، ماوە، بەرواری دەستپێک
   - شێوازی خشتە (ئەنویتی یان سەرەکی هاوسەنگ)
   - ڕێژەی ساڵانەی نوێ یان هەنووکەیی بەپێی مانگ (خشتەی گۆڕانی ڕێژە)

ئەنجام: خشتەی تۆمارەکان لە هەر قسەطێکدا ڕیزدەکات:
- بەرواری قسەط
- بڕی پارەدان
- بەشی سەرەکی
- بەشی سۆد
- ماوەی دواوە

تێبینی:
- گۆڕانی ڕێژە: ئەگەر دانراون، ڕێژەی ساڵانە لە مانگی کاریگەرەکەوە دەگۆڕن.
- دراو: هەموو بڕەکانی خشتە بە دراوی قەرزەکەن.

---

## دووبارە-ژمێرین بەکاریگەر (EIR)‏ (هەڵبژاردەیەتی)
ئەگەر EIR چالاکە، دەتوانیت دووبارە دابەشکردنی سۆد و سەرەکی لەسەر بنەمای ڕێژەی کاریگەر بکەیت:
1) قەرزەکە بکەرەوە → لاپەڕەی بینین
2) کرتە لە Recalculate EIR بکە
3) سیستەمەکە دابەشی سۆد دووبارە دەژمێرێت بە بنەمای EIR لەسەر بڕی هەڵگرتن لە ماوەی خشتەی ماوە

بۆچی: EIR زۆرجار لە ژێر IFRS/GAAP دا داواکراوە بۆ هەندێک قەرز/خەرجی، بۆ ناسینەوەی سۆد بە ڕێژەی ناوخۆی پارەگەڕان لە ژیانی قەرزدا.

---

## کۆکردنەوەی سۆد (ڕۆژانە/مانگانە)
ئامانج: ناسینەوەی خەرجی سۆد (بۆ قەرزی دەربەدە) یان داهێنانی سۆد (بۆ قەرزی وەرگر) تا قسەطێکی دیاریکراو.

هەنگاوەکان:
1) قەرزەکە بکەرەوە → لاپەڕەی بینین → Accrue Interest
2) دیالۆگەکە پڕبکەوە:
   - ژۆرنال: لە کوێ قەید بکرێت
   - هەژماری خەرجی/داهێنانی سۆد
   - هەژماری سۆدی کۆکراوە (دارایی/قەرز)
   - ژمارەی قسەط: کەی دیاریکە
3) دڵنیابوون

قەیدی ژمێرەوانی (نمونە بۆ قەرزی دەربەدە):
- مدین: خەرجی سۆد
- دائن: سۆدی کۆکراوە

تێبینی:
- بڕی کۆکردنەوەی سۆد یەکسانە بە بەشی سۆدی ئەو قسەطەی هەڵتبژاردووە.
- بەرواری قەید یەکسانەیە بە بەرواری قسەط.
- بۆ قەرزی وەرگر، هەژمارەکان بە ئاراستەی گونجاو دەگۆڕن (داهێنان لە جیاتی خەرجی).

---

## پۆستی پارەدان (جوڵەی نەقد)
ئامانج: توماركردنی پارەدانی قسەطێک (یان وەرگرتن) بۆ قسەطێکی دیاریکراو، کە سۆدی کۆکراوە و سەرەکی دەدەستێنێت.

هەنگاوەکان:
1) قەرزەکە بکەرەوە → لاپەڕەی بینین → Post Repayment
2) دیالۆگ پڕبکەوە:
   - ژۆرنال: بانك یان نەقد
   - هەژماری بانك: هەژماری GL ی بانك/نەقد
   - هەژماری قەرز: هەژماری سەرەکی قەرز (دەربەدە یان وەرگر)
   - هەژماری سۆدی کۆکراوە: هەمان ئەوەی لە کۆکردنەوەدا بەکاریان هێنا
   - ژمارەی قسەط: کام قسەط پارەدەدەیت
3) دڵنیابوون

قەیدی ژمێرەوانی (نمونەی قەرزی دەربەدە):
- مدین: سۆدی کۆکراوە (بۆ سۆدی ئەو ماوە)
- مدین: قەرزی دەربەدە (بەشی سەرەکی)
- دائن: بانك (کۆی پارەدان)

بۆ قەرزی وەرگر، ئاراستەکە پێچەوانە دەبێت (مدین: بانك؛ دائن: سۆدی کۆکراوە؛ دائن: قەرزی وەرگر).

گرنگ: پارەدانی قەرز لە ڕێگەی مۆدیولی Payments تومار ناکرێت. کرداری Post Repayment لە لاپەڕەی قەرز بەکاربهێنە. ئەم قەیدانە دواتر لە هاوسەنگکردنی بانکییدا لەگەڵ هاتنەژورەوەی بانکدا هاوسەنگ دەکرێن.

---

## دووبارە هاوپۆلکردنی بەشی کاتی (لە کۆتایی مانگ/ساڵ زۆرجار)
ئامانج: گواستنەوەی ئەو بەشەی قەرزی ماوە-درێژ کە لە ماوەی N مانگی داهاتوودا دەسەنگێت بۆ هاوپۆلی ماوە-کورت.

هەنگاوەکان:
1) قەرزەکە بکەرەوە → لاپەڕەی بینین → Reclassify Current Portion
2) دیالۆگ پڕبکەوە:
   - ژۆرنال
   - هەژماری ماوە-درێژ (وەک: قەرزەکان — ماوە درێژ)
   - هەژماری ماوە-کورت (وەک: بەشی کاتی قەرزی ماوە-درێژ)
   - مانگ (وەک: 12)
   - وەک بەرواری
3) دڵنیابوون

قەیدی ژمێرەوانی (نمونەی قەرزی دەربەدە):
- مدین: قەرزی ماوە-درێژ
- دائن: بەشی کاتی قەرزی ماوە-درێژ

بڕەکە یەکسانەیە بە کۆی بەشی سەرەکی خشتەکە لە N مانگی داهاتوودا لە وەک-بەروارەوە.

---

## هێڵی خەرجی و رەسمەلەکردن
دەتوانیت خەرجیەکان (وەک خەرجی دامەزراندن) تومار بکەیت بە رەسمەلەکردن یان نا.
- ئەگەر رەسمەلە بکرێن، کاریگەری لەسەر بڕی هەڵگرتن هەیە و ڕەنگە لە EIR ـدا کاریگەر بن
- ئەوانەی نا-رەسمەلەکراو زۆرجار کاتێک دروست دەبن خەرجی دەبن

---

## چەند-دراوی
- دراوی قەرز جێگیرە و بۆ هەموو ژمێرەکانی خشتە بەکاردهێنرێت
- بڕە پارەییەکان وەک Money object بە دراوی خۆی قەرز پاشەکەون
- لە کاتێکدا قەیدە ژورنالییەکان پۆست دەکرێن، گۆڕانی بۆ دراوی بنچینەی کۆمپانیا بە بنەمای ڕێکخستنەکانت دەکرێت

---

## تێکەڵبوون لەگەڵ مۆډیولەکانی تر
- Payments: Payments بەکار مەهێنە بۆ پارەدانی قەرز. Payments تەنها بۆ شەڕەی قەرزەکانی کڕیار/فرۆشیار و پێشپارەی هاوبەشە. جوڵەکانی نەقدی قەرز بە کرداری Post Repayment ـی قەرز تومار دەکرێن.
- وەسڵی بانك و هاوسەنگکردن: قەیدەکانی پارەدان لە لێدوانی بانکت دەردەکەون و لە کاتی هاوسەنگکردن لەگەڵ وەسڵی بانکی هاوردەکراو دادەمەزرێن.
- بەروارە قوفڵکراوەکان: بەرواری قەرز، کۆکردنەوەی سۆد و پارەدان ڕێساکانی قوفڵکردن ڕێز دەگرن. ئەگەر ماوەیەک قوفڵ کراوە، پێویستە بەروارێکی گونجاو بەکاربهێنیت یان قوفڵەکە بگۆڕیت.

---

## باشترین کارەکان
- خشتەکە دوای دروستکردن بە فۆری ژمێرە، و هەرکات مەرج یان گۆڕانی ڕێژە زیاد دەکەیت
- هەر مانگ سۆد کۆبکەرەوە؛ چەند-مانگەکانی جیاواز لە یەک قەید مەکەرەوە تا ئەوەی یاسای ناوچەکەت ڕێگە بدات
- لە کۆتایی ماوە، Reclassify Current Portion بەکاربهێنە بۆ پیشاندانی دروستی ماوە-کورت دژ بە ماوە-درێژ
- هەمان هەژمارەکان بۆ سۆد، کۆکراوەکان و سەرەکی بەکارببە بۆ ڕاپۆرتی پاک و ڕوون
- سەرچاوەی ڕوون بخەرە ناو قەیدە ژورنالییەکان (سیستەمەکە ئاماژەی پێکهاتووی وەک LOAN-INT/ID/SEQ، LOAN-PAY/ID/SEQ تێدایە)

## پرسیار و وەڵام (FAQ)
پ: دەتوانم پارەدانی قەرز بە Payments تومار بکەم؟
و: نا. کرداری Post Repayment لە لاپەڕەی ڕێکەوتنی قەرز بەکاربهێنە. Payments بۆ شەڕەی قەرزەکانی کڕیار/فرۆشیار و پێشپارەی هاوبەشە.

پ: دەتوانم بەشێکی قسەط بە جیاجیا پارەبدەم؟
و: ئێستا UI بە پێی قسەط پۆست دەکات. بۆ پیشاندانی پارەدانی بەش بە بەش، دەتوانیت خشتەکە بشکێنیت یان ڕێکخستنی دەستی تۆمار بکەیت؛ لە داهاتوودا ڕەنگە UI پشتگیری پارەدانی بەش بە بەش بکات.

پ: سۆدی گۆڕاو چۆن؟
و: لە Rate Changes بەکاربهێنە بۆ گۆڕینی ڕێژەی ساڵانە لە مانگی کاریگەرەوە؛ خشتەکە دووبارە ژمێرە بۆ جێبەجێکردنی گۆڕانەکان.

پ: کاتێک EIR بەکاربهێنم؟
و: کاتێک کە یاسای بنچینەیی یان یاساکانی ژمێرەوانی داوای دەکەن بۆ ناسینەوەی سۆد بە عایدی کاریگەر لە ژیانی قەرزدا، بەتایبەتی کاتێک خەرجیەکان رەسمەلە دەکرێن.
