# ڕێنماڵی سیستەمی مەرجەکانی پارەدان

## پێشەکی

سیستەمی مەرجەکانی پارەدان ڕێگەت پێدەدات بۆ دیاریکردنی مەرجە ستاندارەکانی پارەدان بۆ پسولە و فاتورە فرۆشیارییەکانت. کاتێک مەرجێکی پارەدان بەکاردەهێنریت لەسەر بەڵگەنامەیەک، سیستەمەکە بە شێوەیەکی خۆکارانە بەرواری پێویستی پارەدان هەژمار دەکات و قیستەکانی پارەدان درووستدەکات.

## دەستپێکردن

### ١. درووستکردنی مەرجە باوەکانی پارەدان

کاتێک سیستەمەکە یەکەم جار دادەمەزرێنیت، دەتوانیت مەرجە باوەکانی پارەدان دروست بکەیت بەکارهێنانی:

```bash
php artisan db:seed --class=PaymentTermsSeeder
```

ئەمە ١٦ مەرجی پارەدانی باو درووست دەکات بۆ هەموو کۆمپانیاکان، کە بریتین لە:

- **فەوری** - پارەدان فەوری لەدوای وەرگرتن
- **مەرجە تازەکان** - Net 15, Net 30, Net 45, Net 60
- **داشکاندنی پارەدانی زووتر** - 2% 10 صافي 30، 1% 15 صافي 30، 3% 7 صافي 21
- **مەرجەکانی کۆتایی مانگ** - EOM، EOM + 15، EOM + 30
- **مەرجەکانی ڕۆژی مانگ** - یەکی مانگی داهاتوو، ١٥ی مانگی داهاتوو
- **مەرجەکانی قیست** - دابەشکردنی 50/50، 30-60-90 ڕۆژ، پارەدانی چارەکانە

### ٢. بەڕێوەبردنی مەرجەکانی پارەدان

بڕۆ بۆ **ڕێکخستنەکان → مەرجەکانی پارەدان** بۆ:
- بینینی مەرجە هەبووەکانی پارەدان
- درووستکردنی مەرجی نوێی پارەدان
- دەستکاریکردنی مەرجە هەبووەکانی پارەدان
- چالاک/ناچالاککردنی مەرجەکانی پارەدان

## بەکارهێنانی مەرجەکانی پارەدان

### لە پسولەکان

١. بڕۆ بۆ **ژمێرایەتی → پسولەکان → درووستکردنی پسولە**
٢. پڕکردنەوەی زانیارییە سەرەکییەکانی پسولە
٣. هەڵبژاردنی **مەرجی پارەدان** لە لیستی دادەبەزێ
٤. سیستەمەکە ئەم مەرجە بەکاردەهێنێت بۆ هەژمارکردنی قیستەکان کاتێک پسولەکە پەسند دەکرێت

### لە پسولەکانی فرۆشیار

١. بڕۆ بۆ **ژمێرایەتی → پسولەکانی فرۆشیار → درووستکردنی پسولەی فرۆشیار**
٢. پڕکردنەوەی زانیارییە سەرەکییەکانی پسولە
٣. هەڵبژاردنی **مەرجی پارەدان** لە لیستی دادەبەزێ
٤. سیستەمەکە ئەم مەرجە بەکاردەهێنێت بۆ هەژمارکردنی قیستەکان کاتێک پسولەکە پەسند دەکرێت

### بینینی مەرجەکانی پارەدان

مەرجەکانی پارەدان لە لیستی بینینەکان پیشان دەدرێن:
- **لیستی پسولەکان**: ناوی مەرجی پارەدان لە ستوونێکی تایبەت پیشان دەدات
- **لیستی پسولەکانی فرۆشیار**: هەم مەرجی پارەدان و هەم بەرواری حیسابکراو پیشان دەدات

## چۆنیەتی کارکردنی مەرجەکانی پارەدان

### ١. درووستکردنی بەڵگەنامە
- بەکارهێنەر مەرجێکی پارەدان هەڵدەبژێرێت کاتێک پسولە یان پسولەی فرۆشیار درووست دەکات
- مەرجی پارەدان لەگەڵ بەڵگەنامەکە هەڵدەگیرێت

### ٢. پەسندکردنی بەڵگەنامە
- کاتێک بەڵگەنامەکە پەسند دەکرێت، سیستەمەکە مەرجی پارەدان دەخوێنێتەوە
- قیستەکان بەپێی ڕێکخستنی مەرجی پارەدان هەژمار دەکات
- تۆمارەکانی قیستی پارەدان بە شێوەیەکی خۆکارانە درووست دەکات

### ٣. پرۆسەی پارەدان
- کاتێک پارەدانەکان تۆمار دەکرێن، لەسەر قیستەکان جێبەجێ دەکرێن
- دۆخی قیستەکان بە شێوەیەکی خۆکارانە نوێ دەکرێنەوە:
  - **چاوەڕوان** ← **بەشێك پارەداو** ← **پارەداو**
  - **هەڵوەشێنراوە** (ئەگەر بەڵگەنامەکە هەڵوەشێنرابێت)

### ٤. شوێنکەوتنی پارەدان
- بەکارهێنەران دەتوانن پێشکەوتنی پارەدان بشوێن کەون لە ڕێگەی ڕووکارە هەبووەکانی پارەدان
- قیستە دواکەوتووەکان بە شێوەیەکی خۆکارانە ناسراو دەبن
- ڕاپۆرتی بەتەمەنی پارەدان زانیارییەکانی قیست بەکاردەهێنێت

## جۆرەکانی مەرجی پارەدان

### ١. پارەدانی فەوری
- پارەدان فەوری لەدوای وەرگرتن پێویستە
- قیستێک درووست دەکات کە بەرواری پێویستی = بەرواری بەڵگەنامە

### ٢. مەرجە تازەکان
- پارەدان لە ماوەی ژمارەیەکی دیاریکراو ڕۆژ پێویستە
- نموونە: Net 30 = پارەدان لە ٣٠ ڕۆژ پێویستە
- قیستێک درووست دەکات لەگەڵ بەرواری هەژمارکراو

### ٣. داشکاندنی پارەدانی زووتر
- داشکاندن بەردەستە ئەگەر لە ماوەی داشکاندن پارەبدرێت
- نموونە: 2% 10، صافي 30 = داشکاندنی 2% ئەگەر لە ١٠ ڕۆژ پارەبدرێت، ئەگەر نا صافي ٣٠
- قیستێک درووست دەکات لەگەڵ زانیارییەکانی داشکاندن

### ٤. کۆتایی مانگ
- پارەدان لە کۆتایی مانگ پێویستە (+ ڕۆژانی زیادە بەنارەوە)
- نموونە: EOM + 15 = پارەدان ١٥ ڕۆژ لەدوای کۆتایی مانگ پێویستە
- هەژمارکردنی کۆتایی مانگ بە شێوەیەکی خۆکارانە بەڕێوە دەبات

### ٥. ڕۆژی مانگ
- پارەدان لە ڕۆژێکی دیاریکراوی مانگی داهاتوو پێویستە
- نموونە: ١٥ی مانگی داهاتوو = پارەدان لە ١٥ی مانگی داهاتوو پێویستە
- گواستنەوەی مانگ و ساڵە دوورەبوونەکان بەڕێوە دەبات

## باشترین ڕێگەکان

### ١. ستاندارکردن
- بەکارهێنانی مەرجی پارەدانی یەکگرتوو بۆ کڕیار/فرۆشیارە هاوشێوەکان
- دانانی مەرجە بنەڕەتییەکانی پارەدان لەسەر تۆمارەکانی هاوبەش
- پێداچوونەوە و نوێکردنەوەی مەرجەکان بە بەردەوامی

### ٢. بەڕێوەبردنی جۆڵەی پارە
- بەکارهێنانی مەرجی کورتتر بۆ جۆڵەی پارەی باشتر
- پێشکەشکردنی داشکاندنی پارەدانی زووتر بۆ هاندانی پارەدانی خێراتر
- ئەندیشەکردن لە مەرجەکانی قیست بۆ بڕە گەورەکان

### ٣. پەیوەندییەکانی کڕیار
- ڕوونکردنەوەی مەرجەکانی پارەدان بە کڕیارەکان
- بێ گۆڕانکاری بوون لە جێبەجێکردنی مەرجەکان
- ئەندیشەکردن لە مێژووی پارەدانی کڕیار کاتێک مەرجەکان دادەنرێن

### ٤. بەڕێوەبردنی فرۆشیار
- گۆڕانکاری لەسەر مەرجی پارەدانی باش لەگەڵ فرۆشیارەکان
- سوودوەرگرتن لە داشکاندنی پارەدانی زووتر کاتێک گونجاوە
- چاودێریکردنی مەرجەکانی پارەدانی فرۆشیار بۆ پلانی جۆڵەی پارە

## وردەکارییە تەکنیکییەکان

### پێکهاتەی بنکەی زانیاری
- `payment_terms` - پێناسەکانی سەرەکی مەرجی پارەدان
- `payment_term_lines` - ڕێکخستنەکانی قیستی تاکەکەسی
- `payment_installments` - قیستە درووستکراوەکان بۆ بەڵگەنامەکان
- `payment_installment_links` - بەستەرەکان نێوان پارەدان و قیستەکان

### هەژمارکردنە خۆکارانەکان
- بەرواری پێویست بەپێی بەرواری بەڵگەنامە و ڕێکخستنی مەرج هەژمار دەکرێت
- بڕی قیستەکان لەگەڵ خولاندنەوەی گونجاو هەژمار دەکرێت
- نرخی ئاڵوگۆڕ بۆ دۆخەکانی زیاتر لە یەک دراو وەرگیراو دەبێت
- بەرواری کۆتایی داشکاندن بە شێوەیەکی خۆکارانە هەژمار دەکرێت

### خاڵەکانی یەکگرتنەوە
- **درووستکردنی پسولە/پسولەی فرۆشیار** - هەڵبژاردنی مەرجی پارەدان
- **پەسندکردنی بەڵگەنامە** - درووستکردنی قیست
- **تۆمارکردنی پارەدان** - نوێکردنەوەی دۆخی قیست
- **ڕاپۆرتکردن** - شیکاری بەتەمەنی پارەدان و جۆڵەی پارە

## چارەسەرکردنی کێشە

### مەرجەکانی پارەدان پیشان نادرێن
- دڵنیابوون لەوەی مەرجەکانی پارەدان چالاکن (`is_active = true`)
- پشکنین ئەوەی مەرجەکانی پارەدان بۆ کۆمپانیای ئێستا هەبن
- ڕاکردنی `php artisan db:seed --class=PaymentTermsSeeder` بۆ درووستکردنی مەرجە بنەڕەتییەکان

### قیستەکان درووست نابن
- پشتڕاستکردنەوەی ئەوەی بەڵگەنامەکە مەرجێکی پارەدانی هەیە
- پشکنین ئەوەی بەڵگەنامەکە پەسند کراوە (ڕەشنووس نییە)
- دڵنیابوون لەوەی هێڵەکانی مەرجی پارەدان بە درووستی ڕێکخراون

### بەرواری پێویستی هەڵە
- پێداچوونەوەی ڕێکخستنی هێڵی مەرجی پارەدان
- پشکنینی درووستی بەرواری بەڵگەنامە
- پشتڕاستکردنەوەی جۆری مەرجی پارەدان و ڕێگەی هەژمارکردن

## پشتگیری

بۆ یارمەتی زیاتر لەگەڵ سیستەمی مەرجەکانی پارەدان:
١. پشکنینی ڕووکاری ڕێکخستنەکان ← مەرجەکانی پارەدان
٢. پێداچوونەوەی ڕێکخستنە هەبووەکانی مەرجی پارەدان
٣. تاقیکردنەوە لەگەڵ بەڵگەنامەی نموونە لە ژینگەی گەشەپێدان
٤. ڕاوێژکردن لەگەڵ بەڵگەنامەی تەکنیکی بۆ ڕێکخستنە پێشکەوتووەکان
