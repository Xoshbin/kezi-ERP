# پسووڵەکانی فرۆشیار: بەڵگەنامەی کڕین و ناسینەوەی تێچوون

ئەم ڕێنمایە ڕوونکردنەوەی چۆنیەتی دروستکردن، بەڕێوەبردن و پرۆسێسکردنی پسووڵەکانی فرۆشیار لە سیستەمی ژمێریاریدا دەکات. پسووڵەکانی فرۆشیار کڕینەکان لە دابینکەرەکان بەڵگەدەکەن و دڵنیایی ناسینەوەی گونجاوی تێچوون و شوێنکەوتنی هەژمارەکانی پارەدانی دەکەن.

---

## پسووڵەی فرۆشیار چییە؟

پسووڵەی فرۆشیار بەڵگەنامەیەکی بازرگانییە کە:

-   **کڕینەکان تۆمار دەکات**: کاڵا یان خزمەتگوزارییە وەرگیراوەکان بەڵگە دەکات
-   **بەرپرسیارێتی دامەزرێنێت**: ئەرکی پارەدان بۆ فرۆشیار دروست دەکات
-   **تێچوونەکان دەناسێنێتەوە**: تۆمارەکانی ژمێریاری بۆ تێچوونەکان دروست دەکات
-   **پارەدانەکان شوێن دەکەوێت**: بڕە قەرزاوەکان بۆ فرۆشیارەکان دامەزرێنێت

**کاریگەری ژمێریاری**: پسووڵە پەسەندکراوەکان تێچوونەکان و هەژمارەکانی پارەدانی زیاد دەکەن، بەپەیڕەوکردنی بنەمای ژمێریاری کۆکردنەوە.

---

## پێداویستییەکانی سیستەم

### ڕێکخستنی فرۆشیار

-   **تۆمارەکانی فرۆشیار**: پرۆفایلی فرۆشیارەکان دەبێت لەگەڵ زانیاری پارەدان دروست بکرێن
-   **مەرجەکانی پارەدان**: مەرجەکانی پارەدان دەبێت بۆ ڕێککەوتنەکانی فرۆشیار ڕێکبخرێن
-   **ڕێکخستنی باج**: ڕێژەکانی باج و یاساکان دەبێت بۆ کڕینە پەیوەستەکان ڕێکبخرێن
-   **پشتگیری دراو**: دراوەکانی پسووڵە دەبێت چالاک و ڕێکخراو بن

### پێشمەرجەکان

1. **ڕێکخستنی فرۆشیار**: فرۆشیار دەبێت لە سیستەمەکەدا لەگەڵ زانیاری تەواو هەبێت
2. **ڕێکخستنی بەرهەم/خزمەتگوزاری**: کاڵا کڕراوەکان دەبێت ڕێکبخرێن ئەگەر کاتالۆگ بەکار دەهێنریت
3. **هەژمارەکانی تێچوون**: هەژمارەکانی تێچوونی گونجاو دەبێت ڕێکبخرێن
4. **مۆڵەتەکانی بەکارهێنەر**: دەسترسی بۆ تایبەتمەندییەکانی کڕین و ژمێریاری پێویستە

---

## لە کوێ لە ڕووکاری بەکارهێنەردا بیدۆزیتەوە

بڕۆ بۆ **ژمێریاری ← کڕین ← پسووڵەکانی فرۆشیار**

پسووڵەکانی فرۆشیار هەروەها دەردەکەون لە:

-   **کردەوەی دروستکردن**: دوگمەی "دروستکردن" بۆ تۆمارکردنی پسووڵەی نوێ
-   **تۆمارەکانی فرۆشیار**: بەشی پسووڵە پەیوەستەکان مێژووی کڕین نیشان دەدەن
-   **پرۆسێسی پارەدان**: پسووڵە بەردەستەکان بۆ پارەدان لە بەشی بانکدا

**ئامۆژگاری**: دوگمەی یارمەتی/بەڵگەکان لە سەرپەڕە ئەم ڕێنماییە دەکاتەوە.

---

## گەشەی ژیانی پسووڵە

### 1. دۆخی ڕەشنووس

-   پسووڵە ئامادە دەکرێت
-   دەتوانرێت دەستکاری یان سڕینەوەی بکرێت
-   کاریگەری ژمێریاری نییە
-   هێشتا بۆ پارەدان پەسەندکراو نییە

### 2. دۆخی پەسەندکراو

-   پسووڵە تەواو و پەسەندکراوە
-   تۆمارەکانی پەرتووکی ڕۆژانە دروست دەکات
-   ناتوانرێت دەستکاری بکرێت (تەنها بەڵگەی ڕاستکردنەوە)
-   ئامادەیە بۆ پارەدان

### 3. دۆخی پارەدراو

-   پارەدان بۆ فرۆشیار کراوە
-   هەژمارەکانی پارەدانی پاک کراونەتەوە
-   مامەڵەکە تەواو بووە

### 4. دۆخی هەڵوەشاندنەوە

-   پسووڵە هەڵوەشێنراوەتەوە
-   کاریگەری ژمێریاری نییە

---

## دروستکردنی پسووڵەی فرۆشیار

بڕۆ بۆ **ژمێریاری → کڕین → پسووڵەکانی فرۆشیار → دروستکردن**

### هەنگاوی 1: سەرپەڕەی پسووڵە

**فرۆشیار**: فرۆشیار لە لیستی هاوبەشەکان هەڵبژێرە
**بەرواری پسووڵە**: بەرواری پسووڵەی فرۆشیار بنووسە
**بەرواری ژمێریاری**: بەرواری ناسینەوەی ژمێریاری
**بەرواری پێویست**: بەرواری کۆتایی پارەدان (خۆکارانە لە مەرجەکانی پارەدانەوە هەژمار دەکرێت)
**دراو**: دراوی پسووڵە هەڵبژێرە (بنەڕەت دراوی فرۆشیارە)
**مەرجەکانی پارەدان**: مەرجەکانی پارەدان هەڵبژێرە ([ڕێنماڵی مەرجەکانی پارەدان](payment-terms-guide.ckb.md))
**سەرچاوەی فرۆشیار**: ژمارەی پسووڵەی فرۆشیار بنووسە
**سەرچاوە**: سەرچاوەی ناوخۆیی یان ژمارەی داواکاری کڕین

### هەنگاوی 2: هێڵەکانی پسووڵە

کلیک لە **زیادکردنی هێڵ** بکە بۆ زیادکردنی کاڵای کڕراو:

#### هێڵەکانی بەرهەم

**بەرهەم**: لە کاتالۆگی بەرهەمەکان هەڵبژێرە
**وەسف**: خۆکارانە لە بەرهەمەوە پڕ دەکرێتەوە (دەتوانرێت دەستکاری بکرێت)
**بڕ**: ژمارەی یەکە کڕراوەکان
**نرخی یەکە**: تێچوونی هەر یەکەیەک
**باج**: ڕێژەکانی باجی پەیوەست هەڵبژێرە (باجی کاڵا، باجی فرۆشتن)
**هەژمار**: هەژماری تێچوون یان سامان (خۆکارانە لە بەرهەمەوە پڕ دەکرێتەوە)

#### هێڵەکانی خزمەتگوزاری

**خزمەتگوزاری**: خزمەتگوزاریەکە هەڵبژێرە
**وەسف**: خزمەتگوزاری وەرگیراو باس بکە
**کاتژمێر/یەکە**: بڕی خزمەتگوزاری
**ڕێژە**: ڕێژەی کاتژمێری یان یەکە
**باج**: باجەکانی پەیوەست هەڵبژێرە
**هەژمار**: هەژماری تێچوونی خزمەتگوزاری

#### هێڵەکانی تێچوون

**وەسف**: وەسفی تێچوون بنووسە
**بڕ**: کۆی بڕی تێچوون
**هەژمار**: هەژماری تێچوونی گونجاو هەڵبژێرە
**پۆلی سامان**: بۆ کڕینی سامان، پۆلی سامان هەڵبژێرە بۆ دروستکردنی سامان
**باج**: باجەکانی پەیوەست هەڵبژێرە

### هەنگاوی 3: باج و کۆی گشتی

**هەژماری باج**: سیستەمەکە خۆکارانە باج بەپێی ئەمانە هەژمار دەکات:

-   ڕێکخستنی باجی بەرهەم/خزمەتگوزاری
-   ڕێکخستنەکانی باجی فرۆشیار
-   یاساکانی پێگەی مالی

**کۆی گشتی پسووڵە**:

-   **کۆی لاوەکی**: کۆی هەموو بڕەکانی هێڵ پێش باج
-   **بڕی باج**: کۆی باجی هەژمارکراو (زۆرجار باجی گەڕاندنەوە)
-   **کۆی گشتی**: بڕی کۆتایی قەرزاو بۆ فرۆشیار

### هەنگاوی 4: زانیاری زیادە

**مەرجەکانی پارەدان**: دیاری دەکات کەی پارەدان پێویستە
**پێگەی مالی**: یاساکانی باج بەپێی شوێنی فرۆشیار
**پەرتووکی ڕۆژانە**: پەرتووکی ڕۆژانەی کڕین بۆ پەسەندکردن (زۆرجار خۆکارانە هەڵدەبژێردرێت)
**تێبینیەکان**: مەرجە زیادەکان یان ڕێنماییەکانی گەیاندن

## تایبەتمەندییە پێشکەوتووەکانی پسووڵە

### پسووڵەکانی دراوی جیاواز

کاتێک پسووڵەکان بە دراوی بیانی وەردەگیریت:

1. **دراوی پسووڵە**: دراوی پسووڵەی فرۆشیار بەکاربهێنە
2. **نرخی ئاڵوگۆڕ**: سیستەمەکە نرخی ئێستا یان تۆمارکردنی دەستی بەکاردەهێنێت
3. **دراوی کۆمپانیا**: سیستەمەکە بۆ تۆمارەکانی ژمێریاری ئاڵوگۆڕ دەکات
4. **جیاوازییەکانی ئاڵوگۆڕ**: لە کاتی پارەدان خۆکارانە مامەڵەی لەگەڵ دەکرێت

**نموونە**: پسووڵە وەرگرتن بۆ $2,000 دۆلاری ئەمریکی کاتێک کۆمپانیا دینار بەکاردەهێنێت

-   بڕی پسووڵە: $2,000 دۆلاری ئەمریکی
-   نرخی ئاڵوگۆڕ: 1 USD = 1,310 IQD
-   بڕی ژمێریاری: 2,620,000 دینار

### هەڵسەنگاندنی سێ لایەنە

بۆ دابینکردنی پەیوەست بە داواکاری کڕین:

1. **داواکاری کڕین**: داواکاری کڕینی ڕەسەن
2. **وەصڵی کاڵا**: پشتڕاستکردنەوەی گەیاندن
3. **پسولەی فرۆشیار**: پسولە لە فرۆشیار

سیستەمەکە دووبارە دەکاتەوە:

-   کەمیەتەکان لە نێوان بەڵگەکان دەگونجێن
-   نرخەکان لە نۆبەی بەرگەگرتندان
-   هەموو کاڵاکان لە ڕاستیدا وەرگیراون

### مامەڵەی باج

**باجی ناوەوە**: باجی پارەدراو بۆ فرۆشیارەکان (زۆرجار گەڕاندنەوە)
**باجی کەمکردنەوە**: باج کەم دەکرێتەوە لە پارەدانەکانی فرۆشیار
**باری هاوردە**: باجی زیادە لەسەر کاڵا هاوردەکراوەکان
**ئازادی لە باج**: هەندێک کڕین لە باج ئازاد بن

## پرۆسێسی پەسەندکردنی پسووڵە

### پێش پەسەندکردن - پشکنینی لیستی چاودێری

✅ **زانیاری فرۆشیار**: فرۆشیاری راست هەڵبژێردراوە
✅ **وردەکارییەکانی پسووڵە**: ژمارەی پسووڵە و بەرواری فرۆشیار دروستە
✅ **هێڵەکانی کاڵا**: هەموو کاڵا کڕراوەکان لەگەڵ بڕی راست لەخۆگیراون
✅ **نرخدانان**: تێچوونی یەکە و کۆی گشتی لەگەڵ پسووڵەی فرۆشیار دەگونجێت
✅ **باجەکان**: ڕێژەی باجی گونجاو جێبەجێ کراوە
✅ **هەژمارەکان**: هەژمارەکانی تێچوون/سامانی راست دیاری کراون
✅ **پەسەندکردن**: پەسەندکردنە پێویستەکان وەرگیراون

### پەسەندکردنی پسووڵە

1. **پاشەکەوتکردن وەک ڕەشنووس**: پاشەکەوتکردن بۆ پێداچوونەوە و پەسەندکردن
2. **پێداچوونەوە**: هەموو وردەکارییەکان لە بەرانبەر پسووڵەی فرۆشیار بپشکنەوە
3. **پەسەندکردن**: پەسەندکردنە پێویستەکان وەربگرە
4. **پەسەندکردن**: کلیک لە "پەسەندکردن" بکە بۆ تەواوکردنی پسووڵە

**⚠️ گرنگ**: دوای پەسەندکردن، پسووڵەکان ناتوانن دەستکاری بکرێن. بەڵگەی ڕاستکردنەوە بۆ چاکسازییەکان بەکاربهێنە.

### تۆمارەکانی پەرتووکی ڕۆژانەی دروستکراو

کاتێک پەسەندکرێت، سیستەمەکە دروست دەکات:

```
قەرزدار. تێچوونی کەلووپەلی ئۆفیس    500,000 دینار
قەرزدار. باجی ناوەوە                  50,000 دینار
    قەرزخواز. هەژمارەکانی پارەدانی           550,000 دینار
```

## بەڕێوەبردنی پسووڵە پەسەندکراوەکان

### پرۆسێسی پارەدان

**لە پسووڵەوە**: کلیک لە "تۆمارکردنی پارەدان" لەسەر پسووڵە
**لە پارەدانەکانەوە**: پارەدان دروست بکە و بە پسووڵە بەستەوە
**پارەدانی دەستە**: چەندین پسووڵە لە یەک مامەڵەدا پارە بدە

بۆ زیاتر دەربارەی پارەدانەکان، سەیری [ڕێنماڵی پارەدانەکان](payments.ckb.md) بکە.

### پارەدانی بەشەیی

فرۆشیارەکان ڕەنگە پارەدانی بەشەیی قبوڵ بکەن:

1. پارەدان تۆمار بکە بۆ بڕی بەشەیی
2. پسووڵە دەمێنێتەوە "بە بەشەیی پارەدراو"
3. یەکەی ماوە خۆکارانە شوێن دەکەوێت
4. چەندین پارەدان دەتوانرێت جێبەجێ بکرێت

### بەڵگەی ڕاستکردنەوە

بۆ گەڕاندنەوە، داشکاندن، یان چاکسازی:

1. کلیک لە "بەڵگەی ڕاستکردنەوە" لەسەر پسووڵەی ڕەسەن
2. کاڵاکان بۆ ڕاستکردنەوە هەڵبژێرە
3. بڕی ڕاستکردنەوە دیاری بکە
4. بەڵگەی ڕاستکردنەوە پەسەندکرە
5. بۆ هەژماری فرۆشیار جێبەجێ بکە

## پۆلەکانی کڕین و هەژمارەکان

### کڕینی کۆگا

**هەژمار**: هەژماری سامانی کۆگا
**کاریگەری**: نرخی کۆگا زیاد دەکات
**ناسینەوەی تێچوون**: کاتێک کاڵاکان دەفرۆشرێن (COGS)

### تێچوونە ڕاستەوخۆکان

**هەژمار**: هەژمارەکانی تێچوونی کارکردن
**کاریگەری**: ناسینەوەی تێچوونی فەوری
**نموونەکان**: کەلووپەلی ئۆفیس، کەرەستەی کارەبا، کرێ

### خەرجی سەرمایە

**هەژمار**: هەژمارەکانی سامانی جێگیر
**کاریگەری**: نرخی سامان زیاد دەکات
**لەناوچوون**: تێچوون لەسەر ماوەی ژیانی سامان دەناسرێتەوە

### خزمەتگوزارییەکان

**هەژمار**: هەژمارەکانی تێچوونی خزمەتگوزاری
**کاریگەری**: ناسینەوەی تێچوونی فەوری
**نموونەکان**: ڕاوێژکاری، چاکسازی، کرێی پیشەیی

## ڕێڕەوەکانی پەسندکردن

### سنووری دەسەڵاتی کڕین

**بەڕێوەبەرانی بەش**: تا 5,000,000 دینار
**بەڕێوەبەران**: تا 25,000,000 دینار
**بەڕێوەبەری دارایی**: تا 100,000,000 دینار
**بەڕێوەبەری گشتی**: لەسەروی 100,000,000 دینار

### پرۆسێسی پەسندکردن

1. **داواکار**: پسولە لە دۆخی ڕەشنووسدا درووست دەکات
2. **پێداچوونەوەی بەش**: بەڕێوەبەر پێداچوونەوە و پەسند دەکات
3. **پێداچوونەوەی دارایی**: ژمێرایەتی وردەکارییەکان دووبارە دەکاتەوە
4. **پەسندکردنی کۆتایی**: کەسی دەسەڵاتدار بۆ پەسندکردن پەسند دەکات
5. **پەسندکردن**: پسولە پەسند دەکرێت و ئامادەیە بۆ پارەدان

### جیاکردنەوەی ئەرکەکان

-   **درووستکردنی پسولە**: بەشی کڕین
-   **پەسندکردنی پسولە**: بەڕێوەبەرانی بەش
-   **دەسەڵاتی پارەدان**: بەشی دارایی
-   **جێبەجێکردنی پارەدان**: کرێکاری خەزێنە

**مەرجەکانی پارەدان**: مەرجەکانی ستاندارد پارەدان
**زانیاری باج**: ناسنامەی باج، دۆخی ئازادی
**وردەکارییەکانی بانک**: زانیاری حیسابی پارەدان

### ئادایی فرۆشیار

**مێژووی پارەدان**: شوێنکەوتنی کاتی پارەدان
**پێوەرەکانی کوالیتی**: ڕێژەکردنی کوالیتی بەرهەم/خزمەتگوزاری
**شیکردنەوەی نرخ**: بەراوردکردنی تێچوونەکان بەدرێژایی کات
**ڕێگەپێدان**: چاودێری بڕوانامە و مۆڵەتەکان

## پسووڵەکانی دووبارەبوونەوە

بۆ تێچوونە ڕێکخستنەکانی فرۆشیار:

### ڕێکخستنی پسووڵەی دووبارەبوونەوە

1. نموونەی پسووڵە دروست بکە
2. نەخشی دووبارەبوونەوە دابنێ (مانگانە، چارەکانە، هتد)
3. بەرواری دەستپێک و کۆتایی دیاری بکە
4. پەسەندکردنی خۆکارانە ڕێکبخە

### پرۆسێسی خۆکارانە

-   سیستەمەکە خۆکارانە پسووڵەکان دروست دەکات
-   ڕێڕەوی پەسەندکردن دەست پێ دەکات
-   تۆمارەکانی ژمێریاری دروست دەبن
-   تۆمارکردنی پارەدان بەردەستە

## ڕاپۆرت و شیکردنەوە

### ڕاپۆرتەکانی کڕین

**کڕین بەپێی فرۆشیار**: شیکردنەوەی خەرجکردن بەپێی دابینکەر
**کڕین بەپێی پۆل**: دابەشکردنی پۆلی تێچوون
**کڕین بەپێی ماوە**: ڕەوتەکانی خەرجکردنی مانگانە/چارەکانە
**ڕاپۆرتەکانی باج**: ڕاپۆرتکردنی باجی کاڵا/باجی ناوەوە

### بەڕێوەبردنی پارەدانەکان

**پارەدانەکانی کۆن**: بڕە ماوەکان بەپێی تەمەن
**ستەیتمێنتی فرۆشیار**: کورتەی چالاکی هەژمار
**تۆمارکردنی پارەدان**: پێداویستییەکانی پارەدانی داهاتوو
**پلانکردنی جۆڕی پارە**: شیکردنەوەی کاتی پارەدان

## باشترین پراکتیسەکان

### دووبارە کردنەوەی پسووڵە

-   **لەگەڵ داواکاری کڕین بگونجێنە**: لە بەرانبەر داواکارییە ڕەسەنەکان دووبارە بکەرەوە
-   **وەصڵی گەیاندن بپشکنەوە**: پشتڕاست بکەرەوە کاڵا/خزمەتگوزارییەکان وەرگیراون
-   **نرخدانان ڕاست بکەرەوە**: لەگەڵ نرخە ڕێککەوتووەکان بەراورد بکە
-   **مەرجەکان پێداچوونەوە بکە**: دڵنیا بە کە مەرجەکانی پارەدان راستن

### بەڵگەسازی

-   **پسووڵەکانی فرۆشیار**: بەڵگەکانی ڕەسەنی فرۆشیار بلکێنە
-   **وەصڵی گەیاندن**: سەلماندنی وەرگرتن پارێزراو بکە
-   **تۆمارەکانی پەسەندکردن**: بەڵگەی پەسەندکردن بپارێزە
-   **تۆمارەکانی پارەدان**: پشتڕاستکردنەوەکانی پارەدان بەستەوە

### کۆنترۆڵی ناوخۆیی

-   **جیاکردنەوەی ئەرکەکان**: داواکردن، وەرگرتن و پارەدان جیا بکەرەوە
-   **سنووری دەسەڵات**: سنووری خەرجکردن جێبەجێ بکە
-   **پێداچوونەوەی ڕێکخراو**: پەیوەندییەکانی فرۆشیار چاودێری بکە
-   **شوێنی پشکنین**: تۆمارەکانی تەواوی مامەڵە بپارێزە

## سیناریۆی باو

### سیناریۆی 1: کڕینی کەلووپەلی ئۆفیس

کڕینی کەلووپەلی ئۆفیس بۆ 500,000 دینار لەگەڵ باجی کاڵای 10%:

1. فرۆشیاری کەلووپەلی ئۆفیس هەڵبژێرە
2. هێڵی کەلووپەلی ئۆفیس زیاد بکە (بڕ: 500,000 دینار)
3. سیستەمەکە باجی کاڵا هەژمار دەکات: 50,000 دینار
4. کۆی پسووڵە: 550,000 دینار
5. پسووڵە پەسەندکرە

### سیناریۆی 2: خزمەتگوزاری پیشەیی

پسووڵەی خزمەتگوزاری یاسایی بۆ 3,000,000 دینار:

1. فرۆشیاری فیرمی یاسایی هەڵبژێرە
2. هێڵی خزمەتگوزاری یاسایی زیاد بکە (کاتژمێر: 20، نرخ: 150,000)
3. باجی کاڵا نییە (خزمەتگوزاری پیشەیی ئازادە)
4. کۆی پسووڵە: 3,000,000 دینار
5. پسووڵە پەسەندکرە

### سیناریۆی 3: کڕینی ئەدەمی

کڕینی ئەدەمی کۆمپیوتەر بۆ 5,000,000 دینار:

1. فرۆشیاری ئەدەمی هەڵبژێرە
2. هێڵی ئەدەمی کۆمپیوتەر زیاد بکە (بڕ: 5، تێچوون: 1,000,000)
3. بۆ هەژماری سامانی جێگیر دیاری بکە
4. پۆلی سامان هەڵبژێرە بۆ دروستکردنی سامان
5. باجی گونجاو جێبەجێ بکە
6. پسووڵە پەسەندکرە

## چارەسەری کێشە

### کێشەی باو

**کێشە**: ناتوانرێت پسووڵە پەسەندکرێت - هەڵەی دووبارەکردنەوە
**چارەسەر**:

-   هەموو خانە پێویستەکان تەواو کراون بپشکنەوە
-   زانیاری فرۆشیار تەواوە دووبارە بکەرەوە
-   دڵنیا بە کە هەموو هێڵەکان هەژماریان هەیە

**کێشە**: هەژماری باج هەڵەیە
**چارەسەر**:

-   ڕێکخستنی باجی فرۆشیار دووبارە بکەرەوە
-   ڕێکخستنەکانی باجی بەرهەم بپشکنەوە
-   یاساکانی پێگەی مالی پێداچوونەوە بکە

**کێشە**: هەڵسەنگاندنی سێ لایەنە سەرناکەوێت
**چارەسەر**:

-   بڕەکانی داواکاری کڕین بپشکنەوە
-   تۆمارەکانی وەصڵی کاڵا دووبارە بکەرەوە
-   ئاستی بەرگەگرتنی نرخ پێداچوونەوە بکە

## خاڵەکانی یەکگرتنەوە

### سیستەمی پارەدان

-   پسووڵەکان لە ڕێگەی بەستەرەکانی بەڵگەوە بە پارەدانەکان دەبەسترێنەوە
-   پشتگیری بۆ پارەدانی بەشەیی و قیست
-   مامەڵەی پارەدانی دراوی جیاواز
-   تەرخانکردنی خۆکارانەی پارەدان

### بەڕێوەبردنی کۆگا

-   پسووڵەکانی کۆگا ئاستی کۆگا نوێ دەکەنەوە
-   نوێکردنەوەی تێچوون کاریگەری لەسەر نرخدانانی کۆگا دەکات
-   یەکگرتنەوە لەگەڵ جوڵەکانی کۆگا

### سامانی جێگیر

-   پسووڵەکانی خەرجی سەرمایە سامان دروست دەکەن لە ڕێگەی پۆلەکانی سامانەوە
-   دروستکردنی خۆکارانەی سامان لە هێڵەکانی پسووڵەوە
-   یەکگرتنەوە لەگەڵ سیستەمی لەناوچوونی سامان

### ژمێریاری شیکارانە

-   پسووڵەکان پشتگیری تەرخانکردنی هەژماری شیکارانە دەکەن
-   شوێنکەوتنی ناوەندی تێچوون و پڕۆژە
-   یەکگرتنەوەی ڕاپۆرتی بەڕێوەبردن

---

پسووڵەکانی فرۆشیار بنەڕەتین بۆ ناسینەوەی ورد تێچوون و بەڕێوەبردنی هەژمارەکانی پارەدانی. پرۆسێسی گونجاوی پسووڵە دڵنیایی ڕێگەپێدان بە ستاندارەکانی ژمێریاری دەکات و بناغە بۆ بەڕێوەبردنی کاریگەری جۆڕی پارە و پارێزراوی پەیوەندییەکانی فرۆشیار دابین دەکات.

## فەرهەنگی زاراوەکان

### زاراوە ژمێریارییەکان

-   **پسووڵەی فرۆشیار** (Vendor Bill): بەڵگەی بازرگانی کە بڕی قەرزاو بۆ فرۆشیار دیاری دەکات
-   **هەژماری پارەدانی** (Accounts Payable): هەژماری پارە قەرزاوەکان بۆ فرۆشیارەکان
-   **ژمێریاری کۆکردنەوە** (Accrual Accounting): ناسینەوەی تێچوون کاتێک ڕوودەدات، نەک کاتی پارەدان
-   **تۆماری پەرتووکی ڕۆژانە** (Journal Entry): تۆمارکردنی مامەڵەی مالی بە دوو لایەنی
-   **باجی ناوەوە** (Input VAT): باجی پارەدراو بۆ فرۆشیارەکان کە دەکرێت بگەڕێنرێتەوە

### زاراوە کڕینەکان

-   **داواکاری کڕین** (Purchase Order): بەڵگەی فەرمی داواکردنی کاڵا یان خزمەتگوزاری
-   **وەصڵی کاڵا** (Goods Receipt): پشتڕاستکردنەوەی وەرگرتنی کاڵاکان
-   **هەڵسەنگاندنی سێ لایەنە** (Three-Way Matching): بەراوردکردنی داواکاری کڕین، وەصڵی کاڵا و پسووڵە
-   **مەرجەکانی پارەدان** (Payment Terms): مەرجەکان کە کاتی پارەدان دیاری دەکات

### زاراوە باجەکان

-   **باجی کاڵا** (VAT): باجی کاڵا و خزمەتگوزارییەکان
-   **باجی بڕدراو** (Withholding Tax): باج بڕدەکرێتەوە لە پارەدان
-   **ئازادی لە باج** (Tax Exemption): ئازادبوون لە پارەدانی باج
-   **پێگەی مالی** (Fiscal Position): یاساکانی باج بەپێی شوێن

### دۆخەکانی پسووڵە

-   **ڕەشنووس** (Draft): پسووڵە لە ئامادەسازیدایە
-   **پەسەندکراو** (Posted): پسووڵە کۆتایی هاتووە و تۆماری ژمێریاری دروستکردووە
-   **پارەدراو** (Paid): پارەدان بۆ فرۆشیار کراوە
-   **هەڵوەشاندراوەتەوە** (Cancelled): پسووڵە هەڵوەشێنراوەتەوە

### جۆرەکانی کڕین

-   **کڕینی کۆگا** (Inventory Purchase): کڕینی کاڵا بۆ فرۆشتنەوە
-   **تێچوونی ڕاستەوخۆ** (Direct Expense): تێچوونی فەوری وەک کەلووپەل
-   **خەرجی سەرمایە** (Capital Expenditure): کڕینی سامانی درێژخایەن
-   **خزمەتگوزاری** (Service): کڕینی خزمەتگوزاری وەک ڕاوێژکاری

### پرۆسێسەکانی بەڕێوەبردن

-   **ڕێڕەوی پەسەندکردن** (Approval Workflow): پرۆسێسی پەسەندکردنی پسووڵەکان
-   **جیاکردنەوەی ئەرکەکان** (Segregation of Duties): جیاکردنەوەی ئەرکە جیاوازەکان بۆ کۆنترۆڵ
-   **پشکنینی ناوخۆیی** (Internal Control): ڕێکارەکان بۆ پارێزگاری لە هەڵە و فڕفێری
