# پسولەکانی فرۆشیار: بەڵگەنامەی کڕین و ناسینەوەی تێچوون

ئەم ڕێنمایە ڕوونکردنەوەی چۆنیەتی درووستکردن، بەڕێوەبردن و پرۆسێسکردنی پسولەکانی فرۆشیار لە سیستەمی ژمێرایەتیدا دەکات. پسولەکانی فرۆشیار کڕینەکان لە دابینکەرەکان بەڵگەدەکەن و دڵنیایی ناسینەوەی گونجاوی تێچوون و شوێنکەوتنی حیسابەکانی پارەدانی دەکەن.

---

## پسولەی فرۆشیار چییە؟

پسولەی فرۆشیار بەڵگەنامەیەکی بازرگانییە کە:
- **کڕینەکان تۆمار دەکات**: کاڵا یان خزمەتگوزارییە وەرگیراوەکان بەڵگە دەکات
- **بەرپرسیارێتی دامەزرێنێت**: ئەرکی پارەدان بۆ فرۆشیار درووست دەکات
- **تێچوونەکان دەناسێنێتەوە**: تۆمارەکانی ژمێرایەتی بۆ تێچوونەکان درووست دەکات
- **پارەدانەکان شوێن دەکەوێت**: بڕە قەرزاوەکان بۆ فرۆشیارەکان دامەزرێنێت

**کاریگەری ژمێرایەتی**: پسولە پەسندکراوەکان تێچوونەکان و حیسابەکانی پارەدانی زیاد دەکەن، بەپەیڕەوکردنی بنەمای ژمێرایەتی کۆکردنەوە.

---

## گەشەی ژیانی پسولە

### 1. دۆخی ڕەشنووس
- پسولە ئامادە دەکرێت
- دەتوانرێت دەستکاری یان سڕینەوەی بکرێت
- کاریگەری ژمێرایەتی نییە
- هێشتا بۆ پارەدان پەسند نەکراوە

### 2. دۆخی پەسندکراو
- پسولە تەواو و پەسند کراوە
- تۆمارەکانی ڕۆژنامچە درووست دەکات
- ناتوانرێت دەستکاری بکرێت (تەنها بەڵگەی ڕاستکردنەوە)
- ئامادەیە بۆ پارەدان

### 3. دۆخی پارەدراو
- پارەدان بۆ فرۆشیار کراوە
- حیسابەکانی پارەدانی پاک کراونەتەوە
- مامەڵەکە تەواو بووە

### 4. دۆخی هەڵوەشاندنەوە
- پسولە هەڵوەشێنراوەتەوە
- کاریگەری ژمێرایەتی نییە

---

## درووستکردنی پسولەی فرۆشیار

بڕۆ بۆ **ژمێرایەتی → کڕین → پسولەکانی فرۆشیار → درووستکردن**

### هەنگاوی 1: سەرپەڕەی پسولە

**فرۆشیار**: فرۆشیار لە لیستی هاوبەشەکان هەڵبژێرە
**بەرواری پسولە**: بەرواری پسولەی فرۆشیار بنووسە
**بەرواری ژمێرایەتی**: بەرواری ناسینەوەی ژمێرایەتی
**بەرواری پێویست**: بەرواری کۆتایی پارەدان (خۆکارانە لە مەرجەکانی پارەدانەوە هەژمار دەکرێت)
**دراو**: دراوی پسولە هەڵبژێرە (بنەڕەت دراوی فرۆشیارە)
**مەرجەکانی پارەدان**: مەرجەکانی پارەدان هەڵبژێرە ([ڕێنماڵی مەرجەکانی پارەدان](payment-terms-guide.ckb.md))
**سەرچاوەی فرۆشیار**: ژمارەی پسولەی فرۆشیار بنووسە
**سەرچاوە**: سەرچاوەی ناوخۆیی یان ژمارەی داواکاری کڕین

### هەنگاوی 2: ڕیزەکانی پسولە

کلیک لە **زیادکردنی ڕیز** بکە بۆ زیادکردنی کاڵای کڕراو:

#### ڕیزەکانی بەرهەم
**بەرهەم**: لە کاتالۆگی بەرهەمەکان هەڵبژێرە
**وەسف**: خۆکارانە لە بەرهەمەوە پڕ دەکرێتەوە (دەتوانرێت دەستکاری بکرێت)
**کەمیەت**: ژمارەی یەکە کڕراوەکان
**نرخی یەکە**: تێچوونی هەر یەکەیەک
**باج**: ڕێژەکانی باجی پەیوەست هەڵبژێرە (باجی کاڵا، باجی فرۆشتن)
**حیساب**: حیسابی تێچوون یان سامان (خۆکارانە لە بەرهەمەوە پڕ دەکرێتەوە)

#### ڕیزەکانی خزمەتگوزاری
**خزمەتگوزاری**: خزمەتگوزاریەکە هەڵبژێرە
**وەسف**: خزمەتگوزاری وەرگیراو باس بکە
**کاتژمێر/یەکە**: کەمیەتی خزمەتگوزاری
**ڕێژە**: ڕێژەی کاتژمێری یان یەکە
**باج**: باجەکانی پەیوەست هەڵبژێرە
**حیساب**: حیسابی تێچوونی خزمەتگوزاری

#### ڕیزەکانی تێچوون
**وەسف**: وەسفی تێچوون بنووسە
**بڕ**: کۆی بڕی تێچوون
**حیساب**: حیسابی تێچوونی گونجاو هەڵبژێرە
**پۆلی سامان**: بۆ کڕینی سامان، پۆلی سامان هەڵبژێرە بۆ درووستکردنی سامان
**باج**: باجەکانی پەیوەست هەڵبژێرە

### هەنگاوی 3: باج و کۆی گشتی

**هەژماری باج**: سیستەمەکە خۆکارانە باج بەپێی ئەمانە هەژمار دەکات:
- ڕێکخستنی باجی بەرهەم/خزمەتگوزاری
- ڕێکخستنەکانی باجی فرۆشیار
- یاساکانی پێگەی مالی

**کۆی گشتی پسولە**:
- **کۆی لاوەکی**: کۆی هەموو بڕەکانی ڕیز پێش باج
- **بڕی باج**: کۆی باجی هەژمارکراو (زۆرجار باجی گەڕاندنەوە)
- **کۆی گشتی**: بڕی کۆتایی قەرزاو بۆ فرۆشیار

### هەنگاوی 4: زانیاری زیادە

**مەرجەکانی پارەدان**: دیاری دەکات کەی پارەدان پێویستە
**پێگەی مالی**: یاساکانی باج بەپێی شوێنی فرۆشیار
**ڕۆژنامچە**: ڕۆژنامچەی کڕین بۆ پەسندکردن (زۆرجار خۆکارانە هەڵدەبژێردرێت)
**تێبینیەکان**: مەرجە زیادەکان یان ڕێنماییەکانی گەیاندن

---

## تایبەتمەندییە پێشکەوتووەکانی پسولە

### پسولەکانی دراوی جیاواز

کاتێک پسولەکان بە دراوی بیانی وەردەگیریت:

1. **دراوی پسولە**: دراوی پسولەی فرۆشیار بەکاربهێنە
2. **ڕێژەی ئاڵوگۆڕ**: سیستەمەکە ڕێژەی ئێستا یان تۆمارکردنی دەستی بەکاردەهێنێت
3. **دراوی کۆمپانیا**: سیستەمەکە بۆ تۆمارەکانی ژمێرایەتی ئاڵوگۆڕ دەکات
4. **جیاوازییەکانی ئاڵوگۆڕ**: لە کاتی پارەدان خۆکارانە مامەڵەی لەگەڵ دەکرێت

**نموونە**: پسولە وەرگرتن بۆ $2,000 دۆلاری ئەمریکی کاتێک کۆمپانیا دینار بەکاردەهێنێت
- بڕی پسولە: $2,000 دۆلاری ئەمریکی
- ڕێژەی ئاڵوگۆڕ: 1 USD = 1,310 IQD
- بڕی ژمێرایەتی: 2,620,000 دینار

### هەڵسەنگاندنی سێ لایەنە

بۆ دابینکردنی پەیوەست بە داواکاری کڕین:

1. **داواکاری کڕین**: داواکاری کڕینی ڕەسەن
2. **وەصڵی کاڵا**: پشتڕاستکردنەوەی گەیاندن
3. **پسولەی فرۆشیار**: پسولە لە فرۆشیار

سیستەمەکە دووبارە دەکاتەوە:
- کەمیەتەکان لە نێوان بەڵگەکان دەگونجێن
- نرخەکان لە نۆبەی بەرگەگرتندان
- هەموو کاڵاکان لە ڕاستیدا وەرگیراون

### مامەڵەی باج

**باجی ناوەوە**: باجی پارەدراو بۆ فرۆشیارەکان (زۆرجار گەڕاندنەوە)
**باجی کەمکردنەوە**: باج کەم دەکرێتەوە لە پارەدانەکانی فرۆشیار
**باری هاوردە**: باجی زیادە لەسەر کاڵا هاوردەکراوەکان
**ئازادی لە باج**: هەندێک کڕین لە باج ئازاد بن

## پرۆسێسی پەسندکردنی پسولە

### پێش پەسندکردن - پشکنینی لیستی چاودێری

✅ **زانیاری فرۆشیار**: فرۆشیاری راست هەڵبژێردراوە
✅ **وردەکارییەکانی پسولە**: ژمارەی پسولە و بەرواری فرۆشیار درووستە
✅ **ڕیزەکانی کاڵا**: هەموو کاڵا کڕراوەکان لەگەڵ کەمیەتی راست لەخۆگیراون
✅ **نرخدانان**: تێچوونی یەکە و کۆی گشتی لەگەڵ پسولەی فرۆشیار دەگونجێت
✅ **باجەکان**: ڕێژەی باجی گونجاو جێبەجێ کراوە
✅ **حیسابەکان**: حیسابەکانی تێچوون/سامانی راست دیاری کراون
✅ **پەسندکردن**: پەسندکردنە پێویستەکان وەرگیراون

### پەسندکردنی پسولە

1. **پاشەکەوت وەک ڕەشنووس**: پاشەکەوت بۆ پێداچوونەوە و پەسندکردن
2. **پێداچوونەوە**: هەموو وردەکارییەکان لە بەرانبەر پسولەی فرۆشیار بپشکنەوە
3. **پەسندکردن**: پەسندکردنە پێویستەکان وەربگرە
4. **پەسندکردن**: کلیک لە "پەسندکردن" بکە بۆ تەواوکردنی پسولە

**⚠️ گرنگ**: دوای پەسندکردن، پسولەکان ناتوانن دەستکاری بکرێن. بەڵگەی ڕاستکردنەوە بۆ چاکسازییەکان بەکاربهێنە.

### تۆمارەکانی ڕۆژنامچەی درووستکراو

کاتێک پەسند دەکرێت، سیستەمەکە درووست دەکات:
```
Dr. تێچوونی کەلووپەلی ئۆفیس    500,000 دینار
Dr. باجی ناوەوە                  50,000 دینار
    Cr. حیسابەکانی پارەدانی           550,000 دینار
```

## بەڕێوەبردنی پسولە پەسندکراوەکان

### پرۆسێسی پارەدان

**لە پسولەوە**: کلیک لە "تۆمارکردنی پارەدان" لەسەر پسولە
**لە پارەدانەکانەوە**: پارەدان درووست بکە و بە پسولە بەستەوە
**پارەدانی دەستە**: چەندین پسولە لە یەک مامەڵەدا پارە بدە

بۆ زیاتر دەربارەی پارەدانەکان، سەیری [ڕێنماڵی پارەدانەکان](payments.ckb.md) بکە.

### پارەدانی بەشەیی

فرۆشیارەکان ڕەنگە پارەدانی بەشەیی قبوڵ بکەن:
1. پارەدان تۆمار بکە بۆ بڕی بەشەیی
2. پسولە دەمێنێتەوە "بە بەشەیی پارەدراو"
3. ماندووی ماوە خۆکارانە شوێن دەکەوێت
4. چەندین پارەدان دەتوانرێت جێبەجێ بکرێت

### بەڵگەی ڕاستکردنەوە

بۆ گەڕاندنەوە، داشکاندن، یان چاکسازی:
1. کلیک لە "بەڵگەی ڕاستکردنەوە" لەسەر پسولەی ڕەسەن
2. کاڵاکان بۆ ڕاستکردنەوە هەڵبژێرە
3. بڕی ڕاستکردنەوە دیاری بکە
4. بەڵگەی ڕاستکردنەوە پەسند بکە
5. بۆ حیسابی فرۆشیار جێبەجێ بکە

## پۆلەکانی کڕین و حیسابەکان

### کڕینی کۆگا
**حیساب**: حیسابی سامانی کۆگا
**کاریگەری**: نرخی کۆگا زیاد دەکات
**ناسینەوەی تێچوون**: کاتێک کاڵاکان دەفرۆشرێن (COGS)

### تێچوونە ڕاستەوخۆکان
**حیساب**: حیسابەکانی تێچوونی کارکردن
**کاریگەری**: ناسینەوەی تێچوونی فەوری
**نموونەکان**: کەلووپەلی ئۆفیس، کەرەستەی کارەبا، کرێ

### خەرجی سەرمایە
**حیساب**: حیسابەکانی سامانی جێگیر
**کاریگەری**: نرخی سامان زیاد دەکات
**لەناوچوون**: تێچوون لەسەر ماوەی ژیانی سامان دەناسرێتەوە

### خزمەتگوزارییەکان
**حیساب**: حیسابەکانی تێچوونی خزمەتگوزاری
**کاریگەری**: ناسینەوەی تێچوونی فەوری
**نموونەکان**: ڕاوێژکاری، چاکسازی، کرێی پیشەیی

## ڕێڕەوەکانی پەسندکردن

### سنووری دەسەڵاتی کڕین
**بەڕێوەبەرانی بەش**: تا 5,000,000 دینار
**بەڕێوەبەران**: تا 25,000,000 دینار
**بەڕێوەبەری دارایی**: تا 100,000,000 دینار
**بەڕێوەبەری گشتی**: لەسەروی 100,000,000 دینار

### پرۆسێسی پەسندکردن
1. **داواکار**: پسولە لە دۆخی ڕەشنووسدا درووست دەکات
2. **پێداچوونەوەی بەش**: بەڕێوەبەر پێداچوونەوە و پەسند دەکات
3. **پێداچوونەوەی دارایی**: ژمێرایەتی وردەکارییەکان دووبارە دەکاتەوە
4. **پەسندکردنی کۆتایی**: کەسی دەسەڵاتدار بۆ پەسندکردن پەسند دەکات
5. **پەسندکردن**: پسولە پەسند دەکرێت و ئامادەیە بۆ پارەدان

### جیاکردنەوەی ئەرکەکان
- **درووستکردنی پسولە**: بەشی کڕین
- **پەسندکردنی پسولە**: بەڕێوەبەرانی بەش
- **دەسەڵاتی پارەدان**: بەشی دارایی
- **جێبەجێکردنی پارەدان**: کرێکاری خەزێنە

## بەڕێوەبردنی فرۆشیار

### زانیاری فرۆشیار
**وردەکارییەکانی پەیوەندی**: ناونیشان، تەلەفۆن، ئیمێڵ
**مەرجەکانی پارەدان**: مەرجەکانی ستاندارد پارەدان
**زانیاری باج**: ناسنامەی باج، دۆخی ئازادی
**وردەکارییەکانی بانک**: زانیاری حیسابی پارەدان

### ئادایی فرۆشیار
**مێژووی پارەدان**: شوێنکەوتنی کاتی پارەدان
**پێوەرەکانی کوالیتی**: ڕێژەکردنی کوالیتی بەرهەم/خزمەتگوزاری
**شیکردنەوەی نرخ**: بەراوردکردنی تێچوونەکان بەدرێژایی کات
**ڕێگەپێدان**: چاودێری بڕوانامە و مۆڵەتەکان

---

## پسولەکانی دووبارەبوونەوە

بۆ تێچوونە ڕێکخستنەکانی فرۆشیار:

### ڕێکخستنی پسولەی دووبارەبوونەوە
1. نموونەی پسولە درووست بکە
2. نەخشی دووبارەبوونەوە دابنێ (مانگانە، چارەکانە، هتد)
3. بەرواری دەستپێک و کۆتایی دیاری بکە
4. پەسندکردنی خۆکارانە ڕێکبخە

### پرۆسێسی خۆکارانە
- سیستەمەکە خۆکارانە پسولەکان درووست دەکات
- ڕێڕەوی پەسندکردن دەست پێ دەکات
- تۆمارەکانی ژمێرایەتی درووست دەبن
- تۆمارکردنی پارەدان بەردەستە

---

## ڕاپۆرت و شیکردنەوە

### ڕاپۆرتەکانی کڕین
**کڕین بەپێی فرۆشیار**: شیکردنەوەی خەرجکردن بەپێی دابینکەر
**کڕین بەپێی پۆل**: دابەشکردنی پۆلی تێچوون
**کڕین بەپێی ماوە**: ڕەوتەکانی خەرجکردنی مانگانە/چارەکانە
**ڕاپۆرتەکانی باج**: ڕاپۆرتکردنی باجی کاڵا/باجی ناوەوە

### بەڕێوەبردنی پارەدانەکان
**پارەدانەکانی کۆن**: بڕە ماوەکان بەپێی تەمەن
**ستەیتمێنتی فرۆشیار**: پوختەی چالاکی حیساب
**تۆمارکردنی پارەدان**: پێداویستییەکانی پارەدانی داهاتوو
**پلانکردنی جۆڕی پارە**: شیکردنەوەی کاتی پارەدان

### دووبارە کردنەوەی پسولە
- **لەگەڵ داواکاری کڕین بگونجێنە**: لە بەرانبەر داواکارییە ڕەسەنەکان دووبارە بکەرەوە
- **وەصڵی گەیاندن بپشکنەوە**: پشتڕاست بکەرەوە کاڵا/خزمەتگوزارییەکان وەرگیراون
- **نرخدانان ڕاست بکەرەوە**: لەگەڵ ڕێژە ڕێککەوتووەکان بەراورد بکە
- **مەرجەکان پێداچوونەوە بکە**: دڵنیا بە کە مەرجەکانی پارەدان راستن

### بەڵگەسازی
- **پسولەکانی فرۆشیار**: بەڵگەکانی ڕەسەنی فرۆشیار بلکێنە
- **وەصڵی گەیاندن**: سەلماندنی وەرگرتن پارێزراو بکە
- **تۆمارەکانی پەسندکردن**: بەڵگەی پەسندکردن بپارێزە
- **تۆمارەکانی پارەدان**: پشتڕاستکردنەوەکانی پارەدان بەستەوە

### کۆنترۆڵی ناوخۆیی
- **جیاکردنەوەی ئەرکەکان**: داواکردن، وەرگرتن و پارەدان جیا بکەرەوە
- **سنووری دەسەڵات**: سنووری خەرجکردن جێبەجێ بکە
- **پێداچوونەوەی ڕێکخراو**: پەیوەندییەکانی فرۆشیار چاودێری بکە
- **شوێنی پشکنین**: تۆمارەکانی تەواوی مامەڵە بپارێزە

## سیناریۆی باو

### سیناریۆی 1: کڕینی کەلووپەلی ئۆفیس
کڕینی کەلووپەلی ئۆفیس بۆ 500,000 دینار لەگەڵ باجی کاڵای 10%:

1. فرۆشیاری کەلووپەلی ئۆفیس هەڵبژێرە
2. ڕیزی کەلووپەلی ئۆفیس زیاد بکە (بڕ: 500,000 دینار)
3. سیستەمەکە باجی کاڵا هەژمار دەکات: 50,000 دینار
4. کۆی پسولە: 550,000 دینار
5. پسولە پەسند بکە

### سیناریۆی 2: خزمەتگوزاری پیشەیی
پسولەی خزمەتگوزاری یاسایی بۆ 3,000,000 دینار:

1. فرۆشیاری فیرمی یاسایی هەڵبژێرە
2. ڕیزی خزمەتگوزاری یاسایی زیاد بکە (کاتژمێر: 20، ڕێژە: 150,000)
3. باجی کاڵا نییە (خزمەتگوزاری پیشەیی ئازادە)
4. کۆی پسولە: 3,000,000 دینار
5. پسولە پەسند بکە

### سیناریۆی 3: کڕینی ئەدەمی
کڕینی ئەدەمی کۆمپیوتەر بۆ 5,000,000 دینار:

1. فرۆشیاری ئەدەمی هەڵبژێرە
2. ڕیزی ئەدەمی کۆمپیوتەر زیاد بکە (کەمیەت: 5، تێچوون: 1,000,000)
3. بۆ حیسابی سامانی جێگیر دیاری بکە
4. پۆلی سامان هەڵبژێرە بۆ درووستکردنی سامان
5. باجی گونجاو جێبەجێ بکە
6. پسولە پەسند بکە

## چارەسەری کێشە

### کێشەی باو

**کێشە**: ناتوانرێت پسولە پەسند بکرێت - هەڵەی دووبارەکردنەوە
**چارەسەر**:
- هەموو خانە پێویستەکان تەواو کراون بپشکنەوە
- زانیاری فرۆشیار تەواوە دووبارە بکەرەوە
- دڵنیا بە کە هەموو ڕیزەکان حیسابیان هەیە

**کێشە**: هەژماری باج هەڵەیە
**چارەسەر**:
- ڕێکخستنی باجی فرۆشیار دووبارە بکەرەوە
- ڕێکخستنەکانی باجی بەرهەم بپشکنەوە
- یاساکانی پێگەی مالی پێداچوونەوە بکە

**کێشە**: هەڵسەنگاندنی سێ لایەنە سەرناکەوێت
**چارەسەر**:
- کەمیەتەکانی داواکاری کڕین بپشکنەوە
- تۆمارەکانی وەصڵی کاڵا دووبارە بکەرەوە
- ئاستی بەرگەگرتنی نرخ پێداچوونەوە بکە

## خاڵەکانی یەکگرتنەوە

### داواکارییەکانی کڕین
- پسولەکان دەتوانن ئاماژە بە داواکارییەکانی کڕین بکەن
- هەڵسەنگاندنی خۆکارانەی کەمیەت و نرخەکان
- ڕاپۆرتکردنی جیاوازی بۆ نەگونجان

### بەڕێوەبردنی کۆگا
- پسولەکانی کۆگا ئاستی کۆگا نوێ دەکەنەوە
- نوێکردنەوەی تێچوون کاریگەری لەسەر نرخدانانی کۆگا دەکات
- یەکگرتنەوە لەگەڵ جوڵەکانی کۆگا

### سامانی جێگیر
- پسولەکانی خەرجی سەرمایە سامان درووست دەکەن
- ڕێکخستنی خۆکارانەی لەناوچوون
- نوێکردنەوەی تۆمارەکانی سامان

---

پسولەکانی فرۆشیار بنەڕەتین بۆ ناسینەوەی ورد تێچوون و بەڕێوەبردنی حیسابەکانی پارەدانی. پرۆسێسی گونجاوی پسولە دڵنیایی ڕێگەپێدان بە ستاندارەکانی ژمێرایەتی دەکات و بناغە بۆ بەڕێوەبردنی کاریگەری جۆڕی پارە و پارێزراوی پەیوەندییەکانی فرۆشیار دابین دەکات.
