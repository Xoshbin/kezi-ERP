# بەڵگەنامەکانی وەرگرتن و پارەدان

ئەم ڕێنماییە ڕوونکردنەوەی چۆنیەتی دروستکردن و بەڕێوەبردنی بەڵگەنامەکانی وەرگرتن و پارەدان لە سیستەمی ژمێریاریتاندا دەکات. ئەم بەڵگانە جوڵەکانی پارە لە نێوان کۆمپانیاکەت و هاوبەشەکان تۆمار دەکەن، دڵنیایی لە شوێنکەوتنی دروستی ئاوی پارە و کۆنتڕۆڵی دارایی دەکەن.

---

## بەڵگەنامەکانی وەرگرتن و پارەدان چین؟

**بەڵگەنامەکانی وەرگرتن**: پارەی وەرگیراو لەلایەن کۆمپانیاکەتەوە تۆمار دەکەن
- پارەدانی کڕیارەکان بۆ پسوڵەکان
- پێش پارەدانی هاوبەشەکان وەرگیراو
- وەرگرتنی پارەی تر

**بەڵگەنامەکانی پارەدان**: پارەی دراو لەلایەن کۆمپانیاکەتەوە تۆمار دەکەن
- پارەدان بۆ پسوڵەکانی فرۆشیار
- پێش پارەدان بۆ هاوبەشەکان
- پارەدانی تر

**کاریگەری ژمێریاری**: هەر بەڵگەنامەیەک تۆمارەکانی ڕۆژنامە دروست دەکات کە هەژمارەکانی پارە/بانکت و هەژمارە پەیوەندیدارەکانی قەرزار/قەرزخوازت نوێ دەکاتەوە.

---

## جۆرەکانی بەڵگەنامەکانی پارەدان

### ١. پارەدانەکانی تەسویەکردن
پارەدانەکان کە پسوڵە یان پسوڵەکانی فرۆشیاری بوونی هەیە تەسویە دەکەن:
- **وەرگرتنی کڕیار**: کۆکردنەوەی پارە بۆ پسوڵە دەرنەکراوەکان
- **پارەدانی فرۆشیار**: پارەدان بۆ پسوڵەکانی فرۆشیاری دەرنەکراو

### ٢. پێش پارەدان/قەرزی هاوبەش
پارەدانەکان بەبێ تەسویەکردنی بەڵگەنامەی تایبەت:
- **پێش پارەدانی کڕیار**: پێش پارەدان لە کڕیارەکانەوە بۆ داواکاریەکانی داهاتوو
- **پێش پارەدانی فرۆشیار**: پێش پارەدان بۆ فرۆشیارەکان بۆ کڕینەکانی داهاتوو
- **پێش پارەدانی هاوبەش**: پێش پارەدان بۆ هاوبەشەکان

---

## دروستکردنی بەڵگەنامەکانی وەرگرتن

بڕۆ بۆ **ژمێریاری ← بانک و پارە ← پارەدانەکان ← دروستکردن**

### هەنگاوی ١: زانیاریە بنەڕەتییەکان

**جۆری پارەدان**: "هاتوو" (وەرگرتن) هەڵبژێرە
**هاوبەش**: کڕیار یان هاوبەشی پارەدەر هەڵبژێرە
**بڕ**: کۆی گشتی وەرگیراو بنووسە
**بەرواری پارەدان**: بەرواری ڕاستەقینەی وەرگرتن بنووسە
**سەرچاوە**: ژمارەی چێک، سەرچاوەی گواستنەوە، یان ژمارەی وەصڵ بنووسە
**ڕۆژنامچە**: ڕۆژنامچەی بانک یان پارەت هەڵبژێرە
**شێوازی پارەدان**: شێواز هەڵبژێرە (گواستنەوەی بانکی، پارە، چێک، هتد)
**دراو**: دراوی پارەدان هەڵبژێرە

### هەنگاوی ٢: تەسویەکردن دژ بە پێش پارەدان

#### بۆ تەسویەکردنی پسوڵە (پارەدانی کڕیار)
ئەمە پارەدانی تەسویەکردن دروست دەکات کە بە پسوڵە تایبەتەکان دەبەستێتەوە. سیستەمەکە بە شێوەیەکی خۆکار دیاری دەکات کە ئەمە پارەدانی تەسویەکردنە کاتێک پسوڵەکان دەبەستیتەوە.

**نموونە**: کڕیار $٥,٠٠٠ دەدات بۆ دوو پسوڵە
- پسوڵە #٠٠١: $٣,٠٠٠ (پارەدانی تەواو)
- پسوڵە #٠٠٢: $٢,٠٠٠ (پارەدانی بەشی لە پسوڵەی $٤,٥٠٠)

#### بۆ پێش پارەدانی کڕیار (پێش پارەدان)
ئەمە پێش پارەدانی هاوبەش دروست دەکات بەبێ بەستنەوە بە بەڵگەنامە تایبەتەکان.

١. **هیچ بەستەری بەڵگەنامەیەک**: هیچ پسوڵەیەک مەبەستەوە
٢. **هاوبەش پێویستە**: دڵنیابە لەوەی کڕیار هەڵبژێردراوە
٣. **سەرچاوە**: تێبینی وەک "پێش پارەدان بۆ داواکاریەکانی داهاتوو" زیاد بکە

### هەنگاوی ٣: پێداچوونەوە و پشتڕاستکردنەوە

١. **بڕەکان بپشکنە**: بپشکنە کۆی گشتی لەگەڵ وەرگرتنی ڕاستەقینە دەگونجێت
٢. **پاشەکەوت**: پارەدانەکە وەک ڕەشنووس پاشەکەوت بکە
٣. **پشتڕاستکردنەوە**: کلیک لە "پشتڕاستکردنەوەی پارەدان" بکە بۆ پۆستکردنی پارەدانەکە

## دروستکردنی بەڵگەنامەکانی پارەدان

بڕۆ بۆ **ژمێریاری ← بانک و پارە ← پارەدانەکان ← دروستکردن**

### هەنگاوی ١: زانیاریە بنەڕەتییەکان

**جۆری پارەدان**: "چووە دەرەوە" (ناردن) هەڵبژێرە
**هاوبەش**: فرۆشیار یان هاوبەشی وەرگری پارە هەڵبژێرە
**بڕ**: کۆی گشتی دراو بنووسە
**بەرواری پارەدان**: بەرواری ڕاستەقینەی پارەدان بنووسە
**سەرچاوە**: ژمارەی چێک، سەرچاوەی گواستنەوە، یان سەرچاوەی پارەدان بنووسە
**ڕۆژنامچە**: ڕۆژنامچەی بانک یان پارەت هەڵبژێرە
**شێوازی پارەدان**: شێواز هەڵبژێرە (گواستنەوەی بانکی، پارە، چێک، هتد)
**دراو**: دراوی پارەدان هەڵبژێرە

### هەنگاوی ٢: تەسویەکردن دژ بە پێش پارەدان

#### بۆ تەسویەکردنی پسوڵە (پارەدانی فرۆشیار)
ئەمە پارەدانی تەسویەکردن دروست دەکات کە بە پسوڵەکانی فرۆشیاری تایبەت دەبەستێتەوە. سیستەمەکە بە شێوەیەکی خۆکار دیاری دەکات کە ئەمە پارەدانی تەسویەکردنە کاتێک پسوڵەکان دەبەستیتەوە.

**نموونە**: فرۆشیار $٨,٠٠٠ بدە بۆ سێ پسوڵە
- پسوڵە #VB٠٠١: $٣,٠٠٠ (پارەدانی تەواو)
- پسوڵە #VB٠٠٢: $٢,٥٠٠ (پارەدانی تەواو)
- پسوڵە #VB٠٠٣: $٢,٥٠٠ (پارەدانی بەشی لە پسوڵەی $٤,٠٠٠)

#### بۆ پێش پارەدانی فرۆشیار (پێش پارەدان)
ئەمە پێش پارەدانی هاوبەش دروست دەکات بەبێ بەستنەوە بە بەڵگەنامە تایبەتەکان.

١. **هیچ بەستەری بەڵگەنامەیەک**: هیچ پسوڵەی فرۆشیارێک مەبەستەوە
٢. **هاوبەش پێویستە**: دڵنیابە لەوەی فرۆشیار هەڵبژێردراوە
٣. **سەرچاوە**: تێبینی وەک "پێش پارەدان بۆ کڕینەکانی داهاتوو" زیاد بکە

### هەنگاوی ٣: پێداچوونەوە و پشتڕاستکردنەوە

١. **بڕەکان بپشکنە**: بپشکنە کۆی گشتی لەگەڵ پارەدانی ڕاستەقینە دەگونجێت
٢. **پاشەکەوت**: پارەدانەکە وەک ڕەشنووس پاشەکەوت بکە
٣. **پشتڕاستکردنەوە**: کلیک لە "پشتڕاستکردنەوەی پارەدان" بکە بۆ پۆستکردنی پارەدانەکە

## پارەدانەکانی فرە دراو

### وەرگرتن/پارەدانەکانی دراوی بیانی

کاتێک وەرگرتن یان پارەدان بە دراوی بیانی:

١. **دراوی پارەدان**: دراوی ڕاستەقینەی وەرگیراو/دراو هەڵبژێرە
٢. **ڕێژەی ئاڵوگۆڕ**: سیستەمەکە ڕێژەی ئێستا بەکار دەهێنێت یان ڕێژەی دەستی بنووسە
٣. **دراوی کۆمپانیا**: سیستەمەکە بە شێوەیەکی خۆکار بۆ دراوی بنەڕەت دەگۆڕێت
٤. **جیاوازیەکانی ئاڵوگۆڕ**: بە شێوەیەکی خۆکار لە هەژماری دیاریکراو تۆمار دەکرێت

**نموونە**: $١,٠٠٠ USD وەربگرە کاتێک دراوی کۆمپانیا IQD یە
- بڕی پارەدان: $١,٠٠٠ USD
- ڕێژەی ئاڵوگۆڕ: ١ USD = ١,٣١٠ IQD
- بڕی دراوی کۆمپانیا: ١,٣١٠,٠٠٠ IQD

## کارپێڕاگەی دۆخی پارەدان

### دۆخی ڕەشنووس
- پارەدان دروستکراوە بەڵام هێشتا پشتڕاست نەکراوەتەوە
- دەتوانرێت دەستکاری یان سڕینەوەی بکرێت
- هێشتا کاریگەری ژمێریاری نییە

### دۆخی پشتڕاستکراو
- پارەدان کۆتایی هاتووە و پۆست کراوە
- تۆمارەکانی ڕۆژنامە دروست کراون
- ناتوانرێت دەستکاری بکرێت (تەنها هەڵوەشاندنەوە)

### دۆخی ڕێکخراو
- پارەدان لەگەڵ ڕاپۆرتی بانک هاوتا کراوە
- پشتڕاستی مامەڵەی ڕاستەقینەی بانک دەکات
- دۆخی کۆتایی لە کارپێڕاگەکەدا

### دۆخی هەڵوەشاوە
- پارەدان هەڵوەشاوەتەوە و ناتوانرێت بەکار بهێنرێت
- بەکار دەهێنرێت لە جیاتی سڕینەوە بۆ شوێنی پشکنین

## باشترین پراکتیسەکان

### بەڵگەکردن
- **بەڵگە پشتگیرەکان**: وەصڵ، پسوڵە، یان پشتڕاستکردنەوەکانی پارەدان بلکێنە
- **سەرچاوە ڕوونەکان**: ژمارە سەرچاوە واتادارەکان بەکار بهێنە
- **پرۆسەی پەسەندکردن**: پرۆسەی پەسەندکردن بۆ بڕە گەورەکان جێبەجێ بکە

### کاتبەندی
- **تۆماری هەمان ڕۆژ**: پارەدانەکان لە بەرواری ڕاستەقینەی مامەڵەکەدا تۆمار بکە
- **ڕێکخستنی بانک**: ڕێکخستنی بەردەوام دڵنیایی لە وردی دەکات
- **پرۆسیجورەکانی بڕین**: دڵنیابە لەوەی پارەدانەکان لە ماوەی دروستدا تۆمار کراون

### ئاسایش
- **مۆڵەتەکانی بەکارهێنەر**: دروستکردنی پارەدان بە بەکارهێنەرە مۆڵەتدارەکان سنووردار بکە
- **جیاکردنەوەی ئەرکەکان**: دروستکردنی پارەدان لە پەسەندکردن جیا بکەرەوە
- **پێداچوونەوە بەردەوامەکان**: شێوازەکانی پارەدان بۆ چالاکی نائاسایی چاودێری بکە

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

**کێشە**: ناتوانم پسوڵە/پسوڵە بدۆزمەوە بۆ بەستنەوە
**چارەسەر**:
- پشتڕاست بکەرەوە پسوڵە/پسوڵە پۆست کراوە (ڕەشنووس نییە)
- هەڵبژاردنی هاوبەش بپشکنەرەوە دەگونجێت
- دڵنیابە لەوەی پسوڵە/پسوڵە بالانسی دەرنەکراوی هەیە

**کێشە**: بڕی پارەدان لەگەڵ کۆی پسوڵە ناگونجێت
**چارەسەر**:
- دابەشکردنی پارەدانی بەشی بەکار بهێنە
- پارەدانی جیا بۆ بڕی ماوە دروست بکە
- داشکاندنی پارەدانی زوو بپشکنەرەوە

**کێشە**: ڕێژەی ئاڵوگۆڕ هەڵە دیارە
**چارەسەر**:
- بەرواری ڕێژەی ئاڵوگۆڕ پشتڕاست بکەرەوە
- ڕێکخستنی ڕێژەی دراو بپشکنەرەوە
- ڕێژەی دروست بە دەستی بنووسە ئەگەر پێویست بوو

## ڕاپۆرت و شیکردنەوە

### ڕاپۆرتەکانی پارەدان
- **تۆماری پارەدان**: لیستی هەموو پارەدانەکان بە مەودای بەروار
- **ڕاپۆرتەکانی هاوبەش**: مێژووی پارەدان بە هاوبەش پیشان بدە
- **ڕاپۆرتەکانی ئاوی پارە**: وەرگرتن و پارەدانی پارە شی بکەرەوە

### بالانسە دەرنەکراوەکان
- **قەرزارەکان**: پسوڵە نەدراوەکانی کڕیار بشوێنە کەوە
- **قەرزخوازەکان**: پسوڵەکانی فرۆشیاری نەدراو چاودێری بکە
- **ڕاپۆرتەکانی تەمەن**: بڕە دواکەوتووەکان شی بکەرەوە

## یەکگرتن لەگەڵ ڕێکخستنی بانک

### هاوتاکردنی ڕاپۆرتی بانک
١. ڕاپۆرتەکانی بانک هاوردە بکە یان دروست بکە
٢. پارەدانە پشتڕاستکراوەکان لەگەڵ هێڵەکانی ڕاپۆرت هاوتا بکە
٣. جیاوازی یان جیاوازیەکانی کاتبەندی دەستنیشان بکە
٤. پرۆسەی ڕێکخستن تەواو بکە

### بڕگە دەرنەکراوەکان
- **چێکەکان لە گواستنەوەدا**: پارەدانە پشتڕاستکراوەکان بەڵام هێشتا پاک نەکراونەتەوە
- **پاشکەوتەکان لە گواستنەوەدا**: وەرگرتنەکان تۆمارکراون بەڵام هێشتا پاشکەوت نەکراون
- **کۆمسیۆنەکانی بانک**: کۆمسیۆنی زیادە لە پارەدانەکاندا تۆمار نەکراوە

---

بەڵگەنامەکانی وەرگرتن و پارەدان گرنگن بۆ پاراستنی تۆمارە وردەکانی ئاوی پارە و دڵنیایی لە کۆنتڕۆڵی دارایی دروست. دروستکردن و ڕێکخستنی بەردەوامی ئەم بەڵگانە بناغەی ڕاپۆرتی دارایی متمانەپێکراو و بەڕێوەبردنی پارە دابین دەکات.
