# بەڵگەنامەکانی وەرگرتن و پارەدان: بەڵگەکردن و کۆنتڕۆڵی جووڵەی پارە

ئەم ڕێنماییە تەواوە ڕوونکردنەوەی چۆنیەتی کارکردنی بەڵگەنامەکانی وەرگرتن و پارەدان لە سیستەمی ژمێریاریتاندا دەکات، دابەشکردنی دروستکردن، پرۆسەی تەسویەکردن، و شوێنکەوتنی جووڵەی پارە دەگرێتەوە. بۆ هەموو بەکارهێنەرەکان نووسراوە — ژمێریار و نا-ژمێریارەکان — ڕێنماییە پراکتیکیەکان دابین دەکات کە شوێن باشترین پراکتیسەکانی ژمێریاری دووانە دەکەوێت.

---

## بەڵگەنامەکانی وەرگرتن و پارەدان چین؟

بەڵگەنامەکانی وەرگرتن و پارەدان بەڵگەنامەکانن کە جووڵەکانی پارە لە نێوان کۆمپانیاکەت و هاوبەشەکان تۆمار دەکەن، میکانیزمە بنەڕەتیەکانی شوێنکەوتنی جووڵەی پارە و کۆنتڕۆڵی دارایی دابین دەکەن.

- **بەڵگەنامەکانی وەرگرتن**: پارەی وەرگیراو لەلایەن کۆمپانیاکەتەوە لە کڕیارەکان، پێش پارەدانەکان، یان سەرچاوەکانی تر تۆمار دەکەن
- **بەڵگەنامەکانی پارەدان**: پارەی دراو لەلایەن کۆمپانیاکەتەوە بۆ فرۆشیارەکان، بۆ پێش پارەدانەکان، یان پارەدانەکانی تر تۆمار دەکەن
- **کارکردی تەسویەکردن**: بەڵگەنامەکان بە پسووڵەکانی فرۆشتن یان پسووڵەکانی کڕینی تایبەت دەبەستێتەوە بۆ تەسویەکردنی هەژمار
- **شوێنکەوتنی جووڵەی پارە**: هەموو جووڵەکانی پارە چاودێری دەکات لەگەڵ ڕێگە تەواوەکانی پشکنین

**مەبەستی ژمێریاری**: هەر بەڵگەنامەیەک تۆمارەکانی پەرتووکی ڕۆژانە دروست دەکات کە هەژمارەکانی پارە/بانک و هەژمارە پەیوەندیدارەکانی وەرگر/دەرکەوتوو نوێ دەکاتەوە، دڵنیایی لە ڕاپۆرتی وردی جووڵەی پارە دەکات.

## پێداویستیەکانی سیستەم

### هاوسەنگی بانکەکان
- **پەرتووکەکانی ڕۆژانەی بانک**: پەرتووکەکانی ڕۆژانەی بانک و پارە دەبێت بۆ پرۆسەی بەڵگەنامەکان ڕێک بخرێن
- **شێوازەکانی پارەدان**: شێوازەکانی پارەدان دەبێت بۆ جۆرە جیاوازەکانی مامەڵە دانرێن
- **تۆمارەکانی هاوبەش**: کڕیار و فرۆشیارەکان دەبێت بوونیان هەبێت بۆ دروستکردنی بەڵگەنامە
- **پشتگیری دراو**: توانای دراوی فرە دەبێت ڕێک بخرێت ئەگەر پێویست بوو

### پێشمەرجەکان
1. **دانانی پەرتووکی ڕۆژانە**: پەرتووکەکانی ڕۆژانەی بانک، پارە، و پارەدان بە شێوەیەکی دروست ڕێکخراون
2. **زانیاریی هاوبەش**: تۆمارەکانی کڕیار و فرۆشیار لەگەڵ وردەکاریەکانی پارەدان
3. **یەکگرتنی بەڵگەنامە**: سیستەمەکانی پسووڵەکانی فرۆشتن و پسووڵەکانی کڕین ڕێکخراون بۆ بەستنەوەی تەسویەکردن
4. **مۆڵەتەکانی بەکارهێنەر**: دەستڕاگەیشتن بە تایبەتمەندیەکانی بانک، بەڕێوەبردنی پارە، و پارەدان

## لە کوێ لە UI دایبدۆزیتەوە

بڕۆ بۆ **ژمێریاری ← بانک و پارە ← پارەدانەکان ← دروستکردن**

بەڵگەنامەکانی وەرگرتن و پارەدان هەروەها دەردەکەون لە:
- **لیستی پارەدانەکان**: تێڕوانینی گشتی لەسەر هەموو بەڵگەنامەکان لەگەڵ زانیاریی دۆخ
- **تۆمارەکانی هاوبەش**: مێژووی بەڵگەنامە پەیوەندیدارەکان شێوازەکانی پارەدان پیشان دەدەن
- **ڕێکخستنی بانک**: بەڵگەنامەکان بەردەستن بۆ هاوتاکردن لەگەڵ ڕاپۆرتەکانی بانک
- **ڕاپۆرتەکانی جووڵەی پارە**: داتای بەڵگەنامەکان بەشدارن لە شیکردنەوەی جووڵەی پارە

**ئامۆژگاری**: دوگمەی یارمەتی/بەڵگەکان لە سەرەوە ئەم ڕێنماییە دەکاتەوە.

## دروستکردنی بەڵگەنامەکانی وەرگرتن

بڕۆ بۆ **ژمێریاری ← بانک و پارە ← پارەدانەکان ← دروستکردن**

### هەنگاوی ١: زانیاریە بنەڕەتییەکان

**جۆری پارەدان**: "هاتوو" (وەرگرتن) هەڵبژێرە
**هاوبەش**: کڕیار یان هاوبەشی پارەدەر هەڵبژێرە
**بڕ**: کۆی گشتی وەرگیراو بنووسە
**بەرواری پارەدان**: بەرواری ڕاستەقینەی وەرگرتن بنووسە
**سەرچاوە**: ژمارەی چێک، سەرچاوەی گواستنەوە، یان ژمارەی وەصڵ بنووسە
**پەرتووکی ڕۆژانە**: پەرتووکی ڕۆژانەی بانک یان پارەت هەڵبژێرە
**شێوازی پارەدان**: شێواز هەڵبژێرە (گواستنەوەی بانکی، پارە، چێک، هتد)
**دراو**: دراوی پارەدان هەڵبژێرە

### هەنگاوی ٢: تەسویەکردن دژ بە پێش پارەدان

#### بۆ تەسویەکردنی پسووڵەکانی فرۆشتن (پارەدانی کڕیار)
ئەمە پارەدانی تەسویەکردن دروست دەکات کە بە پسووڵەکانی فرۆشتنی تایبەت دەبەستێتەوە. سیستەمەکە بە شێوەیەکی خۆکار دیاری دەکات کە ئەمە پارەدانی تەسویەکردنە کاتێک پسووڵەکانی فرۆشتن دەبەستیتەوە.

**نموونە**: کڕیار $٥,٠٠٠ دەدات بۆ دوو پسووڵەکانی فرۆشتن
- پسووڵەکانی فرۆشتن #٠٠١: $٣,٠٠٠ (پارەدانی تەواو)
- پسووڵەکانی فرۆشتن #٠٠٢: $٢,٠٠٠ (پارەدانی بەشی لە پسووڵەکانی فرۆشتنی $٤,٥٠٠)

#### بۆ پێش پارەدانی کڕیار (پێش پارەدان)
ئەمە پێش پارەدانی هاوبەش دروست دەکات بەبێ بەستنەوە بە بەڵگەنامە تایبەتەکان.

١. **هیچ بەستەری بەڵگەنامەیەک**: هیچ پسووڵەکانی فرۆشتنێک مەبەستەوە
٢. **هاوبەش پێویستە**: دڵنیابە لەوەی کڕیار هەڵبژێردراوە
٣. **سەرچاوە**: تێبینی وەک "پێش پارەدان بۆ داواکاریەکانی داهاتوو" زیاد بکە

### هەنگاوی ٣: پێداچوونەوە و پشتڕاستکردنەوە

١. **بڕەکان بپشکنە**: بپشکنە کۆی گشتی لەگەڵ وەرگرتنی ڕاستەقینە دەگونجێت
٢. **پاشەکەوت**: پارەدانەکە وەک ڕەشنووس پاشەکەوت بکە
٣. **پشتڕاستکردنەوە**: کلیک لە "پشتڕاستکردنەوەی پارەدان" بکە بۆ نێردانی پارەدانەکە

## پارەدانەکانی فرە دراو

### وەرگرتن/پارەدانەکانی دراوی بیانی

کاتێک وەرگرتن یان پارەدان بە دراوی بیانی:

١. **دراوی پارەدان**: دراوی ڕاستەقینەی وەرگیراو/دراو هەڵبژێرە
٢. **نرخی ئاڵوگۆڕ**: سیستەمەکە نرخی ئێستا بەکار دەهێنێت یان نرخی دەستی بنووسە
٣. **دراوی کۆمپانیا**: سیستەمەکە بە شێوەیەکی خۆکار بۆ دراوی بنەڕەتی دەگۆڕێت
٤. **جیاوازیەکانی ئاڵوگۆڕ**: بە شێوەیەکی خۆکار لە هەژماری دیاریکراو تۆمار دەکرێت

**نموونە**: $١,٠٠٠ USD وەربگرە کاتێک دراوی کۆمپانیا IQD یە
- بڕی پارەدان: $١,٠٠٠ USD
- نرخی ئاڵوگۆڕ: ١ USD = ١,٣١٠ IQD
- بڕی دراوی کۆمپانیا: ١,٣١٠,٠٠٠ IQD

## کارپێڕاگەی دۆخی پارەدان

### دۆخی ڕەشنووس
- پارەدان دروستکراوە بەڵام هێشتا پشتڕاست نەکراوەتەوە
- دەتوانرێت دەستکاری یان سڕینەوەی بکرێت
- هێشتا کاریگەری ژمێریاری نییە

### دۆخی نێردراو
- پارەدان کۆتایی هاتووە و نێردراوە
- تۆمارەکانی پەرتووکی ڕۆژانە دروست کراون
- ناتوانرێت دەستکاری بکرێت (تەنها هەڵوەشاندنەوە)

### دۆخی هاوسەنگکراو
- پارەدان لەگەڵ ڕاپۆرتی بانک هاوتا کراوە
- پشتڕاستی مامەڵەی ڕاستەقینەی بانک دەکات
- دۆخی کۆتایی لە کارپێڕاگەکەدا

### دۆخی هەڵوەشاوە
- پارەدان هەڵوەشاوەتەوە و ناتوانرێت بەکار بهێنرێت
- بەکار دەهێنرێت لە جیاتی سڕینەوە بۆ شوێنی پشکنین

## باشترین پراکتیسەکان

### بەڵگەکردن
- **بەڵگە پشتگیرەکان**: وەصڵ، پسوڵە، یان پشتڕاستکردنەوەکانی پارەدان بلکێنە
- **سەرچاوە ڕوونەکان**: ژمارە سەرچاوە واتادارەکان بەکار بهێنە
- **پرۆسەی پەسەندکردن**: پرۆسەی پەسەندکردن بۆ بڕە گەورەکان جێبەجێ بکە

### کاتبەندی
- **تۆماری هەمان ڕۆژ**: پارەدانەکان لە بەرواری ڕاستەقینەی مامەڵەکەدا تۆمار بکە
- **ڕێکخستنی بانک**: ڕێکخستنی بەردەوام دڵنیایی لە وردی دەکات
- **پرۆسیجورەکانی بڕین**: دڵنیابە لەوەی پارەدانەکان لە ماوەی دروستدا تۆمار کراون

### ئاسایش
- **مۆڵەتەکانی بەکارهێنەر**: دروستکردنی پارەدان بە بەکارهێنەرە مۆڵەتدارەکان سنووردار بکە
- **جیاکردنەوەی ئەرکەکان**: دروستکردنی پارەدان لە پەسەندکردن جیا بکەرەوە
- **پێداچوونەوە بەردەوامەکان**: شێوازەکانی پارەدان بۆ چالاکی نائاسایی چاودێری بکە

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

**کێشە**: ناتوانم پسووڵەکانی فرۆشتن/پسووڵەکانی کڕین بدۆزمەوە بۆ بەستنەوە
**چارەسەر**:
- پشتڕاست بکەرەوە پسووڵەکانی فرۆشتن/پسووڵەکانی کڕین نێردراوە (ڕەشنووس نییە)
- هەڵبژاردنی هاوبەش بپشکنەرەوە دەگونجێت
- دڵنیابە لەوەی پسووڵەکانی فرۆشتن/پسووڵەکانی کڕین بالانسی دەرنەکراوی هەیە

**کێشە**: بڕی پارەدان لەگەڵ کۆی پسووڵەکان ناگونجێت
**چارەسەر**:
- دابەشکردنی پارەدانی بەشی بەکار بهێنە
- پارەدانی جیا بۆ بڕی ماوە دروست بکە
- داشکاندنی پارەدانی زوو بپشکنەرەوە

**کێشە**: نرخی ئاڵوگۆڕ هەڵە دیارە
**چارەسەر**:
- بەرواری نرخی ئاڵوگۆڕ پشتڕاست بکەرەوە
- هاوسەنگی نرخی دراو بپشکنەرەوە
- نرخی دروست بە دەستی بنووسە ئەگەر پێویست بوو

## ڕاپۆرت و شیکردنەوە

### ڕاپۆرتەکانی پارەدان
- **تۆماری پارەدان**: لیستی هەموو پارەدانەکان بە مەودای بەروار
- **ڕاپۆرتەکانی هاوبەش**: مێژووی پارەدان بە هاوبەش پیشان بدە
- **ڕاپۆرتەکانی ئاوی پارە**: وەرگرتن و پارەدانی پارە شی بکەرەوە

### بالانسە دەرنەکراوەکان
- **قەرزارەکان**: پسووڵەکانی فرۆشتنی نەدراوەکانی کڕیار بشوێنە کەوە
- **قەرزخوازەکان**: پسووڵەکانی کڕینی فرۆشیاری نەدراو چاودێری بکە
- **ڕاپۆرتەکانی تەمەن**: بڕە دواکەوتووەکان شی بکەرەوە

---

## یەکگرتن لەگەڵ هاوسەنگی بانک

### هاوتاکردنی ڕاپۆرتی بانک
١. ڕاپۆرتەکانی بانک هاوردە بکە یان دروست بکە
٢. پارەدانە پشتڕاستکراوەکان لەگەڵ هێڵەکانی ڕاپۆرت هاوتا بکە
٣. جیاوازی یان جیاوازیەکانی کاتبەندی دەستنیشان بکە
٤. پرۆسەی هاوسەنگی تەواو بکە

### بڕگە دەرنەکراوەکان
- **چێکەکان لە گواستنەوەدا**: پارەدانە پشتڕاستکراوەکان بەڵام هێشتا پاک نەکراونەتەوە
- **پاشکەوتەکان لە گواستنەوەدا**: وەرگرتنەکان تۆمارکراون بەڵام هێشتا پاشکەوت نەکراون
- **کۆمسیۆنەکانی بانک**: کۆمسیۆنی زیادە لە پارەدانەکاندا تۆمار نەکراوە

---

## فەرهەنگی زاراوەکان

### زاراوە ژمێریاریەکان
- **قەرزدار (Dr.)**: لای چەپی قیدی ژمێریاری، زیادبوونی سامان یان کەمبوونی قەرز/خاوەندارێتی
- **قەرزخواز (Cr.)**: لای ڕاستی قیدی ژمێریاری، کەمبوونی سامان یان زیادبوونی قەرز/خاوەندارێتی
- **قەرزار**: ئەو بڕانەی کڕیارەکان لە کۆمپانیاکەت قەرزارن
- **قەرزخواز**: ئەو بڕانەی کۆمپانیاکەت لە فرۆشیارەکان قەرزخوازە
- **پەرتووکی ڕۆژانە**: گرووپی هەژمارەکان بۆ جۆرە تایبەتی مامەڵەکان
- **تۆماری پەرتووکی ڕۆژانە**: قیدی ژمێریاری قەرزدار و قەرزخواز

### زاراوە بانکیەکان
- **ڕاپۆرتی بانک**: لیستی مامەڵەکانی بانک لە ماوەیەکی دیاریکراودا
- **چێک لە گواستنەوەدا**: چێکی دەرکراو بەڵام هێشتا لە بانک نەگەیشتووە
- **پاشکەوت لە گواستنەوەدا**: پارەی نێردراو بەڵام هێشتا لە هەژماردا نەکەوتووە
- **هاوسەنگی**: پرۆسەی هاوتاکردنی تۆمارەکانت لەگەڵ ڕاپۆرتی بانک

### زاراوە مامەڵەکان
- **بەڵگەنامەی وەرگرتن**: بەڵگەی تۆماری پارە وەرگیراو
- **بەڵگەنامەی پارەدان**: بەڵگەی تۆماری پارە دراو
- **تەسویەکردن**: بەستنەوەی پارەدان بە پسووڵەکانی فرۆشتن یان پسووڵەکانی کڕینی تایبەت
- **پێش پارەدان**: پارەدان بۆ کارێکی داهاتوو
- **دۆخی ڕەشنووس**: مامەڵە هێشتا تەواو نەبووە و دەتوانرێت گۆڕرا

---

بەڵگەنامەکانی وەرگرتن و پارەدان گرنگن بۆ پاراستنی تۆمارە وردەکانی جووڵەی پارە و دڵنیایی لە کۆنتڕۆڵی دارایی دروست. دروستکردن و هاوسەنگی بەردەوامی ئەم بەڵگانە بناغەی ڕاپۆرتی دارایی متمانەپێکراو و بەڕێوەبردنی پارە دابین دەکات.
