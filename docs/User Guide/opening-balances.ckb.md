# یەکەی دەستپێکردن: دامەزراندنی بنچینەی دارایی

ئەم ڕێنماییە تەواوەتی ڕوونکردنەوەی چۆنێتی کارکردنی یەکەی دەستپێکردن لە سیستەمی ژمێریاریتدا دەکات، کە دەگرێتەوە ئامادەکردنی نەخشەی هەژمارەکان، دروستکردنی تۆماری ڕۆژانە، و پشتڕاستکردنەوەی یەکە. بۆ هەموو بەکارهێنەران نووسراوە — ژمێریاران و غیر-ژمێریاران — ڕێنماییی پراکتیکی پێشکەش دەکات کە باشترین پراکتیکەکانی ژمێریاری دووپات-تۆمارکردن پەیڕەو دەکات.

---

## یەکەی دەستپێکردن چییە؟

یەکەی دەستپێکردن بڕە سەرەتاییەکانن لە هەژمارەکانتدا لە سەرەتای ماوەی ژمێریاریت، نوێنەرایەتی دۆخی داراییی کۆمپانیاکەت دەکەن کاتێک دەستدەکەیت بە بەکارهێنانی سیستەمی ژمێریاری.

- **سامانەکان**: ئەوەی کۆمپانیاکەت خاوەنێتی (کاش، کۆگا، ئامێرەکان، قەرزداری کڕیار)
- **قەرزەکان**: ئەوەی کۆمپانیاکەت قەرزاری (قەرزەکان، قەرزخوازی فرۆشیار، خەرجییە کۆکراوەکان)
- **خاوەندارێتی**: پشکی خاوەن لە بازرگانییەکەدا (سەرمایە، قازانجی پاشماوە)
- **بنچینە**: خاڵی دەستپێک بۆ هەموو مامەڵە ژمێریاریەکانی داهاتوو

**بنەمای ژمێریاری**: سامانەکان = قەرزەکان + خاوەندارێتی. یەکەی دەستپێکردنەکانت دەبێت ئەم هاوکێشەیە بنەڕەتییە ژمێریاری پەیڕەو بکات بۆ دڵنیابوون لە ڕاپۆرتی داراییی ورد.

---

## پێداویستیەکانی سیستەم

### ڕێکخستنی کۆمپانیا
- **دامەزراندنی کۆمپانیا**: پڕۆفایلی کۆمپانیا، دراو، و ساڵی دارایی دەبێت ڕێکخرابن
- **نەخشەی هەژمارەکان**: هەموو هەژمارە پێویستەکان دروست کراون و بە دروستی پۆلێن کراون
- **ڕێکخستنی ڕۆژنامە**: لانیکەم یەک ڕۆژنامەی جیاواز بەردەست بێت بۆ تۆمارکردنەکان
- **ڕێکخستنی دراو**: دراوی بنەڕەت و هەر دراوێکی بیانی دەبێت چالاک بن

### پێداویستیەکان
1. **بوونی زانیاریە داراییەکان**: یەکەی کۆتایی ماوەی پێشوو یان زانیاریی سەرمایەی سەرەتایی
2. **زانیاریی بانک**: ڕاپۆرتی هەژماری بانکی ئێستا کە یەکەی ڕاستەقینەی کاش پیشان دەدەن
3. **بڕگە بەکرێنەدراوەکان**: لیستی قەرزداری کڕیار، قەرزخوازی فرۆشیار، و یەکە بەکرێنەدراوەکانی تر
4. **نرخاندنی سامانەکان**: نرخی سامانی جێگیر، نرخی کۆگا، و بڕی دابەزاندن

---

## لە کوێ لە ڕووکاری بەکارهێنەر دایبدۆزیتەوە

بڕۆ بۆ **ژمێریاری ← تۆمارەکانی پەرتووکی ڕۆژانە ← دروستکردن**

یەکەی دەستپێکردن هەروەها دەردەکەون لە:
- **نەخشەی هەژمارەکان**: زانیاریی یەکەی هەژمار دوای ناردنی تۆمارەکان
- **ڕاپۆرتەکان**: پێداچونەوەی هاوسەنگی و پەڕەی هاوسەنگی کە دۆخی دەستپێکردن پیشان دەدەن
- **داشبۆرد**: کورتەی هەژمارەکان کە کاریگەریی یەکەی دەستپێکردن ڕەنگدانەوە دەکەن

## هەنگاوی یەکەم: ئامادەکردنی نەخشەی هەژمارەکان

بڕۆ بۆ **ڕێکخستنەکان ← هەژمارەکان**

### پۆلە سەرەکیەکانی هەژمارە

**سامانەکان (1000-1999)**
- هەژمارەکانی بانک (110101, 110102)
- هەژمارەکانی کاش (110201, 110202)
- قەرزداری کڕیار (120101)
- کۆگا (130101)
- سامانی جێگیر (140101)

**قەرزەکان (2000-2999)**
- قەرزخوازی فرۆشیار (210101)
- قەرزی قەرزخواز (250201)
- خەرجییە کۆکراوەکان (220101)

**خاوەندارێتی (3000-3999)**
- سەرمایەی خاوەن (310101)
- قازانجی پاشماوە (320101)

### پشتڕاستکردنەوەی ڕێکخستنی هەژمارە

دڵنیابە لەوەی هەر هەژمارەیەک:
- ✅ جۆری هەژمارەی دروست (سامانەکان، قەرزەکان، خاوەندارێتی)
- ✅ ڕێکخستنی دراوی گونجاو
- ✅ ڕێکخستنی چالاککراو بۆ هەژمارەکانی بانک (ئەگەر [هاوسەنگی بانکی](bank-reconciliation.ckb.md) بەکاردەهێنیت)

---

## هەنگاوی دووەم: دروستکردنی تۆماری پەرتووکی ڕۆژانەی یەکەی دەستپێکردن

بڕۆ بۆ **ژمێریاری ← تۆمارەکانی پەرتووکی ڕۆژانە ← دروستکردن**

### سەرپەڕەی تۆماری پەرتووکی ڕۆژانە

1. **پەرتووکی ڕۆژانە**: پەرتووکی ڕۆژانەی بنەڕەتی جیاوازەکانت هەڵبژێرە یان پەرتووکی ڕۆژانەی "یەکەی دەستپێکردن" دروست بکە
2. **دراو**: دراوی بنەڕەتی کۆمپانیاکەت هەڵبژێرە
3. **بەروار**: ڕۆژی یەکەمی ماوەی ژمێریاریت بەکاربهێنە
4. **ژمارەی بەڵگە**: "یەکەی دەستپێکردن" یان "OB-2024"
5. **وەسف**: "یەکەی دەستپێکردن وەک لە [بەروار]"

### زیادکردنی هێڵەکانی تۆماری پەرتووکی ڕۆژانە

بۆ هەر هەژمارەیەک کە یەکەی دەستپێکردنی هەیە:

**هەژمارەکانی سامانەکان** (یەکەی ئاسایی قەرزدار (مدين))
- **قەرزدار (مدين)**: بڕی یەکەی دەستپێکردن بنووسە
- **قەرزخواز**: بەتاڵی بهێڵەوە (0)
- **هەژمار**: هەژماری تایبەتی سامانەکان هەڵبژێرە

**هەژمارەکانی قەرزەکان** (یەکەی ئاسایی قەرزخواز)
- **قەرزدار (مدين)**: بەتاڵی بهێڵەوە (0)
- **قەرزخواز**: بڕی یەکەی دەستپێکردن بنووسە
- **هەژمار**: هەژماری تایبەتی قەرزەکان هەڵبژێرە

**هەژمارەکانی خاوەندارێتی** (یەکەی ئاسایی قەرزخواز)
- **قەرزدار (مدين)**: بەتاڵی بهێڵەوە (0)
- **قەرزخواز**: بڕی یەکەی دەستپێکردن بنووسە  
- **هەژمار**: هەژماری تایبەتی خاوەندارێتی هەڵبژێرە

## هەنگاوی سێیەم: پشتڕاستکردنەوە و ناردن

### پشتڕاستکردنەوەی یەکە

1. **پشکنینی کۆ**: دڵنیابە کۆی گشتی قەرزدار (مدين) یەکسانە لەگەڵ کۆی گشتی قەرزخواز
2. **پێداچوونەوەی بڕەکان**: پشتڕاست بکەوە هەر بڕێک لەگەڵ بەڵگەنامەکانی سەرچاوەت دەگونجێت
3. **پۆلێنکردنی هەژمار**: پشتڕاست بکەوە هەژمارەکان بە دروستی پۆلێن کراون

### هەڵەکانی باوی پشتڕاستکردنەوە

❌ **تۆماری نا-یەکە**: کۆی گشتی قەرزدار (مدين) ≠ کۆی گشتی قەرزخواز
- **چارەسەر**: پێداچوونەوە بە هەر بڕێکی هێڵ و دروستکردنی بڕەکان

❌ **جۆری هەژماری هەڵە**: هەژماری سامانەکان لەگەڵ یەکەی قەرزخواز
- **چارەسەر**: پشکنینی پۆلێنکردنی هەژمار یان ئاراستەی تۆمار

❌ **هەژمار ون**: بڕگە سەرەکیەکانی پەڕەی هاوسەنگی لەناو نین
- **چارەسەر**: زیادکردنی هەژمار ونەکان بۆ نەخشەی هەژمارەکان سەرەتا

### ناردنی تۆمار

1. کلیک لە **پاشەکەوتکردن** بکە بۆ پاشەکەوتکردن وەک ڕەشنووس
2. پێداچوونەوە بە هەموو بڕگەکانی هێڵ بە وردی
3. کلیک لە **ناردن** بکە بۆ کۆتایی هێنان بە یەکەی دەستپێکردنەکان

**⚠️ گرنگ**: دوای ناردن، تۆمارەکانی یەکەی دەستپێکردن نابێت دەستکاری لێ بکرێت. تۆماری ڕاستکردنەوە دروست بکە ئەگەر دروستکردنەوە پێویست بێت.

---

## هەنگاوی چوارەم: پشتڕاستکردنەوەی یەکەی دەستپێکردن

### کارکردنی پەڕەی هاوسەنگی

بڕۆ بۆ **ژمێریاری ← ڕاپۆرتەکان ← پەڕەی هاوسەنگی**

1. مەودای بەروار بۆ بەرواری یەکەی دەستپێکردنت ڕێکبخە
2. پشتڕاست بکەوە هەموو هەژمارەکان بڕی دروستی دەستپێکردن پیشان دەدەن
3. پشتڕاست بکەوە سامانەکان = قەرزەکان + خاوەندارێتی

### پشکنینی یەکەی هەژمار تاکەکەسییەکان

بڕۆ بۆ **ڕێکخستنەکان ← هەژمارەکان**

1. کلیک لە هەر هەژمارەیەک بکە بۆ بینینی یەکەکەی
2. پشتڕاست بکەوە بڕەکانی دەستپێکردن لەگەڵ بەڵگەنامەکانی سەرچاوەت دەگونجێن
3. دڵنیابە یەکەی هەژمارەکانی بانک لەگەڵ ڕاپۆرتی هەژماری بانکی ڕاستەقینە دەگونجێت

### بەکارهێنانی پێداچونەوەی هاوسەنگی

بڕۆ بۆ **ژمێریاری ← ڕاپۆرتەکان ← پێداچونەوەی هاوسەنگی**

1. پێداچونەوەی هاوسەنگی تا بەرواری دەستپێکردنت بەڕێوە ببە
2. پشتڕاست بکەوە هەموو یەکەکان دروستن
3. پشتڕاست بکەوە کۆی گشتی قەرزدار (مدين) یەکسانە لەگەڵ کۆی گشتی قەرزخواز

## لێکۆڵینەوە تایبەتەکان

### ڕێکخستنی فرە-دراو

ئەگەر کۆمپانیاکەت بە چەندین دراو کاردەکات:

1. **دراوی بنەڕەت**: بڕەکان بە دراوی بنەڕەتی کۆمپانیاکەت بنووسە
2. **دراوی بیانی**: نرخی ئاڵوگۆڕ تا بەرواری کردنەوە بەکاربهێنە
3. **جیاوازیی ئاڵوگۆڕ**: هەر ڕاستکردنەوەیەکی وەرگێڕان لە هەژمارەکانی مووڵکدا تۆمار بکە

### یەکەی هاوبەشەکان

بۆ قەرزداری کڕیار و قەرزخوازی فرۆشیاری ورد بە بەکارهێنانی سیستەمی هاوبەش:

1. **تۆمارەکانی هاوبەش**: دڵنیابە هەموو هاوبەشەکان (کڕیار/فرۆشیار) لە سیستەمدا دروست کراون
2. **پسووڵەکانی فرۆشتنی تاکەکەسی**: [تۆمارەکانی پسووڵەی فرۆشتن](customer-invoices.ckb.md) تایبەت بۆ هەر بڕێکی کڕیاری قەرزدار دروست بکە
3. **پسووڵەکانی کڕینی فرۆشیار**: [پسووڵەکانی کڕینی فرۆشیار](vendor-bills.md) تایبەت بۆ هەر قەرزخوازیەکی فرۆشیار دروست بکە
4. **دانانی هاوبەش**: هەر یەکەیێک بە هاوبەشی دروست ببەستەوە
5. **مەرجەکانی پارەدان**: [مەرجەکانی پارەدان](payment-terms-guide.md) گونجاو بۆ هاوبەشەکان ڕێکبخە

### یەکەی دەستپێکردنی کۆگا

بۆ بازرگانییەکانی پشت بە کۆگا:

1. **تۆمارەکانی بەرهەم**: دڵنیابە هەموو بەرهەمەکان لە سیستەمدا دروست کراون
2. **شێوازی نرخاندن**: شێوازی نرخاندنی یەکگرتوو بەکاربهێنە (FIFO، ناوەند، هتد)
3. **شوێنەکانی کۆگا**: کۆگا بۆ شوێنە دروستەکانی کۆگاخانە دابنێ
4. **بەڕێوەبردنی کۆگا**: [ڕێنماییی بەڕێوەبردنی کۆگا](stock-management-user-guide.md) بۆ وردەکارییەکان ببینە

### ڕێکخستنی هاوسەنگی بانکی

ئەگەر تایبەتمەندییەکانی هاوسەنگی بانکی بەکاردەهێنیت:

1. **چالاککردنی هاوسەنگی**: دڵنیابە "چالاککردنی هاوسەنگی" لە ڕێکخستنەکانی کۆمپانیادا چالاک کراوە
2. **ڕێکخستنی هەژمار**: "ڕێدانی هاوسەنگی" چالاک بکە بۆ هەژمارەکانی بانک
3. **ڕاپۆرتی هەژماری بانک**: ئامادەکاری بۆ هاوردەکردنی [ڕاپۆرتی هەژماری بانک](bank-statements.ckb.md) بۆ هاوسەنگی
4. **هاوسەنگی دەستپێکردن**: لێکۆڵینەوە لە ئەنجامدانی [هاوسەنگی بانکی](bank-reconciliation.ckb.md) بۆ بەرواری دەستپێکردن

## باشترین پراکتیکەکان

### بەڵگەنامەکردن

- **بەڵگەنامەکانی سەرچاوە**: کۆپی لە هەموو بەڵگەنامە پشتگیریکەرەکان بپارێزە
- **پرۆسەی پەسەندکردن**: یەکەی دەستپێکردنەکان لەلایەن بەڕێوەبەرایەتییەوە پێداچوونەوە بکرێت
- **شوێنی پشکنینەوە**: سەرچاوەی هەر بڕێکی یەکەی دەستپێکردن بەڵگەنامە بکە

### کاتخایەنی

- **دەستپێکی ماوە**: یەکەی دەستپێکردن لە ڕۆژی یەکەمی ماوەی ژمێریاریت تۆمار بکە
- **هاوسەنگی بانکی**: دڵنیابە یەکەی بانک لەگەڵ ڕاپۆرتەکان لە بەرواری دەستپێکردندا دەگونجێت
- **بڕینەوە**: دڵنیابە هەموو مامەڵەکان لە ماوەی دروستدا تۆمار کراون

### پرۆسەی پێداچوونەوە

1. **پێداچوونەوەی سەربەخۆ**: کەسێکی تر یەکەی دەستپێکردنەکان پشتڕاست بکاتەوە
2. **پێداچونەوەی هاوسەنگی**: پێداچونەوەی هاوسەنگی دەستپێکردن چاپ بکە و پێداچوونەوە بکە
3. **پەسەندکردنی بەڕێوەبەرایەتی**: پەسەندکردنی فەرمی وەربگرە پێش بەردەوامبوون

---

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

**کێشە**: تۆماری یەکەی دەستپێکردن یەکە نابێت
**چارەسەر**: 
1. هەموو قەرزدار (مدين) و قەرزخوازەکان بە جیا کۆ بکەوە
2. بڕی جیاوازی بدۆزەرەوە
3. پشکنین بۆ ژمارە گۆڕدراوەکان یان تۆمار ونەکان

**کێشە**: یەکەی بانک لەگەڵ ڕاپۆرت ناگونجێت
**چارەسەر**:
1. پشتڕاست بکەوە بەرواری ڕاپۆرت لەگەڵ بەرواری دەستپێکردنت دەگونجێت
2. پشکنین بۆ چێک یان پشکەساتە بەکرێنەدراوەکان
3. لێکۆڵینەوە لە دروستکردنی [هاوسەنگی بانکی](bank-reconciliation.ckb.md) بۆ بەرواری دەستپێکردن

**کێشە**: یەکەی هاوبەشەکان وا دیارە هەڵەن
**چارەسەر**:
1. پێداچوونەوە بە ڕاپۆرتەکانی قەرزداری کڕیار/قەرزخوازی فرۆشیاری تەمەنی سیستەمی پێشوو
2. پشتڕاستکردنەوەی بڕەکانی پسووڵەی فرۆشتن/پسووڵەی کڕینی تاکەکەسی
3. پشکنین بۆ پارەدانەکان کە ماوەکانیان پەڕاندووە

**کێشە**: ئاڵۆزیی فرە-دراو
**چارەسەر**:
1. دڵنیابە نرخی ئاڵوگۆڕ بە دروستی بۆ بەرواری کردنەوە ڕێکخراوە
2. دراوی بنەڕەتی کۆمپانیا بۆ تۆمارکردنی باڵانسی کردنەوە بەکاربهێنە
3. لێکۆڵینەوە لە ڕاستکردنەوەی وەرگێڕانی دراو لە هەژمارەکانی مووڵکدا

---

## هەنگاوە دواییەکان

دوای دامەزراندنی یەکەی دەستپێکردن:

1. **هاوسەنگی بانکی**: یەکەی دەستپێکردنی بانک لەگەڵ [ڕاپۆرتی هەژماری بانک](bank-statements.ckb.md) ڕێک بخە
2. **پسووڵەکانی فرۆشتنی کڕیار**: دەستپێکی پرۆسەکردنی [مامەڵە نوێیەکانی کڕیار](customer-invoices.ckb.md)
3. **پسووڵەکانی کڕینی فرۆشیار**: دەستپێکی تۆمارکردنی [مامەڵە نوێیەکانی فرۆشیار](vendor-bills.md)
4. **پرۆسەکردنی پارەدان**: ڕێکخستنی [پرۆسەکردنی پارەدان](payments.ckb.md) بۆ کارەکانی بەردەوام
5. **کارەکانی ئاسایی**: بەردەوامبوون لەگەڵ چالاکیە ڕۆژانەی ژمێریاری

یەکەی دەستپێکردنەکەت ئێستا دامەزراوە و سیستەمی ژمێریاریت ئامادەی کارەکانی ئاسایییە. هەموو مامەڵەکانی داهاتوو لەسەر ئەم بنچینە بەهێزە بنیاد دەنرێن.

---

**بیربهێنەرەوە**: یەکەی دەستپێکردن بنچینەی ڕاپۆرتی داراییی وردن. کات تەرخان بکە بۆ دڵنیابوون لەوەی تەواو و وردن پێش بەردەوامبوون لەگەڵ مامەڵە ئاساییەکان.

---

## فەرهەنگی زاراوەکان

**یەکەی دەستپێکردن** (Opening Balance) - بڕی سەرەتایی لە هەژمارەیەکدا لە سەرەتای ماوەی ژمێریاریدا

**سامانەکان** (Assets) - هەموو ئەو شتانەی کۆمپانیا خاوەنێتی و نرخیان هەیە، وەک پارە، کۆگا، ئامێرەکان

**قەرزەکان** (Liabilities) - هەموو ئەو قەرز و پارانەی کۆمپانیا قەرزاری کەسانی تر، وەک قەرزی بانک، پسووڵەکانی کڕینی فرۆشیار

**خاوەندارێتی** (Equity) - پشکی خاوەنان لە کۆمپانیادا، وەک سەرمایەی سەرەتایی و قازانجی پاشماوە

**ژمێریاری دووپات-تۆمارکردن** (Double-Entry Accounting) - سیستەمی ژمێریاری کە هەر مامەڵەیەک لە لانیکەم دوو هەژماردا تۆمار دەکات

**تۆماری پەرتووکی ڕۆژانە** (Journal Entry) - تۆمارکردنی یەک مامەڵە لە سیستەمی ژمێریاریدا

**پەرتووکی ڕۆژانە** (Journal) - کتێبی یەکەمین تۆمارکردن بۆ مامەڵە ژمێریاریەکان

**قەرزدار (مدين)** (Debit) - لایەنی چەپی هەژمارەیەک، زیادکردن بۆ سامانەکان و خەرجیەکان

**قەرزخواز** (Credit) - لایەنی ڕاستی هەژمارەیەک، زیادکردن بۆ قەرزەکان و داهات

**پێداچونەوەی هاوسەنگی** (Trial Balance) - ڕاپۆرتێک کە هەموو هەژمارەکان و یەکەکانیان پیشان دەدات

**پەڕەی هاوسەنگی** (Balance Sheet) - ڕاپۆرتی دارایی کە سامانەکان، قەرزەکان، و خاوەندارێتی کۆمپانیا پیشان دەدات

**هاوسەنگی بانکی** (Bank Reconciliation) - پرۆسەی هاوتاکردنی ڕیکۆردەکانی کۆمپانیا لەگەڵ ڕاپۆرتی هەژماری بانک

**نەخشەی هەژمارەکان** (Chart of Accounts) - لیستی هەموو هەژمارەکانی ژمێریاری لە کۆمپانیادا

**ماوەی ژمێریاری** (Accounting Period) - ماوەی دیاریکراو کە ژمێریاری بۆی دەکرێت، زۆرجار یەک ساڵ

**دراوی بنەڕەتی** (Base Currency) - دراوی سەرەکی کە کۆمپانیا هەژمارەکانی پێ دەکات
