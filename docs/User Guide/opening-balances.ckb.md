# باڵانسی کردنەوە: دامەزراندنی بنچینەی دارایی

ئەم ڕێنماییە ڕوونکردنەوەی چۆنێتی دامەزراندنی باڵانسی کردنەوە کاتی ڕێکخستنی سیستەمی ژمێریاریتدا دەکات. باڵانسی کردنەوە نوێنەرایەتی دۆخی داراییی کۆمپانیاکەت لە سەرەتای ماوەی ژمێریاریدا دەکات—ئەو بنچینەیەی کە هەموو مامەڵەکانی داهاتوو لەسەری بنیاد دەنرێن.

---

## باڵانسی کردنەوە چییە؟

باڵانسی کردنەوە بڕە سەرەتاییەکانن لە هەژمارەکانتدا لە سەرەتای ماوەی ژمێریاریدا. ئەوان نوێنەرایەتی دەکەن:

- **سامان**: ئەوەی کۆمپانیاکەت خاوەنێتی (پارە، کۆگا، ئامێر، قەرزاریە بەکرێ)
- **قەرزاریە**: ئەوەی کۆمپانیاکەت قەرزاری (قەرز، قەرزاریە بەبەدهی، خەرجییە کۆکراوەکان)  
- **مووڵک**: پشکی خاوەنەکە لە بازرگانییەکەدا (سەرمایە، قازانجی پاشماوە)

**بنەمای ژمێریاری**: سامان = قەرزاریە + مووڵک. باڵانسی کردنەوەکەت دەبێت ئەم هاوکێشەیە بنەڕەتییە پەیڕەو بکات.

---

## پێش دەستپێکردن

### پێداویستیەکان

1. **تەواوکردنی ڕێکخستنی کۆمپانیا**: پڕۆفایلی کۆمپانیاکەت، دراو، و ساڵی دارایی ڕێکخراون
2. **ئامادەکردنی نەخشەی هەژمارەکان**: هەموو هەژمارە پێویستەکان دروست کراون و بە دروستی پۆلێن کراون
3. **بوونی زانیاریە داراییەکان**: زانیاریت لە باڵانسی داخستنی ماوەی پێشوو یان زانیاریی سەرمایەی سەرەتایی هەیە

### زانیاریی پێویست

- ڕاپۆرتی باڵانسی تاقیکردنەوە یان ڕاپۆرتی باڵانسی ماوەی پێشوو
- ڕاپۆرتەکانی بانک کە باڵانسی ڕاستەقینەی پارە پیشان دەدەن
- فاکتورە بەکرێنەدراوەکانی کڕیار (قەرزاریە بەکرێ)
- فاکتورە بەبەدهینەدراوەکانی فرۆشیار (قەرزاریە بەبەدهی)
- نرخاندنی کۆگا
- نرخی سامانی جێگیر و دابەزاندنی کۆکراوە
- باڵانسی قەرز و قەرزاریی تر

---

## هەنگاوی یەکەم: ئامادەکردنی نەخشەی هەژمارەکان

بڕۆ بۆ **ڕێکخستنەکان ← هەژمارەکان**

### پۆلە سەرەکیەکانی هەژمارە

**سامان (1000-1999)**
- هەژمارەکانی بانک (110101, 110102)
- هەژمارەکانی پارەی کاش (110201, 110202)
- قەرزاریە بەکرێ (120101)
- کۆگا (130101)
- سامانی جێگیر (140101)

**قەرزاریە (2000-2999)**
- قەرزاریە بەبەدهی (210101)
- قەرزی بەبەدهی (250201)
- خەرجییە کۆکراوەکان (220101)

**مووڵک (3000-3999)**
- سەرمایەی خاوەن (310101)
- قازانجی پاشماوە (320101)

### پشتڕاستکردنەوەی ڕێکخستنی هەژمارە

دڵنیابە لەوەی هەر هەژمارەیەک:
- ✅ جۆری هەژمارەی دروست (سامان، قەرزاریە، مووڵک)
- ✅ ڕێکخستنی دراوی گونجاو
- ✅ ڕێکخستنی چالاککراو بۆ هەژمارەکانی بانک (ئەگەر [ڕێکخستنی بانک](bank-reconciliation.ckb.md) بەکاردەهێنیت)

## هەنگاوی دووەم: دروستکردنی تۆمارکردنی ڕۆژنامەی باڵانسی کردنەوە

بڕۆ بۆ **ژمێریاری ← تۆمارکردنەکانی ڕۆژنامە ← دروستکردن**

### سەرپەڕەی تۆمارکردنی ڕۆژنامە

1. **ڕۆژنامە**: ڕۆژنامەی بنەڕەتی جیاوازەکانت هەڵبژێرە یان ڕۆژنامەی "باڵانسی کردنەوە" دروست بکە
2. **دراو**: دراوی بنەڕەتی کۆمپانیاکەت هەڵبژێرە
3. **بەروار**: ڕۆژی یەکەمی ماوەی ژمێریاریت بەکاربهێنە
4. **ژمارەی ئاماژە**: "باڵانسی کردنەوە" یان "OB-2024"
5. **وەسف**: "باڵانسی کردنەوە وەک لە [بەروار]"

### زیادکردنی ڕیزەکانی تۆمارکردنی ڕۆژنامە

بۆ هەر هەژمارەیەک کە باڵانسی کردنەوەی هەیە:

**هەژمارەکانی سامان** (باڵانسی ئاسایی مەدین)
- **مەدین**: بڕی باڵانسی کردنەوە بنووسە
- **داین**: بەتاڵی بهێڵەوە (0)
- **هەژمارە**: هەژمارەی تایبەتی سامان هەڵبژێرە

**هەژمارەکانی قەرزاریە** (باڵانسی ئاسایی داین)
- **مەدین**: بەتاڵی بهێڵەوە (0)
- **داین**: بڕی باڵانسی کردنەوە بنووسە
- **هەژمارە**: هەژمارەی تایبەتی قەرزاریە هەڵبژێرە

**هەژمارەکانی مووڵک** (باڵانسی ئاسایی داین)
- **مەدین**: بەتاڵی بهێڵەوە (0)
- **داین**: بڕی باڵانسی کردنەوە بنووسە  
- **هەژمارە**: هەژمارەی تایبەتی مووڵک هەڵبژێرە

## هەنگاوی سێیەم: پشتڕاستکردنەوە و بڵاوکردنەوە

### پشتڕاستکردنەوەی باڵانس

1. **پشکنینی کۆ**: دڵنیابە کۆی گشتی مەدین یەکسانە لەگەڵ کۆی گشتی داین
2. **پێداچوونەوەی بڕەکان**: پشتڕاست بکەوە هەر بڕێک لەگەڵ بەڵگەکانی سەرچاوەت دەگونجێت
3. **پۆلێنکردنی هەژمارە**: پشتڕاست بکەوە هەژمارەکان بە دروستی پۆلێن کراون

### هەڵەکانی باوی پشتڕاستکردنەوە

❌ **تۆمارکردنی نا-باڵانس**: کۆی گشتی مەدین ≠ کۆی گشتی داین
- **چارەسەر**: پێداچوونەوە بە هەر بڕێکی ڕیزە و دروستکردنی بڕەکان

❌ **جۆری هەژمارەی هەڵە**: هەژمارەی سامان لەگەڵ باڵانسی داین
- **چارەسەر**: پشکنینی پۆلێنکردنی هەژمارە یان ئاراستەی تۆمارکردن

❌ **هەژمارە ون**: بڕگە سەرەکیەکانی ڕاپۆرتی باڵانس لەناو نین
- **چارەسەر**: زیادکردنی هەژمارە ونەکان بۆ نەخشەی هەژمارەکان سەرەتا

### بڵاوکردنەوەی تۆمارکردن

1. کلیک لە **پاشەکەوتکردن** بکە بۆ پاشەکەوتکردن وەک ڕەشنووس
2. پێداچوونەوە بە هەموو بڕگەکانی ڕیزە بە وردی
3. کلیک لە **تۆمارکردنی پۆست** بکە بۆ کۆتایی هێنان بە باڵانسی کردنەوەکان

**⚠️ گرنگ**: دوای بڵاوکردنەوە، تۆمارکردنەکانی باڵانسی کردنەوە نابێت گۆڕانکاری لێ بکرێت. تۆمارکردنی ڕاستکردنەوە دروست بکە ئەگەر دروستکردنەوە پێویست بێت.

---

## هەنگاوی چوارەم: پشتڕاستکردنەوەی باڵانسی کردنەوە

### کارکردنی ڕاپۆرتی ڕاپۆرتی باڵانس

بڕۆ بۆ **ژمێریاری ← ڕاپۆرتەکان ← ڕاپۆرتی باڵانس**

1. مەودای بەروار بۆ بەرواری باڵانسی کردنەوەت ڕێکبخە
2. پشتڕاست بکەوە هەموو هەژمارەکان بڕی دروستی کردنەوە پیشان دەدەن
3. پشتڕاست بکەوە سامان = قەرزاریە + مووڵک

### پشکنینی باڵانسی هەژمارە تاکەکەسییەکان

بڕۆ بۆ **ڕێکخستنەکان ← هەژمارەکان**

1. کلیک لە هەر هەژمارەیەک بکە بۆ بینینی باڵانسەکەی
2. پشتڕاست بکەوە بڕەکانی کردنەوە لەگەڵ بەڵگەکانی سەرچاوەت دەگونجێن
3. دڵنیابە باڵانسی هەژمارەکانی بانک لەگەڵ ڕاپۆرتە ڕاستەقینەکانی بانک دەگونجێت

### بەکارهێنانی ڕاپۆرتی باڵانسی تاقیکردنەوە

بڕۆ بۆ **ژمێریاری ← ڕاپۆرتەکان ← باڵانسی تاقیکردنەوە**

1. باڵانسی تاقیکردنەوە تا بەرواری کردنەوەت بەڕێوە ببە
2. پشتڕاست بکەوە هەموو باڵانسەکان دروستن
3. پشتڕاست بکەوە کۆی گشتی مەدین یەکسانە لەگەڵ کۆی گشتی داین

---

## لێکۆڵینەوە تایبەتەکان

### ڕێکخستنی فرە-دراو

ئەگەر کۆمپانیاکەت بە چەندین دراو کاردەکات:

1. **دراوی بنەڕەت**: بڕەکان بە دراوی بنەڕەتی کۆمپانیاکەت بنووسە
2. **دراوی بیانی**: نرخی ئاڵوگۆڕ تا بەرواری کردنەوە بەکاربهێنە
3. **جیاوازیی ئاڵوگۆڕ**: هەر ڕاستکردنەوەیەکی وەرگێڕان لە هەژمارەکانی مووڵکدا تۆمار بکە

### باڵانسی هاوبەشەکان

بۆ قەرزاریە بەکرێ و بەبەدهیی ورد بە بەکارهێنانی سیستەمی هاوبەش:

1. **تۆمارەکانی هاوبەش**: دڵنیابە هەموو هاوبەشەکان (کڕیار/فرۆشیار) لە سیستەمدا دروست کراون
2. **فاکتورە تاکەکەسییەکان**: [تۆمارەکانی فاکتور](customer-invoices.ckb.md) تایبەت بۆ هەر بڕێکی کڕیاری بەکرێ دروست بکە
3. **فاکتورەکانی فرۆشیار**: [فاکتورەکانی فرۆشیار](vendor-bills.md) تایبەت بۆ هەر قەرزاریەکی بەبەدهی دروست بکە
4. **دانانی هاوبەش**: هەر باڵانسێک بە هاوبەشی دروست ببەستەوە
5. **مەرجەکانی پارەدان**: [مەرجەکانی پارەدان](payment-terms-guide.md) گونجاو بۆ هاوبەشەکان ڕێکبخە

### باڵانسی کردنەوەی کۆگا

بۆ بازرگانییەکانی پشت بە کۆگا:

1. **تۆمارەکانی بەرهەم**: دڵنیابە هەموو بەرهەمەکان لە سیستەمدا دروست کراون
2. **شێوازی نرخاندن**: شێوازی نرخاندنی یەکگرتوو بەکاربهێنە (FIFO، ناوەند، هتد)
3. **شوێنەکانی کۆگا**: کۆگا بۆ شوێنە دروستەکانی کۆگاخانە دابنێ
4. **بەڕێوەبردنی کۆگا**: [ڕێنماییی بەڕێوەبردنی کۆگا](stock-management-user-guide.md) بۆ وردەکارییەکان ببینە

### ڕێکخستنی ڕێکخستنی بانک

ئەگەر تایبەتمەندییەکانی ڕێکخستنی بانک بەکاردەهێنیت:

1. **چالاککردنی ڕێکخستن**: دڵنیابە "چالاککردنی ڕێکخستن" لە ڕێکخستنەکانی کۆمپانیادا چالاک کراوە
2. **ڕێکخستنی هەژمارە**: "ڕێدانی ڕێکخستن" چالاک بکە بۆ هەژمارەکانی بانک
3. **ڕاپۆرتەکانی بانک**: ئامادەکاری بۆ هاوردەکردنی [ڕاپۆرتەکانی بانک](bank-statements.ckb.md) بۆ ڕێکخستن
4. **ڕێکخستنی کردنەوە**: لێکۆڵینەوە لە ئەنجامدانی [ڕێکخستنی بانک](bank-reconciliation.ckb.md) بۆ بەرواری کردنەوە

## باشترین پراکتیکەکان

### بەڵگەکردن

- **بەڵگەکانی سەرچاوە**: کۆپی لە هەموو بەڵگە پشتگیریکەرەکان بپارێزە
- **پرۆسەی پەسەندکردن**: باڵانسی کردنەوەکان لەلایەن بەڕێوەبەرایەتییەوە پێداچوونەوە بکرێت
- **شوێنی پشکنینەوە**: سەرچاوەی هەر بڕێکی باڵانسی کردنەوە بەڵگە بکە

### کاتخایەنی

- **دەستپێکی ماوە**: باڵانسی کردنەوە لە ڕۆژی یەکەمی ماوەی ژمێریاریت تۆمار بکە
- **ڕێکخستنی بانک**: دڵنیابە باڵانسی بانک لەگەڵ ڕاپۆرتەکان لە بەرواری کردنەوەدا دەگونجێت
- **بڕینەوە**: دڵنیابە هەموو مامەڵەکان لە ماوەی دروستدا تۆمار کراون

### پرۆسەی پێداچوونەوە

1. **پێداچوونەوەی سەربەخۆ**: کەسێکی تر باڵانسی کردنەوەکان پشتڕاست بکاتەوە
2. **باڵانسی تاقیکردنەوە**: باڵانسی تاقیکردنەوەی کردنەوە چاپ بکە و پێداچوونەوە بکە
3. **پەسەندکردنی بەڕێوەبەرایەتی**: پەسەندکردنی فەرمی وەربگرە پێش بەردەوامبوون

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

**کێشە**: تۆمارکردنی باڵانسی کردنەوە باڵانس نابێت
**چارەسەر**: 
1. هەموو مەدین و داینەکان بە جیا کۆ بکەوە
2. بڕی جیاوازی بدۆزەرەوە
3. پشکنین بۆ ژمارە گۆڕدراوەکان یان تۆمارکردنە ونەکان

**کێشە**: باڵانسی بانک لەگەڵ ڕاپۆرت ناگونجێت
**چارەسەر**:
1. پشتڕاست بکەوە بەرواری ڕاپۆرت لەگەڵ بەرواری کردنەوەت دەگونجێت
2. پشکنین بۆ چێک یان پشکەساتە بەکرێنەدراوەکان
3. لێکۆڵینەوە لە دروستکردنی [ڕێکخستنی بانک](bank-reconciliation.ckb.md) بۆ بەرواری کردنەوە

**کێشە**: باڵانسی هاوبەشەکان وا دیارە هەڵەن
**چارەسەر**:
1. پێداچوونەوە بە ڕاپۆرتەکانی قەرزاریە بەکرێ/بەبەدهیی تەمەنی سیستەمی پێشوو
2. پشتڕاستکردنەوەی بڕەکانی فاکتور/فاکتورەکانی تاکەکەسی
3. پشکنین بۆ پارەدانەکان کە ماوەکانیان پەڕاندووە

**کێشە**: ئاڵۆزیی فرە-دراو
**چارەسەر**:
1. دڵنیابە نرخی ئاڵوگۆڕ بە دروستی بۆ بەرواری کردنەوە ڕێکخراوە
2. دراوی بنەڕەتی کۆمپانیا بۆ تۆمارکردنی باڵانسی کردنەوە بەکاربهێنە
3. لێکۆڵینەوە لە ڕاستکردنەوەی وەرگێڕانی دراو لە هەژمارەکانی مووڵکدا

---

## هەنگاوە دواییەکان

دوای دامەزراندنی باڵانسی کردنەوە:

1. **ڕێکخستنی بانک**: باڵانسی کردنەوەی بانک لەگەڵ [ڕاپۆرتەکانی بانک](bank-statements.ckb.md) ڕێک بخە
2. **فاکتورەکانی کڕیار**: دەستپێکی پرۆسەکردنی [مامەڵە نوێیەکانی کڕیار](customer-invoices.ckb.md)
3. **فاکتورەکانی فرۆشیار**: دەستپێکی تۆمارکردنی [مامەڵە نوێیەکانی فرۆشیار](vendor-bills.md)
4. **پرۆسەکردنی پارەدان**: ڕێکخستنی [پرۆسەکردنی پارەدان](payments.ckb.md) بۆ کارەکانی بەردەوام
5. **کارەکانی ئاسایی**: بەردەوامبوون لەگەڵ چالاکیە ڕۆژانەی ژمێریاری

باڵانسی کردنەوەکەت ئێستا دامەزراوە و سیستەمی ژمێریاریت ئامادەی کارەکانی ئاسایییە. هەموو مامەڵەکانی داهاتوو لەسەر ئەم بنچینە بەهێزە بنیاد دەنرێن.

**بیربهێنەرەوە**: باڵانسی کردنەوە بنچینەی ڕاپۆرتی داراییی وردن. کات تەرخان بکە بۆ دڵنیابوون لەوەی تەواو و وردن پێش بەردەوامبوون لەگەڵ مامەڵە ئاساییەکان.
