# دەستپێکردنی یەکەی هەژمارەکان: ئامادەکردنی بناغەی دارایی

ئەم ڕێبەرییە گشتگیرە ڕوونیدەکاتەوە چۆن دەستپێکردنی یەکەی هەژمارەکان کار دەکەن لە سیستەمی ژمێریاریدا، کە خۆی دەبینێتەوە لە ئامادەکردنی هەژمارەکان، دروستکردنی تۆماری ڕۆژانە، و پشتڕاستکردنەوەی هاوسەنگی. ئەم ڕێبەرییە نووسراوە بۆ هەموو بەکارهێنەرێک — ژمێریار بێت یان بەکارهێنەری ئاسایی — لەم ڕێبەرییەدا هەوڵدراوە بە نموونەی کردەیی هەنگاوەکان باس بکرێت کە پەیڕەوی لە باشترین شێوازەکانی ژمێریاری دوو-تۆماری دەکات.

---

## دەستپێکردنی یەکەی هەژمارەکان چین؟

دەستپێکردنی یەکەی هەژمارەکان بڕە پارە سەرەتاییەکانن لە هەژمارەکانتدا لە سەرەتای ماوەی ژمێریارییەکەت، کە نوێنەرایەتی پێگەی دارایی کۆمپانیاکەت دەکەن کاتێک دەست دەکەیت بە بەکارهێنانی سیستەمی ژمێریاریی.

-   **سامانەکان**: ئەوەی کۆمپانیاکەت خاوەندارێتی دەکات (کاش، کۆگا، ئامێر، وەرگیراوەکان)
-   **قەرزەکان**: ئەوەی کۆمپانیاکەت قەرزارە (قەرز، دراوەکان، خەرجییە کۆکراوەکان)
-   **خاوەندارێتی**: پشکی خاوەن لە بازرگانیدا (سەرمایە، قازانجی پارێزراو)
-   **بنەما**: خاڵی دەستپێک بۆ هەموو مامەڵە ژمێریارییەکانی داهاتوو

**بنەمای ژمێریاری**: سامانەکان = قەرزەکان + خاوەندارێتی. دەستپێکردنی یەکەی هەژمارەکانت دەبێت پەیڕەوی لەم هاوکێشە بنەڕەتییەی ژمێریاری بکات بۆ دڵنیابوون لە دروستی ڕاپۆرتە داراییەکان.

---

## پێداویستیەکانی سیستەم

### ڕێکخستنەکانی کۆمپانیا
-   **دانانی کۆمپانیا**: پرۆفایلی کۆمپانیا، دراو، و ساڵی دارایی دەبێت ڕێکخرابن
-   **نەخشەی هەژمارەکان**: هەموو هەژمارە پێویستەکان دروستکرابن و بە دروستی پۆلێنکرابن
-   **ڕێکخستنی پەرتووکی ڕۆژانە**: لانیکەم یەک پەرتووکی ڕۆژانەی جیاواز بۆ تۆمارکردن بەردەست بێت
-   **دانانی دراو**: دراوی بنەڕەتی و هەر دراوێکی بیانی دەبێت چالاک بن

### پێشمەرجەکان
1.  **داتای دارایی بەردەست**: یەکەی کۆتایی ماوەی پێشووتر یان زانیاری سەرەتایی سەرمایە
2.  **زانیاری بانک**: ڕاپۆرتی هەژماری بانکی ئێستا کە باڵانسی کاشی ڕاستەقینە نیشان دەدات
3.  **بڕگە هەڵواسراوەکان**: لیستی وەرگیراوەکان، دراوەکان، و بڕە باڵانسە هەڵواسراوەکانی تر
4.  **بەهای سامانەکان**: بەهای سامانی جێگیر، تێچووی کۆگا، و بڕی بەها کەمکراوەکان

---

## چۆن لە ڕووکاری بەکارهێنەردا بیدۆزینەوە

بڕۆ بۆ **ژمێریاری → تۆماری ڕۆژانە → دروستکردن**

دەستپێکردنی یەکەی هەژمارەکان لەم بەشانەشدا دەردەکەون:
-   **نەخشەی هەژمارەکان**: زانیاری باڵانسی هەژمار دوای ناردنی تۆمارەکان
-   **ڕاپۆرتەکان**: پێداچونەوەی هاوسەنگی و پەڕەی هاوسەنگی کە پێگەی دەستپێک نیشان دەدەن
-   **داشبۆرد**: کورتەی هەژمارەکان کە کاریگەرییەکانی یەکەی دەستپێک ڕەنگدەدەنەوە

**تێبینی**: دوگمەی یارمەتی/بەڵگەنامە لە سەرەوە ئەم ڕێبەرییە دەکاتەوە.

## هەنگاوی ۱: خشتەی هەژمارەکانت ئامادە بکە

بڕۆ بۆ **ڕێکخستنەکان → هەژمارەکان**

### پۆلێنکردنی سەرەکی هەژمارەکان

**سامانەکان (1000-1999)**
-   هەژماری بانک (110101, 110102)
-   هەژماری کاش (110201, 110202)
-   هەژمارەکانی وەرگرتن (120101)
-   کۆگا (130101)
-   سامانی جێگیر (140101)

**قەرزەکان (2000-2999)**
-   هەژمارەکانی دراوەکان (210101)
-   قەرزی بانک (250201)
-   خەرجییە کۆکراوەکان (220101)

**خاوەندارێتی (3000-3999)**
-   سەرمایەی خاوەن (310101)
-   قازانجی پارێزراو (320101)

### پشتڕاستکردنەوەی دانانی هەژمارەکان

دڵنیابەرەوە هەر هەژمارێک خاوەنی:
-   ✅ جۆری هەژماری دروست (سامان، قەرز، خاوەندارێتی)
-   ✅ ڕێکخستنی دراوی گونجاو
-   ✅ هاوتاکردن چالاککراوە بۆ هەژمارەکانی بانک (ئەگەر [هاوتای بانکی](bank-reconciliation.md) بەکاردەهێنیت)

## هەنگاوی ۲: تۆماری ڕۆژانەی دەستپێکردنی یەکەی هەژمارەکان دروست بکە

بڕۆ بۆ **ژمێریاری → تۆماری ڕۆژانە → دروستکردن**

### بەشی سەرەوەی تۆماری ڕۆژانە

1.  **پەرتووکی ڕۆژانە**: پەرتووکی ڕۆژانەی جیاوازی بنەڕەتی خۆت هەڵبژێرە یان پەرتووکی ڕۆژانەی "دەستپێکردنی یەکەی هەژمارەکان" دروست بکە
2.  **دراو**: دراوی بنەڕەتی کۆمپانیاکەت هەڵبژێرە
3.  **بەروار**: یەکەم ڕۆژی ماوەی ژمێریارییەکەت بەکاربێنە
4.  **ژمارەی بەڵگە**: "دەستپێکردنی یەکەی هەژمارەکان" یان "OB-2024"
5.  **شرۆڤە**: "دەستپێکردنی یەکەی هەژمارەکان تا بەرواری [ڕۆژ]"

### زیادکردنی هێڵەکانی تۆماری ڕۆژانە

بۆ هەر هەژمارێک کە یەکەی دەستپێکی هەیە:

**هەژمارەکانی سامان** (باڵانسی قەرزداری ئاسایی)
-   **قەرزدار(مدين)**: بڕی یەکەی دەستپێک بنووسە
-   **قەرزدەر**: بەتاڵ جێیبهێڵە (0)
-   **هەژمار**: هەژماری سامانی دیاریکراو هەڵبژێرە

**هەژمارەکانی قەرز** (باڵانسی قەرزدەری ئاسایی)
-   **قەرزدار(مدين)**: بەتاڵ جێیبهێڵە (0)
-   -   **قەرزدەر**: بڕی یەکەی دەستپێک بنووسە
-   **هەژمار**: هەژماری قەرزی دیاریکراو هەڵبژێرە

**هەژمارەکانی خاوەندارێتی** (باڵانسی قەرزدەری ئاسایی)
-   **قەرزدار(مدين)**: بەتاڵ جێیبهێڵە (0)
-   **قەرزدەر**: بڕی یەکەی دەستپێک بنووسە
-   **هەژمار**: هەژماری خاوەندارێتی دیاریکراو هەڵبژێرە

## هەنگاوی ۳: پشتڕاستکردنەوە و ناردن

### پشتڕاستکردنەوەی باڵانس

1.  **پشکنینی کۆی گشتی**: دڵنیابەرەوە کۆی قەرزدار(مدين) یەکسانە بە کۆی قەرزدەر
2.  **پێداچوونەوەی بڕەکان**: پشتڕاستکردنەوەی هەر بڕێک کە لەگەڵ بەڵگەنامە سەرچاوەییەکانت یەکبگرێتەوە
3.  **پۆلێنکردنی هەژمار**: پشتڕاستکردنەوەی پۆلێنکردنی دروستی هەژمارەکان

### هەڵە باوەکانی پشتڕاستکردنەوە

❌ **تۆمار نا‌باڵانس**: کۆی قەرزدار(مدين) یەکسان نییە بە کۆی قەرزدەر
-   **چارەسەر**: پێداچوونەوە بە هەر هێڵێکی تۆمار و ڕاستکردنەوەی بڕەکان

❌ **جۆری هەژماری هەڵە**: هەژماری سامان بە باڵانسی قەرزدەر
-   **چارەسەر**: پشکنینی پۆلێنکردنی هەژمار یان ئاڕاستەی تۆمار

❌ **هەژماری ونبوو**: بڕگە سەرەکییەکانی پەڕەی هاوسەنگی تۆمارنەکراون
-   **چارەسەر**: سەرەتا هەژمارە ونبووەکان بۆ نەخشەی هەژمارەکان زیاد بکە

### ناردنی تۆمارەکە

1.  کلیک لە **پاشەکەوتکردن** بکە بۆ پاشەکەوتکردن وەک ڕەشنووس
2.  بە وردی پێداچوونەوە بە هەموو هێڵەکاندا بکە
3.  کلیک لە **ناردنی تۆمار** بکە بۆ کۆتاییهێنان بە دەستپێکردنی یەکەی هەژمارەکان

**⚠️ گرنگ**: کاتێک تۆماری دەستپێکردنی یەکەی هەژمارەکان نێردراوە، نابێت دەستکاری بکرێت. ئەگەر پێویست بە ڕاستکردنەوە هەبوو، تۆماری ڕاستکەرەوە دروست بکە.

## هەنگاوی ٤: پشتڕاستکردنەوەی دەستپێکردنی یەکەی هەژمارەکان

### ڕاپۆرتی پەڕەی هاوسەنگی ڕابگەیەنە

بڕۆ بۆ **ژمێریاری → ڕاپۆرتەکان → پەڕەی هاوسەنگی**

1.  ماوەی بەروار بۆ بەرواری دەستپێکردنی یەکەی هەژمارەکانت دابنێ
2.  دڵنیابەرەوە هەموو هەژمارەکان بڕە دەستپێکییە دروستەکان نیشان دەدەن
3.  پشتڕاستکردنەوەی سامانەکان = قەرزەکان + خاوەندارێتی

### پشکنینی باڵانسی هەژمارە تاکەکەسییەکان

بڕۆ بۆ **ڕێکخستنەکان → هەژمارەکان**

1.  کلیک لەسەر هەر هەژمارێک بکە بۆ بینینی باڵانسەکەی
2.  پشتڕاستکردنەوەی بڕە پارە دەستپێکەکان لەگەڵ بەڵگەنامەکانیان سەرچاوەییەکانت یەکبگرنەوە
3.  دڵنیابەرەوە باڵانسی هەژماری بانک لەگەڵ ڕاپۆرتی بانکی ڕاستەقینە یەکدەگرنەوە

### ڕاپۆرتی پێداچونەوەی باڵانس بەکاربێنە

بڕۆ بۆ **ژمێریاری → ڕاپۆرتەکان → پێداچونەوەی باڵانس**

1.  پێداچونەوەی باڵانس وەک بەرواری دەستپێکردنت دیاریبکە
2.  دڵنیابەرەوە هەموو باڵانسەکان دروستن
3.  پشتڕاستکردنەوەی کۆی قەرزدار(مدين) یەکسانە بە کۆی قەرزدەر

## تێبینی تایبەت

### دانانی فرە-دراو

ئەگەر کۆمپانیاکەت زیاد لە دراوێک بەکاردێنێت:

1.  **دراوی بنەڕەتی**: بڕەکان بە دراوی بنەڕەتی کۆمپانیاکەت بنووسە
2.  **دراوی بیانی**: نرخی ئاڵوگۆڕ وەک بەرواری دەستپێک تۆماربکە
3.  **جیاوازی نرخی ئاڵوگۆڕ**: هەر ڕێکخستنێکی ئاڵوگۆڕ لە خاوەندارێتیدا تۆمار بکە

### باڵانسی هاوبەشەکان

بۆ بەدواداچوونی ووردی قەرزە وەرگیراو و دراوەکان سیسیتمی هاوبەشەکان بەکاربێنە:

1.  **تۆمارەکانی هاوبەش**: دڵنیابەرەوە هەموو هاوبەشەکان (کڕیاران/فرۆشیاران) لە سیستەمەکەدا دروستکراون
2.  **پسووڵە تاکەکەسییەکان**: [تۆمارەکانی پسووڵە](customer-invoices.md)ی دیاریکراو بۆ هەر بڕێکی کڕیار کە قەرزەکەی نەدراوە دروست بکە
3.  **پسووڵەکانی فرۆشیار**: [پسووڵەکانی فرۆشیار](vendor-bills.md)ی دیاریکراو بۆ هەر قەرزێک دراوە (کە پێویستە بدرێت) دروست بکە
4.  **تەرخانکردنی هاوبەش**: هەر باڵانسێک بە هاوبەشی دروستەوە ببەستەوە
5.  **مەرجەکانی پارەدان**: [مەرجەکانی پارەدانی](payment-terms-guide.md) گونجاو بۆ هاوبەشەکان دابنێ

### دەستپێکردنی یەکەی هەژمارەکانی کۆگا

بۆ ئەو بازرگانییانەی پشت بە کۆگا دەبەستن:

1.  **تۆمارەکانی بەرهەم**: دڵنیابەرەوە هەموو بەرهەمەکان لە سیستەمەکەدا تۆمارکراون
2.  **شێوازی بەها دانان**: شێوازی تێچوونی هاوشێوە بەکاربێنە (FIFO, Average, هتد.)
3.  **شوێنەکانی کۆگا**: کۆگا بۆ شوێنەکانی کۆگای بەردەست دروستبکە بکە
4.  **بەڕێوەبردنی کۆگا**: بۆ وردەکاری زیاتر، [ڕێبەری بەڕێوەبردنی کۆگا](stock-management-user-guide.md) ببینە

### دانانی هاوسەنگی بانکی

ئەگەر تایبەتمەندی هاوسەنگی بانکی بەکاردەهێنیت:

1.  **چالاککردنی هاوتاکردن**: دڵنیابەرەوە "چالاککردنی هاوتای" لە ڕێکخستنەکانی کۆمپانیا چالاککراوە
2.  **ڕێکخستنەکانی هەژمار**: "ڕێگەدان بە هاوتاکردن" بۆ هەژمارەکانی بانک چالاک بکە
3.  **ڕاپۆرتەکانی بانک**: ئامادەبە بۆ هاوردەکردنی [ڕاپۆرتەکانی بانک](bank-statements.md) بۆ هاوتاکردن
4.  **هاوتای دەستپێک**: بیر لە ئەنجامدانی [هاوتای بانکی](bank-reconciliation.md) بکەرەوە بۆ بەرواری دەستپێک

## باشترین شێوازەکان

### بەڵگەنامەکردن

-   **بەڵگەنامە سەرچاوەییەکان**: کۆپی هەموو بەڵگەنامە پشتگیریکەرەکان بپارێزە
-   **پڕۆسەی پەسەندکردن**: دەستپێکردنی یەکەی هەژمارەکان لەلایەن بەڕێوەبەرایەتییەوە پێداچوونەوەی بۆ بکرێت
-   **ڕێڕەوی پشکنین**: سەرچاوەی هەر بڕێکی یەکەی دەستپێک بەڵگەی پێوە بلکێنرێت

### کات

-   **سەرەتای ماوەی ژمێریاری**: دەستپێکردنی یەکەی هەژمارەکان لە یەکەم ڕۆژی ماوەی ژمێریارییەکەت بنووسە
-   **هاوتای بانکی**: دڵنیابەرەوە هاوسەنگی بانکی لەگەڵ ڕاپۆرتەکاندا یەکدەگرێتەوە لە بەرواری دەستپێکدا
-   **ڕێکخستنی سنووری کاتی**: دڵنیابەرەوە هەموو مامەڵەکان لە ماوەی دروستدا تۆمارکراون

### پڕۆسەی پێداچوونەوە

1.  **پێداچوونەوەی سەربەخۆ**: با کەسێکی تر دەستپێکردنی یەکەی هەژمارەکان پشتڕاست بکاتەوە
2.  **پێداچونەوەی باڵانس**: پێداچونەوەی باڵانسی دەستپێک چاپ بکە و پێداچوونەوەی بۆ بکە
3.  **پەسەندکردنی بەڕێوەبەرایەتی**: پێش بەردەوامبوون، پەسەندکردنی فەرمی وەربگرە

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

**کێشە**: تۆماری دەستپێکردنی یەکەی هەژمارەکان باڵانس نابێت
**چارەسەر**:
1.  هەموو قەرزدار(مدين) و قەرزدەرەکان بە جیا کۆ بکەرەوە
2.  بڕی جیاوازییەکە بدۆزەرەوە
3.  پشکنین بکە بۆ ژمارە گۆڕدراوەکان یان تۆمارە ونبووەکان

**کێشە**: باڵانسی بانک لەگەڵ ڕاپۆرتەکەدا یەکناگرێتەوە
**چارەسەر**:
1.  پشتڕاستکردنەوەی بەرواری ڕاپۆرتەکە لەگەڵ بەرواری دەستپێکردنت یەکبگرێتەوە
2.  پشکنین بکە بۆ چێکە هەڵواسراوەکان یان پارەدانە گواستراوەکان
3.  بیر لە دروستکردنی [هاوتای بانکی](bank-reconciliation.md) بکەرەوە بۆ بەرواری دەستپێک

**کێشە**: باڵانسەکانی هاوبەش هەڵە دەردەکەون
**چارەسەر**:
1.  پێداچوونەوە بە ڕاپۆرتەکانی قەرزی وەرگیراوە/دراوە کۆنەکانی سیستەمی پێشووتر
2.  پشتڕاستکردنەوەی بڕەکانی پسووڵە کڕیار/پسووڵەی فرۆشیاری تاکەکەسی
3.  پشکنین بکە بۆ پارەدانەکان کە سنووری ماوەیان بەزاندووە

**کێشە**: ئاڵوگۆڕی دراوی فرە-دراو
**چارەسەر**:
1.  دڵنیابەرەوە نرخی ئاڵوگۆڕ بۆ بەرواری دەستپێک بە دروستی دانراون
2.  دراوی بنەڕەتی کۆمپانیا بۆ تۆماری دەستپێکردنی یەکەی هەژمارەکان بەکاربێنە
3.  بیر لە ڕێکخستنی گۆڕینی دراو بکەرەوە لە هەژمارەکانی خاوەندارێتیدا

---

## هەنگاوەکانی داهاتوو

دوای دانانی دەستپێکردنی یەکەی هەژمارەکان:

1.  **هاوتای بانکی**: هاوسەنگی بانکی دەستپێک لەگەڵ [ڕاپۆرتەکانی بانک](bank-statements.md) هاوتا بکە
2.  **پسووڵەکانی کڕیار**: دەست بکە بە تۆمارکردنی [مامەڵە نوێکانی کڕیار](customer-invoices.md)
3.  **پسووڵەکانی فرۆشیار**: دەست بکە بە تۆمارکردنی [مامەڵە نوێکانی فرۆشیار](vendor-bills.md)
4.  **پرۆسەی پارەدان**: [پرۆسەی پارەدان](payments.md) بۆ کارەکان بە بەردەوامی دابنێ
5.  **کارە ئاساییەکانی ڕۆژانە**: بەردەوام بە لەگەڵ چالاکییە ڕۆژانەکانی ژمێریاری

دەستپێکردنی یەکەی هەژمارەکانت ئێستا دامەزراوە و سیستەمی ژمێریارییەکەت ئامادەیە بۆ کارەکانی بە شێوەیەکی ئاسایی. هەموو مامەڵەکانی داهاتوو لەسەر ئەم بنەما پتەوە بنیات دەنرێن.

---

**لەبیرت بێت**: دەستپێکردنی یەکەی هەژمارەکان بنەمای دروستی ڕاپۆرتە داراییەکانن. کات تەرخان بکە بۆ دڵنیابوون لە دروستییان پێش بەردەوامبوون لەگەڵ مامەڵە ئاساییەکان.
