# تۆمارەکانی ڕۆژانە: تۆمارە ژمێریارییە دەستییەکان

ئەم ڕێبەرە ڕوونی دەکاتەوە کە چۆن **تۆمارەکانی ڕۆژانە (Journal Entries)** لە سیستەمەکەدا کاردەکەن. هەرچەندە زۆربەی تۆمارەکان بە شێوەی ئۆتۆماتیکی دروست دەکرێن (وەک دروستکردنی پسووڵەیەک)، بەڵام هەندێک جار پێویستە مامەڵەکان بە دەستی تۆمار بکەیت بۆ ئەوەی دەفتەرەکانت هاوسەنگ بن.

---

## تۆماری ڕۆژانە چییە؟

وەک داتای خاوی ژمێریاری سەیری **تۆماری ڕۆژانە** بکە. بریتییە لە تۆماری مامەڵەیەکی دارایی کە کاریگەری لەسەر لانی کەم دوو هەژمار (Account) هەیە.

**بۆچی ئەمە گرنگە؟**
1.  **ڕاستکردنەوە**: هەڵەکان چاک بکەوە یان باڵانسەکان ڕێکبخەوە.
2.  **مامەڵە نائاساییەکان**: تۆمارکردنی وەک داخوران (Depreciation)، مووچە، یان کرێی بانک بە دەستی ئەگەر پێویست بوو.
3.  **باڵانسی کردنەوە (Opening Balances)**: هێنانی داتا لە سیستەمە کۆنەکانەوە.

---

## لە کوێ دەیدۆزیتەوە

بڕۆ بۆ: **ژمێریاری -> ژمێریاری -> تۆمارەکانی ڕۆژانە**

لیستێک لە هەموو تۆمارەکانی ڕۆژانە دەبینیت (هەم ئۆتۆماتیکی و هەم دەستی).

---

## دروستکردنی تۆمارێکی ڕۆژانەی دەستی

با باسی هەنگاوەکانی دروستکردنی تۆمارێکی نوێ بکەین.

### هەنگاوی 1: دەستپێکردن

بڕۆ بۆ **ژمێریاری -> تۆمارەکانی ڕۆژانە -> دروستکردن**

فۆرمێک دەبینیت بەم زانیارییانەوە:

| بوار | وەسف | نموونە |
|-------|-------------|---------|
| **ڕۆژنامە (Journal)** | ئەو پەرتووکەی ئەم تۆمارەی تێدەچێت (وەک گشتی، بانک) | "کارەکانی جۆراوجۆر" |
| **سەرچاوە (Reference)** | ژمارەیەکی تایبەت بۆ ئەم مامەڵەیە | "ADJ/2024/001" |
| **ڕێکەوت** | ڕێکەوتی ئەنجامدانی مامەڵەکە | ڕێکەوتی ئەمڕۆ |
| **دراو** | جۆری دراوی مامەڵە | "USD" |

### هەنگاوی 2: زیادکردنی بڕگەکانی ڕۆژانە (Journal Items)

لێرەدا بنەمای "دوو-تۆمار" (Double-Entry) جێبەجێ دەکەیت. دەبێت لانی کەم دوو هێڵ زیاد بکەیت.

| بوار | وەسف |
|-------|-------------|
| **هەژمار (Account)** | هەژماری پەرتووکی گشتی (نموونە: 5001 خەرجی ئۆفیس) |
| **هاوبەش (Partner)** | (بژاردە) ئەو کڕیار یان فرۆشیارەی پەیوەندی بەم هێڵەوە هەیە |
| **ناونیشان/تێبینی (Label)** | وەسفی ئەم هێڵە دیاریکراوە |
| **قەرزدار (Debit)** | بڕی پارەی زیادبوونی سامان/خەرجی یان کەمبوونی قەرز/سەرمایە |
| **قەرزدەر (Credit)** | بڕی پارەی زیادبوونی قەرز/سەرمایە/داهات یان کەمبوونی سامان |

> [!IMPORTANT]
> **یاسای دوو-تۆمار**: دەبێت کۆی **قەرزدار** یەکسان بێت بە کۆی **قەرزدەر**. سیستەمەکە ڕێگەت پێ نادات تۆمارێکی هاوسەنگ نەبوو بنێریت (Post).

### هەنگاوی 3: پێداچوونەوە و پاشەکەوتکردن

1.  دڵنیابەرەوە کە بڕی **جیاوازییەکان (Differences)** لە خوارەوە **0.00** بێت.
2.  کلیک لە **دروستکردن** بکە بۆ پاشەکەوتکردن وەک **ڕەشنووس (Draft)**.

---

## تێگەیشتن لە بارودۆخەکان (Statuses)

تۆماری ڕۆژانەکەت بەم قۆناغانەدا دەڕوات:

┌─────────┐      ┌─────────┐
│  ڕەشنووس   │ ──▶ │  نێردراو  │
└─────────┘      └─────────┘
    📝              ✅

### 📝 ڕەشنووس (Draft)
- تۆمارەکە پاشەکەوت کراوە بەڵام هێشتا **کاریگەری نییە** لەسەر ڕاپۆرتە داراییەکانت.
- دەتوانیت دەستکاری بکەیت یان بیسڕیتەوە.
- بەسوودە بۆ ئامادەکردنی تۆمارەکان پێش کۆتایی هێنان.

### ✅ نێردراو/تۆمارکراو (Posted)
- تۆمارەکە "فەرمی"یە و پەرتووکی گشتی (General Ledger) نوێ دەکاتەوە.
- ئێستا قوفڵ بووە و ناتوانرێت بسڕدرێتەوە (بۆ پاراستنی مێژووی وردبینی).
- لە تەرزووی پێداچوونەوە (Trial Balance) و ڕاپۆرتەکانی تر دەردەکەوێت.

---

## پێچەوانەکردنەوەی تۆمار

ئەگەر هەڵەت کرد لە تۆمارێکی **نێردراو (Posted)**، ناتوانیت تەنها بیسڕیتەوە. لە جیاتی ئەوە، دەبێت **پێچەوانەی بکەیتەوە (Reverse)**.

1.  تۆمارە نێردراوەکە بکەوە.
2.  کلیک لە دوگمەی **پێچەوانەکردنەوەی تۆمار** بکە (ئەگەر بەردەست بێت) یان تۆمارێکی نوێ دروست بکە بە پێچەوانەی بەهاکانی قەرزدار/قەرزدەر.
3.  ئەمە تۆمارێکی "هەڵوەشاندنەوە" دروست دەکات، بۆ ئەوەی مێژووی وردبینییەکەت پاک بێت.

---

## چارەسەرکردنی کێشەکان

### پرسیارە باوەکان

**پرسیار: بۆچی ناتوانم تۆمارەکەم بنێرم (Post)؟**
وەڵام: دڵنیابەرەوە کە لایەنی قەرزدار و قەرزدەر یەکسانن. سیستەمەکە پێویستی بە تۆماری هاوسەنگ هەیە. هەروەها، دڵنیابەرەوە ڕێکەوتەکە لە ناو **ماوەی قوفڵدراو (Locked Period)** دا نییە.

**پرسیار: ئایا دەتوانم تۆمارەکان هاوردە (Import) بکەم؟**
وەڵام: بەڵێ! کردارەکانی **هاوردەکردن (Import)** لە پەڕەی لیستەکە بەکاربهێنە بۆ بارکردنی فایلێکی CSV/Excel ە تۆمارەکانی ڕۆژانە.

**پرسیار: پارەکە لە کوێوە دێت؟**
وەڵام: لەبیرت بێت:
- **قەرزدار (Debit)** = زیادبوونی سامان (پارە دێتە ناوەوە/خەرجی کراوە)
- **قەرزدەر (Credit)** = زیادبوونی قەرز/داهات (سەرچاوەی پارە)
