# ڕاپۆرتەکانی بانک: بنەمای بەڵگەکردنی گەرانی پارە و ڕێکخستنی بانک

ئەم ڕێنماییە ڕوونکردنەوەی چۆنێتی دروستکردن و بەڕێوەبردنی ڕاپۆرتەکانی بانک لە سیستەمی ژمێریاریت دەکات. ڕاپۆرتەکانی بانک گرنگن بۆ چاودێریکردنی جووڵەی ڕاستەقینەی پارە و بنەمای [پرۆسەکانی ڕێکخستنی بانک](bank-reconciliation.ckb.md) پێکدەهێنن.

---

## ڕاپۆرتەکانی بانک چین؟

ڕاپۆرتەکانی بانک تۆمارکردنی هەموو ئەو مامەڵانەن کە لە ماوەیەکی دیاریکراودا لە هەژمارەکانی بانکتدا ڕوویاندانە. ئەوان ئەمانە لەخۆدەگرن:
- **بڕی دەستپێک**: بڕی هەژماری لە سەرەتای ماوەکە
- **مامەڵەکان**: هەموو دانەپارە، دەرکردن و کرێی بانکەکان
- **بڕی کۆتایی**: بڕی هەژماری لە کۆتایی ماوەکە

**مەبەست**: ڕاپۆرتەکانی بانک دەرخستنی دەرەکی پێداویستیان دابیندەکەن بۆ دڵنیابوون لەوەی تۆمارە ناوخۆییەکانی پارەت لەگەڵ تۆمارەکانی بانک دادەگرن لە ڕێگەی [پرۆسەی ڕێکخستن](bank-reconciliation.ckb.md)ەوە.

---

## پێکهاتەکانی ڕاپۆرتی بانک

### سەرەوەی ڕاپۆرت
- **ژۆرنالی بانک**: ژۆرنالی تایبەتی بانک (دەبێت جۆری "بانک" بێت)
- **دراو**: دراوی ڕاپۆرت (بنەڕەت دراوی کۆمپانیا)
- **ژمارەی سەرچاوە**: ژمارەی ڕاپۆرت یان ناسێنەر
- **بەروار**: بەرواری ڕاپۆرت
- **بڕی دەستپێک**: بڕی کراوەیی لە ڕاپۆرتی پێشوو
- **بڕی کۆتایی**: بڕی دابەندان بۆ ماوەکە

### ڕیزەکانی ڕاپۆرت (مامەڵەکان)
- **بەروار**: بەرواری مامەڵە
- **پێناسە**: پێناسەی مامەڵە لە بانکەوە
- **هاوبەش**: کڕیار/فرۆشیار ئەگەر هەبێت
- **بڕ**: بڕی مامەڵە (ئەرێنی بۆ دانەپارە، نەرێنی بۆ دەرکردن)
- **دراوی بیانی**: دڵخواز - ئەگەر مامەڵە لە دراوی جیاوازدا بووبێت
- **بڕ بە دراوی بیانی**: بڕی ڕەسەن پێش گۆڕین

---

## دروستکردنی ڕاپۆرتەکانی بانک

بڕۆ بۆ **ژمێریاری → بانک و پارەی نەقد → ڕاپۆرتەکانی بانک → دروستکردن**

### پرۆسەی دەستی تێخستن

#### هەنگاوی ۱: زانیاری سەرەوەی ڕاپۆرت
**دراو**: دراوی ڕاپۆرت هەڵبژێرە (بنەڕەت دراوی کۆمپانیا)
**ژۆرنالی بانک**: ژۆرنالی بانک هەڵبژێرە (دەبێت جۆری "بانک" بێت)  
**ژمارەی سەرچاوە**: ژمارەی ڕاپۆرت تێبخە (نموونە "STMT-2024-001")
**بەروار**: بەرواری ڕاپۆرت تێبخە
**بڕی دەستپێک**: بڕی کراوەیی لە ڕاپۆرتی پێشوو تێبخە
**بڕی کۆتایی**: بڕی دابەندان لە ڕاپۆرتی ئێستا تێبخە

#### هەنگاوی ۲: زیادکردنی ڕیزەکانی ڕاپۆرت
خشتەی ڕیزەکانی ڕاپۆرت بەکاربهێنە بۆ زیادکردنی هەر مامەڵەیەک:

**بەروار**: بەرواری مامەڵە لە ڕاپۆرتی بانکەوە
**پێناسە**: پێناسەی بانک بۆ مامەڵەکە
**هاوبەش**: بەستنەوە بە کڕیار/فرۆشیار ئەگەر هەبێت (دڵخواز)
**بڕ**: 
- **بەهاکانی ئەرێنی**: بۆ دانەپارە، گۆڕاندنی ناوەوە، سوودی بەدەستهاتوو
- **بەهاکانی نەرێنی**: بۆ دەرکردن، چێک، کرێ، گۆڕاندنی دەرەوە

**بوارەکانی دراوی بیانی** (دڵخواز بۆ مامەڵە زۆر دراویەکان):
- **دراوی بیانی**: دراوی مامەڵەی ڕەسەن هەڵبژێرە
- **بڕ بە دراوی بیانی**: بڕی ڕەسەن پێش گۆڕین تێبخە

#### هەنگاوی ۳: دڵنیابوون و پاشەکەوتکردن
1. **دڵنیابوون لە بڕ**: سیستەم بە شێوەی خۆکار بڕی کۆتایی حیسابدەکات
2. **پێداچوونەوەی ڕیزەکان**: دڵنیابە هەموو مامەڵەکان بە دروستی تێخراون
3. **پاشەکەوتکردن**: ڕاپۆرتەکە دروستبکە و ئامادەی بکە بۆ [ڕێکخستن](bank-reconciliation.ckb.md)

---

## جۆرەکانی ڕیزەی ڕاپۆرتی بانک

### دانەپارە و قەرزەکان
**پارەدانی کڕیاران**: پارە وەرگیراو لە کڕیارانەوە
**گۆڕاندنی ناوەوەی بانک**: پارە گۆڕدراو لە هەژمارەکانی ترەوە
**سوودی بەدەستهاتوو**: پارەدانی سوودی بانک
**دەرامەتی قەرز**: پارە وەرگیراو لە قەرزەکانەوە
**داهاتی تر**: دانەپارەی جۆراوجۆر

### دەرکردن و قەرزەکان
**پارەدانی فرۆشیاران**: پارەدراو بە دابینکەران
**گۆڕاندنی دەرەوەی بانک**: پارە گۆڕدراو بۆ هەژمارەکانی تر
**کرێی بانک**: کرێی مانگانە، کرێی مامەڵەکان
**پارەدانی قەرز**: پارەدانی سەرەکی و سوود
**خەرجی تر**: دەرکردنی جۆراوجۆر

### بڕگەکانی تایبەت بە بانک
**کرێی NSF**: کرێی کەمی پارە
**کرێی زیادەپارە**: کرێی زیادە لە هەژمار
**کرێی گۆڕاندنی تەل**: کرێی گۆڕاندنی تەل
**چاپکردنی چێک**: خەرجی داواکاری چێک
**کرێی ATM**: کرێی بانکەی خۆکار

## بیرکردنەوەکانی زۆر دراوی

### مامەڵەی دراوی بیانی
سیستەم پشتگیری مامەڵەکانی سەرەتا لە دراوە جیاوازەکاندا کراو دەکات:

1. **دراوی ڕاپۆرت**: هەموو بڕەکان لە دراوی بنەڕەتی ڕاپۆرتدا هەڵدەگیردرێن
2. **دراوی ڕەسەن**: بەدواداچوونی دراو و بڕی مامەڵەی ڕەسەن
3. **ڕێژەی گۆڕان**: گۆڕان کاتی تۆمارکردنی مامەڵە بەڕێوەدەچێت
4. **پیشاندان**: بڕی ڕەسەن و گۆڕدراو هەردووک دیارن

### هەژمارەکانی بانکی زۆر دراوی
کاتی بەڕێوەبردنی هەژمارەکانی بانکی دراوی بیانی:
- دەبێت هەر ڕاپۆرتێک لەگەڵ دراوی ژۆرنالەکە بگونجێت
- ڕێژەکانی گۆڕان لە کاتی تێخستنی مامەڵەدا جێبەجێدەکرێن
- [ڕێکخستن](bank-reconciliation.ckb.md) جیاوازیەکانی دراو بە شێوەی خۆکار بەڕێوەدەبات

---

## بەڕێوەبردنی ڕاپۆرتی بانک

### دەستپێگەیشتنی ڕاپۆرت
- **دروستکردن**: بەردەستە بۆ بەکارهێنەرانی مۆڵەتدراوی ژمێریاری
- **دەستکاریکردن**: دەستکاریی ڕاپۆرتەکانی ڕێکنەخراو تەنها
- **بینین**: هەموو بەکارهێنەران دەتوانن ڕاپۆرتەکان ببینن بەپێی مۆڵەتەکان
- **ڕێکخستن**: بەردەستە کاتێک ڕێکخستن بۆ کۆمپانیاکە چالاککراوە

### دۆخی ڕاپۆرت
ڕاپۆرتەکان دۆخێکی ناڕاستەوخۆیان هەیە بەپێی دۆخی ڕێکخستنەکەیان:
- **ڕەشنووس**: تازە دروستکراو، دەتوانرێت بە ئازادی دەستکاری بکرێت
- **ئامادە**: ڕاپۆرتی تەواو، ئامادە بۆ [ڕێکخستن](bank-reconciliation.ckb.md)
- **بەشێک ڕێکخراو**: هەندێک ڕیز لە ڕێگەی ڕێکخستنەوە پرۆسەکراون
- **تەواو ڕێکخراو**: هەموو ڕیزەکان لە ڕێگەی ڕێکخستنەوە پرۆسەکراون

### دەستکاریکردنی ڕاپۆرتەکان
- **پێش ڕێکخستن**: توانای دەستکاریکردنی تەواو
- **لە کاتی ڕێکخستندا**: دەستکاریکردنی سنووردار پێشنیاردەکرێت
- **دوای ڕێکخستن**: دروستکردنی تۆمارەی ڕاستکردنەوەی نوێ لە جیاتی دەستکاریکردن

---

## یەکخستن لەگەڵ پارەدانەکان

### بەدواداچوونی پەیوەندی خۆکار
سیستەم بەدواداچوونی پەیوەندیەکان دەکات لە نێوان:
- **ڕیزەکانی ڕاپۆرتی بانک**: مامەڵەکانی بانکی دەرەکی
- **تۆمارەکانی پارەدان**: تۆمارەکانی پارەدانی ناوخۆیی
- **دۆخی ڕێکخستن**: ئایا بڕگەکان یەکگیراون یان نا

### بەستنەوەی پارەدان
لە کاتی [ڕێکخستنی بانکدا](bank-reconciliation.ckb.md):
1. **پارەدانەکانی ڕێکنەخراو**: سیستەم پارەدانە پشتڕاستکراوەکان بە بێ بەستنەوەی ڕاپۆرتی بانک پیشاندەدات
2. **پرۆسەی یەکخستن**: بەکارهێنەران پارەدانەکان لەگەڵ ڕیزەکانی ڕاپۆرت یەکدەخەن
3. **نوێکردنەوەی دۆخ**: پارەدانە یەکخراوەکان دۆخیان بۆ "ڕێکخراو" دەگۆڕێت
4. **پەیوەندی کلیلی بیانی**: بەستنەوەی ڕاستەوخۆ لە نێوان تۆمارەکانی پارەدان و ڕیزەی ڕاپۆرت

### بینینی بەڕێوەبەری پەیوەندی
لە تۆمارەکانی پارەدانەوە، بەکارهێنەران دەتوانن ببینن:
- ڕیزەکانی ڕاپۆرتی بانکی پەیوەندیدار
- دۆخ و بەرواری ڕێکخستن
- سەرچاوە و بڕەکانی ڕاپۆرت

---

## ڕاپۆرت و شیکردنەوە

### بینینەکانی بەڕێوەبردنی ڕاپۆرت
**لیستی ڕاپۆرتی بانک**: تێڕوانینی گشتی هەموو ڕاپۆرتەکان لەگەڵ زانیاری گرنگ
- سەرچاوە، ژۆرنالی بانک، بەروار، دراو
- بڕی دەستپێک و کۆتایی
- بەرواری دروستکردن و دەستکاریکردن

**وردەکاریەکانی ڕاپۆرت**: بینینی تاکی ڕاپۆرت پیشاندانی:
- زانیاری تەواوی سەرەوە
- هەموو ڕیزەکانی مامەڵە لەگەڵ بڕ و پێناسەکان
- نیشانکەرەکانی دۆخی ڕێکخستن

### ڕاپۆرتەکانی یەکخستن
**شیکردنەوەی گەرانی پارە**: دراوەکانی ڕاپۆرت بەشدارین لە:
- ڕاپۆرتکردنی دۆخی پارە
- شیکردنەوەی ئاوی خوێن
- دڵنیابوونەوەی بڕی بانک
- بەدواداچوونی دۆخی ڕێکخستن

---

## باشترین ڕێگاکان

### تێخستنی دراوە
- **زانیاری دروست**: دووجار چاو بە هەموو بڕ و بەرواران بخشێنە
- **پێناسەی تەواو**: پێناسەی واتاداری مامەڵەکان لەخۆبگرە
- **بەستنەوەی هاوبەش**: مامەڵەکان لەگەڵ کڕیار/فرۆشیاران بەستنەوە کاتێک گونجاوە
- **دراوی بیانی**: بڕەکانی ڕەسەن تۆمار بکە بۆ بەدواداچوونی باشتر

### بەڕێوەبردنی کارگێڕی
- **تێخستنی ئاسایی**: ڕاپۆرتەکان بە خێرایی دروست بکە دوای وەرگرتنی ڕاپۆرتەکانی بانک
- **پێداچوونەوەی سیستەماتیک**: بڕە حیسابکراوەکان دڵنیابکەرەوە پێش پاشەکەوتکردن
- **ڕێکخستنی خێرا**: [پرۆسەی ڕێکخستن](bank-reconciliation.ckb.md) دەست پێبکە ڕاستەوخۆ دوای دروستکردن

### کۆنترۆڵی دەستپێگەیشتن
- **مۆڵەتی بەکارهێنەر**: دروستکردنی ڕاپۆرت سنووربکە بۆ ستافی مۆڵەتدراوی ژمێریاری
- **پرۆسەی پێداچوونەوە**: کارگێڕی پەسەندکردن جێبەجێ بکە بۆ مامەڵە گرنگەکان
- **ڕێگای پشکنین**: مێژووی تەواوی گۆڕانکاری پارێزراو بکە

---

## سیناریۆ باوەکان

### سیناریۆی ۱: دروستکردنی ڕاپۆرتی مانگانە
دروستکردنی ڕاپۆرتی بانکی مانگانە لەگەڵ ۲۵ مامەڵە:

1. بڕۆ بۆ ڕاپۆرتەکانی بانک ← دروستکردن
2. دراو و ژۆرنالی بانک هەڵبژێرە
3. سەرچاوە و بەرواری ڕاپۆرت تێبخە
4. بڕی دەستپێک و کۆتایی تێبخە
5. ڕیزەکانی مامەڵە بە بەکارهێنانی ڕووکاری خشتە زیاد بکە
6. حیسابکردنی بڕ دڵنیابکەرەوە
7. ڕاپۆرت پاشەکەوت بکە و بڕۆ بۆ [ڕێکخستن](bank-reconciliation.ckb.md)

### سیناریۆی ۲: تێخستنی مامەڵەی زۆر دراوی
تۆمارکردنی مامەڵەیەکی دراوی بیانی:

1. ڕاپۆرت لە دراوی بنەڕەتدا دروست بکە (نموونە IQD)
2. ڕیزی مامەڵە لەگەڵ بڕی گۆڕدراو زیاد بکە
3. دراوی ڕەسەن هەڵبژێرە (نموونە USD)
4. بڕی ڕەسەن بە دراوی بیانی تێبخە
5. سیستەم هەردوو بڕەکە بۆ بەدواداچوون پارێزگاری دەکات

### سیناریۆی ۳: ڕاستکردنەوەی هەڵە
ڕاستکردنەوەی بڕی هەڵەی مامەڵە:

1. ڕاپۆرتەکە دەستکاری بکە (ئەگەر تا ئێستا ڕێکنەخراوە)
2. ڕیزی مامەڵەی هەڵە دەستکاری بکە
3. دڵنیابەرەوە بڕی کۆتایی هێشتا دروستە
4. گۆڕانکارییەکان پاشەکەوت بکە
5. ئەگەر پێشتر ڕێکخراوە، ڕاپۆرتی ڕاستکردنەوەی نوێ دروست بکە

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

**کێشە**: ناتوانرێت ژۆرنالی بانک هەڵبژێردرێت
**چارەسەر**:
- دڵنیابە جۆری ژۆرنالەکە دامەزراوە بۆ "بانک"
- دڵنیابەرەوە ژۆرنالەکە سەر بە کۆمپانیا ئێستاکەیە
- مۆڵەتەکانی بەکارهێنەر بۆ دەستپێگەیشتنی ژۆرنال بپشکنە

**کێشە**: حیسابکردنی بڕ نایەکگرێتەوە
**چارەسەر**:
- هەموو بڕ و نیشانەکانی مامەڵەکان پێداچوونەوە بکە
- بگەڕێ بۆ تۆمارەکانی ون یان دووبارەبووەوە
- دڵنیابە بڕی دەستپێک دروستە
- دڵنیابە گۆڕینەکانی دراوی بیانی دروستن

**کێشە**: ناتوانرێت ڕاپۆرت دەستکاری بکرێت
**چارەسەر**:
- بپشکنە ئایا ڕیزەکانی ڕاپۆرت ڕێکخراون
- مۆڵەتەکانی دەستکاریکردنی بەکارهێنەر دڵنیابکەرەوە
- بیرلە دروستکردنی ڕاپۆرتی ڕاستکردنەوەی نوێ لە جیاتی

---

## هەنگاوەکانی داهاتوو

کاتێک ڕاپۆرتی بانکەکەت دروست و دڵنیاکراوە:

1. **بڕۆ بۆ ڕێکخستن**: تایبەتمەندی [ڕێکخستنی بانک](bank-reconciliation.ckb.md) بەکاربهێنە بۆ یەکخستنی ڕیزەکانی ڕاپۆرت لەگەڵ پارەدانەکانی سیستەم
2. **چاودێری دۆخ**: پێشکەوتنی ڕێکخستن لە ڕێگەی بینینەکانی ڕاپۆرتەوە بەدوادا بخە
3. **دروستکردنی ڕاپۆرتەکان**: دراوەکانی ڕاپۆرت بەکاربهێنە بۆ شیکردنەوەی گەرانی پارە و ڕاپۆرتکردنی دارایی

---

ڕاپۆرتەکانی بانک بنەمای بەڕێوەبردنی دروستی پارە پێکدەهێنن و دەرخستنی دەرەکی پێداویستی دابیندەکەن بۆ پرۆسە بەڵکداراکانی [ڕێکخستنی بانک](bank-reconciliation.ckb.md).
