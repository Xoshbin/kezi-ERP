# ڕاپۆرتەکانی هەژماری بانک: بنەمای بەڵگەکردنی گەرانی پارە و هاوسەنگی بانکی

ئەم ڕێنماییە فراوانە ڕوونکردنەوەی چۆنێتی دروستکردن و بەڕێوەبردنی ڕاپۆرتەکانی هەژماری بانک لە سیستەمی ژمێریاریت دەکات، و پرۆسەی تێخستنی مامەڵەکان، مامەڵەکردن لەگەڵ چەندین دراو، و یەکخستن لەگەڵ [هاوسەنگی بانکی](bank-reconciliation.ckb.md) دەگرێتەوە. نووسراوە بۆ هەموو بەکارهێنەران — ژمێریاران و نا-ژمێریاران — و ڕێنماییە کراوەییەکان دابیندەکات کە شوێن باشترین ڕێگاکانی ژمێریاری دووقات دەکەوێت.

**گرنگ**: هەموو مامەڵە بانکییەکان پێداویستیان بە ڕاپۆرتی هەژماری بانکی دروست هەیە بۆ پرۆسەکردنی گونجاو و کارەکانی هاوسەنگی.

---

## ڕاپۆرتەکانی هەژماری بانک چین؟

ڕاپۆرتەکانی هەژماری بانک تۆمارکردنی کاتی فراوانی هەموو ئەو مامەڵانەن کە لە ماوەیەکی دیاریکراودا لە هەژمارەکانی بانکی ڕێکخراوەکەتدا ڕوویاندانە. ئەوان چەندین مەبەستی گرنگی خزمەتدەکەن:

- **چاودێری مامەڵەکان**: تۆمارکردنی هەموو گەرانی پارەی هاتووە ناوەوە و چووە دەرەوە
- **یەکخستنی تۆمارەکان**: دابینکردنی سەرچاوە بۆ بەراوردکردن لەگەڵ تۆمارە ناوخۆییەکان
- **کۆنترۆڵی دارایی**: دڵنیابوون لە ڕۆشنایی و بەرپرسیارێتی لە مامەڵە بانکییەکان
- **پابەندبوون و پشکنین**: دابینکردنی پێداویستییەکانی پشکنین و ڕاپۆرتکردنی یاسایی

**بنەمای ژمێریاری**: هەر مامەڵەیەکی بانک دەبێت تۆمارێکی هاوتای لە سیستەمی ژمێریاری تۆدا هەبێت، کە ڕێگەی پشکنینی تەواو دروستدەکات و دڵنیابوون لە وردی ڕاپۆرتەکانی کاش دەکات.

---

## پێداویستییەکانی سیستەم

### ڕێکخستنی کۆمپانیا
- **هەژمارەکانی بانک**: دەبێت هەژمارەکانی بانک لە پەرتووکی گشتیدا ڕێکبخرێن
- **پەرتووکی ڕۆژانەکانی بانک**: دەبێت پەرتووکی ڕۆژانەکان لە جۆری "بانک"ەوە دروستبکرێن
- **مۆڵەتەکانی بەکارهێنەر**: بەکارهێنەران پێداویستیان بە مۆڵەتی گونجاو هەیە بۆ دروستکردنی ڕاپۆرتەکانی هەژماری بانک

### پێشمەرجە سەرەتاییەکان
1. **ڕێکخستنی بانک**: دەبێت وردەکاریەکانی بانک و زانیاری هەژمار ڕێکبخرێت
2. **ڕێکخستنی دراو**: دەبێت دراوەکان و نرخی ئاڵوگۆڕ بۆ مامەڵە چەندین دراوییەکان ڕێکبخرێن
3. **هاوتەریبی دراوە**: دڵنیابوون لەوەی دراوەکانی بانک لەگەڵ ڕێکخستنەکانی سیستەم بگونجێن

---

## شوێنی دۆزینەوەی لە ڕووکاری بەکارهێنەردا

بڕۆ بۆ **ژمێریاری ← بانک و کاش ← ڕاپۆرتەکانی هەژماری بانک**

ڕاپۆرتەکانی هەژماری بانک لە شوێنەکانی ترەوە دەردەکەون:
- **داشبۆردی دارایی**: پیشاندەرەکانی دۆخی هەژمارەکانی بانک
- **تۆمارەکانی پارەدان**: بەستنەوەکانی هاوسەنگی و یەکخستن
- **ڕاپۆرتە داراییەکان**: ڕاپۆرتەکانی گەرانی کاش و لیستی جووڵەی پارە

**ئامۆژگاری**: گەڕانی خێرا بەکاربهێنە بۆ دۆزینەوەی ڕاپۆرتەکانی هەژماری بانکی تایبەت بە بەروار یان ژمارەی بەڵگە.

---

## پێکهاتەکانی ڕاپۆرتی هەژماری بانک

### سەرەوەی ڕاپۆرت
- **پەرتووکی ڕۆژانەی بانک**: پەرتووکی ڕۆژانەی تایبەتی بانک (دەبێت جۆری "بانک" بێت)
- **دراو**: دراوی ڕاپۆرت (دراوی بنەڕەتی کۆمپانیا)
- **ژمارەی بەڵگە**: ژمارەی ڕاپۆرت یان ناسێنەر
- **بەروار**: بەرواری ڕاپۆرت
- **یەکەی دەستپێکردن**: یەکەی کراوەیی لە ڕاپۆرتی پێشوو
- **یەکەی کۆتایی**: یەکەی دابەندان بۆ ماوەکە

### ڕیزەکانی ڕاپۆرت (مامەڵەکان)
- **بەروار**: بەرواری مامەڵە
- **پێناسە**: پێناسەی مامەڵە لە بانکەوە
- **هاوبەش**: کڕیار/فرۆشیار ئەگەر هەبێت
- **بڕ**: بڕی مامەڵە (ئەرێنی بۆ وەرگرتن، نەرێنی بۆ پارەدان)
- **دراوی بیانی**: دڵخواز - ئەگەر مامەڵە لە دراوی جیاوازدا بووبێت
- **بڕ بە دراوی بیانی**: بڕی ڕەسەن پێش ئاڵوگۆڕ

---

## دروستکردنی ڕاپۆرتەکانی هەژماری بانک

بڕۆ بۆ **ژمێریاری ← بانک و کاش ← ڕاپۆرتەکانی هەژماری بانک**

### هەنگاوی ۱: دەستگەیشتن بە ڕووکاری دروستکردنی ڕاپۆرت

**کلیک لە "دروستکردن"**: هەڵبژاردن لە لیستی ڕاپۆرتەکان
**دیاریکردنی پەرتووکی ڕۆژانەی بانک**: هەڵبژاردنی پەرتووکی ڕۆژانەی بانکی گونجاو (دەبێت لە جۆری "بانک"ەوە بێت)
**کۆنترۆڵی دەستگەیشتن**: دڵنیابوون لە هەبوونی مۆڵەتی گونجاو بۆ دروستکردنی ڕاپۆرتەکانی هەژماری بانک

### هەنگاوی ۲: تێگەیشتن لە پلانی ڕووکاری تێخستنی دراوە

ڕووکاری دروستکردنی ڕاپۆرتی هەژماری بانک پلانێکی تەاکتیڤی چەندین بەش بەکاردەهێنێت:

**بەشی سەرەوە - زانیاری ڕاپۆرت**: تێخستنی دراوە سەرەتاییەکانی ڕاپۆرت
- دیاریکردنی پەرتووکی ڕۆژانەی بانک و دراو
- تێخستنی ماوەی ڕاپۆرت (بەرواری دەستپێک و کۆتایی)
- تۆمارکردنی یەکەکانی دەستپێکردن و کۆتایی

**بەشی ناوەڕاست - ڕیزەکانی مامەڵە**: پیشاندان و بەڕێوەبردنی مامەڵە تاکەکان
- پیشاندانی هەموو مامەڵە زیادکراوەکان
- پشتگیری چەندین دراو لەگەڵ ئاڵوگۆڕی خۆکار
- توانای دەستکاری و سڕینەوە بۆ هەر مامەڵەیەک

**بەشی خوارەوە - کورتەی پشکنین**: پشکنینی هاوسەنگی ڕاپۆرت لە کاتی ڕاستەقینەدا
- کۆی مامەڵەکان (کۆی هەموو مامەڵە تێخراوەکان)
- یەکەی حیسابکراو (یەکەی دەستپێکردن + مامەڵەکان)
- پیشاندەری دۆخی هاوسەنگی بۆ پشکنینی وردی

### هەنگاوی ۳: تێخستنی مامەڵەکانی ڕاپۆرت

**تێخستنی زانیاری ڕاپۆرت**: تێخستنی دراوە سەرەتاییەکان لە بەشی سەرەوە
**زیادکردنی ڕیزەکانی مامەڵە**: بەکارهێنانی خشتەی مامەڵەکان بۆ زیادکردنی هەر مامەڵەیەک
**چاودێری هاوسەنگی**: چاودێری بەشی کورتە بۆ پشکنینی دۆخی "هاوسەنگ"
**جێبەجێکردنی پاشەکەوتکردن**: کلیککردن لە "پاشەکەوتکردن" کاتێک هەموو دراوەکان دروست و هاوسەنگن

---

## جۆرەکانی ڕیزەی ڕاپۆرتی هەژماری بانک

### وەرگرتن و قەرزەکان
**پارەدانی کڕیاران**: پارە وەرگیراو لە کڕیارانەوە
**گواستنەوەی ناوەوەی بانک**: پارە گواستراو لە هەژمارەکانی ترەوە
**داهاتی بەدەستهاتوو**: پارەدانی داهاتی بانک
**دەرامەتی قەرز**: پارە وەرگیراو لە قەرزەکانەوە
**داهاتی تر**: وەرگرتنی جۆراوجۆر

### پارەدان و قەرزەکان
**پارەدانی فرۆشیاران**: پارەدراو بە دابینکەران
**گواستنەوەی دەرەوەی بانک**: پارە گواستراو بۆ هەژمارەکانی تر
**خەرجی بانک**: خەرجی مانگانە، خەرجی مامەڵەکان
**پارەدانی قەرز**: پارەدانی سەرەکی و داهات
**خەرجی تر**: پارەدانی جۆراوجۆر

### بڕگەکانی تایبەت بە بانک
**خەرجی NSF**: خەرجی کەمی پارە
**خەرجی زیادەپارە**: خەرجی زیادە لە هەژمار
**خەرجی گواستنەوەی تەل**: خەرجی گواستنەوەی تەل
**چاپکردنی چێک**: خەرجی داواکاری چێک
**خەرجی ATM**: خەرجی بانکەی خۆکار

---

## مامەڵەکردن لەگەڵ چەندین دراو

### ئاڵوگۆڕی خۆکاری دراو
سیستەم بە شێوەی خۆکار لەگەڵ سیناریۆکانی چەندین دراو مامەڵە دەکات:

**مامەڵەکانی دراوی جیاواز**: مامەڵەکان بە دراوی جیاواز بۆ دراوی ڕاپۆرت دەگۆڕدرێن
**نرخە ڕاستەوخۆییەکان**: نرخەکانی ئاڵوگۆڕی ئێستا بەکاردەهێنێت بۆ حیسابەکان
**زانیاری پیشاندان**: هەردوو بڕی ڕەسەن و گۆڕدراو پیشاندەدات
**حیسابی یەکە**: هەموو کۆکان بە دراوی ڕاپۆرت حیسابدەکرێن

### پیشاندەرەکانی دراو
- **هەمان دراو**: پیشاندەری سەوز بۆ دراوە هاوتاکان
- **دراوی جیاواز**: پیشاندەری زەرد کە ئاڵوگۆڕی جێبەجێکراو پیشاندەدات
- **وردەکاریەکانی ئاڵوگۆڕ**: گەڕان یان کلیککردن بۆ زانیاری نرخ

**نموونە**: تۆمارکردنی پارەدانێک بە دۆلاری ئەمریکی لە ڕاپۆرتێکی دینار عێراقیدا:
- پارەدانی سیستەم: 500 دۆلاری ئەمریکی
- نرخی ئاڵوگۆڕ: 1 دۆلاری ئەمریکی = 1,310 دینار عێراقی
- بڕی گۆڕدراو: 655,000 دینار عێراقی
- ڕیزی ڕاپۆرتی هەژماری بانک: 655,000 دینار عێراقی (یەکدەگرێتەوە دوای ئاڵوگۆڕ)

## کاریگەری تۆماری ژمێریاری

### دروستکردنی ڕاپۆرتی هەژماری بانک بە سەرکەوتوویی
```
هیچ تۆماری ژمێریاری دروستناکرێت - ڕاپۆرتی هەژماری بانک بەڵگەی سەرچاوەیە بۆ هاوسەنگی
```

### مامەڵە لەگەڵ خەرجی بانک (نموونە)
```
قەرزدار (مدین) هەژماری خەرجی بانک    25,000 د.ع
    قەرزخواز هەژماری بانک              25,000 د.ع
```

ڕاپۆرتی هەژماری بانک بە خۆیەوە تۆمارە ژمێریاریەکان دروستناکات بەڵکو دراوەی سەرچاوەیی دابیندەکات بۆ [پرۆسەی هاوسەنگی](bank-reconciliation.ckb.md) کە کاریگەری لەسەر هەژمارەکان دەبێت.

## سیناریۆ باوەکان

### سیناریۆی ۱: پرۆسەی دروستکردنی ڕاپۆرتی ستاندارد
دروستکردنی ڕاپۆرتی هەژماری بانکی مانگانە بە ۲۵ مامەڵە:

**هەنگاوەکان**:
1. دەستگەیشتن بە ڕاپۆرتەکانی هەژماری بانک لە لیستی ژمێریاری
2. دروستکردنی ڕاپۆرتی نوێ (پەرتووکی ڕۆژانەی بانکی ڕاجحی، مارتی ۲۰۲۴)
3. تێخستنی یەکەی دەستپێکردن (2,500,000 د.ع)
4. زیادکردنی ۲۵ ڕیزی مامەڵە (وەرگرتنەکان، پارەدانەکان، خەرجەکان)
5. پشکنینی هاوسەنگی یەکەی کۆتایی (2,750,000 د.ع)
6. پاشەکەوتکردنی ڕاپۆرت و ئامادەکردن بۆ [هاوسەنگی](bank-reconciliation.ckb.md)

**ئەنجام**:
- ڕاپۆرتی هەژماری بانکی تەواو ئامادە بۆ هاوسەنگی
- هەموو مامەڵەکان تۆمارکراو و پۆلێنکراون
- ڕووکار دۆخی "ئامادە بۆ هاوسەنگی" پیشاندەدات

### سیناریۆی ۲: مامەڵەی دراوی بیانی
تۆمارکردنی پارەدانێک بە دۆلاری ئەمریکی لە ڕاپۆرتێکی دینار عێراقیدا:

**نموونە**:
- مامەڵەی بانک: 655,000 دینار عێراقی گواستنەوەی دەرەوە
- مامەڵەی سیستەم: 500 دۆلاری ئەمریکی پارەدانی دابینکەر
- نرخی ئاڵوگۆڕ: 1 دۆلاری ئەمریکی = 1,310 دینار عێراقی (پارێزراو لە کاتی پارەداندا)
- بڕی گۆڕدراو: 655,000 دینار عێراقی

**هەنگاوەکان**:
1. زیادکردنی ڕیزی ڕاپۆرت بە دینار عێراقی (655,000 د.ع)
2. دیاریکردنی دراوی ڕەسەن (دۆلاری ئەمریکی)
3. تێخستنی بڕی ڕەسەن (500 دۆلاری ئەمریکی)
4. سیستەم هەردوو بڕی ڕەسەن و گۆڕدراو پاشەکەوت دەکات
5. پیشاندانی زانیاری ئاڵوگۆڕ بۆ پێداچوونەوە

**ئەنجام**:
- مامەڵەی چەندین دراوی بە تەواوی تۆمارکراو
- بەڵگەکردنی نرخی ئاڵوگۆڕ پارێزراو
- هەردوو تۆمارەکە بەستراوەتەوە و ئامادەن بۆ هاوسەنگی

### سیناریۆی ۳: ڕاستکردنەوەی هەڵە لە ڕاپۆرت
ڕاستکردنەوەی بڕی هەڵەی مامەڵە لە ڕاپۆرتی هەژماری بانک:

**هەنگاوەکان**:
1. دۆزینەوەی مامەڵەی هەڵە (بڕی تۆمارکراو: 100,000 د.ع لە جیاتی 150,000 د.ع)
2. دەستکاریکردنی ڕیزی مامەڵەی دیاریکراو
3. نوێکردنەوەی بڕ بۆ بەهای دروست (150,000 د.ع)
4. پشکنینی نوێکردنەوەی یەکەی کۆتایی بە شێوەی خۆکار
5. پاشەکەوتکردنی گۆڕانکارییەکان و پشکنینی هاوسەنگی

**ئەنجام**:
- مامەڵە ڕاستکراوەتەوە بە بڕی دروست
- یەکەی کۆتایی بە شێوەی خۆکار نوێکراوەتەوە
- ڕێگای پشکنینی گۆڕانکارییەکان پارێزراو

## باشترین ڕێگاکان

### ئامادەکردن و ڕێکخستن
- **چالاککردنی ڕێکخستنەکانی بانک**: دڵنیابە لەوەی پەرتووکی ڕۆژانەکانی بانک و هەژمارەکانی پەرتووکی گشتی بە دروستی ڕێکخراون
- **ڕاپۆرتە تەواوەکان**: دڵنیابە لەوەی هەموو مامەڵەکانی بانک لە [ڕاپۆرتەکانی هەژماری بانک](bank-statements.ckb.md)دا تێخراون
- **پشکنینی پارەدان**: دڵنیابە لەوەی پارەدانەکانی سیستەم لە دۆخی "نێردراوە"دان
- **وردی دراو**: پێداچوونەوەی مامەڵە چەندین دراوییەکان بۆ نرخە گونجاوەکان

### پرۆسەی دروستکردنی ڕاپۆرت
- **ڕێگەی سیستەماتیک**: کارکردن لە ڕێگەی مامەڵەکانەوە بە شێوەی سیستەماتیک بەپێی بەروار یان بڕ
- **پشکنینی ڕاستەوخۆیی**: چاودێری پیشاندەری هاوسەنگی بە بەردەوامی لە کاتی تێخستنی دراوەدا
- **بەڵگەکردنی ڕوون**: دابینکردنی پێناسەی ورد بۆ هەموو مامەڵەکان
- **وردی لە تێخستن**: پشکنینی دووجار لە هەموو بڕەکان و بەرواران پێش پاشەکەوتکردن

### کۆنترۆڵی کوالیتی
- **دروستکردنی ئاسایی**: دروستکردنی ڕاپۆرتەکان لە ماوەی چەند ڕۆژێکدا دوای وەرگرتنی ڕاپۆرتەکانی هەژماری بانک
- **پێداچوونەوەی بڕگە نەیەکگیراوەکان**: پشکنینی جیاوازییەکان لە سەرووی ئاستێکی پێشوەختە دیاریکراو
- **بەدواداچوونی پەیتەوەکردن**: چاودێری بڕگە هەڵواسراوەکان لە چەندین ماوەدا
- **ڕاهێنانی بەکارهێنەران**: دڵنیابوون لەوەی تیمەکە تایبەتمەندییەکانی ڕووکاری بەکارهێنەر تێدەگات

### پشکنین و پابەندبوون
- **ڕێگای پشکنین**: سیستەم مێژووی تەواوی دروستکردنی ڕاپۆرتەکانی هەژماری بانک پارێزگاری دەکات
- **بەڵگەنامەکانی پشتگیری**: بەستنەوەی ڕاپۆرتەکانی هەژماری بانک و پشتڕاستکردنەوەکانی پارەدان
- **پرۆسەی پێداچوونەوە**: جێبەجێکردنی پێداچوونەوەی سەرپەرشتیار بۆ مامەڵە گەورەکان
- **پڕۆسیجورەکانی کۆتایی مانگ**: تەواوکردنی هەموو ڕاپۆرتەکانی هەژماری بانک پێش داخستنی ماوەکان

---

## ڕاپۆرت و شیکردنەوە

### بینینەکانی بەڕێوەبردنی ڕاپۆرت
**لیستی ڕاپۆرتی هەژماری بانک**: تێڕوانینی گشتی هەموو ڕاپۆرتەکان لەگەڵ زانیاری گرنگ
- بەڵگە، پەرتووکی ڕۆژانەی بانک، بەروار، دراو
- یەکەی دەستپێکردن و کۆتایی
- بەرواری دروستکردن و دەستکاریکردن

**وردەکاریەکانی ڕاپۆرت**: بینینی تاکی ڕاپۆرت پیشاندانی:
- زانیاری تەواوی سەرەوە
- هەموو ڕیزەکانی مامەڵە لەگەڵ بڕ و پێناسەکان
- نیشانکەرەکانی دۆخی هاوسەنگی

### ڕاپۆرتەکانی یەکخستن
**شیکردنەوەی گەرانی پارە**: دراوەکانی ڕاپۆرت بەشدارین لە:
- ڕاپۆرتکردنی دۆخی کاش
- شیکردنەوەی لیستی جووڵەی پارە
- دڵنیابوونەوەی یەکەی بانک
- بەدواداچوونی دۆخی هاوسەنگی

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

**پ: ناتوانرێت پەرتووکی ڕۆژانەی بانک لە لیستی داکەوتووەکەدا بدۆزرێتەوە**
وەڵام: پشکنینی ڕێکخستنەکانی پەرتووکی ڕۆژانە و مۆڵەتەکان:
- دڵنیابە لەوەی جۆری پەرتووکی ڕۆژانە دامەزراوە لەسەر "بانک"
- پشکنین بکە کە پەرتووکی ڕۆژانە سەر بە کۆمپانیا ئێستاکەیە
- پێداچوونەوەی مۆڵەتەکانی بەکارهێنەر بۆ دەستگەیشتن بە پەرتووکی ڕۆژانەکانی بانک

**پ: کۆکان لە هاوسەنگی چەندین دراودا بە تەواوی هاوسەنگ نابن**
وەڵام: پێداچوونەوەی مامەڵەکردن لەگەڵ ئاڵوگۆڕی دراو:
- پشکنینی نرخەکانی ئاڵوگۆڕی بەکارهاتوو بۆ پارەدان و هاوسەنگی
- دڵنیابوون لەوەی بڕەکان بە دروستی لە بەڵگەنامە سەرچاوەییەکاندا تێخراون
- بیرکردنەوە لە جیاوازیە بچووکەکانی خوارەوەکردن (بە ئاسایی لە ژێر 1,000 د.ع)
- بەکارهێنانی سڕینەوە بۆ جیاوازیە نەگرنگەکانی ئاڵوگۆڕ

**پ: هیچ پارەدانێک لە پانێلی پارەدانەکانی سیستەمدا لە کاتی هاوسەنگیدا دەرناکەوێت**
وەڵام: پشکنینی دۆخی پارەدان و فلتەرەکان:
- دڵنیابوون لەوەی پارەدانەکان لە دۆخی "نێردراوە"دان (نەک ڕەشنووس)
- پشکنین بکە کە پارەدانەکان پێشوەختە هاوسەنگ نەکراون
- دڵنیابوون لەوەی پارەدانەکان سەر بە کۆمپانیا و هەژماری بانکی دروست دەکەن
- نوێکردنەوەی وێبگەڕ ئەگەر ڕووکار کۆن دیارە

**پ: ناتوانرێت پرۆسەی پاشەکەوتکردنی ڕاپۆرتی هەژماری بانک تەواو بکرێت**
وەڵام: پشکنینی پشتڕاستکردنەوە و هاوسەنگی:
- دڵنیابوون لەوەی کۆکان هاوسەنگن (جیاوازی = 0)
- پشکنین بکە کە هەموو خانە پێداویستەکان پڕکراوەتەوە
- گەڕان بۆ هەڵەکانی پشتڕاستکردنەوە یان پەیامەکانی سیستەم
- تاقیکردنەوەی پاککردنەوەی هەڵبژاردنەکان و دەستپێکردنەوە لە نوێ ئەگەر پێویست بوو

### پەیامەکانی هەڵە

**"پەرتووکی ڕۆژانەی هەڵبژێردراو لە جۆری بانک نییە"**
- پەیوەندی بە بەڕێوەبەرەوە بکە بۆ ڕێکخستنی پەرتووکی ڕۆژانەکانی بانک بە دروستی
- پشکنین بکە کە ڕێکخستنی کۆمپانیا پەرتووکی ڕۆژانە گونجاوەکانی بانک لەخۆدەگرێت

**"کۆی مامەڵە هەڵبژێردراوەکان لەگەڵ کۆی بانک ناگونجێت"**
- پێداچوونەوەی بڕە هەڵبژێردراوەکان بە وردی
- پشکنینی کێشەکانی ئاڵوگۆڕی دراو
- بیرکردنەوە لە بەکارهێنانی سڕینەوە بۆ جیاوازیە بچووکەکان

---

## پرسیارە دووپاتکراوەکان

**پ: چی ڕوودەدات بە ڕاپۆرتەکانی هەژماری بانک دوای دروستکردنیان؟**
وەڵام: ڕاپۆرتەکانی هەژماری بانکی دروستکراو بەردەست دەبن بۆ [پرۆسەی هاوسەنگی](bank-reconciliation.ckb.md) و بە ڕیزە مامەڵە تایبەتەکانەوە دەبەسترێنەوە. لە تۆمارەکانی ڕاپۆرتەکانی هەژماری بانکدا بە پیشاندەرەکانی دۆخی هاوسەنگیەوە بەردەوام بە دیار دەکەون.

**پ: ئایا دەتوانم ڕاپۆرتێکی هەژماری بانک بگەڕێنمەوە ئەگەر هەڵەیەکم کردبێت؟**
وەڵام: سیستەم ڕێگاکانی پشکنین پارێزگاری دەکات بەڵام ڕاپۆرتەکانی هەژماری بانک دەبێت وەک هەمیشەیی دابنرێن کاتێک لە هاوسەنگیدا بەکاردەهێنرێن. پەیوەندی بە بەڕێوەبەری سیستەمەوە بکە ئەگەر پێویستت بە گەڕاندنەوەی تۆمارەکانی ڕاپۆرتی هەژماری بانک بۆ ڕاستکردنەوە هەیە.

**پ: چۆن مامەڵە لەگەڵ مامەڵەکانی بانک بکەم کە لە چەندین ماوەدا بڵاودەبنەوە؟**
وەڵام: مامەڵەکانی بانک دەبێت لە ماوەکەدا ڕێکبخرێن کە لە ڕاپۆرتی هەژماری بانکدا دەردەکەون. ئەگەر پارەدانەکانی سیستەم لە ماوەیەکی جیاوازدا تۆمارکرابن، هێشتا دەکرێت لە کاتی هاوسەنگیدا یەکیان بخرێت.

**پ: چی دەبێت لەگەڵ بڕگە گەورە نەیەکگیراوەکان بکەم؟**
وەڵام: بە وردی پشکنین بکە پێش سڕینەوە. گەڕان بۆ پارەدانەکانی سیستەمی ونبوو، یان بڕە هەڵەکان، یان جیاوازیەکانی کاتەکان. جیاوازیە گەورەکان ڕەنگە ئاماژە بە هەڵەکان بدەن کە پێویستیان بە ڕاستکردنەوە هەیە لە جیاتی سڕینەوە.

## فەرهەنگی زاراوەکان (گلۆساری)

ئەم بەشە فەرهەنگی تایبەتی زاراوە ژمێریاری و بانکییەکانی بەکارهاتوو لەم بەڵگەیەدا دەخاتەڕوو، بەپێی [فەرهەنگی فەرمی زاراوەکان](../../lang/Terminology.yaml):

### زاراوەکانی بنەڕەتی ژمێریاری
- **ژمێریاری**: ژمێریاری - زانستی تۆمارکردن و شیکردنەوەی مامەڵە داراییەکان
- **هەژمار**: هەژمار - هەژماری تاک لە سیستەمی ژمێریاریدا
- **ژۆرنال**: پەرتووکی ڕۆژانە - تۆماری ڕۆژانەی مامەڵەکان
- **تۆماری هەژمار**: تۆماری هەژمار - تۆمارکردنی مامەڵەیەکی تاک
- **پەرتووکی گشتی**: پەرتووکی گشتی - کۆی گشتی هەموو هەژمارەکان

### مدین و دائن
- **قەرزدار (مدین)**: قەرزدار (مدین) - لای چەپی هەژمار
- **قەرزخواز**: قەرزخواز - لای ڕاستی هەژمار
- **بڕی قەرزدار**: بڕی قەرزدار - بڕ لە لای قەرزداردا
- **بڕی قەرزخواز**: بڕی قەرزخواز - بڕ لە لای قەرزخوازدا

### ڕاپۆرتەکانی دارایی
- **پەڕەی هاوسەنگی**: پەڕەی هاوسەنگی - ڕاپۆرتی دۆخی دارایی
- **لیستی داهات**: لیستی داهات - ڕاپۆرتی قازانج و زیان
- **لیستی جووڵەی پارە**: لیستی جووڵەی پارە - ڕاپۆرتی گەرانی پارە
- **یەکە**: یەکە - یەکەی هەژمار
- **یەکەی دەستپێکردن**: یەکەی دەستپێکردن - یەکەی سەرەتایی
- **یەکەی کۆتایی**: یەکەی کۆتایی - یەکەی کۆتایی

### بەڵگەنامەکانی بازرگانی
- **پسوولەکانی فرۆشتن**: پسوولەکانی فرۆشتن - پسوولەی فرۆشتن
- **پسوولەکانی کڕین**: پسوولەکانی کڕین - پسوولەی کڕین
- **پسوولەی وەرگرتن**: پسوولەی وەرگرتن - پسوولەی پارەدان
- **بەڵگە(السند)**: بەڵگە(السند) - بەڵگەی پشتگیری
- **ژمارەی بەڵگە**: ژمارەی بەڵگە - ژمارەی سەرچاوەی بەڵگە

### بانک و کاش
- **بانک**: بانک - دامەزراوەی دارایی
- **ڕاپۆرتی هەژماری بانک**: ڕاپۆرتی هەژماری بانک - ڕاپۆرتی هەژماری بانک
- **کاش**: کاش - پارەی فیزیکی
- **هەژماری بانک**: هەژماری بانک - هەژماری بانک
- **هاوسەنگی بانکی**: هاوسەنگی بانکی - ڕێکخستنی بانک
- **پارەدان**: پارەدان - پارەدانی پارە
- **وەرگرتن**: وەرگرتن - وەرگرتنی پارە
- **گواستنەوە**: گواستنەوە - گواستنەوەی پارە

### سامان و قەرزەکان
- **سامان**: سامان - سەرچاوەی ئابووری
- **سامانەکان**: سامانەکان - چەندین سامان
- **قەرز**: قەرز - ئەرکی دارایی
- **قەرزەکان**: قەرزەکان - چەندین قەرز
- **خاوەندارێتی**: خاوەندارێتی - مافی خاوەن
- **سەرمایە**: سەرمایە - سەرمایەی وەبەرهێنان

### داهات و خەرجییەکان
- **داهات**: داهات - داهات لە کارەکانەوە
- **خەرجی**: خەرجی - تێچوونی بازرگانی
- **خەرجیەکان**: خەرجیەکان - چەندین خەرجی
- **تێچوون**: تێچوون - تێچوونی کاڵا/خزمەتگوزارییەکان
- **قازانج**: قازانج - داهاتی پاک
- **زیان**: زیان - زیانی دارایی

### کڕیار و فرۆشیارەکان
- **کڕیار**: کڕیار - کڕیاری فرۆشتن
- **فرۆشیار**: فرۆشیار - دابینکەر
- **دابینکەر**: دابینکەر - دابینکەری کاڵا/خزمەتگوزارییەکان
- **هاوبەش**: هاوبەش - هاوبەشی بازرگانی
- **قەرزدار**: قەرزدار - ئەوەی قەرزی لێیە
- **قەرزخواز**: قەرزخواز - ئەوەی قەرزی بۆ دەکرێت

### دراو و ئاڵوگۆڕ
- **دراو**: دراو - یەکەی پارە
- **نرخی ئاڵوگۆڕ**: نرخی ئاڵوگۆڕ - ڕێژەی گۆڕینی دراو
- **دراوی بیانی**: دراوی بیانی - دراوی نا-ناوخۆیی
- **دراوی بنەڕەتی**: دراوی بنەڕەتی - دراوی سەرەکی
- **ئاڵوگۆڕ**: ئاڵوگۆڕ - ئاڵوگۆڕی دراو

### ماوەکانی ژمێریاری
- **ماوەی ژمێریاری**: ماوەی ژمێریاری - ماوەی دارایی
- **ساڵی دارایی**: ساڵی دارایی - ساڵی دارایی
- **مانگ**: مانگ - مانگی ڕۆژژمێر
- **چارەک**: چارەک - ماوەی سێ مانگ
- **ساڵ**: ساڵ - ساڵی ڕۆژژمێر/دارایی

### دۆخ و بارودۆخ
- **ڕەشنووس**: ڕەشنووس - تۆماری نەنێردراو
- **نێردراوە**: نێردراوە - تۆماری پشتڕاستکراو
- **هەڵوەشێنراوەتەوە**: هەڵوەشێنراوەتەوە - تۆماری باطڵکراو
- **چالاک**: چالاک - ئێستا بەکاردەهێنرێت
- **چالاک نییە**: چالاک نییە - ئێستا بەکارناهێنرێت
- **چاوەڕوان**: چاوەڕوان - چاوەڕوانی کردار
- **پەسەندکراو**: پەسەندکراو - بە فەرمی پەسەندکراو

### زاراوەکانی کردار
- **دروستکردن**: دروستکردن - دروستکردنی نوێ
- **دەستکاری**: دەستکاری - گۆڕانی ئەوەی هەیە
- **سڕینەوە**: سڕینەوە - لابردن
- **نوێکردنەوە**: نوێکردنەوە - نوێکردنەوە/گۆڕانکاری
- **ناردن**: ناردن - پشتڕاستکردنەوەی تۆمار
- **هەڵوەشاندنەوە**: هەڵوەشاندنەوە - باطڵکردن

ڕاپۆرتەکانی هەژماری بانک دڵنیایی لە وردی تۆمارەکانی کاش دەکەن و متمانە لە ڕاپۆرتکردنی دارایی دابیندەکەن. ڕووکاری بەکارهێنەر پرۆسەی تێخستنی دراوە ئاسان دەکات لەگەڵ پاراستنی ڕێگاکانی پشکنینی تەواو و کۆنترۆڵە ژمێریاری گونجاوەکان.
