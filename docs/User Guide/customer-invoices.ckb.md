# پسووڵەکانی فرۆشتنی کڕیاران: بەڵگەکردنی فرۆشتن و ناسینەوەی داهات

ئەم ڕێبەرە تەواوەیە ڕوونکردنەوەی چۆنیەتی کارکردنی پسووڵەکانی فرۆشتنی کڕیاران لە سیستەمی ژمێریاریتاندا، بە گرتنەبەری دروستکردن، پڕۆسێسکردنی پارەدان، پشتگیری فرە دراوی، و ناسینەوەی داهات. نووسراوە بۆ هەموو بەکارهێنەران - ژمێریار و نا ژمێریار - ڕێنماییە کردارییەکان پێشکەش دەکات بە شوێنکەوتنی باشترین پراکتیزەکانی ژمێریاری دوو هاتووچۆیی.

---

## پسووڵەی فرۆشتنی کڕیار چییە؟

پسووڵەی فرۆشتنی کڕیار بەڵگەنامەیەکی بازرگانییە کە بە فەرمی پارەدان داوا دەکات بۆ کاڵا فرۆشراوەکان یان خزمەتگوزارییە پێشکەشکراوەکان، تۆمارە یاساییەکان و ژمێریاریەکان بۆ مامەڵەکە دروست دەکات.

- **فرۆشتن تۆمار دەکات**: کاڵا فرۆشراوەکان یان خزمەتگوزارییە پێشکەشکراوەکان بۆ کڕیاران بەڵگە دەکات
- **پارەدان داوا دەکات**: بە فەرمی لە کڕیار داوا دەکات پارە بدات بۆ کاڵا/خزمەتگوزارییە گەیاندراوەکان
- **داهات دەناسێتەوە**: بڕیارە ژمێریاریەکان بۆ داهات لە ژێر ژمێریاریی ئەستەمکراودا دروست دەکات
- **قەرزە وەرگیراوەکان بەدوادا دەکات**: بڕەکان دیاری دەکات کە لە لایەن کڕیارانەوە لە هەژماری وەرگیراوەکاندا قەرزارن

**کاریگەری ژمێریاری**: پسووڵە نێردراوەکان داهات و هەژماری وەرگیراوەکان زیاد دەکەن، شوێن پێی بنەمای ژمێریاریی ئەستەمکراو دەکەن کە داهات دەناسێتەوە کاتێک دەست دەکەوێت، نەک کاتێک کاش وەردەگیرێت.

---

## پێداویستیەکانی سیستەم

### ڕێکخستنی کۆمپانیا
- **ڕێکخستنی کڕیار**: پێویستە تۆمارەکانی کڕیار پێش پسووڵەی فرۆشتنکردن دروست بکرێن
- **کاتالۆگی بەرهەم**: پێویستە بەرهەم و خزمەتگوزارییەکان لەگەڵ نرخدانان ڕێک بخرێن
- **ڕێکخستنی باج**: پێویستە ڕێژە و یاساکانی باج بۆ دەسەڵاتدارییە جێبەجێکراوەکان دابنرێن
- **مەرجەکانی پارەدان**: پێویستە مەرجەکانی پارەدان بۆ ڕێکەوتننامەکانی کڕیار دیاری بکرێن

### پێشمەرجەکان
1. **تۆماری کڕیار**: پێویستە کڕیار لە سیستەمدا بوونی هەبێت لەگەڵ زانیاری تەواو
2. **ڕێکخستنی بەرهەم/خزمەتگوزاری**: پێویستە ئەو بڕگانەی کە پسووڵەی فرۆشتنی بۆ دەکرێت لە کاتالۆگدا ڕێک بخرێن
3. **ڕێکخستنی دراو**: پێویستە دراوی پسووڵەی فرۆشتن چالاک و ڕێکخراو بێت
4. **ڕێگەپێدانی بەکارهێنەر**: دەستڕاگەیشتنی گونجاو بە تایبەتمەندیەکانی فرۆشتن و ژمێریاری

---

## لە کوێ لە ڕووکاری بەکارهێنەردا بیدۆزیتەوە

بچۆ بۆ **ژمێریاری ← فرۆشتن ← پسووڵەکانی فرۆشتن**

پسووڵەکانی فرۆشتنی کڕیاران هەروەها دەردەکەون لە:
- **کردەوەی دروستکردن**: دوگمەی "دروستکردن" بۆ تۆمارکردنی پسووڵەی فرۆشتنی نوێ
- **تۆمارەکانی کڕیار**: بەشی پسووڵە پەیوەستەکان کە مێژووی پسووڵەکان پیشان دەدات
- **پەڕەی سەرەکی**: پسووڵەکانی دوایی و کورتەی فرۆشتنەکان

**ئامۆژگاری**: دوگمەی یارمەتی/بەڵگەکان لە سەرەوە ئەم ڕێبەرە دەکاتەوە.

## سوڕی ژیانی پسووڵەی فرۆشتن

### دۆخی ڕەشنووس
- پسووڵەی فرۆشتن لە ئامادەکردندا دایە و دەتوانرێت بە ئازادی دەستکاری بکرێت
- هێشتا هیچ کاریگەرییەکی ژمێریاریی دروست نەکراوە
- هێشتا ژمارەی پسووڵەی فرۆشتنی هەمیشەیی دیاری نەکراوە
- بۆ کڕیار بەردەست نییە
- دەتوانرێت بسڕێتەوە ئەگەر چیتر پێویستی پێ نەبێت

### دۆخی نێردراو
- پسووڵەی فرۆشتن بە ژمارەی پسووڵەی فرۆشتنی هەمیشەیی تەواو کراوە
- بڕیارە ژورنالییەکان بۆ ناسینەوەی ژمێریاری دروست کراون
- ناتوانرێت دەستکاری بکرێت (بەڵگەنامەکانی ڕاستکردنەوە بۆ چاکسازی بەکاربهێنە)
- بۆ پارەدانی کڕیار بەردەستە
- جوڵەی کۆگا بۆ بەرهەمە هەڵگیراوەکان دروست دەکرێت
- قیستی پارەدان لەسەر بنەمای مەرجەکانی پارەدان دروست دەکرێت

### دۆخی پارەدراو
- پارەی کڕیار وەرگیراوە و لەسەر پسووڵەی فرۆشتن جێبەجێ کراوە
- دۆخی پارەدان وەک "پارەدراو" دەردەکەوێت
- یەکەی هەژماری وەرگیراوەکان پاک کراوەتەوە
- سوڕی مامەڵەکە تەواو بووە

### دۆخی هەڵوەشاندنەوە
- پسووڵەی فرۆشتن پێش یان دوای ناردن هەڵوەشاندراوەتەوە
- بڕیاری ژورنالیی پێچەوانەکراو ئەگەر پێشتر نێردرابێت دروست کراوە
- ناتوانرێت پارە وەربگرێت
- شوێنپێی پشکنینی تەواو پارێزراوە

## دروستکردنی پسووڵەی فرۆشتنی کڕیار

بچۆ بۆ **ژمێریاری → فرۆشتن → پسووڵەکانی فرۆشتن → دروستکردن**

### هەنگاوی ١: زانیاری کڕیار و دراو

**کڕیار**: کڕیار لە لیستی هاوبەشانەوە هەڵبژێرە
- ئەگەر پێویست کرد کڕیاری نوێ ڕاستەوخۆ لە فۆڕمی پسووڵەی فرۆشتنەوە دروست بکە
- زانیاری کڕیار ناو، ئیمەیڵ، کەسی پەیوەندی و ناونیشان لەخۆدەگرێت

**دراو**: دراوی پسووڵەی فرۆشتن هەڵبژێرە
- بۆ دراوی پەسەندکراوی کڕیار یان دراوی بنەڕەتی کۆمپانیا دەگەڕێتەوە
- پشتگیری فرە دراوی لەگەڵ گۆڕینی نرخی ئاڵوگۆڕ بە شێوەیەکی خۆکارانە
- نرخی ئاڵوگۆڕ لە کاتی ناردنی پسووڵەی فرۆشتندا گیراوە

**نرخی ئاڵوگۆڕی ئێستا**: بۆ دراوە بیانیەکان پیشان دەدرێت
- نرخی ئاڵوگۆڕی ئێستا بۆ ئاماژەدان پیشان دەدات
- کاتێک پسووڵەی فرۆشتن دەنێردرێت قفڵ دەکرێت

### هەنگاوی ٢: وردەکاریەکانی پسووڵەی فرۆشتن

**پێگەی باج**: مەرجەکانی باجی ئیختیاری لەسەر بنەمای شوێن/جۆری کڕیار
**بەرواری پسووڵەی فرۆشتن**: بەرواری فرۆشتن یان گەیاندنی خزمەتگوزاری (پێویستە)
- نابێت لە ماوەیەکی قفڵکراودا بێت
**بەرواری کۆتایی**: کۆتایی پارەدان (پێویستە)
**مەرجەکانی پارەدان**: مەرجەکانی پارەدان هەڵبژێرە (بڕوانە [ڕێبەری مەرجەکانی پارەدان](payment-terms-guide.md))
- مەرجەکان خشتەی قیستی پارەدان دیاری دەکەن
- لەسەر هەژمارکردنی بەرواری کۆتایی کاریگەری هەیە

### هەنگاوی ٣: هێڵەکانی پسووڵەی فرۆشتن

لەسەر **زیادکردنی هێڵ** کلیک بکە بۆ زیادکردنی کاڵا یان خزمەتگوزاری:

#### هێڵەکانی کاڵا
**کاڵا**: لە کاتالۆگی کاڵاکانەوە هەڵبژێرە
**وەسف**: لە کاڵاکانەوە خۆکارانە پڕ دەکرێتەوە (دەتوانرێت دەستکاری بکرێت)
**بڕ**: ژمارەی یەکەکان فرۆشراو
**نرخی یەکە**: نرخ بۆ هەر یەکە (لە کاڵاکانەوە دەگەڕێتەوە)
**باج**: ڕێژەی باجی جێبەجێکراو هەڵبژێرە
**هەژمار**: هەژماری داهات (لە کاڵاکانەوە خۆکارانە پڕ دەکرێتەوە)

#### هێڵەکانی خزمەتگوزاری
**خزمەتگوزاری**: بڕگەی خزمەتگوزاری هەڵبژێرە
**وەسف**: خزمەتگوزاری پێشکەشکراو باس بکە
**کات/یەکە**: بڕی خزمەتگوزاری
**نرخ**: نرخی کاتژمێری یان یەکە
**باج**: ڕێژەی باجی جێبەجێکراو هەڵبژێرە
**هەژمار**: هەژماری داهاتی خزمەتگوزاری

#### هێڵەکانی دەستی
**وەسف**: وەسفی تایبەتمەند بنووسە
**بڕ**: بڕ بنووسە
**نرخی یەکە**: نرخ بنووسە
**هەژمار**: هەژماری داهاتی گونجاو هەڵبژێرە
**باج**: باجی جێبەجێکراو هەڵبژێرە

### هەنگاوی ٤: باج و کۆ

**هەژمارکردنی باج**: سیستەم بە شێوەیەکی خۆکارانە باج هەژمار دەکات لەسەر بنەمای:
- ڕێکخستنی باجی کاڵا/خزمەتگوزاری
- دۆخی ئازادی باجی کڕیار
- مەرجەکانی پێگەی باج (ئەگەر جێبەجێکراو بێت)

**کۆی پسووڵەی فرۆشتن**:
- **کۆی لقی**: کۆی هەموو بڕە هێڵەکان پێش باج
- **بڕی باج**: کۆی باجی هەژمارکراو
- **کۆی گشتی**: بڕی کۆتایی کە لە کڕیار قەرزارە

**پیشاندانی فرە دراوی**: بۆ پسووڵەکانی فرۆشتنی دراوی بیانی:
- بڕەکانی پسووڵەی فرۆشتن لە دراوی پسووڵەی فرۆشتندا پیشان دەدرێن
- هاوتای دراوی کۆمپانیا بە شێوەیەکی خۆکارانە هەژمار دەکرێت
- نرخی ئاڵوگۆڕ لە کاتی ناردندا قفڵ دەکرێت

### هەنگاوی ٥: پێداچوونەوە و پشتڕاستکردنەوە

پێش ناردن، پێداچوونەوە بکە:
- وردەکاریەکانی کڕیار و دراو
- هەموو بڕگە هێڵەکان لەگەڵ بڕی درووست و نرخەکان
- هەژمارکردنەکانی باج
- مەرجەکانی پارەدان و بەرواری کۆتایی

## تایبەتمەندیە پێشکەوتووەکانی پسووڵەی فرۆشتن

### پسووڵەی فرۆشتنکردنی فرە دراوی

کاتێک بە دراوی بیانی پسووڵەی فرۆشتن دەکەیت:

١. **دراوی پسووڵەی فرۆشتن**: دراوی پەسەندکراوی کڕیار هەڵبژێرە
٢. **نرخی ئاڵوگۆڕ**: سیستەم نرخی ئێستا بە شێوەیەکی خۆکارانە دەگرێت
٣. **دراوی کۆمپانیا**: سیستەم بۆ تۆمارە ژمێریاریەکان دەگۆڕێت
٤. **قفڵکردنی نرخی ئاڵوگۆڕ**: نرخ کاتێک پسووڵەی فرۆشتن دەنێردرێت قفڵ دەکرێت
٥. **مامەڵەکردنی پارەدان**: جیاوازیەکانی نرخی ئاڵوگۆڕ لە کاتی پارەداندا مامەڵە دەکرێت

**نموونە**: پسووڵەی فرۆشتنکردنی کڕیار بە ١٠٠٠ دۆلاری ئەمەریکی کاتێک کۆمپانیا دینار عێراقی بەکاردەهێنێت
- بڕی پسووڵەی فرۆشتن: ١٠٠٠ دۆلاری ئەمەریکی
- نرخی ئاڵوگۆڕ: ١ دۆلاری ئەمەریکی = ١٣١٠ دینار عێراقی (لە ناردندا قفڵکراو)
- بڕی ژمێریاری: ١,٣١٠,٠٠٠ دینار عێراقی
- ڕاپۆرتەکانی کۆمپانیا هەردوو دراوەکە پیشان دەدەن

### یەکخستنی مەرجەکانی پارەدان

مەرجەکانی پارەدان بە شێوەیەکی خۆکارانە قیستی پارەدان دروست دەکەن کاتێک پسووڵەی فرۆشتن دەنێردرێت.

**مەرجە بەردەستەکانی ستاندارد**:
- **دەستبەجێ**: پارەدان دەستبەجێ پێویستە
- **مەرجەکانی ناو**: ناو ١٥، ناو ٣٠، ناو ٤٥، ناو ٦٠
- **داشکاندنی پارەدانی زوو**: ٢٪ ١٠ ناو ٣٠، ١٪ ١٥ ناو ٣٠
- **مەرجەکانی کۆتایی مانگ**: کۆتایی مانگ، کۆتایی مانگ + ١٥، کۆتایی مانگ + ٣٠
- **مەرجەکانی قیست**: خشتەی چەندین پارەدان

بۆ وردەکاری، بڕوانە [ڕێبەری مەرجەکانی پارەدان](payment-terms-guide.md).

### بەدواداچوونی دۆخی پارەدان

سیستەم دۆخی پارەدان جیا لە دۆخی پسووڵەی فرۆشتن بەدوادا دەکات:
- **پارەنەدراو**: هیچ پارەیەک وەرنەگیراوە
- **بەشێک پارەدراو**: هەندێک پارە وەرگیراوە
- **پارەدراو**: بەتەواوی پارەدراوە
- **زۆر پارەدراو**: پارەکان لە بڕی پسووڵەی فرۆشتن زیاترن

### مامەڵەکردنی پێگەی باج

پێگەکانی باج بە شێوەیەکی خۆکارانە ڕێک دەخەن:
- **ڕێژەی باج**: لەسەر بنەمای شوێنی کڕیار
- **ئازادی باج**: بۆ کڕیارە شایستەکان
- **نەخشەکردنی هەژمار**: ڕێکخستنی هەژماری داهات
- **پابەندبوون**: ڕێنماییەکانی باجی خۆجێیی

## پڕۆسەی ناردنی پسووڵەی فرۆشتن

### پێش ناردن - لیستی پێداچوونەوە

✅ **زانیاری کڕیار**: کڕیاری درووست هەڵبژێردراوە لەگەڵ وردەکاری تەواو
✅ **دراو & نرخی ئاڵوگۆڕ**: دراوی گونجاو هەڵبژێردراوە
✅ **بڕگە هێڵەکان**: هەموو کاڵا/خزمەتگوزارییەکان لەگەڵ بڕی درووست لەخۆگرتووە
✅ **نرخدانان**: نرخە یەکەکان و کۆکان درووستن
✅ **باج**: ڕێژەی باجی گونجاو بەگوێرەی پێگەی باج جێبەجێکراوە
✅ **مەرجەکانی پارەدان**: مەرجە درووستەکان هەڵبژێردراون
✅ **بەرواری کۆتایی**: کۆتایی پارەدانی مناسب
✅ **هەژمارەکان**: هەژمارەکانی داهات بە درووستی دیاریکراون

### ناردنی پسووڵەی فرۆشتن

١. **پاشەکەوتکردن وەک ڕەشنووس**: بۆ پێداچوونەوە و پەسەندکردن پاشەکەوت بکە
٢. **پێداچوونەوە**: هەموو وردەکارییەکان بە وردی لەگەڵ لیستی سەرەوە بپشکنە
٣. **پشتڕاستکردنەوە**: لەسەر "پشتڕاستکردنەوە" کلیک بکە بۆ تەواوکردنی پسووڵەی فرۆشتن

**⚠️ گرنگ**: دوای ناردن، پسووڵەکانی فرۆشتن ناتوانن دەستکاری بکرێن. بەڵگەنامەی ڕێکخستن بۆ چاکسازی بەکاربهێنە.

### ئەوەی ڕوودەدات کاتێک دەنێردرێت

سیستەم بە شێوەیەکی خۆکارانە:

١. **ژمارەی پسووڵەی فرۆشتن دیاری دەکات**: ژمارەی بەردەوامی هەمیشەیی دروست دەکرێت
٢. **بڕیاری ژمێریاری دروست دەکات**: بڕیارە ژمێریاریەکان بۆ ناسینەوەی داهات
٣. **نرخی ئاڵوگۆڕ قفڵ دەکات**: نرخی دراوی بیانی بۆ یەکخستنەوە جێگیر دەکرێت
٤. **قیستی پارەدان دروست دەکات**: لەسەر بنەمای مەرجە هەڵبژێردراوەکانی پارەدان
٥. **جوڵەی کۆگا دروست دەکات**: بۆ کاڵای هەڵگیراو (کەمکردنەوەی کۆگا)
٦. **دراو دەگۆڕێت**: بڕەکان بۆ دراوی کۆمپانیا دەگۆڕێت
٧. **دۆخی پارەدان نوێ دەکاتەوە**: دۆخی سەرەتایی پارەدان وەک "پارەنەدراو" دادەنێت

### بڕیاری ژمێریاری دروستکراو

بۆ پسووڵەیەکی فرۆشتنی ستاندارد، سیستەم دروست دەکات:
```
ئەستەمکراو: هەژماری وەرگیراوەکان (کڕیار)     ١٠٠ $
    خەرج: داهاتی فرۆشتن                       ١٠٠٠ $
    خەرج: باجی فرۆشتنی پارەدراو                 ١٠٠ $
```

بۆ پسووڵەکانی فرۆشتنی فرە دراوی، بڕەکان بۆ دراوی کۆمپانیا بەکارهێنانی نرخی ئاڵوگۆڕی قفڵکراو دەگۆڕێن.

## بەڕێوەبردنی پسووڵە نێردراوەکانی فرۆشتن

### تۆمارکردنی پارەدان

**لە پسووڵەی فرۆشتنەوە**: ڕاستەوخۆ لەسەر پسووڵەی فرۆشتن لەسەر "تۆمارکردنی پارەدان" کلیک بکە
- ژورنالی پارەدان هەڵبژێرە (هەژماری بانک/کاش)
- بەرواری پارەدان و بڕ بنووسە
- ئاماژەی ئیختیاری زیاد بکە
- سیستەم بە شێوەیەکی خۆکارانە پارەدان دروست دەکات و پشتڕاست دەکاتەوە

**لە مۆدیولی پارەدانەوە**: پارەدان بە جیاکاری دروست بکە و بە پسووڵەکانی فرۆشتنەوە بەستە
**ڕێکخستنی بانک**: پارەدانەکان دەتوانن بە شێوەیەکی خۆکارانە لە کاتی ڕێکخستندا لێک بخرێن

بۆ زانیاری وردی پارەدان، بڕوانە [ڕێبەری پارەدانەکان](payments.md).

### قیستەکانی پارەدان

پسووڵە نێردراوەکانی فرۆشتن بە شێوەیەکی خۆکارانە قیستی پارەدان لەسەر بنەمای مەرجەکانی پارەدان دروست دەکەن:
- **پارەدانی یەک**: بۆ مەرجە سادەکان وەک ناو ٣٠
- **چەندین قیست**: بۆ خشتە ئاڵۆزەکانی پارەدان
- **بەدواداچوونی پارەدان**: هەر قیستێک بە جیاکاری بەدوادا دەکرێت
- **ڕاپۆرتەکانی تەمەنی قەرز**: لەسەر بنەمای بەرواری کۆتایی قیست

### پارەدانی بەشی

سیستەم بەتەواوی پشتگیری پارەدانی بەشی دەکات:
١. پارەدان بۆ بڕی بەشی تۆمار بکە
٢. دۆخی پارەدان بۆ "بەشێک پارەدراو" دەگۆڕێت
٣. ماوەی باقی بە شێوەیەکی خۆکارانە بەدوادا دەکرێت
٤. چەندین پارەدان دەتوانن جێبەجێ بکرێن
٥. قیستەکانی پارەدان بەگوێرەی ئەوەش نوێ دەکرێنەوە

### بەڵگەکانی ڕێکخستن

بۆ چاکسازی دوای ناردن، بەڵگەنامەکانی ڕێکخستن بەکاربهێنە لە جیاتی دەستکاری:

**تێبینی قەرزدار**: بۆ گەڕاندنەوە، داشکاندن، یان چاکسازی
١. بچۆ بۆ پسووڵەی فرۆشتن
٢. بەڕێوەبەری پەیوەندی "بەڵگەنامەکانی ڕێکخستن" بەکاربهێنە
٣. تێبینی قەرزدار بۆ بڕە تایبەتەکان دروست بکە
٤. قەرزدار بە شێوەیەکی خۆکارانە بۆ هەژماری کڕیار جێبەجێ دەکرێت

**تێبینی قەرزخواز**: بۆ تۆژی زیاتر
- پڕۆسەی هاوشێوە بەڵام یەکەی کڕیار زیاد دەکات
- کەمتر باو بەڵام کاتێک پێویست دەبێت بەردەستە

### دروستکردن و گەیاندنی PDF

**بینینی PDF**: لەسەر "بینینی PDF" کلیک بکە بۆ پێشبینی پسووڵەی فرۆشتن
**داگرتنی PDF**: لەسەر "داگرتنی PDF" کلیک بکە بۆ داگرتنی فایل
**قاڵبەکان**: چەندین قاڵبی PDF بۆ هەر کۆمپانیایەک بەردەستە
**گەیاندنی ئیمەیڵ**: PDFەکان دەتوانن بۆ کڕیاران ئیمەیڵ بکرێن (ڕێکخستن پێویستە)

### هەڵوەشاندنەوەی پسووڵەی فرۆشتن

بۆ پسووڵە نێردراوەکانی فرۆشتن کە پێویستیان بە هەڵوەشاندنەوەیە:
- هەڵوەشاندنەوە بڕیاری ژمێریاریی پێچەوانەکراو دروست دەکات
- دۆخی پسووڵەی فرۆشتن بۆ "هەڵوەشێنراوەتەوە" دەگۆڕێت
- شوێنپێی پشکنین بۆ پابەندبوون پارێزراوە
- دوای هەڵوەشاندنەوە ناتوانرێت پووچەڵی بکرێتەوە

- **قاڵبە تایبەتەکان**: دیزاینە تایبەتەکانی کۆمپانیا

## ڕاپۆرت و شیکردنەوە

### ڕاپۆرتەکانی فرۆشتن
**فرۆشتن بە کڕیار**: شیکردنەوەی داهات بە کڕیار لەگەڵ پشتگیری فرە دراوی
**فرۆشتن بە کاڵا**: بەدواداچوونی کارایی کاڵا لە تەواوی دراوەکاندا
**فرۆشتن بە ماوە**: ڕۆیشتنی فرۆشتنی مانگانە/وەرزی لەگەڵ گۆڕینی دراو
**ڕاپۆرتەکانی باج**: ڕاپۆرتکردنی باجی VAT/فرۆشتن لەگەڵ مامەڵەکردنی دراوی درووست
**ڕاپۆرتەکانی فرە دراوی**: ڕاپۆرتە یەکخراوەکان لە دراوی بنەڕەتیدا

### بەڕێوەبردنی وەرگیراوەکان
**وەرگیراوەی تەمەن پێکراو**: بڕە باقیەکان بە تەمەن لەگەڵ گۆڕینی دراو
**ڕاپۆرتی قیستەکانی پارەدان**: خشتەکانی پارەدانی داهاتوو
**ڕاپۆرتەکانی کڕیار**: کورتەکانی چالاکی هەژمار لە دراوی کڕیاردا
**ڕاپۆرتەکانی کۆکردنەوە**: بەدواداچوونی فاکتوری دواکەوتوو لەگەڵ دۆخی پارەدان
**پێشبینی جۆڵەی پارە**: کاتی پارەدانی چاوەڕوانکراو لەسەر بنەمای قیستەکان

### ڕاپۆرتەکانی فرە کۆمپانیا
بۆ بازرگانییەکان لەگەڵ چەندین کۆمپانیا:
- ڕاپۆرتە یەکخراوەکانی فرۆشتن
- شیکردنەوەی مامەڵە نێوان کۆمپانیاکان
- ڕاپۆرتەکانی بەرەنگاربوونەوەی دراو
- شیکردنەوەی کڕیاری خولی کۆمپانیاکان

## ڕێسای باشترین

### ژمارەکردنی پسووڵەی فرۆشتن

- **ئاماژەکانی ڕەشنووس**: پسووڵە ڕەشنووسەکانی فرۆشتن شێوازی "DRAFT-00001" پیشان دەدەن تا دەنێردرێن

- **دراوی یەکگرتوو**: کاتێک گونجاو بێت دراوی پەسەندکراوی کڕیار بەکاربهێنە
- **چاودێری نرخی ئاڵوگۆڕ**: نرخەکان پێش ناردنی پسووڵە گەورەکانی فرۆشتن پێداچوونەوە بکە
- **مەترسی دراو**: بۆ بەرەنگاربوونەوەی بەرچاوی دراوی بیانی پاراستن لە ڕیسک بکە
- **دراوی ڕاپۆرتکردن**: ڕاپۆرتەکان لە هەردوو دراوی پسووڵەی فرۆشتن و کۆمپانیادا پارێزە

### جۆڕی کاری پەسەندکردن
- **پێداچوونەوەی ڕەشنووس**: پەسەندکردنی بەڕێوەبەری پێش ناردن پێشنیار دەکرێت
- **سنووری قەرزدار**: قەرزداری کڕیار پێش پسووڵە گەورەکانی فرۆشتن چاودێری بکە
- **پەسەندکردنی نرخدانان**: نرخدانان یان داشکاندنی تایبەت بپشکنە
- **پابەندبوونی باج**: دڵنیایی لە دابەشکردنی پێگەی باجی درووست

### بەڵگەکردن و شوێنپێی پشکنین
- **بەڵگەنامە پشتگیریکەرەکان**: پاشکۆی گەیاندن و گرێبەستەکان بەستە
- **مێژووی گۆڕانکاری**: سیستەم لۆگی پشکنینی تەواو پارێزگاری دەکات
- **بەڵگەکردنی پارەدان**: پشتڕاستکردنەوە و ڕاپۆرتی هەژماری بانکەکان بەستە
- **تۆمارەکانی فرە دراوی**: بەڵگەکردنی نرخی ئاڵوگۆڕ پارێزگاری بکە

## بارودۆخە باوەکان

### بارودۆخی ١: فرۆشتنی کاڵای ستاندارد (یەک دراو)
فرۆشتنی ١٠ لاپتۆپ لە ١٠٠٠ دۆلار هەریەکە لەگەڵ ١٠٪ باج لە دۆلاری ئەمەریکیدا:

١. کڕیار و دراوی دۆلاری ئەمەریکی هەڵبژێرە
٢. هێڵی کاڵای لاپتۆپ زیاد بکە (بڕ: ١٠، نرخ: ١٠٠٠ دۆلار)
٣. سیستەم باج هەژمار دەکات: ١٠٠٠ دۆلار
٤. کۆی پسووڵەی فرۆشتن: ١١٠٠٠ دۆلاری ئەمەریکی
٥. پسووڵەی فرۆشتن پشتڕاست بکە بۆ دروستکردنی ژمارەی پسووڵەی فرۆشتن و بڕیارەکانی ژمێریاری

### بارودۆخی ٢: پسووڵەی فرۆشتنی خزمەتگوزاری فرە دراوی
کڕیاری ئەمەریکی پارە لە دۆلاری ئەمەریکیدا دەدات لەکاتێکدا کۆمپانیا دینار عێراقی بەکاردەهێنێت:

١. کڕیار و دۆلاری ئەمەریکی وەک دراوی پسووڵەی فرۆشتن هەڵبژێرە
٢. هێڵەکانی خزمەتگوزاری ڕاوێژکاری لە دۆلاری ئەمەریکیدا زیاد بکە
٣. سیستەم نرخی ئاڵوگۆڕی ئێستا پیشان دەدات (وەک ١ دۆلاری ئەمەریکی = ١٣١٠ دینار عێراقی)
٤. پسووڵەی فرۆشتن پشتڕاست بکە - نرخی ئاڵوگۆڕ لە ١٣١٠ قفڵ دەکرێت
٥. بڕیارە ژمێریاریەکان لە دینار عێراقی (دراوی کۆمپانیا) دروست دەکرێن
٦. کڕیار لە دۆلاری ئەمەریکیدا پارە دەدات - جیاوازیەکانی نرخی ئاڵوگۆڕ بە شێوەیەکی خۆکارانە مامەڵە دەکرێن

### بارودۆخی ٣: پسووڵەی فرۆشتنی مەرجەکانی پارەدانی ئاڵۆز
پڕۆژەیەکی گەورە لەگەڵ مەرجەکانی پارەدانی ٣٠-٦٠-٩٠ ڕۆژ:

١. پسووڵەی فرۆشتن لەگەڵ کۆی بڕی ٣٠٠٠٠ دۆلار دروست بکە
٢. مەرجەکانی پارەدانی "٣٠-٦٠-٩٠ ڕۆژ" هەڵبژێرە
٣. پسووڵەی فرۆشتن پشتڕاست بکە
٤. سیستەم ٣ قیستی پارەدان دروست دەکات:
   - ١٠٠٠٠ دۆلار لە ٣٠ ڕۆژدا
   - ١٠٠٠٠ دۆلار لە ٦٠ ڕۆژدا
   - ١٠٠٠٠ دۆلار لە ٩٠ ڕۆژدا
٥. هەر قیستێک بە جیاکاری لە ڕاپۆرتەکانی پارەداندا بەدوادا بکە

### بارودۆخی ٤: پسووڵەی فرۆشتن لەگەڵ داشکاندنی پارەدانی زوو
پاڵنەردان بۆ پارەدانی زوو لەگەڵ مەرجەکانی "٢٪ ١٠ ناو ٣٠":

١. پسووڵەی فرۆشتنی ستاندارد دروست بکە
٢. مەرجەکانی پارەدانی "٢٪ ١٠ ناو ٣٠" هەڵبژێرە
٣. پسووڵەی فرۆشتن پشتڕاست بکە
٤. کڕیار لە ناو ١٠ ڕۆژدا پارە دەدات - ٢٪ داشکاندن لە ڕێگەی بەڵگەنامەی ڕێکخستنەوە جێبەجێ بکە
٥. سیستەم داشکاندن وەک هەژماری داشکاندنی فرۆشتن بەدوادا دەکات

### بارودۆخی ٥: چاکسازی پسووڵەی فرۆشتن لە ڕێگەی بەڵگەنامەی ڕێکخستنەوە
چاککردنی هەڵەی نرخدانان لەسەر پسووڵەی فرۆشتنی نێردراو:

١. پسووڵەی فرۆشتنی سەرەتا بۆ ٥٠٠٠ دۆلار نێردراوە
٢. دۆزینەوەی ئەوەی نرخ دەبێت ٤٥٠٠ دۆلار بێت
٣. بەڵگەنامەی ڕێکخستنی تێبینی قەرزدار بۆ ٥٠٠ دۆلار دروست بکە
٤. قەرزدار بە هەژماری کڕیار جێبەجێ بکە
٥. پسووڵەی فرۆشتنی سەرەتا بۆ شوێنپێی پشکنین بە نەگۆڕاوی دەمێنێتەوە

## چاکسازی کێشەکان

### کێشە باوەکان

**کێشە**: ناتوانرێت پسووڵەی فرۆشتن پشتڕاست بکرێتەوە - هەڵەکانی پشتڕاستکردنەوە
**چارەسەری**: 
- زانیاری تەواوی ناونیشانی کڕیار بپشکنە
- دڵنیایی بکە هەموو هێڵەکانی پسووڵەی فرۆشتن هەژماری داهاتیان دیاریکراوە
- بپشکنە بەرواری پسووڵەی فرۆشتن لە ماوەی قفڵکراوی ژمێریاریدا نییە
- دراو و نرخی ئاڵوگۆڕ بەردەست بێت پشتڕاست بکە

**کێشە**: هەژمارکردنی باج هەڵە دەردەچێت
**چارەسەری**:
- دابەشکردنی پێگەی باجی کڕیار بپشکنە
- ڕێکخستنی باجی کاڵا لە داتا سەرەکیەکاندا بپشکنە
- دۆخی ئازادی باجی کڕیار پێداچوونەوە بکە
- نەخشەکردنی باجی پێگەی باج درووست بێت دڵنیا بکە

**کێشە**: نرخی ئاڵوگۆڕ نوێ ناکرێتەوە
**چارەسەری**:
- ڕێکخستنی نرخی دراو بپشکنە
- مەودای بەرواری نرخ بپشکنە
- خزمەتگوزاری دراو بە درووستی ڕێکخراوە دڵنیا بکە
- بۆ ڕێکخستنی سەرچاوەی نرخ پەیوەندی لەگەڵ بەڕێوەبەری سیستەم بکە

**کێشە**: قیستەکانی پارەدان دروست نابن
**چارەسەری**:
- مەرجەکانی پارەدان بە درووستی ڕێکخراون بپشکنە
- ڕێکخستنی هێڵەکانی مەرجەکانی پارەدان بپشکنە
- مەرجەکانی پارەدان چالاک بن دڵنیا بکە
- یاساکانی هەژمارکردنی مەرجەکانی پارەدان پێداچوونەوە بکە

**کێشە**: دروستکردنی PDF شکست دەهێنێت
**چارەسەری**:
- لۆگۆی کۆمپانیا و ڕێکخستنی قاڵب بپشکنە
- خزمەتگوزاری PDF چالاکە بپشکنە
- زانیاری تەواوی ناونیشانی کڕیار دڵنیا بکە
- ئەگەر قاڵبەکان ون بن پەیوەندی لەگەڵ بەڕێوەبەری سیستەم بکە

### کێشەکانی یەکخستن

**کێشە**: بڕیارە ژمێریاریەکان دروست نابن
**چارەسەری**:
- ڕێکخستنی چارتی هەژمارەکان بپشکنە
- نەخشەکردنی هەژماری داهات بۆ کاڵاکان بپشکنە
- ڕێکخستنی پەرتووکی ڕۆژانە تەواو بێت دڵنیا بکە
- ڕێکخستنی ماوەی ژمێریاری پێداچوونەوە بکە

**کێشە**: جوڵەی کۆگا دروست نابێت
**چارەسەری**:
- کاڵاکان وەک "هەڵگیراو" نیشانکراون بپشکنە
- ڕێکخستنی کۆگا و شوێن بپشکنە
- بەڕێوەبردنی کۆگا چالاک کرابێت دڵنیا بکە
- ڕێکخستنەکانی کۆگای کاڵا پێداچوونەوە بکە

## فەرهەنگی زاراوەکان (گلۆساری)

### **پسووڵەی فرۆشتن (Invoice)**
بەڵگەنامەیەکی فەرمیی بازرگانی کە داواکاری پارەدان دەکات بۆ کاڵا یان خزمەتگوزارییە پێشکەشکراوەکان.

### **دۆخی ڕەشنووس (Draft Status)**
باری پسووڵەی فرۆشتن کە هێشتا لە ئامادەکردندایە و دەتوانرێت دەستکاری بکرێت.

### **دۆخی نێردراو (Posted Status)**
باری پسووڵەی فرۆشتن کە تەواو کراوە و ناتوانرێت دەستکاری بکرێت، بڕیارە ژمێریاریەکان دروست کراون.

### **ناسینەوەی داهات (Revenue Recognition)**
پڕۆسەی ژمێریاری کە داهات لە کاتی بەدەستهێنان تۆمار دەکات، نەک کاتی وەرگرتنی پارە.

### **هەژماری وەرگیراوەکان (Accounts Receivable)**
هەژماری ژمێریاری کە بڕەکان کە لە لایەن کڕیارانەوە قەرزارن تۆمار دەکات.

### **فرە دراوی (Multi-Currency)**
تایبەتمەندی کە ڕێگە دەدات بە بەکارهێنانی چەندین جۆری دراو لە پسووڵەی فرۆشتنکردن و ژمێریاریدا.

### **نرخی ئاڵوگۆڕ (Exchange Rate)**
نرخی گۆڕینی نێوان دوو دراوی جیاواز، وەک ١ دۆلاری ئەمەریکی = ١٣١٠ دینار عێراقی.

### **مەرجەکانی پارەدان (Payment Terms)**
ڕێکخستنەکان کە دیاری دەکەن کاتێک و چۆن کڕیار پێویستە پارە بدات.

### **قیستی پارەدان (Payment Installments)**
بەشەکانی بچووکی بڕی کۆی پسووڵەی فرۆشتن کە لە کاتە جیاوازەکاندا پارە دەدرێن.

### **پێگەی باج (Fiscal Position)**
ڕێکخستنی باج کە دەڵێت چ ڕێژەیەکی باج بۆ کڕیاری دیاریکراو جێبەجێ بکرێت.

### **بەڵگەنامەی ڕاستکردنەوە (Adjustment Document)**
بەڵگەنامەیەک کە بۆ چاکسازی پسووڵەیەکی فرۆشتنی نێردراو بەکاردێت، وەک تێبینی قەرزدار یان تێبینی قەرزخواز.

### **تێبینی قەرزدار (Credit Note)**
بەڵگەنامەیەک کە بڕی پسووڵەی فرۆشتن کەم دەکاتەوە، بۆ گەڕاندنەوە یان داشکاندن بەکاردێت.

### **تۆماری ڕۆژانە (Journal Entry)**
تۆمارکردنی ژمێریاری کە کاریگەری پسووڵەی فرۆشتن لەسەر هەژمارەکان نیشان دەدات.

### **جوڵەی کۆگا (Stock Move)**
تۆمارکردنی گۆڕانی بڕی کۆگا کاتێک کاڵا دەفرۆشرێت.

### **ژمێریاریی ئەستەمکراو (Accrual Accounting)**
سیستەمی ژمێریاری کە مامەڵەکان لە کاتی ڕوودان تۆمار دەکات، نەک لە کاتی پارەدان.

### **دۆخی پارەدان (Payment Status)**
باری پسووڵەی فرۆشتن لە ڕووی پارەدانەوە: پارەنەدراو، بەشێک پارەدراو، پارەدراو، یان زۆر پارەدراو.

### **ڕاپۆرتی تەمەنپێکراو (Aged Report)**
ڕاپۆرتێک کە پیشان دەدات چەند دەمە پسووڵەکانی فرۆشتن بێ پارە ماونەتەوە.

### **هاوسەنگی بانکی (Bank Reconciliation)**
پڕۆسەی لێکخستنی پارەدانەکان لەگەڵ ڕاپۆرتی هەژماری بانک.

### **قاڵبی PDF (PDF Template)**
فۆرماتێکی دیاریکراو بۆ چاپکردن یان گەیاندنی پسووڵەی فرۆشتن بە شێوەی PDF.

### **شوێنپێی پشکنین (Audit Trail)**
تۆمارکردنی هەموو گۆڕانکارییەکان و چالاکییەکان لەسەر پسووڵەی فرۆشتن بۆ مەبەستی پشکنین.
